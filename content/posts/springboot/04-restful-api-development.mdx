## 前言

RESTful 是目前最流行的 API 设计风格，它充分利用 HTTP 协议的特性，设计出简洁、易用、可扩展的 Web API。

本文将系统讲解 RESTful API 设计规范和 SpringBoot 的实现方式。

## RESTful 架构风格

### 什么是 REST？

REST（Representational State Transfer）是一种软件架构风格，而不是标准。

**核心原则**：

- 资源（Resource）：使用 URI 唯一标识
- 表现层（Representation）：JSON、XML 等
- 状态转换（State Transfer）：HTTP 动词（GET、POST、PUT、DELETE）

### RESTful 设计规范

| 操作     | HTTP 方法 | URI             | 说明             |
| -------- | --------- | --------------- | ---------------- |
| 列表     | GET       | /api/users      | 查询所有用户     |
| 详情     | GET       | /api/users/{id} | 查询单个用户     |
| 创建     | POST      | /api/users      | 创建新用户       |
| 更新     | PUT       | /api/users/{id} | 更新用户（全部） |
| 部分更新 | PATCH     | /api/users/{id} | 更新用户（部分） |
| 删除     | DELETE    | /api/users/{id} | 删除用户         |

## 项目准备

### 实体类

```java
@Entity
@Table(name = "articles")
public class Article {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false, length = 200)
    private String title;

    @Column(columnDefinition = "TEXT")
    private String content;

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "user_id")
    private User author;

    @OneToMany(mappedBy = "article")
    private List<Comment> comments;

    @CreationTimestamp
    private LocalDateTime createdAt;

    // Getter 和 Setter
}
```

### Repository

```java
@Repository
public interface ArticleRepository extends JpaRepository<Article, Long> {

    // 根据作者查询文章
    List<Article> findByAuthorId(Long authorId);

    // 根据标题模糊查询
    List<Article> findByTitleContaining(String keyword);

    // 分页查询
    Page<Article> findAll(Pageable pageable);
}
```

## 请求映射

### GET 请求

```java
// 查询所有文章
@GetMapping
public Result<List<Article>> list() {
    List<Article> articles = articleService.findAll();
    return Result.success(articles);
}

// 根据 ID 查询
@GetMapping("/{id}")
public Result<Article> getById(@PathVariable Long id) {
    Article article = articleService.findById(id);
    return Result.success(article);
}

// 分页查询
@GetMapping("/page")
public Result<Page<Article>> page(
    @RequestParam(defaultValue = "0") Integer page,
    @RequestParam(defaultValue = "10") Integer size
) {
    Pageable pageable = PageRequest.of(page, size);
    Page<Article> articles = articleService.findAll(pageable);
    return Result.success(articles);
}

// 搜索
@GetMapping("/search")
public Result<List<Article>> search(@RequestParam String keyword) {
    List<Article> articles = articleService.search(keyword);
    return Result.success(articles);
}
```

### POST 请求

```java
// 创建文章
@PostMapping
public Result<Article> create(@Valid @RequestBody ArticleCreateDTO dto) {
    Article article = articleService.create(dto);
    return Result.success("创建成功", article);
}
```

### PUT 请求

```java
// 更新文章（全部字段）
@PutMapping("/{id}")
public Result<Article> update(
    @PathVariable Long id,
    @Valid @RequestBody ArticleUpdateDTO dto
) {
    Article article = articleService.update(id, dto);
    return Result.success("更新成功", article);
}
```

### PATCH 请求

```java
// 部分更新
@PatchMapping("/{id}")
public Result<Article> partialUpdate(
    @PathVariable Long id,
    @RequestBody Map<String, Object> updates
) {
    Article article = articleService.partialUpdate(id, updates);
    return Result.success("更新成功", article);
}
```

### DELETE 请求

```java
// 删除文章
@DeleteMapping("/{id}")
public Result<Void> delete(@PathVariable Long id) {
    articleService.delete(id);
    return Result.success("删除成功");
}
```

## 请求参数处理

### @PathVariable - 路径参数

```java
// 单个路径参数
@GetMapping("/users/{id}")
public Result<User> getById(@PathVariable Long id) {
    return Result.success(userService.findById(id));
}

// 多个路径参数
@GetMapping("/users/{userId}/articles/{articleId}")
public Result<Article> getArticle(
    @PathVariable Long userId,
    @PathVariable Long articleId
) {
    return Result.success(articleService.findArticle(userId, articleId));
}
```

### @RequestParam - 查询参数

```java
// 单个查询参数
@GetMapping("/users")
public Result<List<User>> list(@RequestParam Integer status) {
    return Result.success(userService.findByStatus(status));
}

// 可选参数
@GetMapping("/users")
public Result<List<User>> list(
    @RequestParam(required = false) Integer status,
    @RequestParam(defaultValue = "0") Integer page
) {
    return Result.success(userService.findUsers(status, page));
}

// 多个查询参数
@GetMapping("/users/search")
public Result<List<User>> search(
    @RequestParam String keyword,
    @RequestParam(defaultValue = "0") Integer minAge,
    @RequestParam(defaultValue = "100") Integer maxAge
) {
    return Result.success(userService.search(keyword, minAge, maxAge));
}
```

### @RequestBody - 请求体

```java
// POST/PUT 请求，接收 JSON 数据
@PostMapping
public Result<Article> create(@RequestBody ArticleCreateDTO dto) {
    return Result.success(articleService.create(dto));
}
```

### @RequestHeader - 请求头

```java
// 获取请求头
@PostMapping
public Result<Void> create(@RequestBody ArticleCreateDTO dto,
                              @RequestHeader("Authorization") String token) {
    // 验证 token
    // 创建文章
    return Result.success();
}
```

### @CookieValue - Cookie

```java
// 获取 Cookie
@GetMapping("/profile")
public Result<User> getProfile(@CookieValue("userId") Long userId) {
    return Result.success(userService.findById(userId));
}
```

## 文件上传

### 单文件上传

```java
@PostMapping("/upload")
public Result<String> upload(@RequestParam("file") MultipartFile file) {
    if (file.isEmpty()) {
        return Result.error("请选择文件");
    }

    String fileName = file.getOriginalFilename();
    String filePath = "/upload/" + fileName;

    try {
        file.transferTo(new File(filePath));
        return Result.success("上传成功", fileName);
    } catch (IOException e) {
        return Result.error("上传失败");
    }
}
```

### 多文件上传

```java
@PostMapping("/uploads")
public Result<List<String>> uploads(@RequestParam("files") MultipartFile[] files) {
    List<String> fileNames = new ArrayList<>();

    for (MultipartFile file : files) {
        // 保存文件
        String fileName = file.getOriginalFilename();
        file.transferTo(new File("/upload/" + fileName));
        fileNames.add(fileName);
    }

    return Result.success("上传成功", fileNames);
}
```

## 统一返回格式

### Result 类

```java
public class Result<T> {

    private Integer code;
    private String message;
    private T data;

    public static <T> Result<T> success(T data) {
        return new Result<>(200, "操作成功", data);
    }

    public static <T> Result<T> error(String message) {
        return new Result<>(500, message, null);
    }
}
```

## 完整示例

### ArticleController

```java
@RestController
@RequestMapping("/api/articles")
@Tag(name = "文章管理", description = "文章增删改查接口")
public class ArticleController {

    @Autowired
    private ArticleService articleService;

    @GetMapping
    @Operation(summary = "查询所有文章")
    public Result<Page<Article>> list(
        @RequestParam(defaultValue = "0") Integer page,
        @RequestParam(defaultValue = "10") Integer size
    ) {
        Pageable pageable = PageRequest.of(page, size);
        Page<Article> articles = articleService.findAll(pageable);
        return Result.success(articles);
    }

    @GetMapping("/{id}")
    @Operation(summary = "查询文章详情")
    public Result<Article> getById(@PathVariable Long id) {
        Article article = articleService.findById(id);
        return Result.success(article);
    }

    @PostMapping
    @Operation(summary = "创建文章")
    public Result<Article> create(@Valid @RequestBody ArticleCreateDTO dto) {
        Article article = articleService.create(dto);
        return Result.success("创建成功", article);
    }

    @PutMapping("/{id}")
    @Operation(summary = "更新文章")
    public Result<Article> update(
        @PathVariable Long id,
        @Valid @RequestBody ArticleUpdateDTO dto
    ) {
        Article article = articleService.update(id, dto);
        return Result.success("更新成功", article);
    }

    @DeleteMapping("/{id}")
    @Operation(summary = "删除文章")
    public Result<Void> delete(@PathVariable Long id) {
        articleService.delete(id);
        return Result.success("删除成功");
    }
}
```

## 最佳实践

### 1. 使用名词复数

```
✅ 推荐：
GET /api/users
GET /api/articles

❌ 避免：
GET /api/getUsers
GET /api/listArticles
```

### 2. 资源嵌套层级不超过 3 层

```
✅ 推荐：
GET /api/users/1/articles/2/comments

❌ 避免：
GET /api/users/1/articles/2/comments/3/replies/4/likes
```

### 3. HTTP 方法语义化

```
✅ 推荐：
GET    /api/users       # 查询
POST   /api/users       # 创建
PUT    /api/users/1     # 更新
DELETE /api/users/1     # 删除

❌ 避免：
GET /api/createUser
GET /api/deleteUser/1
```

### 4. 统一返回格式

```java
// ✅ 推荐
return Result.success(data);
return Result.error("操作失败");

// ❌ 避免
return data;  // 无法统一处理
```
