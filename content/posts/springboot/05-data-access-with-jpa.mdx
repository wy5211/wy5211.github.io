## 前言

Spring Data JPA 是 SpringBoot 生态中最重要的数据访问框架，它大大简化了数据库操作的代码量。

本文将深入讲解 Spring Data JPA 的使用方法和实战技巧。

## JPA 基础

### 添加依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>

<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
    <scope>runtime</scope>
</dependency>
```

### 配置文件

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/demo?useUnicode=true&characterEncoding=utf8
    username: root
    password: root123
    driver-class-name: com.mysql.cj.jdbc.Driver

  jpa:
    hibernate:
      ddl-auto: update
    show-sql: true
    properties:
      hibernate:
        dialect: org.hibernate.dialect.MySQLDialect
        format_sql: true
```

## Entity 实体映射

### 基础注解

```java
@Entity
@Table(name = "users")
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false, unique = true, length = 50)
    private String username;

    @Column(nullable = false, unique = true, length = 100)
    private String email;

    @Column(length = 20)
    private String phone;

    private Integer age;

    @Enumerated(EnumType.STRING)
    private Gender gender;

    @CreationTimestamp
    private LocalDateTime createdAt;

    @UpdateTimestamp
    private LocalDateTime updatedAt;

    public enum Gender { MALE, FEMALE, OTHER }
}
```

### 字段约束

```java
@Column(
    nullable = false,      // 不允许为空
    unique = true,          // 唯一约束
    length = 50,           // 字符串长度
    precision = 10,         // 数字精度
    scale = 2              // 小数位数
)
private String username;
```

## 关系映射

### 一对一（OneToOne）

```java
// 用户和用户详情一对一
@Entity
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @OneToOne(mappedBy = "user", cascade = CascadeType.ALL)
    private UserProfile profile;
}

@Entity
public class UserProfile {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(length = 100)
    private String realName;

    @Column(length = 18)
    private String idCard;

    @OneToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "user_id")
    private User user;
}
```

### 一对多（OneToMany）

```java
// 用户和文章一对多
@Entity
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @OneToMany(mappedBy = "author", cascade = CascadeType.ALL)
    private List<Article> articles;

    // 获取文章数量
    public long getArticleCount() {
        return articles != null ? articles.size() : 0;
    }
}

@Entity
public class Article {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false)
    private String title;

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "user_id")
    private User author;

    @OneToMany(mappedBy = "article", cascade = CascadeType.ALL)
    private List<Comment> comments;
}
```

### 多对多（ManyToMany）

```java
// 文章和标签多对多
@Entity
public class Article {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @ManyToMany(mappedBy = "articles")
    private Set<Tag> tags;
}

@Entity
public class Tag {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false, unique = true, length = 50)
    private String name;

    @ManyToMany(cascade = {CascadeType.PERSIST, CascadeType.MERGE})
    @JoinTable(
        name = "article_tag",
        joinColumns = @JoinColumn(name = "tag_id"),
        inverseJoinColumns = @JoinColumn(name = "article_id")
    )
    private Set<Article> articles;
}
```

## Repository 接口

### 基础方法

```java
@Repository
public interface UserRepository extends JpaRepository<User, Long> {
    // JpaRepository 提供的基础方法：
    // save(), findAll(), findById(), deleteById(), count() 等
}
```

### 方法命名查询

```java
@Repository
public interface UserRepository extends JpaRepository<User, Long> {

    // 根据用户名查询
    Optional<User> findByUsername(String username);

    // 根据邮箱查询
    Optional<User> findByEmail(String email);

    // 检查用户名是否存在
    boolean existsByUsername(String username);

    // 根据年龄查询
    List<User> findByAge(Integer age);

    // 根据年龄大于指定值查询
    List<User> findByAgeGreaterThan(Integer age);

    // 根据用户名模糊查询
    List<User> findByUsernameContaining(String keyword);

    // 根据年龄排序查询
    List<User> findByAgeOrderByCreatedAtDesc(Integer age);

    // 根据状态查询
    List<User> findByStatus(Boolean status);
}
```

### @Query 自定义查询

```java
@Repository
public interface UserRepository extends JpaRepository<User, Long> {

    // JPQL 查询
    @Query("SELECT u FROM User u WHERE u.username = ?1")
    Optional<User> findByUsername(String username);

    // 命名参数
    @Query("SELECT u FROM User u WHERE u.email = :email")
    Optional<User> findByEmail(@Param("email") String email);

    // 原生 SQL 查询
    @Query(value = "SELECT * FROM users WHERE username = :username", nativeQuery = true)
    User findByUsernameNative(@Param("username") String username);

    // 动态查询
    @Query("SELECT u FROM User u WHERE u.username LIKE CONCAT('%', :keyword, '%')")
    List<User> searchByUsername(@Param("keyword") String keyword);

    // 统计查询
    @Query("SELECT COUNT(u) FROM User u")
    long countAllUsers();

    // 分页查询
    @Query("SELECT u FROM User u WHERE u.status = ?1")
    Page<User> findByStatus(Boolean status, Pageable pageable);
}
```

### 修改和删除查询

```java
@Repository
public interface UserRepository extends JpaRepository<User, Long> {

    // 修改查询（需要 @Modifying 和 @Transactional）
    @Modifying
    @Query("UPDATE User u SET u.status = :status WHERE u.id = :id")
    int updateStatus(@Param("id") Long id, @Param("status") Boolean status);

    // 删除查询
    @Modifying
    @Query("DELETE FROM User u WHERE u.lastLoginAt < :date")
    int deleteInactiveUsers(@Param("date") LocalDateTime date);
}
```

## 分页和排序

### Pageable

```java
@Service
public class ArticleService {

    @Autowired
    private ArticleRepository articleRepository;

    public Page<Article> list(Integer page, Integer size) {
        // 创建分页对象（从 0 开始）
        Pageable pageable = PageRequest.of(page, size);

        return articleRepository.findAll(pageable);
    }

    public Page<Article> listSorted(Integer page, Integer size) {
        // 创建排序
        Sort sort = Sort.by(Sort.Direction.DESC, "createdAt");

        // 创建分页
        Pageable pageable = PageRequest.of(page, size, sort);

        return articleRepository.findAll(pageable);
    }

    public Page<Article> listMultiSort(Integer page, Integer size) {
        // 多字段排序
        Sort sort = Sort.by(
            Sort.Order.desc("createdAt"),
            Sort.Order.asc("title")
        );

        Pageable pageable = PageRequest.of(page, size, sort);

        return articleRepository.findAll(pageable);
    }
}
```

## 事务管理

### @Transactional

```java
@Service
public class UserService {

    @Autowired
    private UserRepository userRepository;

    @Autowired
    private ArticleService articleService;

    @Transactional // 默认 RuntimeException 回滚
    public void transfer(Long fromId, Long toId, Integer amount) {
        User fromUser = userRepository.findById(fromId).orElseThrow();
        User toUser = userRepository.findById(toId).orElseThrow();

        fromUser.setBalance(fromUser.getBalance() - amount);
        toUser.setBalance(toUser.getBalance() + amount);

        userRepository.save(fromUser);
        userRepository.save(toUser);
    }

    @Transactional(readOnly = true) // 只读事务
    public User getById(Long id) {
        return userRepository.findById(id).orElse(null);
    }

    @Transactional(rollbackFor = Exception.class) // 所有异常回滚
    public void updateWithException(User user) {
        userRepository.save(user);
        // ...
    }
}
```

## 最佳实践

### 1. 使用懒加载

```java
@ManyToOne(fetch = FetchType.LAZY)
@JoinColumn(name = "user_id")
private User author;
```

### 2. 避免 N+1 问题

```java
// 使用 JOIN FETCH
@Query("SELECT a FROM Article a JOIN FETCH a.author")
List<Article> findAllWithAuthor();

// 使用 @EntityGraph
@EntityGraph(attributePaths = {"author", "comments"})
List<Article> findAll();
```

### 3. 批量操作

```java
// ✅ 推荐：批量插入
List<Article> articles = new ArrayList<>();
// ... 添加文章
articleRepository.saveAll(articles);

// ❌ 避免：逐个保存
for (Article article : articles) {
    articleRepository.save(article);
}
```
