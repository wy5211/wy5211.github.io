每次有人提到年轻人配置红利低波，评论区总会出现一种声音：「你才二三十岁，投资期限几十年，应该全仓成长股最大化收益。红利低波那是退休大爷大妈的菜，年轻人买它就是浪费青春。」

这话听起来很有道理——年轻人最大的资本就是时间，应该用时间换空间，承担更高风险换取更高收益。红利低波年化7-8%，纳指过去十年年化15%+，谁不想多赚？

但这个逻辑有几个被忽略的盲点。这篇就来掰扯清楚：年轻人到底该不该买红利低波？如果买，该占多少？

## 先承认对方说对的部分

「年轻人应该多配成长股」这个建议，在纯理论层面是对的。

如果你做一个纯数学优化：25岁、投资期限40年、能承受任何波动——那全仓高成长资产（纳指、科技股）的期望终值确实最高。红利低波的7-8%年化，长期跑不过纳指的15%+。40年下来，15%年化翻160倍，8%年化翻21倍——差距是数量级的。

这个逻辑的前提是两个「如果」：

1. **如果你能承受极端波动不卖出**
2. **如果你选的成长资产长期确实高增长**

问题就出在这两个「如果」上。

## 盲点一：你能承受-50%不卖吗

理论上，年轻人在-50%的回撤中应该加仓而不是割肉。但现实中？

2022年纳指跌33%，A股很多科技基金跌40-50%。你去看看基金社区，满屏都是「扛不住了」「割肉了」「再也不碰基金了」。2025年初AI行情启动前，多少人是在2024年底绝望清仓的？

**行为金融学有一个反复验证的结论：亏损厌恶。** 亏损1块钱的心理痛苦，大约等于赚2块钱的快乐。这意味着一个-50%的回撤，你需要+100%的反弹才能在心理上"回本"。但大多数人等不到+100%——他们在-30%或-40%就崩溃了。

**结果就是：理论上应该全仓成长股的年轻人，实际操作中因为扛不住回撤而在低点卖出，完美错过后续反弹。** 理论收益很高，实际收益为零甚至为负。

红利低波的价值正在这里。它年化收益不如成长股，但最大回撤只有成长股的一半甚至三分之一。**少赚一点但能拿得住，远好过收益很高但中间被洗出局。**

这不是我瞎说。如果你回看2022年（纳指跌33%、红利低波跌约10%），大量基民在纳指最低点附近割肉，而红利低波的持有人大多安然度过——因为跌10%和跌33%，心理压力完全不是一个级别。

## 盲点二：成长资产一定高增长吗

「年轻人全仓成长股」隐含一个假设：成长股长期一定高增长。但历史告诉我们，这个假设未必成立。

**日经指数**从1990年到2012年跌了22年，最大回撤超过80%。如果你在1990年25岁的日本年轻人全仓日股，到你47岁时账户还是亏损的。你浪费的不是青春，是一整个人生前半段。

**纳斯达克**在2000年互联网泡沫破裂后，花了15年才回到2000年的高点。2000年全仓纳指的人，到2015年才回本。

**A股的创业板**，2015年高点4037点，到2024年还在1500点左右，10年没回去。

我不是说纳指会重演这些悲剧。但你得承认：成长资产的长期高增长不是必然的。它取决于技术周期、货币政策、地缘政治等太多不确定因素。把全部身家押在一个篮子里，赌它未来40年一定高增长——这个赌注太大了。

## 盲点三：混淆了「收益」和「风险调整后收益」

红利低波的真正优势不在于绝对收益，而在于**风险调整后收益（夏普比率）**。

举个对比：

|                    | 纳指100 | 红利低波 |
| ------------------ | ------- | -------- |
| 年化收益（含分红） | ~15%    | ~8%      |
| 年化波动率         | ~22%    | ~15%     |
| 最大回撤           | ~33%    | ~15%     |
| 夏普比率           | ~0.59   | ~0.40    |

纳指的夏普比率确实更高（收益高更多），看起来全仓纳指更优。但注意：这是**过去10年**的数据，恰好是美股历史上最强的10年之一。

如果换到2010-2020年（含2008金融危机余波），或者换到A股的视角，红利低波的夏普比率会更有竞争力——因为在波动更大的市场里，低波策略的风险调整优势更明显。

**红利低波提供的不是「高收益」，而是「不对称收益」：下跌时少亏很多，上涨时少赚一点。** 长期累积下来，这种不对称性产生的复利效应，可能比单纯的"高收益高波动"更实在。

## 所以年轻人该怎么配

既不是"全仓成长"，也不是"全仓红利"——而是**按比例搭配**。

一个适合大多数年轻人的框架是**核心-卫星策略**：

- **核心仓位（60-70%）**：成长型资产（纳指100、标普500、A股科技）。这部分负责长期增长引擎，用时间换取高收益。持有不动，跌了也不卖。
- **压舱仓位（20-30%）**：红利低波。这部分负责下行保护和稳定现金流。当成长仓位暴跌时，红利仓位缓冲你的整体回撤，让你不至于崩溃割肉。
- **机动仓位（0-10%）**：现金或短债。留着等机会——当成长股暴跌时抄底，当红利低波股息率到6%+时加仓。

为什么红利低波占20-30%而不是5%？因为如果只占5%，它的缓冲效果微乎其微——你的整体组合回撤几乎完全跟着成长仓位走。20-30%的红利仓位，能把你的组合最大回撤从-35%降到-25%左右，这是「能扛住」和「扛不住」的分界线。

为什么不超过30%？因为你年轻，你的时间应该用来承担风险换收益。30%的防御仓位已经足够，再多就是在浪费年轻人的风险承受能力。

### 随年龄调整

这个比例不是一成不变的。随着年龄增长，逐步增加红利低波的比重：

- **25-35岁**：成长70% + 红利20% + 现金10%
- **35-45岁**：成长60% + 红利30% + 现金10%
- **45-55岁**：成长50% + 红利40% + 现金10%
- **55岁+**：成长30% + 红利50% + 现金20%

年轻时多配成长（用时间换收益），年纪大了逐步增加红利（用稳定保生活）。这就是生命周期投资的基本框架。

## 一句话总结

「年轻人不该买红利低波」这话，就像说「年轻人不该买保险」一样——理论上你不用保险也能活，但万一出了事呢？

红利低波就是投资组合里的保险：平时拖一点后腿（收益少一点），出事时救命（回撤小一半）。年轻时可以少买，但不该不买。
