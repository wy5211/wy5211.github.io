前面两篇讲清楚了红利低波是什么、怎么选股的。这一篇做一个硬碰硬的对比：红利低波 vs 沪深300，到底差在哪？为什么一个稳步向上一个原地踏步？

这个对比很重要，因为很多人在配置A股仓位时，默认选沪深300当宽基底仓。但如果你看懂了下面的数据，你可能会重新考虑这个选择。

## 20年没涨的指数

先说一个让人难受的事实：**沪深300指数从2007年10月的5891点，到2026年中的4000多点，快20年了，还没回到历史最高点。**

如果你在2007年牛市顶部买入沪深300，到今天依然亏损14%。就算你在2021年初的高点（5930点附近）买入，到现在也还没回本。

这不是沪深300的问题，这是**所有市值加权宽基指数的结构性缺陷**。

### 为什么宽基指数会原地踏步

沪深300是按市值加权的——市值越大的公司，在指数里权重越高。这个设计逻辑上很自然：大公司代表性强嘛。但它有一个致命的问题：**市值最大的时候，往往就是泡沫最大的时候。**

回顾一下沪深300历史上的三次"权重股灾难"：

- **2007年**：牛市顶部，银行、地产、中石油市值冲天。中石油上市当天市值超万亿，随即开始长达十几年的下跌。这些巨无霸被纳入指数时正是它们的巅峰，之后拖累了指数十几年。

- **2015年**：杠杆牛市，各种概念股被炒到天上。中国中车合并后市值过万亿，然后从高处暴跌。这些被炒高的股票在市值最大时被纳入指数，崩盘后砸出了巨大的坑。

- **2021年**：核心资产牛市，茅台、宁德时代、各种"茅指数"成分股市值膨胀。宁德时代市值一度超过1.5万亿，被大量指数基金被动买入。然后新能源板块见顶回落，至今跌幅超过50%。

看到规律了吗？**市值加权指数天然在高位给出最大仓位，在低位给出最小仓位——这是一种系统性的高买低卖。**

这就是为什么沪深300原地踏步。它不是A股不涨——A股很多优质公司涨了很多倍。问题是沪深300的编制规则，让它在泡沫最大的股票上给了最大权重，泡沫一破就深度套牢。

## 红利低波为什么不一样

上一篇讲过，红利低波用**股息率倒数×波动率倒数**加权，每季度调仓。这套规则产生了一个截然不同的动态：

- 某只股票价格暴涨 → 股息率下降（分母变大）→ 权重降低或被剔除 → **自动减仓**
- 某只股票价格暴跌 → 股息率上升（分母变小）→ 权重提高或被纳入 → **自动加仓**

和沪深300的市值加权完全相反——**红利低波天然在低位给出最大仓位，在高位给出最小仓位。**

这不是人为判断，是规则强制执行的。指数编制者不需要看市场情绪、不需要预测涨跌，规则自动帮它低买高卖。

### 一个对比实验

拿具体数字说话。2018年12月底（A股上一次大底），假设你同时投入10万：

|                   | 沪深300 ETF | 红利低波 ETF |
| ----------------- | ----------- | ------------ |
| 投入              | 10万        | 10万         |
| 起始价格          | 3.079元     | 1.003元      |
| 2026.06价格       | 4.984元     | 1.115元      |
| 价格涨幅          | +62%        | +11%         |
| 年均股息率        | ~2%         | ~5.5%        |
| 含分红年化        | ~6.8%       | ~7.0%        |
| 含分红7.5年总收益 | ~63%        | ~68%         |
| 最大回撤          | -45%        | -15~20%      |

价格收益看起来沪深300赢了很多（62% vs 11%），但这只是表面。红利低波每年分红5-6%，七年半累积的分红再投入，把总收益追了上来甚至反超。

更关键的是最后一行：**最大回撤，沪深300是-45%，红利低波只有-15到20%。** 同样的收益，承担的风险完全不同。

如果你按风险调整后的收益（夏普比率）来衡量，红利低波远胜沪深300。赚同样的钱，少挨一半的打——这就是低波的价值。

## 价格收益 vs 全收益：差出一个世界

理解这个对比的关键，是搞清楚「价格收益」和「全收益」的区别。

**价格收益**（Price Return）：只看股价涨跌。10块买入，11块卖出，价格收益10%。

**全收益**（Total Return）：股价涨跌 + 分红再投入。10块买入，分红0.5块拿去再买入更多份额，股价涨到11块——你的总资产是11块 + 再投入产生的复利，全收益超过10%。

两者差距有多大？取决于股息率。股息率越高，差距越大。

对沪深300来说，年股息率约2%，7年下来价格和全收益差约15个百分点。

对红利低波来说，年股息率5-6%，7年下来价格和全收益差约45个百分点。**这就是为什么红利低波的价格曲线看起来平平无奇，但含分红的净值曲线其实一直在稳步向上。**

很多只看K线图的人会误判红利低波——因为K线图显示的是价格，不含分红。你看到红利低波的价格几乎没涨，以为它表现很差。但实际上，每年5-6%的分红一直在往你口袋里塞钱，如果再投入，复利效应会让你的实际收益远超K线图上的价格涨幅。

**这也是为什么红利低波适合「买了不看」的持有方式**——你不盯盘就不会被平淡的价格走势吓到，分红默默到账，净值默默增长。

## 为什么不直接全仓红利低波

既然红利低波长期表现更好、回撤更小，为什么不把A股仓位全换成它？三个原因：

**第一，它有风格周期。** 红利低波不是什么时候都赢。2019-2020年成长股牛市时，红利低波大幅跑输沪深300。2025年AI科技行情虹吸资金时，红利低波又持续下跌。它会阶段性失宠，有时候一输就是两三年。如果你在它跑输的周期里全仓持有，心态会很煎熬。

**第二，行业集中度太高。** 红利低波重仓银行、煤炭、高速公路这些传统行业，几乎不碰科技、医药、消费升级。如果你全仓红利低波，等于赌传统行业永远赢——这个赌注太大。

**第三，它缺少进攻弹性。** 牛市来了，沪深300能涨30-50%，红利低波可能只涨10-15%。红利低波是防守型选手，进攻能力有限。如果你需要一只能够在牛市里跟着大盘跑的ETF，沪深300（或中证A500）仍然是更好的选择。

所以更合理的配置不是二选一，而是搭配使用：**宽基做进攻引擎（吃牛市弹性），红利做防守压舱石（抗回撤 + 稳定现金流）。** 至于具体比例怎么定，第九篇会展开聊。

## 沪深300不是没有价值

说了这么多沪深300的问题，也得给它正名几句。

沪深300仍然是A股最核心的宽基指数，覆盖了最大的300家公司，代表性强、流动性好、外资配置A股的首选。如果你看好A股整体会涨，沪深300是最直接的工具。

而且，沪深300的问题正在被新一代指数解决。**中证A500**（2024年推出）在编制时加入了行业均衡规则——不再单纯按市值选，而是从各行业中选代表性公司，避免重仓少数泡沫板块。它一定程度上缓解了沪深300"市值加权高买低卖"的问题。

但要注意：中证A500仍然是市值加权为核心的宽基指数，只是加了一些行业平衡规则。它的波动率和回撤仍然大于红利低波，长期全收益也未必跑赢——因为加权方式的根本逻辑没变。

如果你要一个"更好的沪深300"，A500值得考虑。但如果你要的是"稳步向上的复利引擎"，红利低波仍然更合适。两者解决的是不同的问题。

## 这一节的核心

- **沪深300的20年原地踏步**，根源是市值加权规则——天然在泡沫高点给出最大仓位
- **红利低波的反向逻辑**：股息率加权 + 季度调仓，天然在低位加仓、高位减仓
- **价格收益 vs 全收益**：红利低波的真正价值藏在高股息率里，K线图上看不到全貌
- **搭配使用**比二选一更合理——宽基进攻，红利防守
