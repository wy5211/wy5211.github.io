React 19 终于来了！这是 React 团队自 2020 年发布 Hooks 以来最大的一次更新。本文将全面解析 React 19 的核心新特性，帮助你快速上手并应用到实际项目中。

## 🎯 React 19 的重大变化

React 19 的核心目标是**简化数据获取和状态管理**，减少开发者需要编写的样板代码。主要更新包括：

- **Actions**：统一处理表单提交和数据变更
- **新的 Hooks**：`useActionState`、`useFormStatus`、`useOptimistic`、`use`
- **Server Components 增强**：更好的服务端渲染支持
- **ref 作为 prop**：不再需要 `forwardRef`
- **文档元数据支持**：原生支持 title、meta 等标签

## 🚀 Actions：表单处理的革命

### 什么是 Actions？

Actions 是 React 19 引入的新概念，用于统一处理表单提交、数据变更等异步操作。

```tsx
// 传统方式
function Form() {
  const [isPending, setIsPending] = useState(false);
  const [error, setError] = useState(null);

  const handleSubmit = async (e) => {
    e.preventDefault();
    setIsPending(true);
    setError(null);

    try {
      await submitForm(new FormData(e.target));
    } catch (err) {
      setError(err.message);
    } finally {
      setIsPending(false);
    }
  };

  return (
    &lt;form onSubmit={handleSubmit}&gt;
      {/* 表单内容 */}
      &lt;button type="submit" disabled={isPending}&gt;
        {isPending ? '提交中...' : '提交'}
      &lt;/button&gt;
      {error && &lt;p&gt;{error}&lt;/p&gt;}
    &lt;/form&gt;
  );
}

// React 19 Actions 方式
function Form() {
  const [error, submitAction] = useActionState(async (prevState, formData) => {
    try {
      await submitForm(formData);
      return null;
    } catch (err) {
      return err.message;
    }
  }, null);

  return (
    &lt;form action={submitAction}&gt;
      {/* 表单内容 */}
      &lt;SubmitButton /&gt;
      {error && &lt;p&gt;{error}&lt;/p&gt;}
    &lt;/form&gt;
  );
}
```

### useActionState Hook

`useActionState` 用于管理 Action 的状态：

```tsx
import { useActionState } from 'react';

function LoginForm() {
  const [state, formAction, isPending] = useActionState(
    async (prevState, formData) => {
      const email = formData.get('email');
      const password = formData.get('password');

      // 验证
      if (!email || !password) {
        return { error: '请填写所有字段' };
      }

      // 提交登录
      const result = await login({ email, password });

      if (result.success) {
        redirect('/dashboard');
        return {};
      }

      return { error: result.message };
    },
    { error: null }
  );

  return (
    &lt;form action={formAction}&gt;
      &lt;input name="email" type="email" /&gt;
      &lt;input name="password" type="password" /&gt;
      &lt;button type="submit" disabled={isPending}&gt;
        {isPending ? '登录中...' : '登录'}
      &lt;/button&gt;
      {state.error && &lt;div className="error"&gt;{state.error}&lt;/div&gt;}
    &lt;/form&gt;
  );
}
```

### useFormStatus Hook

`useFormStatus` 用于获取父级表单的提交状态：

```tsx
import { useFormStatus } from 'react-dom';

function SubmitButton() {
  const { pending, data, method } = useFormStatus();

  return (
    &lt;button type="submit" disabled={pending}&gt;
      {pending ? (
        &lt;&gt;
          &lt;Spinner /&gt;
          处理中...
        &lt;/&gt;
      ) : (
        '提交'
      )}
    &lt;/button&gt;
  );
}

// 使用
function ContactForm() {
  return (
    &lt;form action={submitContact}&gt;
      &lt;input name="name" /&gt;
      &lt;input name="email" /&gt;
      &lt;textarea name="message" /&gt;
      &lt;SubmitButton /&gt;
    &lt;/form&gt;
  );
}
```

## 🎨 useOptimistic：乐观更新

乐观更新是提升用户体验的重要手段。React 19 提供了 `useOptimistic` Hook 来简化这一过程：

```tsx
import { useOptimistic } from 'react';

function TodoList({ todos, updateTodo }) {
  const [optimisticTodos, addOptimistic] = useOptimistic(
    todos,
    (state, newTodo) => [...state, { ...newTodo, pending: true }]
  );

  async function handleSubmit(formData) {
    const title = formData.get('title');

    // 立即显示乐观更新
    addOptimistic({ id: Date.now(), title, completed: false });

    // 实际提交
    await updateTodo({ title });
  }

  return (
    &lt;&gt;
      &lt;form action={handleSubmit}&gt;
        &lt;input name="title" placeholder="添加待办事项" /&gt;
        &lt;SubmitButton /&gt;
      &lt;/form&gt;

      &lt;ul&gt;
        {optimisticTodos.map(todo => (
          &lt;li key={todo.id} className={todo.pending ? 'opacity-50' : ''}&gt;
            {todo.title}
            {todo.pending && &lt;span&gt;保存中...&lt;/span&gt;}
          &lt;/li&gt;
        ))}
      &lt;/ul&gt;
    &lt;/&gt;
  );
}
```

### 实际应用：点赞功能

```tsx
function LikeButton({ postId, initialLikes, isLiked }) {
  const [likes, setLikes] = useState(initialLikes);
  const [optimisticLikes, addLike] = useOptimistic(
    likes,
    (currentLikes, increment) => currentLikes + increment
  );

  const handleLike = async () => {
    // 乐观更新
    addLike(isLiked ? -1 : 1);

    try {
      const result = await toggleLike(postId);
      setLikes(result.likes);
    } catch (error) {
      // 失败时回滚（乐观更新会自动恢复）
      console.error('点赞失败:', error);
    }
  };

  return (
    &lt;button onClick={handleLike}&gt;
      ❤️ {optimisticLikes}
    &lt;/button&gt;
  );
}
```

## 🪝 use Hook：灵活的资源读取

React 19 引入了新的 `use` Hook，可以在组件中读取 Promise 和 Context：

### 读取 Promise

```tsx
import { use, Suspense } from 'react';

function UserProfile({ userPromise }) {
  // 直接在组件中读取 Promise
  const user = use(userPromise);

  return (
    &lt;div&gt;
      &lt;h1&gt;{user.name}&lt;/h1&gt;
      &lt;p&gt;{user.email}&lt;/p&gt;
    &lt;/div&gt;
  );
}

// 父组件
function App() {
  const userPromise = fetchUser(userId);

  return (
    &lt;Suspense fallback={&lt;Loading /&gt;}&gt;
      &lt;UserProfile userPromise={userPromise} /&gt;
    &lt;/Suspense&gt;
  );
}
```

### 读取 Context

```tsx
import { use, createContext } from 'react';

const ThemeContext = createContext('light');

function Button() {
  // 使用 use 读取 Context
  const theme = use(ThemeContext);

  return (
    &lt;button className={`btn btn-${theme}`}&gt;
      点击我
    &lt;/button&gt;
  );
}
```

### 条件中使用 use

注意：`use` Hook 可以在条件语句中使用（这与其他 Hooks 不同）：

```tsx
function ConditionalData({ shouldLoad, dataPromise }) {
  if (!shouldLoad) {
    return &lt;p&gt;未加载数据&lt;/p&gt;;
  }

  // ✅ 可以在条件中使用
  const data = use(dataPromise);

  return &lt;div&gt;{data.content}&lt;/div&gt;;
}
```

## 🔄 ref 作为 prop

React 19 中，`ref` 现在可以作为普通的 prop 传递，不再需要 `forwardRef`：

```tsx
// React 18 及之前
const Input = forwardRef((props, ref) => {
  return &lt;input ref={ref} {...props} /&gt;;
});

// React 19
function Input({ ref, ...props }) {
  return &lt;input ref={ref} {...props} /&gt;;
}

// 使用
function Form() {
  const inputRef = useRef(null);

  return (
    &lt;form&gt;
      &lt;Input ref={inputRef} placeholder="输入内容" /&gt;
      &lt;button onClick={() => inputRef.current?.focus()}&gt;
        聚焦
      &lt;/button&gt;
    &lt;/form&gt;
  );
}
```

## 📄 文档元数据支持

React 19 支持在组件中直接渲染 `title`、`meta`、`link` 等标签，React 会自动将它们提升到 `&lt;head&gt;` 中：

```tsx
function BlogPost({ post }) {
  return (
    &lt;article&gt;
      {/* 这些会自动提升到 head 中 */}
      &lt;title&gt;{post.title} - 我的博客&lt;/title&gt;
      &lt;meta name="description" content={post.excerpt} /&gt;
      &lt;meta property="og:title" content={post.title} /&gt;
      &lt;meta property="og:image" content={post.cover} /&gt;

      &lt;h1&gt;{post.title}&lt;/h1&gt;
      &lt;div dangerouslySetInnerHTML={{ __html: post.content }} /&gt;
    &lt;/article&gt;
  );
}
```

## 🎭 样式表支持

React 19 提供了新的 `preload` 和 `preinit` API 来优化资源加载：

```tsx
import { preload, preinit } from 'react-dom';

function App() {
  // 预加载关键资源
  preload('/fonts/inter.woff2', { as: 'font', type: 'font/woff2' });
  preinit('/styles/critical.css', { as: 'style' });

  return &lt;div&gt;...&lt;/div&gt;;
}
```

## 🔧 其他重要改进

### 1. 自动批处理增强

React 19 增强了自动批处理，包括 Promise、setTimeout 等场景：

```tsx
function Example() {
  const [count, setCount] = useState(0);
  const [flag, setFlag] = useState(false);

  async function handleClick() {
    // React 18: 可能会触发两次渲染
    // React 19: 自动批处理，只触发一次渲染
    const data = await fetchData();
    setCount(data.count);
    setFlag(true);
  }

  return &lt;button onClick={handleClick}&gt;{count}&lt;/button&gt;;
}
```

### 2. 更好的错误处理

```tsx
import { useErrorBoundary } from 'react';

function ErrorBoundary({ children }) {
  const [error, resetErrorBoundary] = useErrorBoundary();

  if (error) {
    return (
      &lt;div role="alert"&gt;
        &lt;p&gt;出错了：{error.message}&lt;/p&gt;
        &lt;button onClick={resetErrorBoundary}&gt;重试&lt;/button&gt;
      &lt;/div&gt;
    );
  }

  return children;
}
```

### 3. 自定义元素支持

React 19 改进了对 Web Components 的支持：

```tsx
// 自定义元素现在可以正确处理
function App() {
  return (
    &lt;my-custom-element
      customProp="value"
      onCustomEvent={(e) => console.log(e.detail)}
    /&gt;
  );
}
```

## 📊 迁移指南

### 升级步骤

```bash
# 1. 升级 React
npm install react@19 react-dom@19

# 2. 升级相关库
npm install @types/react@19 @types/react-dom@19

# 3. 运行 codemod（可选）
npx react-codemod@latest rename-unsafe-lifecycles
```

### 常见迁移问题

**1. 废弃的 API**

| 废弃 API                    | 替代方案                |
| --------------------------- | ----------------------- |
| `componentWillMount`        | `useEffect`             |
| `componentWillReceiveProps` | `useEffect` + 依赖数组  |
| `componentWillUpdate`       | `useEffect` + 依赖数组  |
| `findDOMNode`               | `ref`                   |
| `string refs`               | `createRef` 或 `useRef` |

**2. 类型更新**

```tsx
// React 19 类型变化
interface Props {
  ref?: React.Ref&lt;HTMLInputElement&gt;;  // ref 现在是普通 prop
  children?: React.ReactNode;
}
```

## 💡 最佳实践总结

### 1. 表单处理优先使用 Actions

```tsx
// ✅ 推荐
&lt;form action={submitAction}&gt;

// ❌ 避免
&lt;form onSubmit={handleSubmit}&gt;
```

### 2. 善用乐观更新提升体验

```tsx
// 对于即时反馈的操作，使用 useOptimistic
const [optimistic, add] = useOptimistic(data, reducer);
```

### 3. 合理使用 Suspense 边界

```tsx
function App() {
  return (
    &lt;Suspense fallback={&lt;PageSkeleton /&gt;}&gt;
      &lt;Header /&gt;
      &lt;Suspense fallback={&lt;ContentSkeleton /&gt;}&gt;
        &lt;MainContent /&gt;
      &lt;/Suspense&gt;
      &lt;Footer /&gt;
    &lt;/Suspense&gt;
  );
}
```

### 4. 使用新的错误边界模式

```tsx
// 根组件包裹错误边界
&lt;ErrorBoundary fallback={&lt;ErrorPage /&gt;}&gt;
  &lt;App /&gt;
&lt;/ErrorBoundary&gt;
```

## 🎯 总结

React 19 带来的主要改进：

| 特性               | 优势                            |
| ------------------ | ------------------------------- |
| **Actions**        | 简化表单处理，统一异步操作模式  |
| **useActionState** | 减少 loading/error 状态样板代码 |
| **useFormStatus**  | 轻松获取表单提交状态            |
| **useOptimistic**  | 实现乐观更新变得简单            |
| **use Hook**       | 灵活读取 Promise 和 Context     |
| **ref as prop**    | 不再需要 forwardRef             |
| **文档元数据**     | 组件内声明 SEO 信息             |

React 19 是一次重大更新，它让数据获取和状态管理变得更加简单直观。建议在新项目中积极采用，现有项目可以逐步迁移。
