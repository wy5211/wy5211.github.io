# 现代 CSS 技巧：从 Flexbox 到 Grid 到 Container Queries

CSS 发展迅速，Flexbox、Grid、Container Queries 等现代布局技术让复杂的响应式布局变得简单。本文系统讲解这些技术，帮你写出更优雅的 CSS。

## 一、Flexbox 弹性布局

### 1.1 基础概念

```css
.container {
  display: flex; /* 水平排列 */
  flex-direction: column; /* 垂直排列 */
  flex-direction: row-reverse; /* 反向排列 */

  /* 主轴对齐 */
  justify-content: flex-start | flex-end | center | space-between | space-around
    | space-evenly;

  /* 交叉轴对齐 */
  align-items: stretch | flex-start | flex-end | center | baseline;

  /* 换行 */
  flex-wrap: nowrap | wrap | wrap-reverse;

  /* 多行对齐 */
  align-content: flex-start | flex-end | center | stretch | space-between |
    space-around;

  /* 间距 */
  gap: 16px;
  row-gap: 16px;
  column-gap: 16px;
}

.item {
  /* 放大比例 */
  flex-grow: 0;

  /* 缩小比例 */
  flex-shrink: 1;

  /* 基础大小 */
  flex-basis: auto;

  /* 简写 */
  flex: 1; /* flex-grow: 1; flex-shrink: 1; flex-basis: 0%; */
  flex: auto; /* flex-grow: 1; flex-shrink: 1; flex-basis: auto; */
  flex: none; /* flex-grow: 0; flex-shrink: 0; flex-basis: auto; */

  /* 单独对齐 */
  align-self: flex-start | flex-end | center | stretch | baseline;

  /* 排序 */
  order: 0;
}
```

### 1.2 常用布局模式

**水平居中：**

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

**两端对齐：**

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

**等宽列：**

```css
.container {
  display: flex;
  gap: 16px;
}

.item {
  flex: 1;
}
```

**圣杯布局：**

```css
.layout {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.header,
.footer {
  flex-shrink: 0;
}

.main {
  display: flex;
  flex: 1;
}

.sidebar {
  flex: 0 0 250px;
}

.content {
  flex: 1;
}
```

**自适应宽度：**

```css
.nav {
  display: flex;
  gap: 16px;
}

.nav-item {
  flex-shrink: 0; /* 不缩小 */
}

.search {
  flex: 1; /* 占据剩余空间 */
  max-width: 400px;
}
```

## 二、Grid 网格布局

### 2.1 基础概念

```css
.container {
  display: grid;

  /* 定义列 */
  grid-template-columns: 200px 1fr 200px; /* 固定 + 弹性 + 固定 */
  grid-template-columns: repeat(3, 1fr); /* 3 列等宽 */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* 自适应 */

  /* 定义行 */
  grid-template-rows: auto 1fr auto;

  /* 间距 */
  gap: 16px;
  row-gap: 16px;
  column-gap: 16px;

  /* 对齐 */
  justify-items: start | end | center | stretch;
  align-items: start | end | center | stretch;
  justify-content: start | end | center | space-between | space-around |
    space-evenly;
  align-content: start | end | center | space-between | space-around |
    space-evenly;
}

.item {
  /* 跨列 */
  grid-column: 1 / 3; /* 从第 1 列到第 3 列 */
  grid-column: span 2; /* 跨 2 列 */

  /* 跨行 */
  grid-row: 1 / 4;
  grid-row: span 3;

  /* 单独对齐 */
  justify-self: start | end | center | stretch;
  align-self: start | end | center | stretch;
}
```

### 2.2 常用布局模式

**响应式网格：**

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 24px;
}
```

**圣杯布局：**

```css
.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main aside"
    "footer footer footer";
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

.header {
  grid-area: header;
}
.sidebar {
  grid-area: sidebar;
}
.main {
  grid-area: main;
}
.aside {
  grid-area: aside;
}
.footer {
  grid-area: footer;
}
```

**卡片布局：**

```css
.cards {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 24px;
}

.card {
  display: grid;
  grid-template-rows: auto 1fr auto;
  gap: 16px;
}

.card-image {
  /* 图片 */
}
.card-content {
  /* 内容自动撑开 */
}
.card-footer {
  /* 底部按钮 */
}
```

**复杂仪表盘布局：**

```css
.dashboard {
  display: grid;
  grid-template-columns: 250px repeat(3, 1fr);
  grid-template-rows: 60px repeat(3, 1fr);
  gap: 16px;
  min-height: 100vh;
}

.sidebar {
  grid-column: 1;
  grid-row: 1 / -1;
}

.header {
  grid-column: 2 / -1;
}

.main-chart {
  grid-column: 2 / 4;
  grid-row: 2 / 4;
}

.stats {
  grid-column: 4;
  grid-row: 2 / 4;
}
```

## 三、Container Queries 容器查询

### 3.1 基础用法

Container Queries 让组件根据容器大小而非视口大小调整样式。

```css
.card-container {
  container-type: inline-size;
  container-name: card; /* 可选，命名容器 */
}

@container card (min-width: 400px) {
  .card {
    display: flex;
    gap: 16px;
  }

  .card-image {
    flex: 0 0 200px;
  }
}

@container card (max-width: 399px) {
  .card {
    display: block;
  }

  .card-image {
    width: 100%;
    aspect-ratio: 16 / 9;
  }
}
```

### 3.2 实用场景

**响应式卡片：**

```css
.card-container {
  container-type: inline-size;
}

.card {
  display: grid;
  gap: 16px;
}

/* 容器宽度 > 500px */
@container (min-width: 500px) {
  .card {
    grid-template-columns: 200px 1fr;
  }

  .card-title {
    font-size: 1.5rem;
  }
}

/* 容器宽度 < 500px */
@container (max-width: 499px) {
  .card {
    grid-template-columns: 1fr;
  }

  .card-title {
    font-size: 1.25rem;
  }
}
```

**侧边栏布局：**

```css
.layout-container {
  container-type: inline-size;
}

@container (min-width: 800px) {
  .layout {
    display: grid;
    grid-template-columns: 250px 1fr;
  }
}

@container (max-width: 799px) {
  .layout {
    display: block;
  }

  .sidebar {
    display: none; /* 或折叠 */
  }
}
```

## 四、CSS 变量

### 4.1 基础用法

```css
:root {
  /* 颜色 */
  --color-primary: #3b82f6;
  --color-secondary: #64748b;
  --color-text: #1e293b;
  --color-bg: #ffffff;

  /* 间距 */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --spacing-xl: 32px;

  /* 字体 */
  --font-sans: "Inter", -apple-system, sans-serif;
  --font-mono: "JetBrains Mono", monospace;

  /* 圆角 */
  --radius-sm: 4px;
  --radius-md: 8px;
  --radius-lg: 16px;

  /* 阴影 */
  --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.button {
  background: var(--color-primary);
  padding: var(--spacing-sm) var(--spacing-md);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
}
```

### 4.2 动态主题

```css
:root {
  --color-bg: #ffffff;
  --color-text: #1e293b;
}

[data-theme="dark"] {
  --color-bg: #0f172a;
  --color-text: #f1f5f9;
}

body {
  background: var(--color-bg);
  color: var(--color-text);
}
```

### 4.3 响应式变量

```css
:root {
  --container-padding: 16px;
}

@media (min-width: 768px) {
  :root {
    --container-padding: 32px;
  }
}

@media (min-width: 1024px) {
  :root {
    --container-padding: 48px;
  }
}

.container {
  padding: 0 var(--container-padding);
}
```

## 五、实用技巧

### 5.1 aspect-ratio 宽高比

```css
/* 16:9 视频 */
.video {
  aspect-ratio: 16 / 9;
  width: 100%;
}

/* 1:1 头像 */
.avatar {
  aspect-ratio: 1;
  width: 48px;
  border-radius: 50%;
}

/* 4:3 卡片图片 */
.card-image {
  aspect-ratio: 4 / 3;
  object-fit: cover;
}
```

### 5.2 gap 代替 margin

```css
/* 之前 */
.list-item + .list-item {
  margin-top: 8px;
}

/* 现在 */
.list {
  display: flex;
  flex-direction: column;
  gap: 8px;
}
```

### 5.3 min() / max() / clamp()

```css
/* 最小宽度 300px，最大 50% */
.sidebar {
  width: min(50%, 300px);
}

/* 最小 16px，最大 32px，理想 2vw */
.title {
  font-size: clamp(16px, 2vw, 32px);
}

/* 最小高度 100vh，但至少占满内容 */
.main {
  min-height: max(100vh, 600px);
}
```

### 5.4 逻辑属性

```css
/* 代替物理属性 */
.element {
  /* margin-top/bottom/left/right */
  margin-block: 16px; /* 上下 */
  margin-inline: 24px; /* 左右 */

  /* padding */
  padding-block-start: 8px;
  padding-inline-end: 16px;

  /* 边框 */
  border-inline-start: 1px solid #ccc;

  /* 位置 */
  inset: 0; /* top/right/bottom/left: 0 */
  inset-block: 0; /* top/bottom: 0 */
  inset-inline: 0; /* left/right: 0 */
}
```

### 5.5 滚动捕捉

```css
/* 卡片轮播 */
.slider {
  display: flex;
  overflow-x: auto;
  scroll-snap-type: x mandatory;
  scroll-behavior: smooth;
}

.slide {
  flex: 0 0 100%;
  scroll-snap-align: start;
}

/* 全屏滚动 */
.fullpage {
  height: 100vh;
  overflow-y: auto;
  scroll-snap-type: y mandatory;
}

.section {
  height: 100vh;
  scroll-snap-align: start;
}
```

### 5.6 子网格

```css
.parent {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

.child {
  display: grid;
  grid-template-rows: subgrid;
  grid-row: span 3; /* 子元素继承父网格的行 */
}
```

## 六、现代布局实战

### 6.1 响应式导航

```css
.nav {
  display: flex;
  align-items: center;
  gap: 16px;
  padding: 16px;
}

.nav-logo {
  flex-shrink: 0;
}

.nav-links {
  display: flex;
  gap: 24px;
  flex: 1;
}

.nav-search {
  flex: 1;
  max-width: 400px;
}

.nav-toggle {
  display: none;
}

@media (max-width: 768px) {
  .nav-links,
  .nav-search {
    display: none;
  }

  .nav-toggle {
    display: block;
  }
}
```

### 6.2 瀑布流布局

```css
.masonry {
  columns: 3 280px; /* 3列，最小280px */
  column-gap: 24px;
}

.masonry-item {
  break-inside: avoid;
  margin-bottom: 24px;
}
```

### 6.3 粘性定位

```css
.header {
  position: sticky;
  top: 0;
  z-index: 100;
  background: white;
}

.sidebar {
  position: sticky;
  top: 80px; /* 头部高度 */
  height: fit-content;
}
```

## 总结

现代 CSS 让布局变得简单而强大。

**核心要点：**

1. **Flexbox** - 一维布局，适合导航、卡片组
2. **Grid** - 二维布局，适合页面整体结构
3. **Container Queries** - 组件级响应式
4. **CSS 变量** - 主题和一致性
5. **实用函数** - clamp()、min()、max()

**学习路径：**

```
Flexbox → Grid → Container Queries → CSS 变量 → 实战技巧
```

现代 CSS 已经非常强大，大部分布局不需要 JavaScript。多练习，用 CSS 解决问题。
