# 全栈错误监控与日志追踪实战

生产环境中最可怕的不是出现错误，而是**不知道出现了错误**，或者**知道了错误却找不到原因**。本文将系统讲解如何构建完整的全栈错误监控与日志追踪体系，让你的应用具备生产级的可观测性。

## 一、为什么需要错误监控和日志追踪？

### 1.1 真实案例教训

```typescript
// ❌ 典型的生产事故场景
// 某天凌晨3点，用户投诉页面打不开
// 查看监控：服务器正常，响应时间正常
// 查看日志：海量日志中找不到任何错误信息
// 结果：排查4小时，发现是某个 API 返回了特殊字符导致前端解析崩溃

// ✅ 完善的监控体系
// 错误发生后1分钟内收到告警
// 错误详情：具体 API、请求参数、错误堆栈、用户信息
// 5分钟内定位问题并修复
```

### 1.2 可观测性三要素

| 要素               | 作用             | 工具示例                        |
| ------------------ | ---------------- | ------------------------------- |
| **Metrics (指标)** | 了解系统健康状态 | Prometheus, Grafana, CloudWatch |
| **Logs (日志)**    | 了解系统行为细节 | ELK, Loki, CloudWatch Logs      |
| **Traces (链路)**  | 了解请求完整路径 | Jaeger, Zipkin, OpenTelemetry   |

## 二、前端错误监控实战

### 2.1 全局错误捕获

```typescript
// utils/error-monitor.ts
interface ErrorContext {
  userId?: string;
  userAgent: string;
  url: string;
  timestamp: number;
  breadcrumbs: Breadcrumb[];
  extra?: Record&lt;string, any&gt;;
}

interface Breadcrumb {
  type: 'click' | 'navigation' | 'http' | 'error';
  message: string;
  timestamp: number;
  data?: any;
}

class ErrorMonitor {
  private breadcrumbs: Breadcrumb[] = [];
  private maxBreadcrumbs = 50;
  private dsn: string;
  private userId?: string;

  constructor(dsn: string) {
    this.dsn = dsn;
    this.setupGlobalHandlers();
    this.setupBreadcrumbs();
  }

  // 设置全局错误处理器
  private setupGlobalHandlers() {
    // 1. JavaScript 错误
    window.onerror = (message, source, lineno, colno, error) =&gt; {
      this.captureError(error || new Error(String(message)), {
        source,
        lineno,
        colno,
      });
      return false;
    };

    // 2. Promise 未捕获错误
    window.addEventListener('unhandledrejection', (event) =&gt; {
      this.captureError(new Error(event.reason), {
        type: 'unhandledrejection',
      });
    });

    // 3. 资源加载错误
    window.addEventListener('error', (event) =&gt; {
      if (event.target !== window) {
        this.captureError(new Error('Resource loading failed'), {
          tagName: (event.target as HTMLElement).tagName,
          src: (event.target as HTMLScriptElement).src,
        });
      }
    }, true);

    // 4. React 错误边界（需要配合 ErrorBoundary 组件）
    // 5. Vue 错误处理器（Vue.config.errorHandler）
  }

  // 记录用户行为轨迹
  private setupBreadcrumbs() {
    // 点击事件
    document.addEventListener('click', (e) =&gt; {
      this.addBreadcrumb({
        type: 'click',
        message: e.target?.toString() || 'unknown',
        timestamp: Date.now(),
        data: {
          xpath: this.getXPath(e.target as Element),
          text: (e.target as HTMLElement).innerText?.slice(0, 50),
        },
      });
    });

    // 路由变化
    if (typeof window !== 'undefined') {
      const originalPushState = history.pushState;
      history.pushState = function(...args) {
        originalPushState.apply(history, args);
        this.addBreadcrumb({
          type: 'navigation',
          message: args[2]?.toString() || 'navigation',
          timestamp: Date.now(),
        });
      };
    }

    // HTTP 请求（拦截 fetch/XMLHttpRequest）
    this.interceptHTTPRequests();
  }

  // 拦截 HTTP 请求
  private interceptHTTPRequests() {
    const originalFetch = window.fetch;
    window.fetch = async (...args) =&gt; {
      const startTime = Date.now();
      try {
        const response = await originalFetch(...args);

        this.addBreadcrumb({
          type: 'http',
          message: `Fetch: ${args[0]}`,
          timestamp: startTime,
          data: {
            method: (args[1]?.method || 'GET').toUpperCase(),
            status: response.status,
            duration: Date.now() - startTime,
          },
        });

        return response;
      } catch (error) {
        this.addBreadcrumb({
          type: 'http',
          message: `Fetch Failed: ${args[0]}`,
          timestamp: startTime,
          data: {
            error: error.message,
          },
        });
        throw error;
      }
    };
  }

  // 捕获错误
  captureError(error: Error, extra?: any) {
    const errorContext: ErrorContext = {
      userId: this.userId,
      userAgent: navigator.userAgent,
      url: window.location.href,
      timestamp: Date.now(),
      breadcrumbs: this.breadcrumbs.slice(-this.maxBreadcrumbs),
      extra: {
        ...extra,
        errorName: error.name,
        errorMessage: error.message,
        stackTrace: error.stack,
      },
    };

    this.sendError(errorContext);
  }

  // 发送错误到服务端
  private async sendError(context: ErrorContext) {
    try {
      // 使用 sendBeacon 确保页面关闭时也能发送
      if (navigator.sendBeacon) {
        const blob = new Blob([JSON.stringify(context)], {
          type: 'application/json',
        });
        navigator.sendBeacon(this.dsn, blob);
      } else {
        await fetch(this.dsn, {
          method: 'POST',
          body: JSON.stringify(context),
          keepalive: true,
        });
      }
    } catch (err) {
      console.error('Failed to send error report:', err);
    }
  }

  // 添加面包屑
  private addBreadcrumb(breadcrumb: Breadcrumb) {
    this.breadcrumbs.push(breadcrumb);
    if (this.breadcrumbs.length &gt; this.maxBreadcrumbs) {
      this.breadcrumbs.shift();
    }
  }

  // 设置用户信息
  setUser(userId: string) {
    this.userId = userId;
  }

  // 获取 XPath（用于定位元素）
  private getXPath(element: Element): string {
    // 实现略...
    return '';
  }
}

// 使用示例
const monitor = new ErrorMonitor('/api/errors');
monitor.setUser('user-123');

// 手动捕获错误
try {
  // 业务逻辑
} catch (error) {
  monitor.captureError(error, { context: 'order creation' });
}
```

### 2.2 React 错误边界

```typescript
// components/ErrorBoundary.tsx
import React from 'react';

interface Props {
  children: React.ReactNode;
  fallback?: React.ReactNode;
  onError?: (error: Error, errorInfo: React.ErrorInfo) =&gt; void;
}

interface State {
  hasError: boolean;
  error?: Error;
}

class ErrorBoundary extends React.Component&lt;Props, State&gt; {
  constructor(props: Props) {
    super(props);
    this.state = { hasError: false };
  }

  static getDerivedStateFromError(error: Error): State {
    return { hasError: true, error };
  }

  componentDidCatch(error: Error, errorInfo: React.ErrorInfo) {
    // 上报错误
    console.error('Error caught by boundary:', error, errorInfo);

    // 发送到监控服务
    if (this.props.onError) {
      this.props.onError(error, errorInfo);
    }

    // 也可以在这里调用全局错误监控实例
    // monitor.captureError(error, { componentStack: errorInfo.componentStack });
  }

  render() {
    if (this.state.hasError) {
      return this.props.fallback || (
        &lt;div style={{ padding: '20px', textAlign: 'center' }}&gt;
          &lt;h2&gt;出现了一些问题&lt;/h2&gt;
          &lt;p&gt;我们已收到错误报告，正在修复中&lt;/p&gt;
          &lt;button onClick={() =&gt; window.location.reload()}&gt;
            刷新页面
          &lt;/button&gt;
        &lt;/div&gt;
      );
    }

    return this.props.children;
  }
}

// 使用示例
function App() {
  return (
    &lt;ErrorBoundary
      onError={(error, info) =&gt; {
        // 上报到监控系统
      }}
      fallback={&lt;CustomErrorPage /&gt;}
    &gt;
      &lt;Router&gt;
        &lt;Routes /&gt;
      &lt;/Router&gt;
    &lt;/ErrorBoundary&gt;
  );
}
```

### 2.3 性能监控（Web Vitals）

```typescript
// utils/performance-monitor.ts
import { onCLS, onFID, onLCP, onFCP, onTTFB } from 'web-vitals';

interface PerformanceMetrics {
  cls: number;  // Cumulative Layout Shift
  fid: number;  // First Input Delay
  lcp: number;  // Largest Contentful Paint
  fcp: number;  // First Contentful Paint
  ttfb: number; // Time to First Byte
}

class PerformanceMonitor {
  private metrics: Partial&lt;PerformanceMetrics&gt; = {};

  constructor() {
    this.init();
  }

  private init() {
    // 核心 Web Vitals
    onCLS((metric) =&gt; {
      this.metrics.cls = metric.value;
      this.report('CLS', metric);
    });

    onFID((metric) =&gt; {
      this.metrics.fid = metric.value;
      this.report('FID', metric);
    });

    onLCP((metric) =&gt; {
      this.metrics.lcp = metric.value;
      this.report('LCP', metric);
    });

    // 其他性能指标
    onFCP((metric) =&gt; {
      this.metrics.fcp = metric.value;
      this.report('FCP', metric);
    });

    onTTFB((metric) =&gt; {
      this.metrics.ttfb = metric.value;
      this.report('TTFB', metric);
    });

    // 页面加载完成后上报
    window.addEventListener('load', () =&gt; {
      setTimeout(() =&gt; {
        this.reportAll();
      }, 5000); // 延迟5秒确保所有指标已收集
    });
  }

  private report(name: string, metric: any) {
    // 发送到性能监控服务
    navigator.sendBeacon('/api/performance', JSON.stringify({
      name,
      value: metric.value,
      rating: metric.rating, // 'good' | 'needs-improvement' | 'poor'
      id: metric.id,
      timestamp: Date.now(),
      url: window.location.href,
    }));
  }

  private reportAll() {
    // 汇总上报
    fetch('/api/performance/batch', {
      method: 'POST',
      body: JSON.stringify({
        metrics: this.metrics,
        url: window.location.href,
        userAgent: navigator.userAgent,
        timestamp: Date.now(),
      }),
      keepalive: true,
    });
  }
}

// 性能预算检查
const PERFORMANCE_BUDGETS = {
  lcp: { good: 2500, poor: 4000 }, // ms
  fid: { good: 100, poor: 300 },   // ms
  cls: { good: 0.1, poor: 0.25 },  // score
};

function checkBudget(metric: string, value: number) {
  const budget = PERFORMANCE_BUDGETS[metric];
  if (!budget) return 'unknown';

  if (value &lt;= budget.good) return 'good';
  if (value &lt;= budget.poor) return 'needs-improvement';
  return 'poor';
}
```

// 日志级别使用规范
export class AppLogger {
  private context: LogContext;

  constructor(context: LogContext = {}) {
    this.context = context;
  }

  // 调试信息（仅开发环境）
  debug(message: string, data?: any) {
    logger.debug({ ...this.context, ...data }, message);
  }

  // 一般信息
  info(message: string, data?: any) {
    logger.info({ ...this.context, ...data }, message);
  }

  // 警告（不影响主流程）
  warn(message: string, data?: any) {
    logger.warn({ ...this.context, ...data }, message);
  }

  // 错误（需要关注）
  error(message: string, error?: Error, data?: any) {
    logger.error(
      {
        ...this.context,
        ...data,
        error: {
          name: error?.name,
          message: error?.message,
          stack: error?.stack,
        },
      },
      message
    );
  }

  // 关键操作（审计日志）
  audit(action: string, data: any) {
    logger.info(
      {
        ...this.context,
        audit: true,
        action,
        timestamp: Date.now(),
      },
      `AUDIT: ${action}`
    );
  }

  // 性能日志
  metric(name: string, value: number, unit: string = 'ms') {
    logger.info(
      {
        ...this.context,
        metric: { name, value, unit },
      },
      `METRIC: ${name}`
    );
  }
}

### 3.2 Express 日志中间件

```typescript
// middleware/request-logger.ts
import { Request, Response, NextFunction } from 'express';
import { v4 as uuidv4 } from 'uuid';
import { AppLogger } from '../utils/logger';

// 生成 Trace ID
declare global {
  namespace Express {
    interface Request {
      traceId: string;
      logger: AppLogger;
    }
  }
}

export function requestLogger() {
  return (req: Request, res: Response, next: NextFunction) =&gt; {
    const startTime = Date.now();

    // 生成或继承 Trace ID
    req.traceId = req.headers['x-trace-id'] as string || uuidv4();

    // 创建请求级别的 logger
    req.logger = new AppLogger({
      traceId: req.traceId,
      userId: (req as any).user?.id,
      ip: req.ip,
      userAgent: req.headers['user-agent'],
    });

    // 请求开始日志
    req.logger.info('Request started', {
      method: req.method,
      path: req.path,
      query: req.query,
    });

    // 响应完成日志
    const originalEnd = res.end;
    res.end = function(...args: any[]) {
      const duration = Date.now() - startTime;

      req.logger.info('Request completed', {
        method: req.method,
        path: req.path,
        statusCode: res.statusCode,
        duration,
        contentLength: res.get('content-length'),
      });

      // 慢请求告警
      if (duration &gt; 1000) {
        req.logger.warn('Slow request detected', {
          duration,
          threshold: 1000,
        });
      }

      return originalEnd.apply(res, args);
    };

    // 设置响应头
    res.setHeader('X-Trace-Id', req.traceId);

    next();
  };
}

// 错误日志中间件
export function errorLogger(err: Error, req: Request, res: Response, next: NextFunction) {
  req.logger.error('Unhandled error', err, {
    method: req.method,
    path: req.path,
    body: req.body,
    query: req.query,
  });

  next(err);
}

// 使用示例
import express from 'express';

const app = express();

app.use(requestLogger());

app.get('/api/users', (req, res) =&gt; {
  req.logger.info('Fetching users', { page: req.query.page });
  // ...
});

app.use(errorLogger);
```

export async function processOrder(orderId: string) {

      span.end();
    });

    // 步骤3: 创建支付
    await tracer.startActiveSpan('create-payment', async (span) =&gt; {
      span.setAttributes({ orderId });
      await createPayment(orderId);
      span.end();
    });

    span.setStatus({ code: 0 }); // OK
  } catch (error) {
    span.recordException(error);
    span.setStatus({ code: 2, message: error.message }); // ERROR
    throw error;
  } finally {
    span.end();
  }
}

import { context, trace, propagation } from '@opentelemetry/api';

// 从请求头提取 Trace Context
export function extractTraceContext(headers: Record&lt;string, string&gt;) {
  return propagation.extract(context.active(), headers);
}

// 向请求头注入 Trace Context
export function injectTraceContext(carrier: Record&lt;string, string&gt;) {
  propagation.inject(context.active(), carrier);
}

// HTTP 客户端示例（axios）
import axios from 'axios';

export async function callExternalAPI(url: string, data: any) {
  const headers: Record&lt;string, string&gt; = {};

  // 注入 Trace Context
  injectTraceContext(headers);

  const span = tracer.startSpan('http-call', {
    attributes: {
      'http.url': url,
      'http.method': 'POST',
    },
  });

  try {
    const response = await axios.post(url, data, { headers });
    span.setAttributes({ 'http.status_code': response.status });
    span.setStatus({ code: 0 });
    return response.data;
  } catch (error) {
    span.recordException(error);
    span.setStatus({ code: 2, message: error.message });
    throw error;
  } finally {
    span.end();
  }
}

// Express 中间件：提取并设置 Trace Context
export function traceMiddleware() {
  return (req: Request, res: Response, next: NextFunction) =&gt; {
    const extractedContext = extractTraceContext(req.headers as any);

    context.with(extractedContext, () =&gt; {
      const span = tracer.startSpan(`${req.method} ${req.route?.path || req.path}`, {
        attributes: {
          'http.method': req.method,
          'http.url': req.url,
          'http.host': req.headers.host,
        },
      });

      (req as any).span = span;

      res.on('finish', () =&gt; {
        span.setAttributes({ 'http.status_code': res.statusCode });
        span.setStatus({ code: res.statusCode &lt; 400 ? 0 : 2 });
        span.end();
      });

      next();
    });
  };
}

// 错误聚合器（避免重复告警）
export class ErrorAggregator {
  private errors = new Map&lt;string, { count: number; firstSeen: number; lastSeen: number }&gt;();
  private threshold = 5; // 同一错误出现5次才告警
  private windowMs = 300000; // 5分钟窗口

  addError(errorKey: string): boolean {
    const now = Date.now();
    const entry = this.errors.get(errorKey);

    if (!entry) {
      this.errors.set(errorKey, {
        count: 1,
        firstSeen: now,
        lastSeen: now,
      });
      return false;
    }

    // 检查是否在时间窗口内
    if (now - entry.firstSeen &gt; this.windowMs) {
      // 重置窗口
      this.errors.set(errorKey, {
        count: 1,
        firstSeen: now,
        lastSeen: now,
      });
      return false;
    }

    // 增加计数
    entry.count++;
    entry.lastSeen = now;

    // 达到阈值才告警
    return entry.count === this.threshold;
  }

  // 清理过期条目
  cleanup() {
    const now = Date.now();
    for (const [key, entry] of this.errors.entries()) {
      if (now - entry.lastSeen &gt; this.windowMs) {
        this.errors.delete(key);
      }
    }
  }
}

// utils/sensitive-data-filter.ts
const SENSITIVE_FIELDS = [
  'password',
  'token',
  'secret',
  'apiKey',
  'creditCard',
  'ssn',
  'email',
  'phone',
];

export function sanitizeLogData(data: any): any {
  if (typeof data !== 'object' || data === null) {
    return data;
  }

  if (Array.isArray(data)) {
    return data.map(sanitizeLogData);
  }

  const sanitized: any = {};

  for (const [key, value] of Object.entries(data)) {
    const lowerKey = key.toLowerCase();

    // 检查是否是敏感字段
    if (SENSITIVE_FIELDS.some(field =&gt; lowerKey.includes(field))) {
      sanitized[key] = '[REDACTED]';
    } else if (typeof value === 'object') {
      sanitized[key] = sanitizeLogData(value);
    } else {
      sanitized[key] = value;
    }
  }

  return sanitized;
}

// 使用示例

  userId: '123',
  email: 'user@example.com',    // → [REDACTED]
  password: 'secret123',         // → [REDACTED]

}));

## 八、总结

### 关键要点

1. **监控是必需品，不是奢侈品** - 生产环境必须有完整的监控体系
2. **结构化日志** - 使用 JSON 格式，便于查询和分析
3. **链路追踪** - 微服务架构下必备的调试工具
4. **告警策略** - 避免告警疲劳，只对重要事件告警
5. **持续优化** - 定期回顾告警规则，减少误报

### 实施路径

```
Week 1: 搭建基础日志体系
  ├─ 集成结构化日志
  ├─ 添加 Trace ID
  └─ 配置日志聚合

Week 2: 完善前端监控
  ├─ 错误捕获
  ├─ 性能监控
  └─ 用户行为追踪

Week 3: 添加链路追踪
  ├─ OpenTelemetry 集成
  ├─ 关键路径埋点
  └─ 调用链可视化

Week 4: 配置告警和看板
  ├─ 定义告警规则
  ├─ 配置通知渠道
  └─ 创建监控看板
```

### 避免的坑

❌ **过度日志** - 日志太多反而难以发现问题  
❌ **忽略性能** - 日志和监控本身也会影响性能  
❌ **缺少上下文** - 只有错误信息没有上下文，难以调试  
❌ **告警过多** - 导致告警疲劳，真正重要的告警被忽视  
❌ **没有 Runbook** - 收到告警却不知道如何处理

### 进阶方向

- **AI 辅助故障诊断** - 使用机器学习识别异常模式
- **自动化根因分析** - 自动定位问题根源
- **预测性告警** - 在问题发生前预警
- **混沌工程** - 主动注入故障测试监控能力

---

完善的错误监控和日志追踪体系是生产级应用的基石。投入时间搭建好这套体系，能在关键时刻节省大量排查时间，提升系统可靠性和用户体验。

记住：**没有监控的系统就像闭着眼睛开车，迟早会出问题。** 🔍
