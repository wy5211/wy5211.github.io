## 前言

CRUD（Create, Read, Update, Delete）是数据库操作的核心。Mongoose 提供了丰富而灵活的 API 来执行这些操作。

本文将深入探讨 Mongoose 的 CRUD 操作，包括基础用法、高级查询技巧、批量操作和性能优化。

## 核心概念

### CRUD 操作映射

```
┌─────────────────────────────────────────────────────────────┐
│                    Mongoose CRUD 操作                         │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  Create（创建）                                               │
│  ├─ new Model().save()  创建单个文档                        │
│  ├─ Model.create()      创建并保存文档                       │
│  └─ Model.insertMany()  批量插入文档                         │
│                                                               │
│  Read（读取）                                                │
│  ├─ Model.find()         查找多个文档                        │
│  ├─ Model.findOne()      查找单个文档                        │
│  ├─ Model.findById()    通过 ID 查找                        │
│  └─ Model.countDocuments() 统计文档数量                      │
│                                                               │
│  Update（更新）                                              │
│  ├─ document.save()        保存修改                          │
│  ├─ Model.updateOne()      更新一个文档                      │
│  ├─ Model.updateMany()     更新多个文档                      │
│  ├─ Model.findOneAndUpdate() 查找并更新                     │
│  └─ Model.findByIdAndUpdate() 通过 ID 更新                    │
│                                                               │
│  Delete（删除）                                              │
│  ├─ document.deleteOne()   删除文档本身                      │
│  ├─ Model.deleteOne()      删除一个文档                      │
│  ├─ Model.deleteMany()     删除多个文档                      │
│  ├─ Model.findOneAndDelete() 查找并删除                    │
│  └─ Model.findByIdAndDelete() 通过 ID 删除                 │
│                                                               │
└─────────────────────────────────────────────────────────────┘
```

## 实战演练

### Create（创建操作）

创建 `src/examples/crud-create.ts`：

```typescript
import User from "../models/User";
import Post from "../models/Post";
import connectDB from "../config/database";

await connectDB();
```

### Read（读取操作）

创建 `src/examples/crud-read.ts`：

```typescript
import Post from "../models/Post";
import connectDB from "../config/database";

await connectDB();

// ==================== 基础查询 ====================
console.log("\n=== 基础查询 ===");

// 查找所有文档
const allPosts = await Post.find();
console.log(`📋 总文章数: ${allPosts.length}`);

// 条件查询
const publishedPosts = await Post.find({ status: "published" });
console.log(`✅ 已发布文章: ${publishedPosts.length}`);

// ==================== 查询运算符 ====================
console.log("\n=== 查询运算符 ===");

// 比较运算符
const popularPosts = await Post.find({
  views: { $gte: 100 }, // 大于等于
});
console.log(`🔥 热门文章 (>=100浏览): ${popularPosts.length}`);

// 组合条件
const recentPopularPosts = await Post.find({
  status: "published",
  views: { $gte: 50 },
  createdAt: { $gte: new Date(Date.now() - 7 * 24 * 60 * 60 * 1000) }, // 最近7天
});
console.log(`📈 最近热门文章: ${recentPopularPosts.length}`);

// 逻辑运算符
const tagPosts = await Post.find({
  $or: [{ tags: "mongodb" }, { tags: "nodejs" }],
});
console.log(`🏷️  特定标签文章: ${tagPosts.length}`);

// ==================== 投影（Projection） ====================
console.log("\n=== 投影（选择字段）===");

// 只返回指定字段
const postTitles = await Post.find({ status: "published" })
  .select("title views createdAt")
  .limit(3);

console.log("📝 文章列表（只包含标题、浏览量、创建时间）:");
postTitles.forEach((post) => {
  console.log(`  - ${post.title} (${post.views} 浏览)`);
});

// 排除字段
const postsWithoutContent = await Post.find()
  .select("-content -metadata")
  .limit(2);

// ==================== 排序 ====================
console.log("\n=== 排序 ===");

// 单字段排序
const latestPosts = await Post.find({ status: "published" })
  .sort({ createdAt: -1 }) // -1 降序，1 升序
  .limit(3);

console.log("🆕 最新文章:");
latestPosts.forEach((post) => console.log(`  - ${post.title}`));

// 多字段排序
const sortedPosts = await Post.find({ status: "published" })
  .sort({ views: -1, createdAt: -1 }) // 先按浏览量降序，再按创建时间降序
  .limit(3);

console.log("🔥 热门最新文章:");
sortedPosts.forEach((post) =>
  console.log(`  - ${post.title} (${post.views} 浏览)`)
);

// ==================== 分页 ====================
console.log("\n=== 分页 ===");

const page = 1;
const limit = 5;
const skip = (page - 1) * limit;

const paginatedPosts = await Post.find({ status: "published" })
  .skip(skip)
  .limit(limit)
  .sort({ createdAt: -1 });

const totalCount = await Post.countDocuments({ status: "published" });
const totalPages = Math.ceil(totalCount / limit);

console.log(`📄 第 ${page} 页 (共 ${totalPages} 页, ${totalCount} 篇文章):`);
paginatedPosts.forEach((post) => console.log(`  - ${post.title}`));

// ==================== 查询单个文档 ====================
console.log("\n=== 查询单个文档 ===");

// findOne
const firstPost = await Post.findOne({ status: "published" }).sort({
  createdAt: 1,
});
console.log("🥇 第一篇文章:", firstPost?.title);

// findById
const postById = await Post.findById(firstPost?._id);
console.log("🆔 通过 ID 查询:", postById?.title);

// ==================== 链式查询 ====================
console.log("\n=== 链式查询 ===");

const result = await Post.find({ status: "published" })
  .where("views")
  .gte(50) // 使用 where 子句
  .select("title views")
  .sort({ views: -1 })
  .limit(5)
  .lean(); // 返回普通 JS 对象（性能更好）

console.log("🔍 链式查询结果:");
result.forEach((post: any) => console.log(`  - ${post.title} (${post.views})`));

// ==================== 字段匹配 ====================
console.log("\n=== 字段匹配 ===");

// 精确匹配
const exactMatch = await Post.find({ status: "published" });
console.log(`✅ 精确匹配: ${exactMatch.length}`);

// in 操作符
const tagInPosts = await Post.find({
  tags: { $in: ["mongodb", "nodejs"] },
});
console.log(`🏷️  包含标签: ${tagInPosts.length}`);

// all 操作符（数组包含所有值）
const allTagsPosts = await Post.find({
  tags: { $all: ["mongodb", "nodejs"] },
});
console.log(`🏷️  包含所有标签: ${allTagsPosts.length}`);

// size 操作符（数组长度）
const postsWithTags = await Post.find({
  tags: { $size: 3 },
});
console.log(`📏 有3个标签的文章: ${postsWithTags.length}`);

// ==================== 正则表达式 ====================
console.log("\n=== 正则表达式 ===");

const regexPosts = await Post.find({
  title: { $regex: /mongoose/i }, // i 忽略大小写
});
console.log(`🔍 标题包含 "mongoose": ${regexPosts.length}`);
regexPosts.forEach((post) => console.log(`  - ${post.title}`));

// ==================== 元素操作符 ====================
console.log("\n=== 元素操作符 ===");

// exists（字段存在）
const postsWithMetadata = await Post.find({
  "metadata.seoTitle": { $exists: true },
});
console.log(`📋 有 SEO 标题的文章: ${postsWithMetadata.length}`);

// type（字段类型）
const postsWithStringId = await Post.find({
  _id: { $type: "string" },
});
console.log(`📝 ID 为字符串的文章: ${postsWithStringId.length}`);

// ==================== 统计查询 ====================
console.log("\n=== 统计查询 ===");

// countDocuments（精确计数）
const publishedCount = await Post.countDocuments({ status: "published" });
console.log(`✅ 已发布文章数: ${publishedCount}`);

// estimatedDocumentCount（快速估算）
const totalCount = await Post.estimatedDocumentCount();
console.log(`📊 总文档数（估算）: ${totalCount}`);

// distinct（去重）
const uniqueTags = await Post.distinct("tags", { status: "published" });
console.log(`🏷️  唯一标签: ${uniqueTags.join(", ")}`);

process.exit(0);
```

创建 `src/examples/crud-update.ts`：

```typescript
import Post from "../models/Post";
import connectDB from "../config/database";

await connectDB();

// 创建测试文章
const post = await Post.create({
  title: "测试文章",
  content: "这是一篇测试文章的内容...",
  status: "draft",
  tags: ["test"],
});

console.log("\n=== 测试文章创建成功 ===");
console.log("🆔 ID:", post._id);
console.log("📝 标题:", post.title);
console.log("📊 状态:", post.status);
console.log("👀 浏览量:", post.views);

// ==================== 方式一：修改后保存 ====================
console.log("\n=== 方式一：document.save() ===");

post.title = "更新后的标题";
post.views = 100;

const savedPost = await post.save();
console.log("✅ 文档更新成功:");
console.log("📝 新标题:", savedPost.title);
console.log("👀 浏览量:", savedPost.views);

// ==================== 方式二：updateOne ====================
console.log("\n=== 方式二：Model.updateOne() ===");

const updateResult1 = await Post.updateOne(
  { _id: post._id },
  { $set: { title: "第二次更新的标题" } }
);
console.log("✅ 更新结果:", updateResult1);
console.log("  - 匹配文档数:", updateResult1.matchedCount);
console.log("  - 修改文档数:", updateResult1.modifiedCount);

// ==================== 方式三：findOneAndUpdate ====================
console.log("\n=== 方式三：Model.findOneAndUpdate() ===");

const updatedPost1 = await Post.findOneAndUpdate(
  { _id: post._id },
  { $set: { status: "published" } },
  { new: true } // 返回更新后的文档
);
console.log("✅ 查找并更新成功:");
console.log("📝 标题:", updatedPost1?.title);
console.log("📊 状态:", updatedPost1?.status);

// ==================== 方式四：findByIdAndUpdate ====================
console.log("\n=== 方式四：Model.findByIdAndUpdate() ===");

const updatedPost2 = await Post.findByIdAndUpdate(
  post._id,
  {
    $set: {
      title: "最终标题",
      views: 200,
    },
    $push: { tags: "updated" }, // 添加到数组
  },
  {
    new: true,
    runValidators: true, // 运行验证器
  }
);
console.log("✅ 通过 ID 更新成功:");
console.log("📝 标题:", updatedPost2?.title);
console.log("👀 浏览量:", updatedPost2?.views);
console.log("🏷️  标签:", updatedPost2?.tags);

// ==================== 更新运算符 ====================
console.log("\n=== 更新运算符 ===");

// $inc（增加/减少）
await Post.findByIdAndUpdate(post._id, {
  $inc: { views: 10, likes: 5 },
});
console.log("✅ $inc: 浏览量+10, 点赞+5");

// $mul（乘法）
await Post.findByIdAndUpdate(post._id, {
  $mul: { views: 2 }, // 浏览量乘以2
});
console.log("✅ $mul: 浏览量×2");

// $rename（重命名字段）
await Post.findByIdAndUpdate(post._id, {
  $rename: { "metadata.seoTitle": "metadata.title" },
});
console.log("✅ $rename: seoTitle → title");

// $unset（删除字段）
await Post.findByIdAndUpdate(post._id, {
  $unset: { "metadata.seoDescription": "" },
});
console.log("✅ $unset: 删除 seoDescription");

// ==================== 数组更新运算符 ====================
console.log("\n=== 数组更新运算符 ===");

// $push（添加到数组）
await Post.findByIdAndUpdate(post._id, {
  $push: { tags: "new-tag" },
});
console.log("✅ $push: 添加标签");

// $addToSet（添加到集合，避免重复）
await Post.findByIdAndUpdate(post._id, {
  $addToSet: { tags: "mongodb" },
});
console.log("✅ $addToSet: 添加唯一标签");

// $pull（从数组删除）
await Post.findByIdAndUpdate(post._id, {
  $pull: { tags: "test" },
});
console.log("✅ $pull: 删除标签");

// $pop（删除数组首尾）
await Post.findByIdAndUpdate(post._id, {
  $pop: { tags: 1 }, // 1 删除最后一个，-1 删除第一个
});
console.log("✅ $pop: 删除数组元素");

// ==================== 批量更新 ====================
console.log("\n=== 批量更新 ===");

const bulkUpdateResult = await Post.updateMany(
  { status: "draft" },
  {
    $set: { status: "published" },
    $inc: { views: 1 },
  }
);
console.log("✅ 批量更新结果:");
console.log("  - 匹配文档数:", bulkUpdateResult.matchedCount);
console.log("  - 修改文档数:", bulkUpdateResult.modifiedCount);

// ==================== replaceOne ====================
console.log("\n=== replaceOne ===");

// 完全替换文档（保留 _id）
await Post.replaceOne(
  { _id: post._id },
  {
    title: "完全替换的文章",
    content: "新的内容...",
    author: post.author,
    status: "published",
    tags: ["replaced"],
  }
);
console.log("✅ 文档已完全替换");

process.exit(0);
```

### 综合示例：文章管理 API

创建 `src/examples/crud-api.ts`：

```typescript
import Post from "../models/Post";
import connectDB from "../config/database";

await connectDB();

// ==================== 文章管理类 ====================
class ArticleManager {
  // 创建文章
  async createArticle(data: any) {
    const post = await Post.create(data);
    return await Post.findById(post._id).populate("author", "name email");
  }

  // 获取文章列表（分页）
  async getArticles(page = 1, limit = 10, status = "published") {
    const skip = (page - 1) * limit;
    const [articles, total] = await Promise.all([
      Post.find({ status })
        .skip(skip)
        .limit(limit)
        .sort({ createdAt: -1 })
        .populate("author", "name avatar")
        .lean(),
      Post.countDocuments({ status }),
    ]);

    return {
      articles,
      pagination: {
        page,
        limit,
        total,
        totalPages: Math.ceil(total / limit),
      },
    };
  }

  // 获取文章详情
  async getArticleById(id: string) {
    const article = await Post.findById(id)
      .populate("author", "name email avatar bio")
      .populate("comments.author", "name avatar");

    if (!article) {
      throw new Error("文章不存在");
    }

    // 增加浏览量
    await article.incrementViews();

    return article;
  }

  // 更新文章
  async updateArticle(id: string, updates: any) {
    const article = await Post.findByIdAndUpdate(
      id,
      { $set: updates },
      {
        new: true,
        runValidators: true,
      }
    ).populate("author", "name email");

    if (!article) {
      throw new Error("文章不存在");
    }

    return article;
  }

  // 删除文章
  async deleteArticle(id: string) {
    const article = await Post.findByIdAndDelete(id);

    if (!article) {
      throw new Error("文章不存在");
    }

    return { message: "文章已删除", id };
  }

  // 搜索文章
  async searchArticles(keyword: string, page = 1, limit = 10) {
    const skip = (page - 1) * limit;

    const [articles, total] = await Promise.all([
      Post.find({
        status: "published",
        $or: [
          { title: { $regex: keyword, $options: "i" } },
          { content: { $regex: keyword, $options: "i" } },
          { tags: { $regex: keyword, $options: "i" } },
        ],
      })
        .skip(skip)
        .limit(limit)
        .sort({ createdAt: -1 })
        .populate("author", "name avatar")
        .lean(),
      Post.countDocuments({
        status: "published",
        $or: [
          { title: { $regex: keyword, $options: "i" } },
          { content: { $regex: keyword, $options: "i" } },
          { tags: { $regex: keyword, $options: "i" } },
        ],
      }),
    ]);

    return {
      articles,
      pagination: {
        page,
        limit,
        total,
        totalPages: Math.ceil(total / limit),
      },
    };
  }
}
```

### 4. 分页实现

```typescript
async function paginate(model: any, query: any, page = 1, limit = 10) {
  const skip = (page - 1) * limit;

  const [data, total] = await Promise.all([
    model.find(query).skip(skip).limit(limit).sort({ createdAt: -1 }),
    model.countDocuments(query),
  ]);

  return {
    data,
    pagination: {
      page,
      limit,
      total,
      totalPages: Math.ceil(total / limit),
      hasNext: page * limit < total,
      hasPrev: page > 1,
    },
  };
}
```

## 常见问题

### Q1: save() 和 updateOne() 的区别？

**A**:

- **save()**: Document 方法，需要先查询文档
- **updateOne()**: Model 方法，直接更新，不触发验证中间件

```typescript
// save() - 触发中间件
const post = await Post.findById(id);
post.title = "New Title";
await post.save(); // 触发 pre/post save 钩子

// updateOne() - 不触发中间件
await Post.updateOne({ _id: id }, { $set: { title: "New Title" } });
```

### Q2: lean() 什么时候使用？

**A**:

- ✅ 只读数据、需要性能时
- ❌ 需要调用文档方法、虚拟字段时

```typescript
// ✅ API 返回（性能优先）
const posts = await Post.find().lean();

// ❌ 需要使用文档方法
const post = await Post.findById(id);
await post.incrementViews(); // lean() 后不能调用
```

### Q3: 如何处理大数据量分页？

**A**:

```typescript
// ✅ 使用基于游标的分页（性能更好）
async function cursorPaginate(lastId: string | null, limit = 10) {
  const query = lastId ? Post.find({ _id: { $gt: lastId } }) : Post.find();

  const posts = await query.sort({ _id: 1 }).limit(limit).lean();

  const newLastId = posts.length === limit ? posts[posts.length - 1]._id : null;

  return { posts, lastId: newLastId, hasMore: posts.length === limit };
}
```

### Q4: 如何避免 N+1 查询问题？

**A**:

```typescript
// ❌ N+1 查询
const posts = await Post.find();
for (const post of posts) {
  const author = await User.findById(post.author); // N 次查询
}

// ✅ 使用 populate
const posts = await Post.find().populate("author"); // 2 次查询

// ✅ 或者使用聚合
const posts = await Post.aggregate([
  {
    $lookup: {
      from: "users",
      localField: "author",
      foreignField: "_id",
      as: "author",
    },
  },
]);
```
