## 前言

Mongoose 的中间件（Middleware）和 Hooks 机制允许我们在文档操作的生命周期中插入自定义逻辑。这是实现业务规则、数据验证和自动化操作的关键功能。

## 核心概念

### 中间件类型

```
Document Hooks:
  - save, validate
  - init (创建时)

Query Hooks:
  - find, findOne, updateOne, deleteOne
  - count (支持正则匹配: /^find/)

Model Hooks:
  - insertMany, validate, bulkWrite
```

### Hook 执行顺序

```
pre (前置钩子)
  ↓
操作 (save, find, update 等)
  ↓
post (后置钩子)
```

## 实战演练

### pre/post save Hooks

```typescript
// 自动更新时间戳
userSchema.pre("save", function (next) {
  this.updatedAt = new Date();
  next();
});

// 密码加密
userSchema.pre("save", async function (next) {
  if (!this.isModified("password")) {
    return next();
  }

  const bcrypt = require("bcrypt");
  const salt = await bcrypt.genSalt(10);
  this.password = await bcrypt.hash(this.password, salt);
  next();
});

// 发送通知
postSchema.post("save", async function (doc, next) {
  if (doc.status === "published") {
    await Notification.create({
      type: "post_published",
      postId: doc._id,
      userId: doc.author,
    });
  }
  next();
});
```

### Query Hooks

```typescript
// 软删除过滤
postSchema.pre(/^find/, function (next) {
  this.where({ deletedAt: { $exists: false } });
  next();
});

// 更新时间
postSchema.pre("findOneAndUpdate", function (next) {
  this.set({ updatedAt: new Date() });
  next();
});

// 删除前验证
userSchema.pre(
  "deleteOne",
  { query: true, document: false },
  async function (next) {
    const user = await this.model.findOne(this.getFilter());
    if (user.role === "admin") {
      next(new Error("Cannot delete admin user"));
    } else {
      next();
    }
  }
);
```

### 虚拟字段更新

```typescript
// 评论计数器
postSchema.virtual("commentCount").get(function () {
  return this.comments.length;
});

// 或使用钩子更新
postSchema.post("save", async function (doc) {
  await User.findByIdAndUpdate(doc.author, {
    $inc: { postCount: 1 },
  });
});
```

## 最佳实践

### Hook 错误处理

```typescript
schema.pre("save", function (next) {
  if (someCondition) {
    next(new Error("Validation failed"));
  } else {
    next();
  }
});
```

### 异步 Hook

```typescript
schema.pre("save", async function (next) {
  await doSomething();
  next();
});
```

## 常见问题

### Q1: Hook 中可以使用 this 吗？

**A**: 可以，但要注意：

- pre Hook: this 是文档实例
- post Hook: this 是已保存的文档

### Q2: 如何跳过某些 Hook？

**A**:

```typescript
const user = new User({ ... });
await user.save({ validateBeforeSave: false });
```

3. ✅ pre/post Hooks
