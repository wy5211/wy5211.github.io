## 前言

MongoDB 是最流行的 NoSQL 数据库，而 Mongoose 是 MongoDB 最优秀的 ODM（对象文档映射）库。为什么需要 Mongoose？

- **Schema 验证**：为 MongoDB 文档提供结构化定义和数据验证
- **类型安全**：配合 TypeScript 提供完整的类型推导
- **中间件系统**：通过 Hooks 机制实现业务逻辑
- **查询构建**：链式 API 让查询更直观
- **虚拟字段**：方便地计算和组合数据

本文将带你快速搭建开发环境，完成第一个 Mongoose 项目。

## 核心概念

### MongoDB vs 关系型数据库

| 特性     | MongoDB          | MySQL/PostgreSQL |
| -------- | ---------------- | ---------------- |
| 数据模型 | 文档型（BSON）   | 表格型（行/列）  |
| Schema   | 灵活的 Schema    | 严格的表结构     |
| 关系     | 嵌入文档 + 引用  | 外键关联         |
| 扩展性   | 水平扩展容易     | 垂直扩展为主     |
| 查询语言 | MQL（类似 JSON） | SQL              |

### 连接 MongoDB

创建 `src/config/database.ts`：

```typescript
import mongoose from "mongoose";

const connectDB = async () => {
  try {
    const options = {
      // 连接池配置
      maxPoolSize: 10,
      minPoolSize: 5,
      // 服务器选择超时
      serverSelectionTimeoutMS: 5000,
      // Socket 超时
      socketTimeoutMS: 45000,
    };

    const conn = await mongoose.connect(
      process.env.MONGODB_URI || "mongodb://localhost:27017/cms",
      options
    );

    console.log(`✅ MongoDB 连接成功: ${conn.connection.host}`);

    // 监听连接事件
    mongoose.connection.on("error", (err) => {
      console.error("❌ MongoDB 连接错误:", err);
    });

    mongoose.connection.on("disconnected", () => {
      console.log("⚠️  MongoDB 连接断开");
    });

    // 优雅关闭
    process.on("SIGINT", async () => {
      await mongoose.connection.close();
      console.log("MongoDB 连接已关闭");
      process.exit(0);
    });
  } catch (error) {
    console.error("❌ MongoDB 连接失败:", error);
    process.exit(1);
  }
};

export default connectDB;
```

### 创建第一个 Schema

创建 `src/models/User.ts`：

```typescript
import mongoose, { Document, Schema } from "mongoose";

// 定义 Document 接口
export interface IUser extends Document {
  name: string;
  email: string;
  age?: number;
  createdAt: Date;
  updatedAt: Date;
  getFullName(): string;
}

// 定义 Schema
const userSchema = new Schema<IUser>(
  {
    name: {
      type: String,
      required: [true, "姓名是必填项"],
      trim: true,
      maxlength: [50, "姓名不能超过50个字符"],
    },
    email: {
      type: String,
      required: [true, "邮箱是必填项"],
      unique: true,
      lowercase: true,
      trim: true,
      match: [/^\S+@\S+\.\S+$/, "请输入有效的邮箱地址"],
    },
    age: {
      type: Number,
      min: [0, "年龄不能为负数"],
      max: [150, "年龄不能超过150"],
    },
  },
  {
    timestamps: true, // 自动添加 createdAt 和 updatedAt
  }
);

// 添加实例方法
userSchema.methods.getFullName = function () {
  return this.name;
};

// 添加静态方法
userSchema.statics.findByEmail = function (email: string) {
  return this.findOne({ email });
};

// 创建 Model
const User = mongoose.model<IUser>("User", userSchema);

export default User;
```

### CRUD 操作示例

创建 `src/examples/crud.ts`：

```typescript
import User from "../models/User";
import connectDB from "../config/database";

// 连接数据库
await connectDB();

// ==================== CREATE ====================
console.log("\n=== 创建用户 ===");

// 方式一：使用 new + save
const user1 = new User({
  name: "张三",
  email: "zhangsan@example.com",
  age: 25,
});
await user1.save();
console.log("✅ 创建用户1:", user1.name);

// 方式二：使用 create
const user2 = await User.create({
  name: "李四",
  email: "lisi@example.com",
  age: 30,
});
console.log("✅ 创建用户2:", user2.name);

// 方式三：批量创建
const users = await User.create([
  { name: "王五", email: "wangwu@example.com", age: 28 },
  { name: "赵六", email: "zhaoliu@example.com", age: 35 },
]);
console.log(`✅ 批量创建 ${users.length} 个用户`);

// ==================== READ ====================
console.log("\n=== 查询用户 ===");

// 查询所有用户
const allUsers = await User.find();
console.log(`📋 总用户数: ${allUsers.length}`);

// 条件查询
const youngUsers = await User.find({ age: { $lt: 30 } });
console.log(
  `👶 年轻用户 (${youngUsers.length}):`,
  youngUsers.map((u) => u.name)
);

// 查询单个用户
const user = await User.findOne({ email: "zhangsan@example.com" });
console.log("🔍 查找结果:", user?.name);

// 通过 ID 查询
const userById = await User.findById(user2._id);
console.log("🆔 ID查询:", userById?.name);

// 使用静态方法
const userByEmail = await User.findByEmail("lisi@example.com");
console.log("📧 邮箱查询:", userByEmail?.name);

// ==================== UPDATE ====================
console.log("\n=== 更新用户 ===");

// 方式一：修改后保存
const updateUser1 = await User.findOne({ name: "张三" });
if (updateUser1) {
  updateUser1.age = 26;
  await updateUser1.save();
  console.log("✅ 更新年龄:", updateUser1.name, updateUser1.age);
}

// 方式二：使用 updateOne
await User.updateOne({ name: "李四" }, { $set: { age: 31 } });
console.log("✅ 批量更新完成");

// 方式三：使用 findOneAndUpdate（返回更新后的文档）
const updatedUser = await User.findOneAndUpdate(
  { name: "王五" },
  { $set: { age: 29 } },
  { new: true } // 返回更新后的文档
);
console.log("✅ 查找并更新:", updatedUser?.name, updatedUser?.age);

// ==================== DELETE ====================
console.log("\n=== 删除用户 ===");

// 方式一：查找后删除
const deleteUserName = "赵六";
const deleteUser = await User.findOne({ name: deleteUserName });
if (deleteUser) {
  await deleteUser.deleteOne();
  console.log("🗑️  删除用户:", deleteUserName);
}

// 方式二：使用 deleteOne
await User.deleteOne({ name: "王五" });
console.log("🗑️  批量删除完成");

// 方式三：使用 findOneAndDelete
const deletedUser = await User.findOneAndDelete({ name: "李四" });
console.log("🗑️  查找并删除:", deletedUser?.name);

// ==================== 查询统计 ====================
console.log("\n=== 统计信息 ===");

const remainingUsers = await User.find();
console.log(`📊 剩余用户数: ${remainingUsers.length}`);

process.exit(0);
```

## 常见问题

### Q1: 连接失败怎么办？

**A**: 检查以下几点：

1. MongoDB 是否正在运行：

```bash
# Docker
docker ps

# 或本地
mongosh --version
```

2. 连接字符串是否正确：

```bash
# 本地 MongoDB
mongodb://localhost:27017/cms

# Docker MongoDB
mongodb://admin:password123@localhost:27017/cms?authSource=admin
```

3. 防火墙是否开放端口 27017

### Q2: Schema 和 Model 的区别？

**A**:

- **Schema**：定义文档的结构、类型、验证器、默认值等（类似类的定义）
- **Model**：基于 Schema 创建的构造函数，用于操作数据库（类似类本身）
- **Document**：Model 的实例，代表具体的数据库记录（类似类的实例）

### Q3: 什么时候使用 lean()？

**A**:

- 需要 **完整功能**（虚拟字段、方法、保存等）→ 使用默认查询
- 只需要 **数据读取**（性能优先）→ 使用 `lean()`

```typescript
// 普通查询（返回 Mongoose Document）
const user = await User.findById(id);

// Lean 查询（返回普通 JS 对象，性能更好）
const user = await User.findById(id).lean();
```

### Q4: 如何处理 ObjectId？

**A**:

```typescript
import mongoose from "mongoose";

// 字符串转 ObjectId
const id = new mongoose.Types.ObjectId("507f1f77bcf86cd799439011");

// 验证 ObjectId
const isValid = mongoose.Types.ObjectId.isValid("507f1f77bcf86cd799439011");

// 生成新的 ObjectId
const newId = new mongoose.Types.ObjectId();
```
