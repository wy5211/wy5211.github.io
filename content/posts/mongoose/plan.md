# Mongoose 渐进式学习计划

## 背景

你用 Node.js 写后端，选了 MongoDB 做数据库。但直接用 MongoDB 官方驱动写代码，没有数据验证，没有类型提示，查询代码散落各处。同事说用 Mongoose，但你不知道 Schema 怎么设计，populate 是什么意思，事务怎么用。

这个系列从"为什么需要 Mongoose"出发，带你逐步掌握 Mongoose 的核心能力和生产实践。

### 技术栈

| 技术       | 版本   | 为什么选它           |
| ---------- | ------ | -------------------- |
| MongoDB    | 7.x    | 最流行的文档数据库   |
| Mongoose   | 8.x    | MongoDB 最成熟的 ODM |
| Node.js    | 22 LTS | 示例代码技术栈       |
| TypeScript | 5.x    | 类型安全             |
