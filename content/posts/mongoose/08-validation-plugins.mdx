## 前言

数据验证确保数据库中只存储有效数据。Mongoose 提供了丰富的验证器和插件系统。

## 内置验证器

### 常用验证器

```typescript
const schema = new Schema({
  // required: 必填
  name: { type: String, required: true },

  // minlength/maxlength: 字符串长度
  title: {
    type: String,
    minlength: 5,
    maxlength: 200,
  },

  // min/max: 数字范围
  age: { type: Number, min: 0, max: 150 },

  // enum: 枚举值
  status: {
    type: String,
    enum: ["draft", "published", "archived"],
  },

  // match: 正则匹配
  email: {
    type: String,
    match: /^\S+@\S+\.\S+$/,
  },

  // unique: 唯一性
  username: {
    type: String,
    unique: true,
  },
});
```

## 自定义验证器

```typescript
// 同步验证器
userSchema.path("email").validate(function (email) {
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regex.test(email);
}, "无效的邮箱格式");

// 异步验证器
userSchema.path("email").validate(async function (email) {
  const user = await this.constructor.findOne({ email });
  return !user || this._id.equals(user._id);
}, "邮箱已被使用");
```

## 插件开发

### 软删除插件

```typescript
function softDeletePlugin(schema) {
  schema.add({ deletedAt: Date, deletedBy: Schema.Types.ObjectId });

  schema.methods.softDelete = async function (userId) {
    this.deletedAt = new Date();
    this.deletedBy = userId;
    return this.save();
  };

  schema.pre(/^find/, function () {
    this.where({ deletedAt: { $exists: false } });
  });
}

// 使用
userSchema.plugin(softDeletePlugin);
```

### 时间戳插件

```typescript
function timestampsPlugin(schema) {
  schema.add({
    createdAt: { type: Date, default: Date.now },
    updatedAt: { type: Date, default: Date.now },
  });

  schema.pre("save", function (next) {
    if (this.isModified()) {
      this.updatedAt = new Date();
    }
    next();
  });
}
```
