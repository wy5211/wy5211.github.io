## 为什么需要文件操作？

在任务队列 API 中，你遇到了几个实际问题：如何将任务执行日志持久化到文件？如何导入批量任务数据？如何配置任务队列的参数？这些都需要文件操作。

Go 的文件操作 API 设计得很实用，提供了简单易用的接口。本文将介绍如何在 Go 中进行文件读写、目录操作和路径处理。

## 读取文件

### 读取整个文件

对于小文件，最简单的方法是一次性读取全部内容：

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    // 读取整个文件
    data, err := os.ReadFile("config.json")
    if err != nil {
        fmt.Printf("读取文件失败: %v\n", err)
        return
    }

    fmt.Printf("文件内容:\n%s\n", string(data))
}
```

`os.ReadFile` 是 Go 1.16 引入的便捷函数，适合读取小文件（小于 10MB）。

### 按行读取大文件

对于大文件，应该逐行读取避免内存溢出：

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Open("tasks.txt")
    if err != nil {
        fmt.Printf("打开文件失败: %v\n", err)
        return
    }
    defer file.Close()  // 确保文件被关闭

    scanner := bufio.NewScanner(file)
    lineNum := 0

    // 逐行扫描
    for scanner.Scan() {
        lineNum++
        line := scanner.Text()
        fmt.Printf("行 %d: %s\n", lineNum, line)
    }

    // 检查扫描过程中是否有错误
    if err := scanner.Err(); err != nil {
        fmt.Printf("扫描错误: %v\n", err)
    }
}
```

**为什么需要 defer？** 文件是系统资源，必须显式关闭。使用 `defer file.Close()` 确保函数返回前一定会关闭文件，即使发生 panic。

### 分块读取文件

对于非常大的文件（如日志文件），可以分块读取：

```go
package main

import (
    "fmt"
    "io"
    "os"
)

func main() {
    file, err := os.Open("large.log")
    if err != nil {
        fmt.Printf("打开文件失败: %v\n", err)
        return
    }
    defer file.Close()

    // 创建缓冲区，每次读取 4KB
    buffer := make([]byte, 4096)

    for {
        n, err := file.Read(buffer)
        if err != nil {
            if err == io.EOF {
                // 读取到文件末尾
                break
            }
            fmt.Printf("读取错误: %v\n", err)
            return
        }

        // 处理读取到的数据（buffer[:n]）
        fmt.Printf("读取了 %d 字节\n", n)
    }
}
```

### 使用 io.ReadAll

```go
package main

import (
    "fmt"
    "io"
    "os"
)

func main() {
    file, err := os.Open("config.json")
    if err != nil {
        fmt.Printf("打开文件失败: %v\n", err)
        return
    }
    defer file.Close()

    // 读取所有内容
    data, err := io.ReadAll(file)
    if err != nil {
        fmt.Printf("读取失败: %v\n", err)
        return
    }

    fmt.Printf("文件内容:\n%s\n", string(data))
}
```

`io.ReadAll` 和 `os.ReadFile` 的区别：

- `os.ReadFile` 直接接收文件路径
- `io.ReadAll` 接收 `io.Reader` 接口，更通用

## 写入文件

### 写入整个文件

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    data := []byte("Hello, Go!\n这是第二行。")

    // 写入文件，权限 0o644（所有者读写，其他人只读）
    err := os.WriteFile("output.txt", data, 0o644)
    if err != nil {
        fmt.Printf("写入文件失败: %v\n", err)
        return
    }

    fmt.Println("文件写入成功")
}
```

### 追加写入

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    // 以追加模式打开文件
    file, err := os.OpenFile("log.txt", os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0o644)
    if err != nil {
        fmt.Printf("打开文件失败: %v\n", err)
        return
    }
    defer file.Close()

    // 写入数据
    _, err = file.WriteString("新的日志条目\n")
    if err != nil {
        fmt.Printf("写入失败: %v\n", err)
        return
    }

    fmt.Println("追加成功")
}
```

**文件打开模式**：

- `os.O_RDONLY`：只读
- `os.O_WRONLY`：只写
- `os.O_RDWR`：读写
- `os.O_CREATE`：不存在则创建
- `os.O_APPEND`：追加
- `os.O_TRUNC`：清空文件

### 使用 bufio 写入

bufio 提供了缓冲写入，可以提高性能：

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("output.txt")
    if err != nil {
        fmt.Printf("创建文件失败: %v\n", err)
        return
    }
    defer file.Close()

    writer := bufio.NewWriter(file)
    defer writer.Flush()  // 确保缓冲区数据被写入

    for i := 1; i <= 1000; i++ {
        fmt.Fprintf(writer, "行 %d: 一些内容\n", i)
    }

    fmt.Println("写入完成")
}
```

**为什么需要 Flush？** bufio 使用缓冲区，数据先写入内存缓冲区，只有缓冲区满或显式调用 Flush 时才会真正写入文件。`defer writer.Flush()` 确保所有数据都被写入。

## 在任务队列中使用文件操作

### 持久化任务日志

```go
package main

import (
    "bufio"
    "fmt"
    "os"
    "sync"
    "time"
)

type TaskLog struct {
    TaskID    int
    Status    string
    Message   string
    Timestamp time.Time
}

type TaskLogger struct {
    file   *os.File
    writer *bufio.Writer
    mu     sync.Mutex
}

func NewTaskLogger(filename string) (*TaskLogger, error) {
    file, err := os.OpenFile(filename, os.O_CREATE|os.O_APPEND|os.O_WRONLY, 0o644)
    if err != nil {
        return nil, err
    }

    return &TaskLogger{
        file:   file,
        writer: bufio.NewWriter(file),
    }, nil
}

func (l *TaskLogger) Log(log TaskLog) error {
    l.mu.Lock()
    defer l.mu.Unlock()

    // 格式化日志
    logLine := fmt.Sprintf(
        "[%s] Task %d: %s - %s\n",
        log.Timestamp.Format("2006-01-02 15:04:05"),
        log.TaskID,
        log.Status,
        log.Message,
    )

    // 写入缓冲区
    if _, err := l.writer.WriteString(logLine); err != nil {
        return err
    }

    // 立即刷新到文件
    return l.writer.Flush()
}

func (l *TaskLogger) Close() error {
    // 先刷新缓冲区
    if err := l.writer.Flush(); err != nil {
        return err
    }
    // 再关闭文件
    return l.file.Close()
}

func main() {
    logger, err := NewTaskLogger("tasks.log")
    if err != nil {
        fmt.Printf("创建日志器失败: %v\n", err)
        return
    }
    defer logger.Close()

    // 模拟记录任务日志
    for i := 1; i <= 5; i++ {
        log := TaskLog{
            TaskID:    i,
            Status:    "completed",
            Message:   fmt.Sprintf("任务 %d 处理完成", i),
            Timestamp: time.Now(),
        }

        if err := logger.Log(log); err != nil {
            fmt.Printf("记录日志失败: %v\n", err)
        }

        time.Sleep(100 * time.Millisecond)
    }

    fmt.Println("日志记录完成")
}
```

### 从文件导入任务

```go
package main

import (
    "bufio"
    "encoding/json"
    "fmt"
    "os"
    "strconv"
)

type Task struct {
    ID          int    `json:"id"`
    Title       string `json:"title"`
    Description string `json:"description"`
}

func ImportTasksFromFile(filename string) ([]Task, error) {
    file, err := os.Open(filename)
    if err != nil {
        return nil, err
    }
    defer file.Close()

    var tasks []Task
    scanner := bufio.NewScanner(file)
    lineNum := 0

    for scanner.Scan() {
        lineNum++
        line := scanner.Text()

        // 跳过空行
        if line == "" {
            continue
        }

        // 解析 JSON
        var task Task
        if err := json.Unmarshal([]byte(line), &task); err != nil {
            fmt.Printf("行 %d 解析失败: %v\n", lineNum, err)
            continue
        }

        tasks = append(tasks, task)
    }

    if err := scanner.Err(); err != nil {
        return nil, err
    }

    return tasks, nil
}

func main() {
    // 创建示例文件
    createSampleFile("tasks.jsonl")

    // 导入任务
    tasks, err := ImportTasksFromFile("tasks.jsonl")
    if err != nil {
        fmt.Printf("导入失败: %v\n", err)
        return
    }

    fmt.Printf("成功导入 %d 个任务:\n", len(tasks))
    for _, task := range tasks {
        fmt.Printf("- 任务 %d: %s\n", task.ID, task.Title)
    }
}

func createSampleFile(filename string) {
    file, _ := os.Create(filename)
    defer file.Close()

    tasks := []Task{
        {ID: 1, Title: "任务1", Description: "描述1"},
        {ID: 2, Title: "任务2", Description: "描述2"},
        {ID: 3, Title: "任务3", Description: "描述3"},
    }

    for _, task := range tasks {
        data, _ := json.Marshal(task)
        file.Write(data)
        file.WriteString("\n")
    }
}
```

## 目录操作

### 创建目录

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    // 创建单个目录
    err := os.Mkdir("data", 0o755)
    if err != nil {
        fmt.Printf("创建目录失败: %v\n", err)
    }

    // 创建多级目录
    err = os.MkdirAll("data/tasks/2024/01", 0o755)
    if err != nil {
        fmt.Printf("创建多级目录失败: %v\n", err)
    }

    fmt.Println("目录创建成功")
}
```

**权限说明**：`0o755` 是八进制权限：

- `7`（所有者）：读+写+执行
- `5`（组）：读+执行
- `5`（其他人）：读+执行

### 读取目录

```go
package main

import (
    "fmt"
    "os"
    "path/filepath"
)

func main() {
    entries, err := os.ReadDir(".")
    if err != nil {
        fmt.Printf("读取目录失败: %v\n", err)
        return
    }

    fmt.Println("当前目录内容:")
    for _, entry := range entries {
        info, _ := entry.Info()
        if entry.IsDir() {
            fmt.Printf("[目录] %s\n", entry.Name())
        } else {
            fmt.Printf("[文件] %s (%d 字节)\n", entry.Name(), info.Size())
        }
    }
}
```

### 遍历目录树

```go
package main

import (
    "fmt"
    "os"
    "path/filepath"
)

func walkDir(dir string) error {
    return filepath.Walk(dir, func(path string, info os.FileInfo, err error) error {
        if err != nil {
            return err
        }

        // 跳过隐藏文件
        if filepath.Base(path)[0] == '.' {
            if info.IsDir() {
                return filepath.SkipDir
            }
            return nil
        }

        relPath, _ := filepath.Rel(dir, path)
        if info.IsDir() {
            fmt.Printf("[目录] %s\n", relPath)
        } else {
            fmt.Printf("[文件] %s (%d 字节)\n", relPath, info.Size())
        }

        return nil
    })
}

func main() {
    err := walkDir(".")
    if err != nil {
        fmt.Printf("遍历目录失败: %v\n", err)
    }
}
```

## 文件路径处理

### path/filepath 包

```go
package main

import (
    "fmt"
    "path/filepath"
)

func main() {
    // Join - 构建路径（自动处理分隔符）
    path := filepath.Join("data", "tasks", "2024", "report.txt")
    fmt.Println("路径:", path)

    // Base - 获取文件名
    fmt.Println("文件名:", filepath.Base(path))

    // Dir - 获取目录
    fmt.Println("目录:", filepath.Dir(path))

    // Ext - 获取扩展名
    fmt.Println("扩展名:", filepath.Ext(path))

    // Abs - 获取绝对路径
    abs, err := filepath.Abs(path)
    if err == nil {
        fmt.Println("绝对路径:", abs)
    }

    // Clean - 清理路径（处理 .. 和 .）
    dirtyPath := "data/../data/./tasks/report.txt"
    fmt.Println("清理前:", dirtyPath)
    fmt.Println("清理后:", filepath.Clean(dirtyPath))

    // Split - 分割路径和文件名
    dir, file := filepath.Split(path)
    fmt.Printf("分割: 目录=%s 文件=%s\n", dir, file)

    // Rel - 计算相对路径
    from := "/data/tasks/2024"
    to := "/data/tasks/2024/report.txt"
    rel, err := filepath.Rel(from, to)
    if err == nil {
        fmt.Printf("相对路径: %s -> %s = %s\n", from, to, rel)
    }
}
```

### 检查文件是否存在

```go
package main

import (
    "fmt"
    "os"
)

func fileExists(path string) bool {
    info, err := os.Stat(path)
    if os.IsNotExist(err) {
        return false
    }
    return !info.IsDir()
}

func dirExists(path string) bool {
    info, err := os.Stat(path)
    if os.IsNotExist(err) {
        return false
    }
    return info.IsDir()
}

func main() {
    fmt.Println("config.json 存在:", fileExists("config.json"))
    fmt.Println("data 目录存在:", dirExists("data"))
}
```

## 配置文件读写

### JSON 配置文件

```go
package main

import (
    "encoding/json"
    "fmt"
    "os"
)

type Config struct {
    Server   ServerConfig   `json:"server"`
    Database DatabaseConfig `json:"database"`
    Queue    QueueConfig    `json:"queue"`
}

type ServerConfig struct {
    Host string `json:"host"`
    Port int    `json:"port"`
}

type DatabaseConfig struct {
    Host     string `json:"host"`
    Port     int    `json:"port"`
    Username string `json:"username"`
    Password string `json:"password"`
    Database string `json:"database"`
}

type QueueConfig struct {
    MaxWorkers int `json:"max_workers"`
    QueueSize  int `json:"queue_size"`
}

func LoadConfig(filename string) (*Config, error) {
    data, err := os.ReadFile(filename)
    if err != nil {
        return nil, err
    }

    var config Config
    if err := json.Unmarshal(data, &config); err != nil {
        return nil, err
    }

    return &config, nil
}

func SaveConfig(filename string, config *Config) error {
    data, err := json.MarshalIndent(config, "", "  ")
    if err != nil {
        return err
    }

    return os.WriteFile(filename, data, 0o644)
}

func main() {
    // 创建默认配置
    config := &Config{
        Server: ServerConfig{
            Host: "localhost",
            Port: 8080,
        },
        Database: DatabaseConfig{
            Host:     "localhost",
            Port:     5432,
            Username: "user",
            Password: "pass",
            Database: "taskdb",
        },
        Queue: QueueConfig{
            MaxWorkers: 10,
            QueueSize:  1000,
        },
    }

    // 保存配置
    if err := SaveConfig("config.json", config); err != nil {
        fmt.Printf("保存配置失败: %v\n", err)
        return
    }

    // 加载配置
    loaded, err := LoadConfig("config.json")
    if err != nil {
        fmt.Printf("加载配置失败: %v\n", err)
        return
    }

    fmt.Printf("配置加载成功:\n")
    fmt.Printf("服务器: %s:%d\n", loaded.Server.Host, loaded.Server.Port)
    fmt.Printf("队列: MaxWorkers=%d, QueueSize=%d\n",
        loaded.Queue.MaxWorkers, loaded.Queue.QueueSize)
}
```

## Trade-off 分析

### os.ReadFile vs bufio.Reader

**os.ReadFile**：

- 适合小文件（小于 10MB）
- 代码简单，一次性读取
- 整个文件加载到内存

**bufio.Reader**：

- 适合大文件
- 按需读取，节省内存
- 代码稍复杂，需要管理文件句柄

### 文件锁

Go 标准库没有提供跨平台的文件锁。如果需要文件锁：

- 使用第三方库（如 `github.com/gofrs/flock`）
- 或者在应用层实现锁机制

### 性能考虑

1. **缓冲区大小**：bufio 默认 4KB，对于大文件可以调整
2. **批量写入**：积累一定量数据再写入，减少 IO 次数
3. **异步写入**：使用 goroutine 将文件操作异步化

```go
// 异步写入日志的例子
type AsyncLogger struct {
    logChan chan string
    file    *os.File
}

func NewAsyncLogger(filename string) (*AsyncLogger, error) {
    file, err := os.OpenFile(filename, os.O_CREATE|os.O_APPEND|os.O_WRONLY, 0o644)
    if err != nil {
        return nil, err
    }

    logger := &AsyncLogger{
        logChan: make(chan string, 1000),
        file:    file,
    }

    go logger.writer()
    return logger, nil
}

func (l *AsyncLogger) writer() {
    for log := range l.logChan {
        l.file.WriteString(log + "\n")
    }
}

func (l *AsyncLogger) Log(message string) {
    l.logChan <- message
}
```

## 总结

文件操作是程序与外部世界交互的基础。Go 的文件 API 简洁实用：

- `os` 包提供基础文件操作
- `bufio` 提供缓冲 IO，提高性能
- `path/filepath` 处理路径，跨平台兼容

在任务队列 API 中，文件操作用于：

- 持久化日志和配置
- 导入导出任务数据
- 存储任务执行结果

记住几个要点：

1. 使用 `defer` 确保文件被关闭
2. 大文件用缓冲读取，避免内存溢出
3. 并发写入需要加锁保护
4. 选择合适的文件格式（JSON、CSV、JSONL）
