# 看懂行情：从蜡烛图到基本面（进阶版）

## 背景

你不是一个刚入门的人。

你读过《金钱心理学》《投资最重要的事》《富爸爸穷爸爸》《全球投资机会》——复利、风险溢价、市场钟摆、行为偏差，这些心法你心里有数。你还实打实定投 ETF 一年，体会过账户浮亏、体会过涨了想加仓跌了想割肉的滋味。

所以这个系列**不教你什么是 PE、不教你怎么开软件、不教你为什么要定投**——这些你早会了。

你真正卡住的是下一层：你已经会「被动持有」，但想学会「主动看懂」——

- 同样是 PE 22 倍，为什么银行股嫌贵、科技股嫌便宜？周期股低 PE 为什么是陷阱？
- 估值分位这种工具，你以为客观，结果同一个沪深 300 不同口径能差出 90 个点，到底信谁？
- 均线金叉死叉，为什么你一用就亏？技术指标那些「胜率 80%」的说法，凭什么大多是骗局？
- 你定投一年，最大的敌人不是市场，是你自己的处置效应、锚定效应——怎么用系统对抗？

这个系列就是补这一层。读完之后，你能做到：

> **不再被动接受任何一个「专家结论」。给你一个标的，你能用自己的框架拆解它——技术面背后的资金博弈、基本面背后的行业逻辑、估值背后的统计陷阱，以及最重要的：把它们串成一个能执行、能复盘的决策系统。**

不扫盲，不灌鸡汤，不预测涨跌。定位是**让你看深一层**。

### 为什么这个系列要「三合一」且「讲深」

市面上理财内容两极分化：要么是 K 线技术派（讲金叉死叉、讲形态，但回避统计真相），要么是价值投资派（讲 PE 讲护城河，但无视价格行为）。两派还互相看不起。

但你真正做决策时，这三层信息同时挤在屏幕上，而且**每一层都有它骗你的方式**：

- **价格行为（K 线 / 量价 / 均线）**——反映资金博弈，但技术指标会过拟合、会滞后、会在震荡市反复打脸
- **基本面估值（PE / PB / 股息率 / FCF）**——反映公司价值，但不同行业逻辑不同、周期股会骗你、单一指标会失真
- **市场结构数据（估值分位 / ERP / 复权 / 资金流）**——反映历史位置，但分位有统计陷阱、复权会扭曲指标、数据本身可能被污染

只懂一层，你看到的是残缺且可能被骗的画面。所以这个系列把它们**串成一条线**，而且每一层都讲透它的**原理、实战用法、以及它骗你的方式**。

## 一条必须先校准的认知

这是全系列的认知地基：

> **没有任何单一指标能告诉你「该不该买」。基本面、技术面、市场结构数据，各自回答不同的问题，各自都有盲区。成熟的判断 = 多层信号交叉验证。**

| 你想问的问题                         | 看哪一层                  | 这一层会怎么骗你                                                            |
| ------------------------------------ | ------------------------- | --------------------------------------------------------------------------- |
| 这公司值不值钱、赚不赚钱、能不能持续 | 基本面（PE/PB/FCF/ROE）   | 周期股利润顶部 PE 低；净利润可调节，FCF 更难造假；不同行业估值逻辑不可比    |
| 现在什么趋势、资金在干什么           | 技术面（K 线/量价/均线）  | 指标滞后；震荡市金叉死叉反复打脸；形态胜率多为过拟合神话                    |
| 现在在历史什么位置、贵不贵           | 市场结构（分位/ERP/复权） | 分位受成分股变更/加权方式污染；同一指数分位能差 90 个点；复权会扭曲技术指标 |

校准完这个，你才不会犯进阶者也会犯的错：「**这个 PE 分位 10%，闭眼买**」「**均线金叉了，追**」「**这公司 ROE 高，长期持有必赚**」——每个都是单层信号，每个都有反例。

## 系列写作规范（Voice & 质量门槛）

> 依据 ECC `article-writing` skill + 项目 `blog-series-writer` skill（类型 D 投资类）。

### Voice Profile（写作 DNA）

延续你已有的风格（提炼自《金钱心理学》读书笔记、《前端转 AI 全栈》系列）：

1. **第二人称「你」贯穿**——像跟朋友聊，不是讲课
2. **真实数据 / 真实标的切入**——用证据，不用形容词
3. **短段落，一句话成段是常态**，节奏快
4. **「不是 X，而是 Y」对比句式**制造张力
5. **先给结论，再展开**
6. **括号补充、破折号、口语连接**，不要书面腔
7. **数据诚实分层**——硬证据 / 观察性统计 / 利益相关方信号，标注来源口径

### 五条核心写作规则

1. **具体先行**——每篇开头用真实标的 + 真实数字 + 真实日期
2. **先摆例子，再讲道理**
3. **句子要紧**
4. **证据代替形容词**——不说「均线有用」，说「多头排列时做多，历史胜率约 X」（需查证）
5. **不编造**——见数据铁律

### 🔴 进阶深度标准（本版核心，区别于入门版）

**每一篇必须包含这五层，缺一不可，禁止停留在「是什么」的科普：**

1. **深层原理**——为什么是这个机制（不只是定义）
2. **实战用法**——进阶者具体怎么用（不是「低买高卖」的废话）
3. **统计陷阱 / 失效场景**——这东西什么时候骗你、为什么
4. **与其他概念的联动**——它和系列里其他工具怎么配合
5. **边界条件**——什么情况下彻底不能用

凡是能用一句话科普完的内容，要么删掉（读者已懂），要么深化（讲到读者没想到的层面）。

### 专有名词简释（进阶 ≠ 傲慢）

系列假设读者懂基础（定投 / PE / K 线基本概念），但遇到**进阶专有名词**（席勒 PE、股权风险溢价 ERP、杜邦分析、凯利公式、处置效应、过拟合、未来函数、订单流等）时，**在首次出现处用一句话或括号简短解释**，让读者不卡壳。解释要轻——半句或括号即可，不展开成独立小节，不打断主线。

### 篇幅标准

**每篇 6000–9000 字。** 深度长文，把一个主题讲透（原理 + 实战 + 陷阱 + 联动 + 边界）。核心实战篇（09/11/14）可到 9000+。

### 纯文本，不用图片

全系列纯文本，不使用任何截图、示意图、流程图。所有「图」的内容用文字、表格、列表表达。frontmatter 的 `cover` 留空。

### Banned Patterns（一律删掉重写）

- 「在当今快速发展的……」「随着 XXX 的普及」式空洞开头
- 「game-changer」「颠覆性」「天花板」「yyds」
- 「此外」「值得一提的是」「综上所述」「不难看出」
- 「具体情况具体分析」「凡事都有两面性」——正确的废话
- 「希望这篇文章对你有帮助」「让我们一起加油」——空洞收尾
- 复选框式总结（用 ✅ 重复章节标题）
- 假的「我也曾经迷茫过」脆弱性叙事（除非真有真实经历）

### 数据真实性铁律

1. **所有具体数字（PE/PB/分位/涨跌幅/胜率/财务数据）必须联网核对真实值**，不凭印象写。
2. **数据标来源与时间点**——「截至 2026 年 X 月，来源：理杏仁 / 同花顺 / 公司年报」。
3. **本 plan 里的数字均为方向性示例**，正文以联网查证的真实数据替换。
4. **回测 / 胜率类数据尤其谨慎**——必须能指出研究来源，绝不编造「某指标胜率 80%」。
5. **加风险提示**——涉及具体标的的数据用作方法演示，非投资建议。

查证工具：理杏仁（分位/估值历史）、同花顺/东方财富（行情/基本面）、巨潮/公司年报（财报）、富途/Yahoo（美股）。

## 每篇文章大纲（进阶版）

> ⚠️ 下文公司、数字均为方向性示例，正文写作时联网核对真实数据替换。所有篇纯文本无图。

### 01. 开篇：从被动持有到主动看懂

> 写作纪律：进阶定位，不扫盲。假设读者已懂定投/复利/PE 基础。讲清「为什么被动定投 + 看个 PE 还不够」，建立全系列坐标。

- **核心问题**：我已经会定投、会看 PE 了，为什么还需要这套进阶能力？
- **钩子**：一个被动定投者真实的困境——长期定投在漫长熊市里的心理损耗、单一 PE 指标的误判案例（待查真实案例）
- **内容要点**：
  1. 被动定投的局限：它在什么周期有效、在什么周期折磨人（定投不是万能解药）
  2. 三层信息（基本面/技术面/市场结构）各自怎么骗你（用上文那张表）
  3. 进阶者的判断方式：多层信号交叉验证，而非单一指标
  4. 全系列 14 篇地图 + 5 模块里程碑
- **结尾动作**：列一张「你目前做投资决策时依赖的单一信号」清单，标出哪些可能是盲区
- **字数**：~6000

### 02. K 线的订单流本质：一根 K 线是买卖博弈的压缩记录

> 不讲红涨绿跌、实体影线（你早懂）。讲 K 线背后的订单流博弈。

- **核心问题**：K 线到底在记录什么？为什么同样一根长上影，有的位置是见顶、有的位置是洗盘？
- **内容要点**：
  1. K 线的本质：一段时间内连续竞价撮合的结果，是订单流的压缩记录
  2. 实体长短 = 趋势强度，影线 = 极端试探后的回落——长上影 = 上方抛压 / 套牢盘，长下影 = 下方承接
  3. **位置权重**：关键位置（前高前低 / 整数关 / 均线位 / 缺口）的 K 线信号权重远高于普通位置，脱离位置谈 K 线是噪声
  4. 陷阱：孤立看单根 K 线 / 形态胜率多为过拟合；K 线是结果不是原因
  5. 联动：K 线必须配成交量（03）和位置（04 均线）才有意义
- **字数**：~6000

### 03. 量价关系：成交量的深层信号

> 不讲"放量上涨好"。讲成交量的深层博弈含义。

- **核心问题**：为什么光看价格不够？成交量到底在度量什么？
- **内容要点**：
  1. 成交量的本质：市场分歧度的度量——缩量 = 共识（趋势继续）、放量 = 分歧（转折可能）
  2. 量价八法 / 量价背离的预警逻辑（量在价先的真实统计基础）
  3. 主力建仓（缩量横盘 + 放量突破）与出货（放量滞涨）的量能特征
  4. 换手率与筹码集中度：高位高换手 vs 低位高换手的天壤之别
  5. 陷阱：放量也可能是对敲诱多；天量天价 / 地量地价的反例；成交量的滞后与操纵
- **字数**：~6000

### 04. 均线系统：趋势的骨架与它的失效区

> 不讲"金叉买死叉卖"。讲均线作为趋势骨架的本质和失效区。

- **核心问题**：均线金叉死叉，为什么你一用就亏？
- **内容要点**：
  1. 均线的本质：趋势的骨架，不是预测器；它是滞后滤波器
  2. 多周期共振（5/20/60/250 同向）的统计基础；葛兰碧八大买卖法则
  3. 多头 / 空头排列的真实含义和有效性
  4. **失效区**：均线在趋势市有效、震荡市反复打脸的数学原因（这是散户用均线亏钱的主因）
  5. 正确用法：用均线做「过滤」（只在多头排列时考虑做多）而非「预测」；先判趋势 vs 震荡再决定信不信
- **字数**：~6000

### 05. 复权、除权与数据陷阱

> 你最初问过复权。这篇讲透，并延伸到数据陷阱。

- **核心问题**：前复权后复权到底什么区别？为什么它会影响你的技术指标和历史回测？
- **内容要点**：
  1. 除权除息的总资产不变原理（为什么分红要扣股价）
  2. 前复权 / 后复权 / 不复权的区别与各自适用场景
  3. 分红再投的复利效应（为什么后复权曲线远超不复权）
  4. **复权对技术指标的扭曲**：均线、MACD 在除权日前后的失真
  5. **数据陷阱**：历史回测里不复权数据导致的假突破；指数成分股调整对估值序列的污染；为什么你的回测结果可能是错的
- **字数**：~6000

### 06. 市盈率的进阶：行业差异、周期陷阱与席勒 PE

> 不讲 PE 公式。讲 PE 背后的行业逻辑和周期陷阱。

- **核心问题**：同样是 PE 20 倍，为什么有的便宜有的贵？周期股低 PE 为什么是陷阱？
- **内容要点**：
  1. 行业 PE 差异的根源：资本结构 / 增长预期 / 风险溢价——银行个位数、科技五六十倍都正常
  2. **周期股低 PE 陷阱**的数学本质：利润在周期顶，PE 用峰值利润算自然低 → 席勒 PE（CAPE，10 年通胀调整均值）为什么更稳
  3. TTM / 静态 / 动态在利润拐点的滞后差异
  4. PEG（PE / 增速）怎么用、怎么被滥用
  5. 边界：亏损股、超级成长股为什么不能用 PE
- **字数**：~7000

### 07. 市净率、股息率与自由现金流：超越表面估值

> 不讲"破净=便宜、高股息=好"。讲可持续性和 FCF 视角。

- **核心问题**：破净就是黄金坑？股息率 8% 就是好票？怎么分辨真假？
- **内容要点**：
  1. 破净的统计真相：A 股银行长期破净却 2025 年大涨——市场错杀 vs 资产质量担忧（待核实真实数据）
  2. 股息率可持续性分析：分红率 / 自由现金流 / 盈利稳定性，区分真高息与周期顶假高息
  3. **自由现金流（FCF）视角**：为什么 FCF 比净利润更难造假，是更硬的估值锚
  4. ROE 的杜邦分解预告（净利率 × 周转率 × 杠杆）——连到 09
  5. PB / 股息率的适用边界（成熟重资产行业 vs 轻资产科技）
- **字数**：~7000

### 08. 估值分位与股权风险溢价：别被分位骗了

> 不讲"分位低就买"。讲分位的统计陷阱和更优指标。

- **核心问题**：估值分位你以为客观，为什么同一个沪深 300 能差出 90 个点？
- **内容要点**：
  1. 分位的统计陷阱：成分股变更 / 市值加权 vs 等权 / 比较年限 / 幸存者偏差（真实案例：沪深 300 分位 6.76% vs 98%）
  2. 为什么单一分位不可靠，要多维度（PE + PB + 股息率 + ERP）
  3. **股权风险溢价（ERP = 盈利收益率 − 无风险利率）**：为什么它比单纯 PE 分位更有跨资产比较意义
  4. 当下（2026）A 股 vs 美股的 ERP 对比与含义（待核实）
  5. 边界：分位只能告诉你「相对历史」，不能告诉你「绝对价值」
- **字数**：~7000

### 09. 个股深度体检：从指标到判断

> 不讲"看 PE/PB"。讲杜邦分析、现金流排雷、完整决策流程。

- **核心问题**：怎么把 06-08 的估值深度 + 02-04 的价格行为，串成对一个标的的完整判断？
- **内容要点**：
  1. **杜邦分析拆解 ROE**：净利率 × 周转率 × 杠杆——区分「靠本事赚钱」vs「靠杠杆撑场面」
  2. **现金流排雷**：经营现金流 vs 净利润背离、应收 / 存货异常、商誉雷
  3. 商业模式与护城河定性（不是玄学，是可检验的）
  4. 用历史估值区间 + 分位锚定买卖点
  5. 完整体检一只真实票（茅台或宁德，深度数据，全程演示）
- **字数**：~9000

### 10. ETF 进阶：超越无脑定投

> 不讲"每月定投"。讲核心-卫星、ETF 选择、止盈、配置。

- **核心问题**：定投一年了，怎么从"无脑定投"升级到"有策略地配置"？
- **内容要点**：
  1. 核心-卫星策略：宽基打底 + 行业 / 主题卫星，比例与再平衡
  2. ETF 选择硬指标：费率 / 跟踪误差 / 规模 / 流动性 / 折溢价——为什么两只同名 ETF 收益会差
  3. 行业与风格轮动的逻辑与陷阱（轮动很难，别高估自己）
  4. 什么时候该止盈：估值 + 趋势双信号，而非拍脑袋
  5. 资产配置入门：股债平衡、再平衡的纪律
- **字数**：~8000

### 11. 周期、仓位与风控：决定生死的不是选股

> 讲仓位和风控——这是大多数人忽视、却决定生死的一层。

- **核心问题**：为什么说"仓位比选股重要"？怎么管仓位和回撤？
- **内容要点**：
  1. 周期位置判断：美林时钟简化版、信贷周期、库存周期的直觉
  2. **仓位管理**：为什么仓位比选股重要、凯利公式的直觉、分批建仓 / 金字塔加仓
  3. 回撤控制与止损纪律：为什么"活得久"比"赚得多"重要
  4. 风险预算：单标的 / 单行业 / 总仓位的上限
  5. 边界：风控不能消除亏损，只能让你留在牌桌上
- **字数**：~8000

### 12. 行为金融实战：你的大脑是怎么让你亏钱的

> 讲行为偏差在实战中的具体表现。你定投一年，这些偏差你大概率都犯过。

- **核心问题**：你定投一年，最大的敌人是谁？
- **内容要点**：
  1. 损失厌恶与处置效应：卖赢持亏的真实心理机制
  2. 锚定效应：成本价锚、历史高点锚怎么扭曲判断

  4. 羊群效应、近因偏差、过度自信
  5. **应对**：用规则和系统对抗本能（连到 14）——认知偏差无法消除，只能用纪律绕开
- **字数**：~7000

### 13. 技术指标为什么会失效：过拟合、参数寻优与未来函数

> 讲清那些"胜率 80% 的指标"为什么大多是骗局。

- **核心问题**：网上那些高胜率技术指标，凭什么信不得？
- **内容要点**：
  1. **过拟合**：历史拟合得好，未来未必——参数越多越容易过拟合
  2. **参数寻优**：挑表现好的参数回测，必然美化结果
  3. **未来函数**：用了当时根本拿不到的数据，回测无敌实盘暴亏
  4. 幸存者偏差：只看活下来的策略 / 个股
  5. 正确看待回测：样本量 / 稳健性 / 参数敏感性 / 样本外验证；哪些信号相对可靠（趋势 > 震荡、大周期 > 小周期、共振 > 单一）
- **字数**：~7000

  3. 为什么「系统」比「判断」重要：判断会错、会受情绪干扰，系统是纪律的物化
  4. 复盘机制：记录每一笔决策的逻辑与结果，迭代系统
  5. 全系列心法收束：多层信号交叉验证、用系统对抗本能、看懂≠预测
