## 主线案例

整个系列围绕**电商商品搜索系统**展开，案例逐步演进：

1. 一条商品文档 → 完整索引设计 → 关键词搜索 → 筛选面板 → 中文搜索优化
2. 数据同步管道 → 性能调优 → 集群扩容 → 完整系统上线
3. 最后拓展到日志分析（ELK）和向量检索（AI 语义搜索）

---

## 技术栈

- **搜索引擎**：Elasticsearch 8.x
- **主数据库**：MySQL / PostgreSQL
- **后端**：Node.js
- **数据同步**：Logstash / Canal / Debezium
- **中文分词**：IK Analyzer、拼音分词器
