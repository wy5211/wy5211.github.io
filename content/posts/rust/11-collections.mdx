## 前言

前面的章节我们学习了基本类型、结构体和枚举，但很多时候我们需要存储多个值——比如一个用户列表、一个键值配置、一组去重的标签。这就是**集合（Collections）**的用武之地。

Rust 标准库提供了三个最常用的集合：

- **Vec\<T\>**：动态数组，存储有序的相同类型元素
- **HashMap\<K, V\>**：哈希表，存储键值对
- **HashSet\<T\>**：哈希集合，存储去重的元素

## Vec 动态数组

### 创建 Vec

```rust
fn main() {
    // 方式 1：使用 vec! 宏（最常用）
    let v1 = vec![1, 2, 3, 4, 5];

    // 方式 2：使用 Vec::new() 创建空 Vec
    let mut v2: Vec<i32> = Vec::new();
    v2.push(6);
    v2.push(7);

    // 方式 3：带初始值
    let v3 = vec![0; 10];  // 10 个 0

    println!("{:?}", v1);
    println!("{:?}", v2);
    println!("{:?}", v3);
}
```

### 常用操作

```rust
fn main() {
    let mut v = vec![1, 2, 3];

    // 添加元素
    v.push(4);
    v.push(5);

    // 读取元素（两种方式）
    let third = &v[2];           // 索引访问，越界会 panic
    println!("第三个: {}", third);

    let third = v.get(2);        // get 方法，返回 Option
    match third {
        Some(value) => println!("第三个: {}", value),
        None => println!("索引越界"),
    }

    // 修改元素
    v[0] = 10;

    // 删除元素
    let removed = v.pop();       // 删除末尾，返回 Option
    println!("删除: {:?}", removed);

    // 插入元素
    v.insert(1, 99);

    // 删除指定位置
    v.remove(0);

    println!("{:?}", v);
}
```

### 遍历 Vec

```rust
fn main() {
    let v = vec![10, 20, 30, 40, 50];

    // 不可变遍历
    for item in &v {
        println!("{}", item);
    }

    // 可变遍历
    let mut v = vec![1, 2, 3];
    for item in &mut v {
        *item *= 2;  // 解引用后修改
    }
    println!("{:?}", v);  // [2, 4, 6]

    // 带索引遍历
    for (i, item) in v.iter().enumerate() {
        println!("索引 {}: 值 {}", i, item);
    }
}
```

### Vec 与所有权

```rust
fn main() {
    let v = vec![String::from("hello"), String::from("world")];

    // 获取引用遍历
    for s in &v {
        println!("{}", s);  // s 是 &String
    }
    // v 仍然有效

    // ❌ move 遍历后 Vec 失效
    // for s in v {
    //     println!("{}", s);
    // }
    // println!("{:?}", v);  // v 已失效

    // ✅ 如果需要消耗所有权
    let v = vec![String::from("hello"), String::from("world")];
    let v2: Vec<String> = v.into_iter().collect();  // 转移所有权
}
```

### Vec 常用方法速查

```rust
fn main() {
    let mut v = vec![3, 1, 4, 1, 5, 9, 2, 6];

    // 长度和容量
    println!("长度: {}", v.len());
    println!("容量: {}", v.capacity());

    // 排序
    v.sort();
    println!("排序: {:?}", v);  // [1, 1, 2, 3, 4, 5, 6, 9]

    // 反转
    v.reverse();
    println!("反转: {:?}", v);

    // 去重
    v.sort();
    v.dedup();
    println!("去重: {:?}", v);  // [1, 2, 3, 4, 5, 6, 9]

    // 查找
    let v = vec![10, 20, 30, 40, 50];
    println!("包含 30: {}", v.contains(&30));       // true
    println!("位置: {:?}", v.iter().position(|&x| x == 30)); // Some(2)

    // 切片
    println!("切片: {:?}", &v[1..3]);  // [20, 30]

    // 连接
    let a = vec![1, 2];
    let b = vec![3, 4];
    let c = [a, b].concat();
    println!("连接: {:?}", c);  // [1, 2, 3, 4]
}
```

## HashMap 哈希表

### 创建 HashMap

```rust
use std::collections::HashMap;

fn main() {
    // 方式 1：使用 new 和 insert
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Red"), 50);

    // 方式 2：从 Vec 创建
    let teams = vec![String::from("Blue"), String::from("Red")];
    let initial_scores = vec![10, 50];
    let scores: HashMap<_, _> = teams.into_iter()
        .zip(initial_scores.into_iter())
        .collect();

    println!("{:?}", scores);
}
```

### 访问和修改

```rust
use std::collections::HashMap;

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Red"), 50);

    // 获取值
    let team_name = String::from("Blue");
    let score = scores.get(&team_name);  // 返回 Option<&V>
    match score {
        Some(s) => println!("Blue 的分数: {}", s),
        None => println!("没找到"),
    }

    // 遍历
    for (key, value) in &scores {
        println!("{}: {}", key, value);
    }

    // 仅在键不存在时插入
    scores.entry(String::from("Green")).or_insert(30);
    scores.entry(String::from("Blue")).or_insert(100);  // Blue 已存在，不插入
    println!("{:?}", scores);

    // 覆盖值
    scores.insert(String::from("Blue"), 20);
    println!("Blue 更新后: {:?}", scores.get("Blue"));
}
```

### entry API

`entry` API 是处理 HashMap 的利器：

```rust
use std::collections::HashMap;

fn main() {
    let text = "hello world wonderful world hello rust";

    let mut word_count = HashMap::new();

    for word in text.split_whitespace() {
        let count = word_count.entry(word).or_insert(0);
        *count += 1;
    }

    println!("{:?}", word_count);
    // {"hello": 2, "world": 2, "wonderful": 1, "rust": 1}
}
```

### HashMap 常用方法

```rust
use std::collections::HashMap;

fn main() {
    let mut map = HashMap::new();
    map.insert("a", 1);
    map.insert("b", 2);
    map.insert("c", 3);

    // 长度
    println!("长度: {}", map.len());

    // 是否包含
    println!("包含 'a': {}", map.contains_key(&"a"));

    // 删除
    let removed = map.remove(&"b");
    println!("删除: {:?}", removed);

    // 保留满足条件的
    map.retain(|&k, &mut v| v > 1);

    // 清空
    // map.clear();

    println!("{:?}", map);
}
```

## HashSet 哈希集合

### 创建和使用

```rust
use std::collections::HashSet;

fn main() {
    let mut set = HashSet::new();

    // 插入
    set.insert(1);
    set.insert(2);
    set.insert(3);
    set.insert(2);  // 重复，不会插入

    println!("长度: {}", set.len());  // 3
    println!("包含 1: {}", set.contains(&1));  // true

    // 删除
    set.remove(&2);

    // 遍历
    for item in &set {
        println!("{}", item);
    }

    // 从 Vec 创建
    let set2: HashSet<i32> = vec![1, 2, 3, 2, 1].into_iter().collect();
    println!("{:?}", set2);  // {1, 2, 3}
}
```

### 集合运算

```rust
use std::collections::HashSet;

fn main() {
    let a: HashSet<_> = vec![1, 2, 3, 4].into_iter().collect();
    let b: HashSet<_> = vec![3, 4, 5, 6].into_iter().collect();

    // 并集
    let union: HashSet<_> = a.union(&b).collect();
    println!("并集: {:?}", union);  // {1, 2, 3, 4, 5, 6}

    // 交集
    let intersection: HashSet<_> = a.intersection(&b).collect();
    println!("交集: {:?}", intersection);  // {3, 4}

    // 差集（a 有但 b 没有）
    let difference: HashSet<_> = a.difference(&b).collect();
    println!("差集: {:?}", difference);  // {1, 2}

    // 对称差集（只在其中一个集合中的元素）
    let symmetric: HashSet<_> = a.symmetric_difference(&b).collect();
    println!("对称差集: {:?}", symmetric);  // {1, 2, 5, 6}
}
```

## 实战演练

### 示例 1：统计单词频率

```rust
use std::collections::HashMap;

fn word_frequency(text: &str) -> Vec<(&str, usize)> {
    let mut map = HashMap::new();

    for word in text.split_whitespace() {
        let word = word.to_lowercase();
        let clean: String = word.chars().filter(|c| c.is_alphanumeric()).collect();
        if !clean.is_empty() {
            *map.entry(clean.as_str()).or_insert(0) += 1;
        }
    }

    let mut result: Vec<_> = map.into_iter().collect();
    result.sort_by(|a, b| b.1.cmp(&a.1));
    result
}

fn main() {
    let text = "The quick brown fox jumps over the lazy dog. The dog was not amused.";

    let freq = word_frequency(text);
    println!("单词频率（降序）:");
    for (word, count) in &freq {
        println!("  {:>10}: {} 次", word, count);
    }
}
```

### 示例 2：学生成绩管理

```rust
use std::collections::HashMap;

struct GradeManager {
    students: HashMap<String, Vec<f64>>,
}

impl GradeManager {
    fn new() -> Self {
        Self { students: HashMap::new() }
    }

    fn add_grade(&mut self, name: &str, grade: f64) {
        self.students
            .entry(name.to_string())
            .or_insert_with(Vec::new)
            .push(grade);
    }

    fn average(&self, name: &str) -> Option<f64> {
        self.students.get(name).map(|grades| {
            grades.iter().sum::<f64>() / grades.len() as f64
        })
    }

    fn top_student(&self) -> Option<(&str, f64)> {
        self.students.iter().filter_map(|(name, grades)| {
            if grades.is_empty() { None } else {
                let avg = grades.iter().sum::<f64>() / grades.len() as f64;
                Some((name.as_str(), avg))
            }
        }).max_by(|a, b| a.1.partial_cmp(&b.1).unwrap())
    }

    fn summary(&self) {
        for (name, grades) in &self.students {
            let avg = grades.iter().sum::<f64>() / grades.len() as f64;
            println!("{}: 成绩 {:?}, 平均 {:.1}", name, grades, avg);
        }
    }
}

fn main() {
    let mut manager = GradeManager::new();

    manager.add_grade("Alice", 90.0);
    manager.add_grade("Alice", 85.0);
    manager.add_grade("Alice", 92.0);
    manager.add_grade("Bob", 78.0);
    manager.add_grade("Bob", 88.0);
    manager.add_grade("Charlie", 95.0);

    manager.summary();

    if let Some((name, avg)) = manager.top_student() {
        println!("最高平均分: {} ({:.1})", name, avg);
    }
}
```

### 示例 3：标签去重与匹配

```rust
use std::collections::{HashMap, HashSet};

fn main() {
    // 用户标签
    let user1_tags: HashSet<_> = vec!["rust", "programming", "systems"].into_iter().collect();
    let user2_tags: HashSet<_> = vec!["rust", "web", "programming"].into_iter().collect();
    let user3_tags: HashSet<_> = vec!["web", "design", "ui"].into_iter().collect();

    // 共同兴趣
    let common: HashSet<_> = user1_tags.intersection(&user2_tags).collect();
    println!("user1 和 user2 的共同标签: {:?}", common);

    // 推荐标签（user1 有但 user3 没有的）
    let recommend: HashSet<_> = user1_tags.difference(&user3_tags).collect();
    println!("推荐给 user3: {:?}", recommend);

    // 标签到用户数
    let mut tag_users: HashMap<&str, Vec<&str>> = HashMap::new();
    let users = [("user1", &user1_tags), ("user2", &user2_tags), ("user3", &user3_tags)];

    for (name, tags) in users {
        for tag in tags {
            tag_users.entry(tag).or_insert_with(Vec::new).push(name);
        }
    }

    println!("\n标签使用统计:");
    for (tag, users) in &tag_users {
        println!("  {}: {:?}", tag, users);
    }
}
```

## 常见错误与修复

### 错误 1：Vec 索引越界

```rust
fn main() {
    let v = vec![1, 2, 3];

    // ❌ 越界访问会 panic
    // let x = v[10];

    // ✅ 使用 get 安全访问
    if let Some(x) = v.get(10) {
        println!("{}", x);
    } else {
        println!("索引越界");
    }
}
```

### 错误 2：遍历时修改 Vec

```rust
fn main() {
    let mut v = vec![1, 2, 3, 4, 5];

    // ❌ 遍历时插入/删除可能出问题
    // for item in &v {
    //     if *item == 3 { v.push(99); }
    // }

    // ✅ 使用 retain 过滤
    v.retain(|&x| x != 3);
    println!("{:?}", v);  // [1, 2, 4, 5]
}
```

### 错误 3：HashMap 所有权

```rust
use std::collections::HashMap;

fn main() {
    let mut map = HashMap::new();

    let key = String::from("name");
    let value = String::from("alice");

    // insert 会获取 key 和 value 的所有权
    map.insert(key, value);
    // println!("{}", key);    // ❌ key 已被移动
    // println!("{}", value);  // ❌ value 已被移动

    // ✅ 如果之后还需要用，插入克隆
    let key = String::from("name");
    map.insert(key.clone(), String::from("bob"));
    println!("{}", key);  // ✅
}
```

## 最佳实践

1. **优先用 `vec!` 宏创建 Vec**：简洁直观
2. **用 `get` 而不是索引**：避免 panic
3. **善用 `entry` API**：处理 HashMap 键存在/不存在的逻辑
4. **用 `collect()` 从迭代器创建集合**：灵活且高效
5. **需要去重时用 HashSet**：比 Vec+去重更高效

## 常见问题

**Q: Vec 和数组有什么区别？**

A: 数组长度固定，存储在栈上；Vec 长度可变，数据存储在堆上。大多数情况下用 Vec。

**Q: HashMap 的键需要什么 trait？**

A: 键类型必须实现 `Eq` 和 `Hash` trait。基本类型和 String 都已实现。

**Q: 如何选择 Vec、HashMap、HashSet？**

A: 有序数据用 Vec；键值对用 HashMap；需要去重和快速查找用 HashSet。

## 总结

本文学习了 Rust 的三个核心集合：

**Vec\<T\>**

- 动态数组，自动扩容
- `push`、`pop`、`insert`、`remove`
- 迭代器遍历，支持不可变和可变

**HashMap\<K, V\>**

- 键值对存储
- `entry` API 处理键存在/不存在
- 键类型需实现 `Eq + Hash`

**HashSet\<T\>**

- 自动去重
- 支持并集、交集、差集运算
