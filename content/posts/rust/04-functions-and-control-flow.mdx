## 前言

函数是组织代码的基本单元，控制流决定程序的执行路径。Rust 的函数语法与其他语言类似，但有一些独特的特性，比如**函数体是表达式**、**没有三元运算符**等。

本文将深入学习 Rust 的函数和控制流，为后续学习所有权打下基础。

## 函数定义

### 基本语法

使用 `fn` 关键字定义函数：

```rust
fn main() {
    println!("Hello, main!");

    // 调用函数
    greet();
    greet();
}

// 定义一个简单函数
fn greet() {
    println!("Hello from greet function!");
}
```

**Rust 函数的命名规范**：

- 使用 snake_case（蛇形命名法）
- 小写字母，单词间用下划线分隔

### 函数参数

Rust 函数的参数必须**显式声明类型**：

```rust
fn main() {
    // 传递参数
    print_labeled_measurement(5, 'h');
}

// 参数必须标注类型
// 参数: 类型
fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("The measurement is: {}{}", value, unit_label);
}
```

**多个相同类型参数**：每个参数都需要单独标注类型

```rust
// ✅ 正确：每个参数都标注类型
fn add(x: i32, y: i32) -> i32 {
    x + y
}

// ❌ 错误：不能省略类型
// fn add(x, y) { ... }
```

### 函数返回值

使用 `->` 指定返回值类型：

```rust
fn main() {
    let result = add(5, 3);
    println!("5 + 3 = {}", result);
}

// 返回值类型用 -> 指定
fn add(x: i32, y: i32) -> i32 {
    // 返回值：最后一个表达式的值（没有分号）
    x + y
}
```

**两种返回方式**：

```rust
// 方式 1：隐式返回（最后一个表达式，无分号）
fn add_one(x: i32) -> i32 {
    x + 1  // 这是表达式，返回它的值
}

// 方式 2：显式 return
fn add_one_explicit(x: i32) -> i32 {
    return x + 1;  // 使用 return 关键字
}

// 提前返回
fn abs(x: i32) -> i32 {
    if x < 0 {
        return -x;  // 提前返回
    }
    x  // 正常返回
}
```

### 语句 vs 表达式

理解语句和表达式的区别是掌握 Rust 的关键：

```
┌─────────────────────────────────────────────────────────────┐
│                    语句 vs 表达式                            │
├─────────────────────────────────────────────────────────────┤
│  语句（Statement）                                           │
│  - 执行某个动作                                               │
│  - 不返回值                                                   │
│  - 以分号结尾                                                 │
│  - 例：let x = 5;                                            │
│                                                             │
│  表达式（Expression）                                         │
│  - 计算并返回一个值                                           │
│  - 不以分号结尾                                               │
│  - 例：5、x + 1、函数调用                                     │
└─────────────────────────────────────────────────────────────┘
```

```rust
fn main() {
    // 语句：不返回值
    let x = 5;  // 这是一个语句

    // 表达式：返回值
    let y = {
        let x = 3;
        x + 1  // 表达式，返回 4（注意没有分号）
    };

    println!("y = {}", y);  // y = 4

    // 如果加了分号，就变成语句，返回 ()
    let z = {
        let x = 3;
        x + 1;  // 语句，返回 ()
    };

    println!("z = {:?}", z);  // z = ()
}
```

**代码块也是表达式**：

```rust
fn main() {
    // 代码块可以返回值
    let condition = true;
    let number = if condition { 5 } else { 6 };

    println!("number = {}", number);

    // 更复杂的代码块
    let result = {
        let a = 10;
        let b = 20;
        a + b  // 返回这个表达式的值
    };

    println!("result = {}", result);  // 30
}
```

### 无返回值函数

没有返回值的函数实际上返回单元类型 `()`：

```rust
// 这两种写法等价
fn do_something() {
    println!("Doing something");
}

fn do_something_explicit() -> () {
    println!("Doing something");
}
```

### 发散函数

使用 `!` 表示永不返回的函数（发散函数）：

```rust
// 这个函数永不返回，总是 panic
fn crash_and_burn() -> ! {
    panic!("程序崩溃！");
}

// 循环也是发散的
fn infinite_loop() -> ! {
    loop {
        // 永远执行
    }
}
```

## 控制流

### if 表达式

Rust 的 `if` 是**表达式**，可以返回值：

```rust
fn main() {
    let number = 6;

    // 基本 if-else
    if number % 4 == 0 {
        println!("number is divisible by 4");
    } else if number % 3 == 0 {
        println!("number is divisible by 3");
    } else if number % 2 == 0 {
        println!("number is divisible by 2");
    } else {
        println!("number is not divisible by 4, 3, or 2");
    }

    // if 作为表达式（必须返回相同类型）
    let condition = true;
    let number = if condition { 5 } else { 6 };

    println!("number = {}", number);
}
```

**注意**：`if` 表达式的所有分支必须返回相同类型：

```rust
fn main() {
    let condition = true;

    // ❌ 错误：类型不匹配
    // let number = if condition {
    //     5
    // } else {
    //     "six"  // 一个是 i32，一个是 &str
    // };
}
```

### loop 循环

`loop` 创建无限循环：

```rust
fn main() {
    let mut count = 0;

    // 无限循环
    loop {
        println!("count = {}", count);
        count += 1;

        if count >= 3 {
            break;  // 退出循环
        }
    }

    println!("循环结束");
}
```

**从循环返回值**：

```rust
fn main() {
    let mut counter = 0;

    // loop 可以返回值
    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2;  // 返回 20
        }
    };

    println!("result = {}", result);  // 20
}
```

**循环标签**：嵌套循环时可以用标签指定 break/continue 的目标

```rust
fn main() {
    let mut count = 0;

    // 外层循环标签 'outer
    'outer: loop {
        println!("外层循环: {}", count);

        let mut remaining = 10;

        // 内层循环
        loop {
            println!("  内层循环: {}", remaining);

            if remaining == 9 {
                break;  // 只退出内层循环
            }
            if count == 2 {
                break 'outer;  // 退出外层循环
            }
            remaining -= 1;
        }

        count += 1;
    }

    println!("所有循环结束");
}
```

### while 循环

条件循环：

```rust
fn main() {
    let mut number = 3;

    // while 循环
    while number != 0 {
        println!("{}!", number);
        number -= 1;
    }

    println!("LIFTOFF!!!");
}
```

**遍历数组**：

```rust
fn main() {
    let a = [10, 20, 30, 40, 50];
    let mut index = 0;

    // 不推荐：使用索引遍历
    while index < 5 {
        println!("a[{}] = {}", index, a[index]);
        index += 1;
    }

    // 推荐：使用 for 循环（更安全）
    for element in a {
        println!("element = {}", element);
    }
}
```

### for 循环

`for` 是最常用的循环，用于遍历迭代器：

```rust
fn main() {
    // 遍历范围
    for number in 1..4 {
        println!("{}", number);  // 1, 2, 3
    }

    // 包含结束值
    for number in 1..=4 {
        println!("{}", number);  // 1, 2, 3, 4
    }

    // 反向遍历
    for number in (1..=4).rev() {
        println!("{}!", number);  // 4, 3, 2, 1
    }
    println!("LIFTOFF!!!");
}
```

**遍历数组**：

```rust
fn main() {
    let a = [10, 20, 30, 40, 50];

    // 遍历元素
    for element in a {
        println!("element = {}", element);
    }

    // 同时获取索引和元素
    for (index, element) in a.iter().enumerate() {
        println!("a[{}] = {}", index, element);
    }
}
```

**Range 语法**：

```rust
fn main() {
    // 1..4  = [1, 2, 3]     （不包含 4）
    // 1..=4 = [1, 2, 3, 4]  （包含 4）

    for i in 0..5 {
        print!("{} ", i);  // 0 1 2 3 4
    }
    println!();

    for i in 0..=5 {
        print!("{} ", i);  // 0 1 2 3 4 5
    }
}
```

### match 表达式

`match` 是强大的模式匹配工具：

```rust
fn main() {
    let number = 13;

    // match 必须穷尽所有可能
    match number {
        // 匹配单个值
        1 => println!("One"),
        // 匹配多个值
        2 | 3 | 5 | 7 => println!("Prime"),
        // 范围匹配
        13..=19 => println!("A teen"),
        // 默认分支（_ 是通配符）
        _ => println!("Something else"),
    }

    // match 是表达式，可以返回值
    let description = match number {
        1 => "one",
        2 => "two",
        3..=10 => "between three and ten",
        _ => "something else",
    };

    println!("number is {}", description);
}
```

**匹配元组**：

```rust
fn main() {
    let pair = (2, -2);

    match pair {
        (0, y) => println!("x is zero, y is {}", y),
        (x, 0) => println!("x is {}, y is zero", x),
        (x, y) if x == -y => println!("x and y are opposites"),
        (x, y) => println!("x = {}, y = {}", x, y),
    }
}
```

### if let 语法糖

当你只关心一种模式时，`if let` 比 `match` 更简洁：

```rust
fn main() {
    let some_value = Some(3);

    // 使用 match
    match some_value {
        Some(3) => println!("three"),
        _ => (),
    }

    // 使用 if let（更简洁）
    if let Some(3) = some_value {
        println!("three");
    }

    // 带 else
    if let Some(x) = some_value {
        println!("got {}", x);
    } else {
        println!("got nothing");
    }
}
```

### while let 循环

```rust
fn main() {
    let mut optional = Some(0);

    // 只要模式匹配就继续循环
    while let Some(i) = optional {
        if i > 5 {
            optional = None;
        } else {
            println!("i = {}", i);
            optional = Some(i + 1);
        }
    }
}
```

### break 和 continue

```rust
fn main() {
    // break：退出循环
    for i in 1..=10 {
        if i == 5 {
            break;  // 到 5 就退出
        }
        print!("{} ", i);  // 1 2 3 4
    }
    println!();

    // continue：跳过本次迭代
    for i in 1..=5 {
        if i % 2 == 0 {
            continue;  // 跳过偶数
        }
        print!("{} ", i);  // 1 3 5
    }
    println!();
}
```

## 实战演练

### 示例 1：斐波那契函数

```rust
/// 计算第 n 个斐波那契数
fn fibonacci(n: u32) -> u64 {
    if n == 0 {
        return 0;
    }
    if n == 1 {
        return 1;
    }

    let mut a: u64 = 0;
    let mut b: u64 = 1;

    for _ in 2..=n {
        let temp = a + b;
        a = b;
        b = temp;
    }

    b
}

fn main() {
    println!("斐波那契数列前 20 项：");
    for i in 0..20 {
        println!("fib({:2}) = {}", i, fibonacci(i));
    }
}
```

### 示例 2：温度转换器

```rust
/// 温度单位
enum TemperatureUnit {
    Celsius,
    Fahrenheit,
}

/// 转换温度
fn convert_temperature(value: f64, from: TemperatureUnit) -> f64 {
    match from {
        TemperatureUnit::Celsius => value * 1.8 + 32.0,   // C to F
        TemperatureUnit::Fahrenheit => (value - 32.0) / 1.8,  // F to C
    }
}

fn main() {
    println!("温度转换器");
    println!("-------------------");

    let temperatures = [0.0, 100.0, -40.0, 37.0];

    for temp in temperatures {
        let fahrenheit = convert_temperature(temp, TemperatureUnit::Celsius);
        println!("{:6.2}°C = {:6.2}°F", temp, fahrenheit);
    }

    println!();

    for temp in temperatures {
        let celsius = convert_temperature(temp, TemperatureUnit::Fahrenheit);
        println!("{:6.2}°F = {:6.2}°C", temp, celsius);
    }
}
```

### 示例 3：简单计算器

```rust
/// 运算类型
enum Operation {
    Add,
    Subtract,
    Multiply,
    Divide,
}

/// 执行计算
fn calculate(a: f64, op: Operation, b: f64) -> Result<f64, String> {
    match op {
        Operation::Add => Ok(a + b),
        Operation::Subtract => Ok(a - b),
        Operation::Multiply => Ok(a * b),
        Operation::Divide => {
            if b == 0.0 {
                Err("不能除以零".to_string())
            } else {
                Ok(a / b)
            }
        }
    }
}

/// 获取运算符对应的操作
fn get_operation(symbol: char) -> Option<Operation> {
    match symbol {
        '+' => Some(Operation::Add),
        '-' => Some(Operation::Subtract),
        '*' => Some(Operation::Multiply),
        '/' => Some(Operation::Divide),
        _ => None,
    }
}

fn main() {
    let calculations = [
        (10.0, '+', 5.0),
        (10.0, '-', 3.0),
        (4.0, '*', 5.0),
        (20.0, '/', 4.0),
        (10.0, '/', 0.0),
        (5.0, '%', 2.0),
    ];

    for (a, op_char, b) in calculations {
        // 使用 if let 处理 Option
        if let Some(op) = get_operation(op_char) {
            // 使用 match 处理 Result
            match calculate(a, op, b) {
                Ok(result) => println!("{} {} {} = {}", a, op_char, b, result),
                Err(e) => println!("错误：{} {} {} - {}", a, op_char, b, e),
            }
        } else {
            println!("不支持的运算符：{}", op_char);
        }
    }
}
```

### 示例 4：打印图案

```rust
fn print_triangle(height: usize) {
    for i in 1..=height {
        // 打印空格
        for _ in 0..(height - i) {
            print!(" ");
        }
        // 打印星号
        for _ in 0..(2 * i - 1) {
            print!("*");
        }
        println!();
    }
}

fn print_multiplication_table(n: u32) {
    for i in 1..=n {
        for j in 1..=i {
            print!("{:2}x{:2}={:2} ", j, i, i * j);
        }
        println!();
    }
}

fn main() {
    println!("三角形：");
    print_triangle(5);

    println!("\n九九乘法表：");
    print_multiplication_table(9);
}
```

## 最佳实践

### 1. 函数保持简洁

```rust
// ✅ 好：单一职责
fn is_even(n: i32) -> bool {
    n % 2 == 0
}

fn process_numbers(numbers: &[i32]) {
    for &n in numbers {
        if is_even(n) {
            println!("{} is even", n);
        }
    }
}

// ❌ 不好：函数做太多事情
fn process_numbers_bad(numbers: &[i32]) {
    for &n in numbers {
        if n % 2 == 0 {
            println!("{} is even", n);
        }
    }
}
```

### 2. 使用有意义的函数名

```rust
// ✅ 好：函数名清楚表达意图
fn calculate_total_price(items: &[f64], tax_rate: f64) -> f64 {
    let subtotal: f64 = items.iter().sum();
    subtotal * (1.0 + tax_rate)
}

// ❌ 不好：名字不清晰
fn calc(items: &[f64], t: f64) -> f64 {
    let s: f64 = items.iter().sum();
    s * (1.0 + t)
}
```

### 3. 优先使用 for 循环而非 while

```rust
// ✅ 好：使用 for
for i in 0..5 {
    println!("{}", i);
}

// ❌ 不如 for 简洁
let mut i = 0;
while i < 5 {
    println!("{}", i);
    i += 1;
}
```

### 4. 使用 match 而非多个 if-else

```rust
// ✅ 好：match 清晰
let result = match status {
    200 => "OK",
    404 => "Not Found",
    500 => "Internal Error",
    _ => "Unknown",
};

// ❌ 不如 match 清晰
let result = if status == 200 {
    "OK"
} else if status == 404 {
    "Not Found"
} else if status == 500 {
    "Internal Error"
} else {
    "Unknown"
};
```

## 常见问题

**Q: 为什么 Rust 没有三元运算符？**

A: Rust 的 `if` 是表达式，可以直接返回值，所以不需要三元运算符：

```rust
// Rust 风格
let x = if condition { 1 } else { 2 };

// 其他语言的三元运算符
// let x = condition ? 1 : 2;
```

**Q: 如何在循环中修改外部变量？**

A: 使用 `mut` 声明变量：

```rust
let mut sum = 0;
for i in 1..=10 {
    sum += i;
}
```

**Q: match 和 if let 怎么选？**

A:

- 需要处理多种情况 → 用 `match`
- 只关心一种情况 → 用 `if let`

**Q: 如何跳出嵌套循环？**

A: 使用循环标签：

```rust
'outer: loop {
    loop {
        break 'outer;  // 跳出外层循环
    }
}
```

## 总结

本文我们学习了：

**函数**

- `fn` 关键字定义函数
- 参数必须标注类型
- 返回值用 `->` 指定
- 最后一个表达式是返回值（无分号）

**语句 vs 表达式**

- 语句：执行动作，无返回值，有分号
- 表达式：返回值，无分号

**控制流**

- `if`：条件表达式
- `loop`：无限循环，可返回值
- `while`：条件循环
- `for`：遍历迭代器
- `match`：模式匹配
- `if let`：简化单分支匹配
