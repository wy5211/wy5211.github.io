## 前言

在上一篇文章中，我们搭建了 Rust 开发环境并运行了第一个程序。现在让我们深入学习 Rust 的基础语法——变量和数据类型。

Rust 的变量系统有一个独特的设计：**默认不可变**。这是 Rust 安全性的基石之一，也是很多初学者需要适应的地方。

## 变量声明

### let 语句

在 Rust 中，使用 `let` 关键字声明变量：

```rust
fn main() {
    // 声明一个变量
    let x = 5;

    // 声明并显式指定类型
    let y: i32 = 10;

    println!("x = {}, y = {}", x, y);
}
```

### 不可变性（Immutability）

**Rust 的变量默认是不可变的**。这是一个重要的设计决策：

```rust
fn main() {
    let x = 5;
    // x = 10;  // ❌ 编译错误！不能对不可变变量重新赋值

    println!("x = {}", x);
}
```

编译错误：

```
error[E0384]: cannot assign twice to immutable variable `x`
 --> src/main.rs:3:5
  |
2 |     let x = 5;
  |         - first assignment to `x`
3 |     x = 10;
  |     ^^^^^^ cannot assign twice to immutable variable
```

**为什么默认不可变？**

1. **减少 bug**：很多 bug 源于意外的变量修改
2. **并发安全**：不可变数据天生线程安全
3. **编译器优化**：编译器可以做更激进的优化

### 可变变量（mut）

如果你确实需要修改变量，使用 `mut` 关键字：

```rust
fn main() {
    // 使用 mut 声明可变变量
    let mut x = 5;
    println!("x = {}", x);  // x = 5

    x = 10;  // ✅ 可以修改
    println!("x = {}", x);  // x = 10
}
```

**什么时候用 mut？**

- 需要累积结果（如循环中的计数器）
- 需要修改状态
- 性能考虑（避免复制）

**最佳实践**：默认使用不可变，只有确实需要修改时才加 `mut`。

### 变量遮蔽（Shadowing）

Rust 允许声明一个与之前变量同名的新变量，这叫做**遮蔽（Shadowing）**：

```rust
fn main() {
    let x = 5;

    // 遮蔽：声明一个新的 x，类型可以不同
    let x = x + 1;  // 新的 x = 6

    // 再次遮蔽，甚至可以改变类型
    let x = "现在我是字符串";  // 新的 x 是 &str 类型

    println!("{}", x);  // 输出：现在我是字符串
}
```

**遮蔽 vs 可变变量**：

```rust
fn main() {
    // 使用遮蔽：类型可以改变
    let spaces = "   ";           // &str 类型
    let spaces = spaces.len();    // usize 类型 ✅

    // 使用 mut：类型不能改变
    let mut spaces2 = "   ";      // &str 类型
    // spaces2 = spaces2.len();   // ❌ 编译错误！类型不匹配
}
```

**遮蔽的用途**：

- 类型转换后保留相同的变量名
- 修改变量但不希望后续代码修改

## 常量（const）

常量与不可变变量类似，但有重要区别：

```rust
// 常量必须注明类型
// 常量名约定使用大写字母和下划线
const MAX_POINTS: u32 = 100_000;  // 数字中可以加下划线提高可读性

fn main() {
    println!("最大分数：{}", MAX_POINTS);
}
```

**const vs let**：

| 特性             | const      | let（不可变） |
| ---------------- | ---------- | ------------- |
| 必须注明类型     | ✅         | 可选          |
| 可以在全局作用域 | ✅         | ❌            |
| 可以在运行时计算 | ❌         | ✅            |
| 可以遮蔽         | ❌         | ✅            |
| 内存位置         | 编译期内联 | 运行时分配    |

```rust
const MAX: u32 = 100;        // ✅ 常量
// const VALUE: u32 = get_value();  // ❌ 常量不能用函数初始化

fn main() {
    let value = get_value(); // ✅ 不可变变量可以用函数初始化
}

fn get_value() -> u32 {
    42
}
```

### 静态变量（static）

`static` 声明静态变量，它在整个程序运行期间存在：

```rust
// 静态变量
static GREETING: &str = "Hello";

// 可变静态变量（需要 unsafe 才能修改）
static mut COUNTER: u32 = 0;

fn main() {
    println!("{}", GREETING);

    // 修改可变静态变量需要 unsafe
    unsafe {
        COUNTER += 1;
        println!("Counter: {}", COUNTER);
    }
}
```

**const vs static**：

- `const`：编译期内联，每次使用都是独立副本
- `static`：有固定内存地址，全局唯一

**最佳实践**：优先使用 `const`，只有需要固定地址或可变全局状态时才用 `static`。

## 基本数据类型

Rust 是**静态类型语言**，所有变量在编译期都必须有确定的类型。

### 类型分类

```
┌─────────────────────────────────────────────────────────────┐
│                    Rust 数据类型                             │
├─────────────────────────────────────────────────────────────┤
│  标量类型（Scalar）                                          │
│  ├── 整数：i8, i16, i32, i64, i128, isize                   │
│  │          u8, u16, u32, u64, u128, usize                  │
│  ├── 浮点数：f32, f64                                       │
│  ├── 布尔：bool                                             │
│  └── 字符：char                                             │
│                                                             │
│  复合类型（Compound）                                        │
│  ├── 元组：(T1, T2, ...)                                    │
│  └── 数组：[T; N]                                           │
└─────────────────────────────────────────────────────────────┘
```

### 整数类型

**有符号整数**（可以表示负数）：

| 类型  | 大小     | 范围                  |
| ----- | -------- | --------------------- |
| i8    | 1 字节   | -128 到 127           |
| i16   | 2 字节   | -32,768 到 32,767     |
| i32   | 4 字节   | -2³¹ 到 2³¹-1         |
| i64   | 8 字节   | -2⁶³ 到 2⁶³-1         |
| i128  | 16 字节  | -2¹²⁷ 到 2¹²⁷-1       |
| isize | 指针大小 | 取决于架构（32/64位） |

**无符号整数**（只能表示非负数）：

| 类型  | 大小     | 范围        |
| ----- | -------- | ----------- |
| u8    | 1 字节   | 0 到 255    |
| u16   | 2 字节   | 0 到 65,535 |
| u32   | 4 字节   | 0 到 2³²-1  |
| u64   | 8 字节   | 0 到 2⁶⁴-1  |
| u128  | 16 字节  | 0 到 2¹²⁸-1 |
| usize | 指针大小 | 取决于架构  |

**整数字面量**：

```rust
fn main() {
    // 十进制
    let decimal = 98_222;        // 下划线提高可读性

    // 十六进制
    let hex = 0xff;              // 255

    // 八进制
    let octal = 0o77;            // 63

    // 二进制
    let binary = 0b1111_0000;    // 240

    // 字节（仅限 u8）
    let byte = b'A';             // 65

    println!("十进制: {}, 十六进制: {}", decimal, hex);
}
```

**默认类型**：如果没有指定类型，Rust 默认使用 `i32`。

```rust
let x = 42;  // 类型推断为 i32
```

**整数溢出**：

在 Debug 模式下，整数溢出会导致 panic；在 Release 模式下，会回绕。

```rust
fn main() {
    let mut x: u8 = 255;
    // x = x + 1;  // Debug 模式会 panic！

    // 显式处理溢出
    x = x.wrapping_add(1);  // 回绕：0
    println!("x = {}", x);  // 0
}
```

### 浮点类型

Rust 有两种浮点类型：`f32`（单精度）和 `f64`（双精度）。

```rust
fn main() {
    let x = 2.0;      // f64（默认）

    let y: f32 = 3.0; // f32

    // 浮点数运算
    let sum = x + 2.5;
    let difference = 95.5 - 4.3;
    let product = 4.0 * 30.0;
    let quotient = 56.7 / 32.2;
    let remainder = 43.0 % 5.0;  // 取余

    println!("加法: {}", sum);
    println!("取余: {}", remainder);
}
```

**注意**：浮点数比较要小心精度问题：

```rust
fn main() {
    let x = 0.1 + 0.2;
    // x != 0.3  // 浮点精度问题！

    // 使用容差比较
    let diff = (x - 0.3).abs();
    if diff < 1e-10 {
        println!("近似相等");
    }
}
```

### 布尔类型

```rust
fn main() {
    let t = true;
    let f: bool = false;  // 显式类型标注

    // 布尔运算
    let and = t && f;     // 逻辑与
    let or = t || f;      // 逻辑或
    let not = !t;         // 逻辑非

    println!("true && false = {}", and);  // false
    println!("true || false = {}", or);   // true
    println!("!true = {}", not);          // false

    // 用于条件判断
    if t {
        println!("条件为真");
    }
}
```

### 字符类型

Rust 的 `char` 类型是 Unicode 字符，占 4 字节：

```rust
fn main() {
    let c = 'z';
    let z: char = 'ℤ';
    let heart_eyed_cat = '😻';
    let chinese = '中';

    println!("字符: {} {} {} {}", c, z, heart_eyed_cat, chinese);

    // char 的方法
    println!("'中' 是字母吗？ {}", '中'.is_alphabetic());
    println!("'5' 是数字吗？ {}", '5'.is_numeric());
}
```

**注意**：`char` 和字符串是不同的类型：

```rust
fn main() {
    let c: char = 'a';      // char，4 字节
    let s: &str = "a";      // &str，字符串切片

    // 它们不能直接比较
    // println!("{}", c == s);  // ❌ 编译错误
}
```

## 复合类型

### 元组（Tuple）

元组可以包含不同类型的值，长度固定：

```rust
fn main() {
    // 创建元组
    let tup: (i32, f64, u8) = (500, 6.4, 1);

    // 解构元组
    let (x, y, z) = tup;
    println!("y = {}", y);  // 6.4

    // 通过索引访问
    let five_hundred = tup.0;
    let six_point_four = tup.1;
    let one = tup.2;

    println!("{}, {}, {}", five_hundred, six_point_four, one);
}
```

**单元类型**：空元组 `()` 叫做单元类型，表示"无值"：

```rust
fn main() {
    let unit = ();
    println!("单元类型大小：{} 字节", std::mem::size_of_val(&unit));  // 0
}

// 没有返回值的函数实际返回 ()
fn do_something() {
    println!("做点什么");
    // 隐式返回 ()
}
```

### 数组（Array）

数组是固定长度的同类型元素集合：

```rust
fn main() {
    // 创建数组
    let a = [1, 2, 3, 4, 5];

    // 指定类型和长度
    let b: [i32; 5] = [1, 2, 3, 4, 5];

    // 重复初始化
    let c = [3; 5];  // 等同于 [3, 3, 3, 3, 3]

    // 访问元素
    let first = a[0];
    let second = a[1];

    println!("第一个元素：{}", first);
    println!("数组 c：{:?}", c);

    // 数组长度
    println!("数组 a 的长度：{}", a.len());
}
```

**数组越界**：Rust 会在运行时检查数组边界：

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];

    // let index = 10;
    // let element = a[index];  // 💥 运行时 panic！
}
```

**数组 vs Vector**：

| 特性   | 数组 | Vector |
| ------ | ---- | ------ |
| 长度   | 固定 | 动态   |
| 内存   | 栈上 | 堆上   |
| 性能   | 更快 | 稍慢   |
| 灵活性 | 低   | 高     |

```rust
fn main() {
    // 数组：栈上，固定长度
    let array = [1, 2, 3];

    // Vector：堆上，可变长度
    let mut vec = vec![1, 2, 3];
    vec.push(4);  // 可以添加元素
}
```

## 类型推断

Rust 有强大的类型推断能力：

```rust
fn main() {
    // 编译器根据上下文推断类型
    let x = 5;              // 推断为 i32
    let y = 1.0;            // 推断为 f64

    // 根据后续使用推断
    let mut v = Vec::new(); // 此时类型未知
    v.push(1);              // 推断为 Vec<i32>

    // 有时需要显式标注
    let z: i64 = 100;       // 显式指定 i64
}
```

## 类型转换

Rust 不支持隐式类型转换，必须显式转换：

```rust
fn main() {
    let x: i32 = 5;
    // let y: i64 = x;  // ❌ 隐式转换不允许

    // 显式转换（as）
    let y: i64 = x as i64;  // ✅

    // 数值到字符串
    let s = x.to_string();

    // 字符串到数值
    let num: i32 = "42".parse().expect("不是有效的数字");

    println!("转换后：{}, {}, {}", y, s, num);
}
```

## 实战演练

### 示例 1：温度转换器

```rust
/// 摄氏度转华氏度
fn celsius_to_fahrenheit(celsius: f64) -> f64 {
    celsius * 1.8 + 32.0
}

/// 华氏度转摄氏度
fn fahrenheit_to_celsius(fahrenheit: f64) -> f64 {
    (fahrenheit - 32.0) / 1.8
}

fn main() {
    // 使用遮蔽进行多次转换
    let temp = 0.0;               // 0 摄氏度
    let temp = celsius_to_fahrenheit(temp);  // 转为华氏度
    println!("0°C = {}°F", temp); // 32°F

    let temp = 100.0;             // 100 华氏度
    let temp = fahrenheit_to_celsius(temp);  // 转为摄氏度
    println!("100°F = {}°C", temp);  // 约 37.8°C

    // 使用元组存储温度对
    let temperatures: [(f64, f64); 3] = [
        (0.0, celsius_to_fahrenheit(0.0)),
        (100.0, celsius_to_fahrenheit(100.0)),
        (-40.0, celsius_to_fahrenheit(-40.0)),
    ];

    println!("\n温度对照表：");
    for (c, f) in temperatures {
        println!("{:6.2}°C = {:6.2}°F", c, f);
    }
}
```

### 示例 2：斐波那契数列

```rust
fn main() {
    // 使用数组存储斐波那契数列
    let mut fib: [u64; 20] = [0; 20];
    fib[0] = 0;
    fib[1] = 1;

    // 计算前 20 个斐波那契数
    let mut i = 2;
    while i < 20 {
        fib[i] = fib[i - 1] + fib[i - 2];
        i += 1;
    }

    println!("前 20 个斐波那契数：");
    for (index, value) in fib.iter().enumerate() {
        println!("fib[{:2}] = {}", index, value);
    }
}
```

### 示例 3：位运算

```rust
fn main() {
    let a: u8 = 0b1010_1010;  // 170
    let b: u8 = 0b1111_0000;  // 240

    println!("a = {:08b} ({})", a, a);
    println!("b = {:08b} ({})", b, b);

    // 位运算
    println!("a & b  = {:08b} (与)", a & b);
    println!("a | b  = {:08b} (或)", a | b);
    println!("a ^ b  = {:08b} (异或)", a ^ b);
    println!("!a     = {:08b} (非)", !a);
    println!("a << 2 = {:08b} (左移)", a << 2);
    println!("a >> 2 = {:08b} (右移)", a >> 2);
}
```

## 最佳实践

### 1. 选择合适的整数类型

```rust
// ✅ 好的做法
let age: u8 = 25;           // 年龄不会超过 255
let population: u64 = 7_000_000_000;  // 人口用大整数

// ❌ 避免不必要的过大类型
// let age: u128 = 25;  // 浪费内存
```

### 2. 使用类型别名提高可读性

```rust
// 类型别名
type Kilometers = i32;
type UserId = u64;

fn main() {
    let distance: Kilometers = 100;
    let user_id: UserId = 12345;

    println!("距离：{} km", distance);
    println!("用户ID：{}", user_id);
}
```

### 3. 优先使用不可变变量

```rust
// ✅ 优先这样写
let result = calculate(x, y);

// ❌ 只有确实需要修改时才用 mut
let mut sum = 0;
for i in 1..=10 {
    sum += i;
}
```

### 4. 善用数字分隔符

```rust
// ✅ 可读性好
let million = 1_000_000;
let hex = 0xff_ff;
let binary = 0b1111_0000;

// ❌ 难以阅读
// let million = 1000000;
```

## 常见问题

**Q: 为什么变量默认不可变？**

A: 不可变性可以减少很多意外的 bug，特别是在并发编程中。Rust 选择默认安全，需要修改时显式声明 `mut`。

**Q: i32 和 usize 有什么区别？**

A: `i32` 是固定 4 字节，`usize` 取决于架构（32位或64位）。`usize` 通常用于索引和大小。

**Q: 数组和 Vector 怎么选？**

A: 如果长度固定且已知，用数组（性能更好）；如果长度需要变化，用 Vector。

**Q: 如何查看变量的大小？**

A:

```rust
use std::mem;

fn main() {
    println!("i32 大小：{} 字节", mem::size_of::<i32>());
    println!("i64 大小：{} 字节", mem::size_of::<i64>());
}
```
