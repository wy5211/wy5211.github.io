# Rust 渐进式学习计划

## 背景

你用过 C++ 或 Java，但遇到过这些问题：C++ 性能强悍但内存 bug 频发，Java 有 GC 但停顿让你头疼。你想要一门既有 C 的性能，又有 Java 的安全性的语言。

Rust 就是为了解决这个问题而生的。它通过所有权系统在编译期保证内存安全，无需垃圾回收器。同时拥有现代工具链和零成本抽象。

### 技术栈

| 技术           | 用途         |
| -------------- | ------------ |
| Rust           | 1.70+        |
| Cargo          | 包管理和构建 |
| tokio          | 异步运行时   |
| axum/actix-web | Web 框架     |
