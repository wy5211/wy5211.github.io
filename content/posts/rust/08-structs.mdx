## 前言

到目前为止，我们学习了 Rust 的基本类型和所有权系统。但如果要表示一个"用户"、"矩形"、"商品"这样的复杂概念，基本类型就不够用了。

Rust 提供了**结构体（Struct）**来定义自定义数据类型。结构体是面向对象编程的基础，也是组织数据的核心工具。

## 定义结构体

### 基本语法

```rust
struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
}
```

关键字 `struct` 定义结构体，花括号内定义字段，每个字段需要标注类型。

### 创建实例

```rust
fn main() {
    let user = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        sign_in_count: 1,
        active: true,
    };

    println!("用户名: {}", user.username);
    println!("邮箱: {}", user.email);
}
```

访问字段使用点号 `.` 语法。

### 可变结构体

整个结构体必须是 `mut` 才能修改字段（Rust 不支持只修改部分字段）：

```rust
fn main() {
    let mut user = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        sign_in_count: 1,
        active: true,
    };

    user.email = String::from("new@example.com");
    user.sign_in_count += 1;

    println!("更新后邮箱: {}", user.email);
}
```

### 字段初始化简写

当变量名和字段名相同时，可以简写：

```rust
fn main() {
    let username = String::from("alice");
    let email = String::from("alice@example.com");

    // 完整写法
    let user1 = User {
        username: username,
        email: email,
        sign_in_count: 1,
        active: true,
    };

    // 简写：变量名和字段名相同时
    let user2 = User {
        username,   // 和 let username = ... 同名
        email,      // 和 let email = ... 同名
        sign_in_count: 1,
        active: true,
    };
}
```

### 结构体更新语法

从已有结构体创建新结构体，使用 `..` 语法：

```rust
fn main() {
    let user1 = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        sign_in_count: 1,
        active: true,
    };

    // 使用 ..user1 复制其余字段
    let user2 = User {
        username: String::from("bob"),
        email: String::from("bob@example.com"),
        ..user1  // sign_in_count 和 active 从 user1 复制
    };

    println!("user2 用户名: {}", user2.username);       // bob
    println!("user2 登录次数: {}", user2.sign_in_count); // 1（来自 user1）

    // 注意：user1 的 username 和 email 已被移动，不能再使用
    // 但 sign_in_count 和 active 是 Copy 类型，仍然可用
    println!("user1 登录次数: {}", user1.sign_in_count); // ✅
    // println!("{}", user1.username);  // ❌ 已被移动到 user2
}
```

## 结构体的三种形式

### 1. 经典结构体（最常用）

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let rect = Rectangle {
        width: 30,
        height: 50,
    };
    println!("矩形: {}x{}", rect.width, rect.height);
}
```

### 2. 元组结构体

有名字的元组，适合"整个结构体有意义但各字段没有独立名称"的场景：

```rust
struct Color(u8, u8, u8);
struct Point(f64, f64, f64);
struct Kg(f64);

fn main() {
    let red = Color(255, 0, 0);
    let origin = Point(0.0, 0.0, 0.0);
    let weight = Kg(72.5);

    // 通过索引访问
    println!("红色: R={} G={} B={}", red.0, red.1, red.2);
    println!("原点: ({}, {}, {})", origin.0, origin.1, origin.2);
    println!("体重: {} kg", weight.0);
}
```

### 3. 单元结构体

没有任何字段的结构体，类似于空枚举，用于实现 trait：

```rust
struct AlwaysEqual;

fn main() {
    let subject = AlwaysEqual;
    // 可以用来标记类型或实现某些行为
}
```

## 方法（Method）

方法是定义在结构体上的函数，第一个参数总是 `self`（或 `&self`、`&mut self`），表示方法被调用的实例。

### 定义方法

使用 `impl` 块定义方法：

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    // &self 是 self: &Self 的简写
    fn area(&self) -> u32 {
        self.width * self.height
    }

    // &mut self 可以修改结构体
    fn scale(&mut self, factor: u32) {
        self.width *= factor;
        self.height *= factor;
    }

    // self 获取所有权（较少使用）
    fn into_components(self) -> (u32, u32) {
        (self.width, self.height)
    }
}

fn main() {
    let mut rect = Rectangle {
        width: 30,
        height: 50,
    };

    println!("面积: {}", rect.area());  // 1500

    rect.scale(2);
    println!("放大后: {}x{}", rect.width, rect.height);  // 60x100

    let (w, h) = rect.into_components();
    println!("组件: {} {}", w, h);  // 60 100
    // rect 在这里已无效（所有权被 into_components 消耗）
}
```

### 方法的自动引用和解引用

调用方法时，Rust 会自动添加 `&`、`&mut` 或移动：

```rust
impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
}

fn main() {
    let rect = Rectangle { width: 30, height: 50 };

    // 这两种写法等价
    rect.area();       // Rust 自动转换为 (&rect).area()
    (&rect).area();    // 显式写法
}
```

### 更多方法示例

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    // 判断能否容纳另一个矩形
    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }

    // 判断是否为正方形
    fn is_square(&self) -> bool {
        self.width == self.height
    }

    // 计算周长
    fn perimeter(&self) -> u32 {
        2 * (self.width + self.height)
    }

    // 创建正方形（关联函数）
    fn square(size: u32) -> Self {
        Self { width: size, height: size }
    }
}

fn main() {
    let rect1 = Rectangle { width: 30, height: 50 };
    let rect2 = Rectangle { width: 20, height: 40 };
    let rect3 = Rectangle { width: 60, height: 45 };

    println!("rect1 能容纳 rect2: {}", rect1.can_hold(&rect2));  // true
    println!("rect1 能容纳 rect3: {}", rect1.can_hold(&rect3));  // false
    println!("rect1 是正方形: {}", rect1.is_square());            // false

    let sq = Rectangle::square(10);  // 关联函数调用
    println!("正方形: {:?}", sq);     // Rectangle { width: 10, height: 10 }
}
```

## 关联函数

`impl` 块中不接收 `self` 参数的函数叫做**关联函数**，类似于其他语言中的"静态方法"：

```rust
struct User {
    username: String,
    email: String,
    active: bool,
}

impl User {
    // 关联函数，用作"构造函数"
    fn new(username: &str, email: &str) -> Self {
        Self {
            username: username.to_string(),
            email: email.to_string(),
            active: true,
        }
    }

    // 带默认值的构造函数
    fn default_user() -> Self {
        Self {
            username: String::from("guest"),
            email: String::from("guest@example.com"),
            active: true,
        }
    }
}

fn main() {
    // 使用 :: 调用关联函数
    let user1 = User::new("alice", "alice@example.com");
    let user2 = User::default_user();

    println!("{} <{}>", user1.username, user1.email);
    println!("{} <{}>", user2.username, user2.email);
}
```

## Debug 输出

手动为每个结构体实现 `println!` 很麻烦，Rust 提供了 `Debug` trait 的派生宏：

```rust
// 添加 derive 宏自动实现 Debug
#[derive(Debug)]
struct User {
    username: String,
    email: String,
    active: bool,
}

fn main() {
    let user = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        active: true,
    };

    // 使用 {:?} 格式化输出
    println!("{:?}", user);
    // User { username: "alice", email: "alice@example.com", active: true }

    // 使用 {:#?} 美化输出
    println!("{:#?}", user);
    // User {
    //     username: "alice",
    //     email: "alice@example.com",
    //     active: true,
    // }
}
```

常用的派生宏：

```rust
#[derive(Debug, Clone, PartialEq)]  // 一次派生多个 trait
struct Rectangle {
    width: u32,
    height: u32,
}
```

| 派生宏      | 作用                         |
| ----------- | ---------------------------- |
| `Debug`     | 允许用 `{:?}` 格式化         |
| `Clone`     | 允许 `.clone()` 深拷贝       |
| `Copy`      | 允许 Copy 语义（需要 Clone） |
| `PartialEq` | 允许 `==` 比较               |
| `Eq`        | 完全相等比较                 |

## 实战演练

### 示例 1：实现矩形计算器

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn new(width: u32, height: u32) -> Self {
        Self { width, height }
    }

    fn square(size: u32) -> Self {
        Self { width: size, height: size }
    }

    fn area(&self) -> u32 {
        self.width * self.height
    }

    fn perimeter(&self) -> u32 {
        2 * (self.width + self.height)
    }

    fn is_square(&self) -> bool {
        self.width == self.height
    }

    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width >= other.width && self.height >= other.height
    }

    fn scale(&mut self, factor: f64) {
        self.width = (self.width as f64 * factor) as u32;
        self.height = (self.height as f64 * factor) as u32;
    }

    fn info(&self) {
        println!(
            "矩形 {}x{}: 面积={}, 周长={}, {}",
            self.width,
            self.height,
            self.area(),
            self.perimeter(),
            if self.is_square() { "正方形" } else { "非正方形" }
        );
    }
}

fn main() {
    let mut rect = Rectangle::new(30, 50);
    rect.info();

    rect.scale(2.0);
    rect.info();

    let square = Rectangle::square(10);
    square.info();

    println!("rect 能容纳 square: {}", rect.can_hold(&square));
}
```

### 示例 2：实现简易用户系统

```rust
#[derive(Debug)]
struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
}

impl User {
    fn new(username: &str, email: &str) -> Self {
        Self {
            username: username.to_string(),
            email: email.to_string(),
            sign_in_count: 0,
            active: true,
        }
    }

    fn sign_in(&mut self) {
        self.sign_in_count += 1;
        println!("{} 登录成功！第 {} 次登录", self.username, self.sign_in_count);
    }

    fn deactivate(&mut self) {
        self.active = false;
        println!("{} 已停用", self.username);
    }

    fn summary(&self) -> String {
        format!(
            "用户: {}, 邮箱: {}, 登录次数: {}, 状态: {}",
            self.username,
            self.email,
            self.sign_in_count,
            if self.active { "活跃" } else { "已停用" }
        )
    }
}

fn main() {
    let mut alice = User::new("alice", "alice@example.com");
    let mut bob = User::new("bob", "bob@example.com");

    alice.sign_in();
    alice.sign_in();
    bob.sign_in();

    println!("{}", alice.summary());
    println!("{}", bob.summary());

    bob.deactivate();
    println!("{}", bob.summary());
}
```

### 示例 3：使用元组结构体表示不同类型的数值

```rust
struct Meters(f64);
struct Kilometers(f64);
struct Celsius(f64);
struct Fahrenheit(f64);

impl Celsius {
    fn new(degrees: f64) -> Self {
        Self(degrees)
    }

    fn to_fahrenheit(&self) -> Fahrenheit {
        Fahrenheit(self.0 * 9.0 / 5.0 + 32.0)
    }

    fn to_kilometers(&self) -> Kilometers {
        Kilometers(self.0)
    }
}

impl Fahrenheit {
    fn to_celsius(&self) -> Celsius {
        Celsius((self.0 - 32.0) * 5.0 / 9.0)
    }
}

impl Kilometers {
    fn to_meters(&self) -> Meters {
        Meters(self.0 * 1000.0)
    }
}

impl Meters {
    fn to_kilometers(&self) -> Kilometers {
        Kilometers(self.0 / 1000.0)
    }
}

fn main() {
    let temp = Celsius::new(100.0);
    let fahrenheit = temp.to_fahrenheit();
    let back_to_celsius = fahrenheit.to_celsius();

    println!("{}°C = {}°F = {}°C", temp.0, fahrenheit.0, back_to_celsius.0);

    let km = Kilometers(5.0);
    let meters = km.to_meters();
    println!("{} km = {} m", km.0, meters.0);
}
```

## 常见错误与修复

### 错误 1：部分可变性

```rust
struct User {
    username: String,
    active: bool,
}

fn main() {
    // ❌ Rust 不支持部分可变
    // let user = User { username: String::from("alice"), active: true };
    // user.active = false;  // 编译错误

    // ✅ 整个结构体设为 mut
    let mut user = User { username: String::from("alice"), active: true };
    user.active = false;
}
```

### 错误 2：使用未初始化的结构体

```rust
fn main() {
    // ❌ 结构体必须初始化所有字段
    // let user = User { username: String::from("alice") };

    // ✅ 初始化所有字段
    let user = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        active: true,
    };
}
```

### 错误 3：在方法签名中使用错误的 self

```rust
impl Rectangle {
    // ❌ 修改数据但用了 &self
    // fn scale(&self, factor: u32) { self.width *= factor; }

    // ✅ 需要修改用 &mut self
    fn scale(&mut self, factor: u32) {
        self.width *= factor;
    }
}
```

## 最佳实践

1. **字段命名用小写蛇形**：`sign_in_count` 而不是 `signInCount`
2. **用 `new` 作为关联函数名**：这是 Rust 的惯例，表示构造函数
3. **优先使用 `&self`**：方法默认用不可变引用，只在需要修改时用 `&mut self`
4. **善用派生宏**：`#[derive(Debug)]` 几乎每个结构体都应该加
5. **结构体应该是一个有意义的整体**：字段之间应该有逻辑关联

## 常见问题

**Q: 结构体字段可以是引用吗？**

A: 可以，但需要添加生命周期标注（后面章节会学）。初学时建议用 `String` 而不是 `&str` 作为字段类型。

**Q: 什么时候用结构体，什么时候用元组？**

A: 当各字段有独立含义时用结构体；当整体有意义但各字段只是位置上的不同时用元组结构体；当数据之间没有语义关联时用普通元组。

**Q: 一个结构体可以有多个 impl 块吗？**

A: 可以，每个方法可以放在不同的 impl 块中。这在实现 trait 时很常见。

**Q: 结构体可以嵌套吗？**

A: 可以，结构体字段可以是另一个结构体。

## 总结

本文我们学习了结构体的核心用法：

**结构体定义**

- 经典结构体：命名字段，最常用
- 元组结构体：有名字的元组
- 单元结构体：无字段，用于标记

**方法与关联函数**

- `impl` 块定义方法
- `&self` 读取数据，`&mut self` 修改数据
- 关联函数（不带 self）用作构造函数
- `Self` 指代当前结构体类型

**实用技巧**

- 字段初始化简写
- 结构体更新语法 `..`
- `#[derive(Debug)]` 快速调试输出
- `::` 调用关联函数
