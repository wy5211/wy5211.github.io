## 不像煤炭也不像酱油的行业

煤炭周期看煤价——一个指标就能定位周期。酱油几乎没有周期——稳稳增长就好。汽车介于两者之间，而且更复杂——它同时被三种周期驱动：

- **消费周期**：汽车是最大的可选消费品，销量跟着居民收入和利率走
- **库存周期**：经销商有囤货-去货的节奏，库存波动放大销量波动
- **技术周期**：燃油车→电动车的替代是一个 10-20 年的大趋势

三层周期叠加，让汽车行业成为周期分析最复杂的行业之一。同样是汽车股，比亚迪翻倍的时候传统车企可能在阴跌——因为它们处于技术周期的不同位置。

这篇把三层周期逐一拆开，看它们怎么分别影响汽车股。

## 第一层：消费周期

汽车是家庭第二大支出（仅次于房子）。买车的决策受三个因素驱动：

**居民收入和信心。** 换车不是刚需——旧车还能开就再凑合一年。经济不确定性高的时候，消费者第一反应就是推迟大额支出。GDP 增速从 6% 降到 4%，汽车销量可能降 10-15%——杠杆效应明显。

反过来，经济好转、收入预期改善时，被推迟的换车需求集中释放，销量可能跳升 15-20%。这种「推迟-释放」的节奏本身就是一种周期。

**利率和信贷条件。** 大部分人买车用贷款（汽车金融渗透率超过 50%）。利率降 1 个百分点，月供少几百块——对价格敏感的消费者来说足够改变决策。

中国的汽车消费还受政策周期影响：购置税减免、以旧换新补贴、限购政策松紧——每一次政策变动都会造成销量的脉冲式波动。2009 年购置税减半 → 销量暴增 46%；2018 年购置税优惠退出 → 销量首次负增长。

**置换周期。** 一辆车的平均使用寿命约 8-12 年。中国 2009-2010 年的第一波汽车消费高峰，到 2020 年前后进入集中置换期——置换需求本身就是周期性的。

消费周期对汽车股的影响是直接的：销量增速变化 → 营收变化。但这只是第一层。

## 第二层：库存周期——真正的放大器

如果消费周期只影响销量，汽车股的波动不会那么大。真正让汽车股暴涨暴跌的是库存周期。

汽车的流通链条：整车厂 → 4S 店（经销商）→ 消费者。整车厂不直接卖给消费者，而是通过经销商。经销商从厂家「进货」（批发），再卖给消费者（零售）。

这里有一个关键的时间差和利益博弈：

**厂家想多卖**：汽车厂的固定成本极高（工厂折旧、模具、研发），产能利用率越高单辆成本越低。所以厂家有强烈的动机向经销商「压库」——不管终端卖得好不好，先批发给经销商再说。

**经销商的痛苦**：经销商被迫吃下的库存需要占压资金（一辆车十几万，1000 辆就是上亿资金），而且要承担跌价风险。库存太多时，经销商只能降价促销——甚至亏本卖车来回笼资金。

这就形成了库存周期：

```
终端需求回暖 → 经销商库存下降 → 经销商信心恢复 → 加大从厂家进货（批发量 > 零售量）
    ↓
厂家看到批发量增长 → 满负荷生产 → 大量车涌入经销商 → 库存被动堆积
    ↓
经销商库存过高 → 降价促销去库存（批发量 < 零售量）→ 厂家被迫减产
    ↓
经销商去库完毕 → 库存低位 → 终端需求回暖时再次加大进货 → 循环
```

库存周期的放大效应：终端零售可能只波动 10%，但批发量（厂家卖给经销商的量，也是厂家确认收入的量）可能波动 30-40%。因为经销商在补库存时大量进货、去库存时几乎不进货——批发量的波动远大于终端零售。

**厂家利润的波动更大**——因为固定成本高（经营杠杆）。批发量降 20%，在固定成本不变的情况下，利润可能降 40-60%。

这就是为什么汽车股的波动总是比汽车销量数据大得多——你看到的销量是终端零售，但股价定价的是厂家的利润（受批发量和经营杠杆双重影响）。

怎么跟踪汽车库存周期？几个信号：

- **经销商库存系数**（中国汽车流通协会每月发布）：库存系数 = 库存车辆数 / 月销量。高于 1.5 说明库存偏高，低于 1.0 说明库存健康。
- **厂家批发销量 vs 零售销量的差异**：批发远大于零售 = 厂家在压库（可能是周期顶部信号）；批发远小于零售 = 厂家在控制产量（可能是去库存尾声）。
- **终端优惠幅度**：大幅降价促销 = 库存压力大的信号。

## 第三层：技术周期——燃油到电动的替代

如果说消费周期和库存周期是中短期波动，技术周期则是长期的结构性力量。

燃油车→电动车的替代是不可逆的趋势。中国新能源汽车渗透率从 2019 年的 5% 升到 2025 年的 50%+——5 年时间完成了从「边缘」到「主流」的跨越。

这个替代过程带来了一个新的周期维度——**产能周期**：

2020-2022 年：电动化前景明确 → 大量资本涌入（蔚来、小鹏、理想、比亚迪扩产 + 新玩家入场）→ 产能快速扩张

2023-2024 年：产能投放高峰 + 竞争加剧 → 价格战（比亚迪降价、特斯拉降价）→ 利润率压缩 → 部分新势力亏损倒闭

2025-2026 年：弱势玩家出清 → 产能增速放缓 → 供需重新平衡 → 龙头企业利润修复

这和第 04 篇讲的煤炭产能周期本质上完全一样——只是驱动因素从「基建需求」变成了「技术替代」。

技术周期带来的投资含义：

**不能把所有汽车股放在一起看。** 燃油车企业和电动车企业处于完全不同的周期位置。燃油车面临的是「需求长期萎缩」（被电动车替代），这是结构性下行，不是周期波动。电动车面临的是「渗透率提升中的产能出清」——长期向上但短期波动大。

**比亚迪翻倍不是因为「汽车行业好」。** 是因为比亚迪在技术周期中占据了有利位置（垂直整合 + 规模效应 + 产品力）。同期某些传统车企阴跌，不是因为「汽车行业差」，而是因为它们在技术替代中处于被淘汰的位置。

## 三层周期叠加的实战分析

拿 2024-2025 年的汽车行业做三层周期叠加分析：

**消费周期**：中等偏上。居民收入温和增长，汽车以旧换新政策刺激，换车周期进入释放期。但经济不确定性仍在制约可选消费。

**库存周期**：从去库存转向补库存。2023 年经销商经历了痛苦的库存去化，2024 年库存系数回到健康水平，开始温和补库。

**技术周期**：产能出清中期。弱势新势力退出或被收购，但行业整体产能仍然偏过剩。价格战持续但烈度减弱。龙头集中度提升。

三层叠加的结论：汽车行业处于「消费温和复苏 + 库存周期向上 + 技术周期出清中期」的叠加状态。不是全面牛市（产能还在出清），但最差的时期（2023 年的去库存+价格战）大概率过去了。

对于投资决策的含义：这个阶段适合关注行业龙头（在产能出清中受益的集中度提升），不适合追二三线品牌（可能成为出清的牺牲品）。对于 ETF 投资者，新能源车 ETF 的周期弹性更大（含比亚迪、宁德时代等龙头），传统汽车 ETF 可能受到燃油车衰退拖累。

## 汽车行业的经营杠杆

最后量化一下经营杠杆。汽车厂是典型的重资产制造业——工厂、模具、产线、机器人，固定成本极高。

简化模型：一家年产 100 万辆的车企

- 固定成本（折旧、研发摊销、管理人员）：400 亿
- 每辆变动成本（零部件、材料、直接人工）：12 万
- 平均售价：15 万
- 每辆边际贡献：3 万

满产时（100 万辆）：边际贡献 300 亿，减固定成本 400 亿...亏损了？这就是为什么规模对车企生死攸关——必须卖够一定量才能覆盖固定成本。

假设设计产能 120 万辆，实际卖 110 万辆：

- 边际贡献：110 × 3 = 330 亿，利润 = 330 - 400 = ... 还是亏？

换个更真实的参数。假设每辆边际贡献 5 万（售价更高或变动成本更低）：

- 卖 100 万辆：边际贡献 500 亿，利润 100 亿
- 卖 110 万辆：边际贡献 550 亿，利润 150 亿——销量涨 10%，利润涨 50%
- 卖 90 万辆：边际贡献 450 亿，利润 50 亿——销量跌 10%，利润跌 50%

**销量波动 10%，利润波动 50%——经营杠杆系数 5。** 这就是为什么汽车股对销量数据极其敏感——一个季度的销量不及预期，利润预期可能直接腰斩。

而且，当销量下滑时，厂家通常还会**降价促销**——变动成本不变但售价降低，边际贡献进一步收窄。双重打击下，利润的跌幅远超销量的跌幅。

## 三种周期股的对比

到这一篇为止，你已经看了三种典型的强周期行业：

| 维度       | 煤炭                   | 银行                 | 汽车                   |
| ---------- | ---------------------- | -------------------- | ---------------------- |
| 周期类型   | 商品周期（产能驱动）   | 信贷周期（滞后经济） | 消费+库存周期          |
| 驱动力     | 煤价（供需失衡）       | 不良率+NIM           | 销量+经销商库存        |
| 经营杠杆   | 极高（固定成本75%）    | 中等                 | 高（固定成本60%）      |
| 定价权     | 无（同质化大宗）       | 中等（利差受管制）   | 中等（品牌+降价）      |
| 最直接信号 | 动力煤价格             | 不良率、拨备覆盖率   | 经销商库存系数         |
| PE 低 = ?  | 周期顶（利润峰值陷阱） | 利润质量打折         | 可能是真的低（成熟期） |

三种行业，三种周期驱动，三种分析方法。但它们有一个共同点：**利润波动远大于收入/销量波动**——因为经营杠杆。这是所有强周期行业的数学共性。
