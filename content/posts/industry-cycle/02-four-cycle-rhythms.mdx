## 「周期」不是一个词，是四种完全不同的东西

你在研究一个行业时，经常碰到一个困惑：2023 年汽车销量反弹了，这到底是短期的库存回补，还是长期的增长趋势？2024 年光伏产能过剩严重，这是一两年能出清的短期问题，还是要磨很多年的产能周期下行？

不区分周期的类型，就会犯方向性错误：把短期反弹当成长期趋势去追，或者把长期下行当成短期波动去抄底。

经济学里其实早就把周期分成了四种，以发现者的名字命名。听起来很学术，但实际非常实用——因为不同类型的周期影响完全不同的行业，时间尺度差了一个数量级。

## 基钦周期：库存周期（3-4 年）

**驱动力：企业的库存调整。**

经济好的时候，企业预期需求旺盛，大量囤货（原材料 + 成品库存）。经济转差时，需求不及预期，库存堆积，企业开始去库存——减产、促销、清仓。

库存的「加」和「减」放大了真实需求的波动。真实需求可能只降了 5%，但企业从「囤货」切换到「去库存」，采购量可能降了 20%——这就是库存周期把小波动放大成大波动的过程。

一个完整的库存周期大约 3-4 年（40 个月左右），分为四个阶段：

```
被动去库存 → 需求回暖，但企业还没反应过来，库存被动下降
    ↓
主动补库存 → 企业发现需求好了，开始大量囤货、扩产
    ↓
被动补库存 → 需求开始走弱，但企业还在惯性生产，库存被动堆积
    ↓
主动去库存 → 企业发现需求不行了，开始减产、清仓、降价促销
    ↓
（回到起点）
```

**影响行业**：有明显库存周期的行业通常是制造业——汽车、家电、电子、化工、服装。这些行业有「生产-库存-销售」的时间差，库存波动明显。

服务业（餐饮、教育、医疗）几乎没有库存周期——服务不能存储，随产随消。

**怎么识别库存周期**：

- PMI（采购经理指数）中的「产成品库存」和「原材料库存」分项
- 工业企业产成品库存增速（国家统计局每月公布）
- 行业龙头公司的库存周转天数变化

**库存周期对股价的影响**：库存周期是短周期，影响的是企业的短期盈利波动。它不会改变行业的长期趋势，但会造成股价的短期波动——主动去库存阶段（减产+降价）利润最差，股价通常在底部；主动补库存阶段（扩产+提价）利润最好，股价通常在高位。

## 朱格拉周期：产能周期（8-10 年）

**驱动力：固定资产投资的周期性。**

这是对行业影响最大、最值得关注的周期类型。

逻辑链条是这样的：

```
行业盈利好 → 企业大规模投资扩产 → 产能快速扩张
    ↓
产能超过需求 → 供过于求 → 价格下跌 → 利润下滑
    ↓
行业亏损 → 企业停止投资、退出市场 → 产能收缩
    ↓
需求增长消化过剩产能 → 供需重新平衡 → 价格回升
    ↓
行业再次盈利 → 新一轮投资扩产 → 循环
```

产能周期比库存周期长得多——因为从投资决策到产能投产需要 2-3 年（轻工业）到 5-7 年（矿业）。加上产能过剩后的出清过程（企业不会立刻退出，会硬扛一段时间），一个完整周期通常 8-10 年。

**影响行业**：重资产、重投资的行业——煤炭、钢铁、有色金属、化工、水泥、航运、造船。这些行业的固定资产投资大、建设周期长，产能调整特别缓慢。

**经典案例——钢铁行业的中国产能周期**：

- 2009-2013 年：四万亿刺激 → 钢铁产能疯狂扩张 → 产能严重过剩
- 2014-2016 年：行业全面亏损 → 大量钢厂停产 → 供给侧改革「地条钢」清退
- 2017-2021 年：产能出清 → 供需改善 → 钢价回升 → 钢铁利润恢复
- 2022-2025 年：地产下行 → 钢铁需求萎缩 → 再次进入周期下行

**怎么识别产能周期**：

- 固定资产投资增速（行业层面）
- 产能利用率（越低说明产能过剩越严重）
- 在建工程/资本开支增速（大量在建 = 未来产能投放 = 潜在过剩信号）
- 行业集中度变化（周期底部通常伴随并购整合、中小企业退出）

**产能周期对投资的指导意义**：这是做周期股投资最重要的周期类型。产能周期底部（行业大面积亏损、产能开始出清）是布局强周期行业的最佳时机。产能周期顶部（行业普遍盈利、大量在建产能）是风险最大的时刻。

## 库兹涅茨周期：地产/建筑周期（15-20 年）

**驱动力：人口结构 + 信贷周期 + 城镇化。**

房地产是人类最大的单一资产类别，也是对经济影响最深远的长周期力量。

一个地产周期通常持续 15-20 年，因为：

- 从拿地到交房到消化需要 5-8 年
- 信贷周期（利率升降）一个完整周期约 10 年
- 人口代际更替（一波婴儿潮长大成年 → 购房需求集中释放 → 需求退潮）需要 20-30 年

**中国地产周期的特殊性**：

中国 1998 年房改启动，开启了人类历史上规模最大的房地产扩张周期。从 2000 年到 2021 年，商品房销售面积从 1.8 亿平方米增长到 17.9 亿平方米——翻了近 10 倍。这个超长上行周期由三个力量驱动：城镇化（农村人口进城）、人口红利（80-90 年代婴儿潮成年）、信贷扩张（居民杠杆率从 10% 升到 60%+）。

2021 年是拐点——三道红线+恒大暴雷+人口见顶+城镇化放缓，地产进入下行周期。这个下行可能持续很多年，因为三个驱动力量同时减弱：城镇化率已经 65%+（放缓空间小）、人口开始负增长、居民杠杆率已经很高（再加不动了）。

**影响行业**：房地产本身，以及整个后地产链——建材（水泥、玻璃）、家居、家电（空调冰箱洗衣机跟着新房走）、工程机械（挖掘机、起重机）、装饰装修。这些行业的景气度高度依赖房地产周期。

**地产周期对投资的含义**：在地产下行周期中，后地产链的行业会长期承压。不能用「PE 低」来判断这些行业便宜——利润下行可能刚刚开始。你需要等待地产周期企稳的信号（销售面积止跌、库存去化完毕、政策转向），才能考虑布局后地产链。

## 康德拉季耶夫周期：技术周期（50-60 年）

这个周期的尺度远超行业层面，它描述的是由技术革命驱动的超长经济波动。详细内容在 [全球宏观系列](/posts/global-macro/02-kondratiev-waves) 里讲过，这里只做简要关联：

- 每一轮康波周期由一次重大技术革命驱动：蒸汽机（1780s）→ 铁路（1830s）→ 电力（1880s）→ 汽车（1930s）→ 信息技术（1970s-2020s）
- 每轮周期约 50-60 年，经历繁荣→衰退→萧条→回升四个阶段
- 当前共识：第五波（信息技术）的衰退到萧条期，AI 有可能开启第六波

**对本系列的关联**：技术周期决定了哪些行业处于「长期风口」（AI、新能源），哪些处于「长期衰退」（传统燃油车、煤炭的远期需求）。但它太长，对短期投资决策的指导意义有限——大多数投资者不会持有一个标的 50 年。

## 多周期叠加：最重要的分析视角

真正强大的分析框架不是单独看某一种周期，而是看**多个周期叠加**的效果。

当一个行业同时受到多种周期的影响时，方向一致还是矛盾，决定了行业的走势强度：

**同向叠加（共振）——最强信号**：

- 2009 年中国「四万亿」：库存周期见底回升 + 产能周期扩张 + 信贷宽松 → 几乎所有周期股暴涨
- 2021 年新能源：技术周期（碳中和政策）+ 产能周期扩张初期 + 库存周期向上 → 新能源ETF 暴涨

**反向叠加（抵消）——走势纠结**：

- 2024 年光伏：技术周期向上（渗透率还在提升）+ 产能周期向下（严重过剩）→ 行业方向矛盾，利润被产能过剩拖累，但长期前景还在
- 2025 年汽车：技术周期向上（电动化）+ 库存周期波动 → 短期销量有波动但长期趋势不变

**同向下行（共振底）——最危险但也最有机**：

- 2024 年中国地产链：地产周期下行 + 库存周期去库 + 信贷周期紧缩 → 全面下行，但这些行业在周期底部，一旦反转弹性大

判断多周期叠加的口诀：

- 库存 + 产能 + 地产同时向上 = 大牛市的基础
- 三者同时向下 = 大熊市或长期阴跌
- 方向不一致 = 震荡，看哪个周期占主导

## 实战：怎么用周期类型避免误判

回到开头的问题：2023 年汽车销量反弹，是短期还是长期？

用周期框架拆解：

- **库存周期**：2022 年疫情导致汽车库存高企，2023 年初经销商完成去库存 → 进入被动去库存和主动补库存阶段 → 销量数据改善。这是短周期的库存反弹。
- **产能周期**：传统燃油车产能过剩、新能源车产能快速扩张。产能结构调整中期持续。
- **技术周期**：燃油→电动的替代是 10-20 年的大趋势，渗透率还在提升。长期向上。

结论：2023 年的销量反弹主要是**库存周期驱动**的短期改善。长期看新能源替代的大趋势不变，但短期不排除库存周期再次波动（2024-2025 年可能再次经历补库存→去库存的波动）。

如果你分不清周期类型，可能会把库存反弹误读为「行业景气度全面恢复」，在补库存高峰追高买入——然后在去库存阶段承受回调。

**周期分析的纪律**：

1. 先判断当前信号属于哪种周期（库存？产能？地产？）
2. 再判断该周期处于什么阶段（上行？下行？见底？见顶？）
3. 最后综合多周期方向（共振？抵消？）
4. 不要用短周期信号判断长周期趋势（库存反弹 ≠ 产能出清）

## 一个简化的周期跟踪清单

不同周期需要跟踪不同的信号：

| 周期类型 | 时间尺度 | 关键信号                                 | 数据来源               |
| -------- | -------- | ---------------------------------------- | ---------------------- |
| 库存周期 | 3-4 年   | PMI 库存分项、产成品库存增速             | 国家统计局、财新 PMI   |
| 产能周期 | 8-10 年  | 产能利用率、固定资产投资增速、在建工程   | 行业协会、上市公司年报 |
| 地产周期 | 15-20 年 | 商品房销售面积、新开工面积、库存去化周期 | 国家统计局、克而瑞     |
| 技术周期 | 50-60 年 | 渗透率（新能源/电动车/AI）、专利数量     | 行业研究报告           |

你不需要每周追踪所有信号——大多数时候只需要关注你持仓行业的核心周期信号。比如你持有煤炭ETF，重点关注产能利用率 + 煤价位置就够判断产能周期阶段了。持有汽车相关 ETF，关注经销商库存和乘用车销量增速就够跟踪库存周期。
