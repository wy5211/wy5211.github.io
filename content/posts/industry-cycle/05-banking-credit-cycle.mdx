## 最不像周期股的周期股

煤炭周期看煤价，一眼能看出在周期什么位置。银行呢？你看不到「银行价格」这种东西，也看不到银行的「库存」。银行年报上的利润数字永远很体面——不管经济好不好，银行似乎都在赚钱。

但银行股的 PE 常年在 5-8 倍，PB 在 0.5 左右——如果利润真的那么稳定、风险真的那么低，为什么市场给这么低的估值？

因为银行的周期性藏在了你看不到的地方——**信贷质量**。银行利润的周期波动不体现在收入端，而体现在「风险成本」端。经济好的时候风险成本低，利润好看；经济差的时候不良贷款暴露，利润被拨备吞噬。这种周期性比煤炭更隐蔽，但威力不亚于商品周期。

## 银行利润的三要素

理解银行，先拆利润。一家银行的利润取决于三个变量：

```
利润 = 贷款规模 × 净息差（NIM） − 风险成本（拨备） − 运营成本
```

### 要素一：贷款规模——跟着信贷周期走

经济扩张期，企业想借钱扩产、居民想借钱买房买车，信贷需求旺盛。银行放量放贷，资产规模快速增长。

经济收缩期，企业去杠杆不愿借钱，居民还贷不借钱，信贷需求萎缩。银行有钱贷不出去，规模增速放缓甚至收缩。

中国的信贷周期有一个特殊变量——**政策**。央行通过窗口指导、MLF/SLF、存款准备金率等工具调控银行的信贷投放节奏。「宽信用」（鼓励放贷）和「紧信用」（控制放贷）的切换，直接决定了银行的规模增速。

### 要素二：净息差（NIM）——跟着利率周期走

净息差 = 贷款利率 − 存款利率。它衡量银行「低息揽储、高息放贷」赚取的利差。

中国的利率环境对银行 NIM 有系统性影响：

- **降息周期**：央行降 LPR（贷款基准利率），银行贷款利率下降。但存款利率下降通常滞后且幅度小——因为银行不敢大幅降低存款利率（怕储户把钱转走）。结果是 NIM 收窄，银行利润承压。
- **加息周期**：反向操作，NIM 扩张。但中国近年几乎没有加息周期——利率趋势性下行。

2022-2025 年中国持续降息，银行业 NIM 从 2.1% 附近降到 1.7% 以下。这意味着：同样的贷款规模，银行赚的利差少了 20%+。

### 要素三：风险成本（拨备）——最隐蔽也最致命

这是银行利润最大的「调节阀」，也是银行周期性的主要来源。

银行放出去的贷款，总有一部分收不回来——企业倒闭了、开发商跑路了、个人断供了。银行必须为这些可能的损失提前计提「拨备」。拨备从利润中扣除——拨备提得越多，当期利润越低。

经济好的时候：不良率低，银行少提拨备，利润好看。有些银行甚至「释放拨备」（把以前多提的拨备转回利润）来粉饰业绩。

经济差的时候：不良率上升，银行必须大幅增加拨备。一笔大额贷款违约可能吃掉一个季度甚至半年的利润。2023-2024 年地产暴雷链上的银行（如某些股份行），就是被地产不良贷款大幅拖累。

**拨备是银行利润的「海绵」**——经济好的时候吸进去（多提），经济差的时候挤出来（大幅计提）。这让银行报告的利润看起来很平稳，但实际盈利能力是在波动的。市场不傻，它用低 PE 来定价这种「利润的不确定性」。

## 银行的信贷周期

银行自身的周期性和实体经济互相强化，形成信贷周期：

```
经济复苏 → 企业盈利改善 → 信贷需求上升 → 银行放量放贷
    ↓
经济繁荣 → 资产价格上涨（地产/股市）→ 抵押物价值上升 → 银行更敢放贷
    ↓
信贷过度扩张 → 资产泡沫 + 企业过度负债 → 风险积累
    ↓
触发因素（政策收紧/外部冲击）→ 资产价格下跌 → 抵押物缩水 → 不良上升
    ↓
银行收紧信贷 → 企业融资困难 → 更多违约 → 不良进一步上升
    ↓
经济衰退 → 信贷冻结 → 周期触底
    ↓
政策刺激 → 信贷重新扩张 → 回到起点
```

这个循环的关键特征是**滞后性**：经济已经转差了，但不良贷款还没有大规模暴露——因为贷款还没到期，企业还在用新贷还旧贷。银行利润的拐点通常滞后经济拐点 1-2 年。

这就是为什么银行股经常在经济见顶之后还在涨（信贷还在扩张），在经济见底之后还在跌（不良还在暴露）。它是一个滞后的周期股。

## 为什么银行 PE 长期只有 5-8 倍

A 股银行板块 PE 常年 5-8 倍，PB 0.5 左右。相比之下，食品饮料 PE 20-40 倍，医药 25-40 倍，科技 30-100 倍。银行为什么被市场「歧视」到这种程度？

三个原因：

### 原因一：利润质量可疑

银行报告的利润高度依赖拨备假设。假设不良率 1%，拨备充足，利润很漂亮。但如果实际不良率是 2%、3% 呢？之前几年少提的拨备要回吐，之前几年的利润要追减。

市场不信任银行报告的利润数字——不是觉得银行造假，而是认为真实的不良率可能比报告的高。你给一个「可能要打折」的利润定估值，自然不敢给高 PE。

### 原因二：隐性风险未定价

中国银行业有大量贷款投向了地方融资平台和房地产——这两个领域的风险都没有充分反映在财报里。

地方债：地方政府融资平台借了大量贷款用于基建。这些项目本身的回报率很低（修路、造桥不赚钱），还款高度依赖地方财政。如果土地财政萎缩（卖地收入下降），偿债压力骤增。

房地产：开发贷和个人按揭贷。地产下行周期中，开发商违约率上升，按揭断供增加。

这些风险的规模可能达到数万亿。市场用 0.5 的 PB 告诉你：银行的账面净资产（每股净资产）可能要打折——因为一部分资产（贷款）最终可能收不回来。

### 原因三：成长性见顶

中国银行业的规模扩张期已经过了。2010 年前后 M2 增速 15-20%，银行资产增速也是这个水平。现在 M2 增速降到 7-8%，银行规模增速跟着放缓。一个增速只有 5% 的行业，市场不愿意给高 PE。

## 2024-2026 年银行股的特殊行情

这里有个有意思的现象。2024 年下半年到 2025 年，A 股银行板块出现了一波明显上涨——几家大行股价创下多年新高。很多人不理解：经济增速放缓、NIM 持续收窄、地产不良还在消化，为什么银行股反而涨了？

答案是**估值修复 + 资产荒驱动**，不是基本面改善。

2024 年中国进入持续降息周期。存款利率从 2% 降到 1.5% 以下，理财收益率从 4% 降到 2.5% 以下。大量资金找不到收益率够高的安全资产——国债收益率 2% 不到，货基收益率 1.5%。

这时候投资者发现：大行银行股的股息率有 5-6%，PE 才 5-6 倍，PB 0.5——虽然成长性差，但每年分红 5-6%，比国债和理财高多了。而且四大行不可能倒闭（系统重要性银行），安全性接近国债。

于是大量「追求稳定收益」的资金涌入银行股——不是赌业绩增长，而是把银行股当「高息债券」买。这推动银行股估值从 PE 4-5 倍修复到 6-8 倍。

**这是估值修复，不是业绩驱动。** 银行的基本面（NIM 收窄、不良压力）并没有改善，只是在一个利率下行的环境里，5% 的股息率变得格外有吸引力。

这对你的投资含义是：银行股的这波上涨不是可持续的业绩增长——当利率企稳或反转时，银行股的「替代债券」逻辑就会弱化。但作为红利低波策略的核心持仓，银行的低波动 + 高股息特性在组合里仍然有价值。

## 银行周期 vs 煤炭周期：对比

把银行和煤炭的周期做个对比，帮助理解为什么它们的分析方法完全不同：

| 维度         | 煤炭                      | 银行                      |
| ------------ | ------------------------- | ------------------------- |
| 周期驱动力   | 商品价格（供需错配）      | 信贷质量（经济周期滞后）  |
| 利润波动来源 | 煤价波动（经营杠杆放大）  | 拨备波动（风险成本调整）  |
| 周期信号     | 煤价、库存、产能利用率    | 不良率、NIM、信贷增速     |
| 周期顶部信号 | 煤价历史高位 + PE 极低    | 信贷疯狂扩张 + NIM 扩张   |
| 周期底部信号 | 煤价跌破成本 + 大面积亏损 | 不耐大面积暴露 + 拨备大增 |
| 长期趋势     | 需求趋势性下降（双碳）    | 规模增速放缓（M2 降速）   |
| 低 PE 的含义 | 周期顶部利润峰值（陷阱）  | 利润质量打折 + 成长性低   |

核心区别：煤炭周期看价格（直接的、透明的），银行周期看信用（间接的、不透明的）。煤炭的不良你能算出来（煤价 - 成本），银行的不良你需要判断（报告数字 vs 实际数字）。这种不透明性是银行长期低估值的根本原因。

## 红利低波里的银行

你可能不直接持有银行 ETF，但红利低波 ETF 里通常有大量银行股——银行因为低 PE、低 PB、高股息率，是红利策略的天然核心持仓。

这意味着：你的红利低波持仓表现，部分取决于银行股的周期位置。当银行处于信贷周期下行（不良暴露期）时，红利低波的回撤中有一部分是银行拖累的。反过来，当银行进入估值修复期（如 2024 年的高股息行情），红利低波也会受益。

理解银行周期，不只是为了投资银行 ETF——它帮你理解红利低波持仓中一个重要成分的驱动力。
