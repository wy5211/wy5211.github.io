## 拿到一个新行业，怎么判断它周期性多强

有人问你：新能源算周期股吗？光伏呢？半导体呢？

大多数人凭感觉回答——「新能源是成长股」「光伏是周期股」「半导体不好说」。但如果你追问「为什么」，很多人答不上来。

这不是靠经验积累的玄学。行业周期性的强弱有明确的、可分析的结构性原因。拿到任何行业，从四个维度打分，就能判断它的周期性有多强、周期驱动力是什么。

这四个维度就是这一篇的核心：**需求弹性、供给刚性、固定成本占比、产品同质化程度。**

## 维度一：需求弹性——经济好才买 vs 反正要买

需求弹性衡量的是「需求对经济变化的敏感度」。

高弹性需求：经济好的时候需求暴增，经济差的时候需求暴跌。典型代表是**可选消费和工业中间品**——汽车、旅游、钢铁、化工、机械。经济好的时候企业扩产、消费者换车，需求井喷；经济差的时候企业停工、消费者推迟大额支出，需求断崖。

低弹性需求：不管经济好坏，需求都比较稳定。典型代表是**必需消费和公共服务**——食品、饮料、医药、水电气。经济好的时候你不会多吃饭，经济差的时候你也不会不看病。

弹性高低取决于几个因素：

**产品是刚需还是改善型。** 电是刚需——工厂不能停电，居民不能没照明。但用电量随经济波动，因为高耗能产业（钢铁、水泥、化工）的产量跟着经济走。煤炭作为火电主要原料，需求间接地被经济周期驱动。

**购买决策的时间跨度。** 买车是个大决策，消费者会反复权衡，经济不确定性高时就推迟。买酱油不需要决策，用完了就买。决策跨度越长，弹性越大——因为可以被推迟或提前。

**支出占比。** 房地产占家庭支出 30-70%，利率和收入一变，购房意愿立刻变。酱油占家庭月支出不到 1%，收入变化对酱油消费几乎没有影响。

给你一张需求弹性排序表，从高到低：

| 弹性等级 | 代表行业                 | 驱动因素               |
| -------- | ------------------------ | ---------------------- |
| 极高弹性 | 航运、工业金属、化工     | 跟着工业产出和贸易量走 |
| 高弹性   | 汽车、家电、旅游、奢侈品 | 可选消费，可推迟       |
| 中等弹性 | 银行、证券、保险         | 跟着经济和信贷走       |
| 低弹性   | 食品饮料、日用品、医药   | 必需消费               |
| 极低弹性 | 公用事业（水电气）、殡葬 | 刚需中的刚需           |

## 维度二：供给刚性——产能调不动才是大问题

需求变化决定了价格变化的方向，供给响应速度决定了价格偏离正常水平的时间长度。

供给刚性的意思是：**产能调整需要很长时间或很大成本。**

煤矿为什么供给极刚性？几个原因：

- **勘探审批周期长**：发现新煤田 → 地质勘探 → 环评审批 → 采矿证，在中国至少 2-3 年
- **建设周期长**：大型矿井从开工到投产 2-4 年
- **产能不可逆**：你不可能「先关掉一半产能，等价格涨了再开」。矿井停产后再复工成本极高，而且长期停产会导致安全隐患
- **资源禀赋约束**：煤炭不是想产多少就产多少——每个矿井有设计产能，受地质条件限制

酱油厂为什么供给灵活？

- 建一条酱油生产线 6-12 个月
- 可以灵活调整班次（三班倒变两班倒）
- 原料（大豆、食盐）供应充足，不存在资源约束

供给刚性越高，供需失衡持续的时间越长，价格偏离正常水平的幅度越大，周期波动越剧烈。

一张供给刚性排序：

| 刚性等级 | 代表行业               | 产能调整时间 | 原因               |
| -------- | ---------------------- | ------------ | ------------------ |
| 极刚性   | 矿业（煤/铁/铜）、油田 | 4-7 年       | 资源勘探+建设+审批 |
| 高刚性   | 芯片制造、大型化工     | 2-3 年       | 技术门槛+资本密集  |
| 中等刚性 | 汽车、钢铁、水泥       | 1-2 年       | 产线建设+供应链    |
| 低刚性   | 食品、服装、家电       | 6-12 月      | 轻资产+技术成熟    |
| 极低刚性 | 软件、服务、零售       | 数月         | 几乎不需要固定资产 |

## 维度三：固定成本占比——经营杠杆的魔法

这个维度最容易被忽视，但它解释了一个核心现象：**为什么周期股的利润波动远大于收入波动。**

一个钢铁厂年收入 200 亿，成本 180 亿，利润 20 亿。它的成本结构是这样的：

- 固定成本（折旧、利息、人员基本工资）：120 亿——不管产多少钢都要花
- 变动成本（铁矿石、焦炭、电费）：60 亿——跟着产量走

如果明年钢材需求下降，收入降到 160 亿（降 20%），变动成本也按比例降到 48 亿。但固定成本还是 120 亿——折旧不会因为你少产钢就减少。

利润 = 160 - 120 - 48 = -8 亿。亏损了。

收入只降了 20%，利润从 +20 亿变成 -8 亿——跌了 140%。

这就是**经营杠杆**。固定成本占比越高，利润波动对收入波动的放大倍数越大。

用数学公式表达：

```
经营杠杆系数 = 边际贡献 / 利润 = (收入 - 变动成本) / (收入 - 总成本)
```

上面那个例子：边际贡献 = 200 - 60 = 140 亿，利润 = 20 亿，经营杠杆系数 = 140 / 20 = 7。意思是收入每变动 1%，利润变动 7%。

煤炭企业的经营杠杆更高。一个煤矿的成本结构：

- 固定成本（矿井建设折旧、安全设备、人员）：占 70-80%
- 变动成本（炸药、运输、部分能耗）：占 20-30%

煤价涨 50%，收入涨 50%，变动成本涨 15%（占小头），固定成本不变。利润可能涨 200-300%。

煤价跌 30%，收入跌 30%，变动成本跌 10%，固定成本不变。利润可能直接从正变负。

**固定成本占比是周期性的放大器。** 同样幅度的需求/价格变化，在高固定成本行业（矿业、航空、芯片）造成的利润波动，远大于在低固定成本行业（软件、服务、品牌消费）。

固定成本占比排序：

| 等级 | 代表行业             | 固定成本占比 | 利润放大倍数 |
| ---- | -------------------- | ------------ | ------------ |
| 极高 | 航空、矿业、芯片制造 | 70-85%       | 5-10x        |
| 高   | 钢铁、水泥、化工     | 60-70%       | 3-5x         |
| 中等 | 汽车、机械           | 40-55%       | 2-3x         |
| 低   | 消费品牌、零售       | 25-35%       | 1.5-2x       |
| 极低 | 软件、咨询、服务     | 10-20%       | 1-1.5x       |

## 维度四：产品同质化——你有定价权吗

最后一个维度：你的产品和别人的有没有区别。

同质化程度高的行业（大宗商品），价格由市场供需决定，单个企业没有定价权。煤炭、铁矿石、铜、铝——你的产品和竞争对手的产品没有本质区别，买家只看价格。

差异化程度高的行业（品牌、专利），企业有定价权。海天味业的酱油比杂牌贵 30%，消费者愿意买——因为品牌信任和口味习惯。恒瑞医药的专利药可以定高价——因为别人不能仿制。

定价权为什么重要？因为它决定了企业能不能「对抗周期」。

煤炭价格暴跌时，煤炭企业能做的只有跟着跌价——不降价就卖不出去。酱油成本上升时，海天可以提价 5%——消费者不会因为一瓶酱油贵了 2 块钱就换品牌。

**定价权是利润的缓冲垫。** 有定价权的企业可以在成本上升时转嫁压力、在需求下行时维持利润。没有定价权的企业只能被动接受市场价格的摆布。

同质化程度排序：

| 等级     | 代表行业               | 定价权                 |
| -------- | ---------------------- | ---------------------- |
| 完全同质 | 煤炭、铁矿石、铜、原油 | 无                     |
| 高同质   | 钢材、水泥、基础化工   | 极弱                   |
| 中等同质 | 汽车、家电、手机       | 中等（品牌有一定影响） |
| 低同质   | 食品饮料、日化         | 强（品牌溢价）         |
| 极低同质 | 专利药、高端软件       | 极强（专利/技术壁垒）  |

## 四维度综合：行业周期性评分

把四个维度综合起来，可以给任何行业打一个「周期性评分」：

| 行业     | 需求弹性           | 供给刚性 | 固定成本 | 同质化 | 综合周期性 |
| -------- | ------------------ | -------- | -------- | ------ | ---------- |
| 煤炭     | 高（下游周期）     | 极强     | 极高     | 高     | **极强**   |
| 航运     | 极高               | 极强     | 极高     | 高     | **极强**   |
| 钢铁     | 高                 | 强       | 高       | 高     | **强**     |
| 有色金属 | 高                 | 极强     | 高       | 高     | **极强**   |
| 化工     | 高                 | 中强     | 高       | 中     | **强**     |
| 汽车     | 高                 | 中       | 高       | 中     | **强**     |
| 银行     | 中                 | 中       | 中       | 中     | **中强**   |
| 券商     | 极高（跟着交易量） | 弱       | 中       | 中     | **强**     |
| 半导体   | 高                 | 强       | 极高     | 中     | **强**     |
| 食品饮料 | 低                 | 弱       | 低       | 低     | **弱**     |
| 医药     | 低                 | 弱       | 中       | 低     | **弱**     |
| 公用事业 | 低                 | 中       | 高       | 低     | **弱**     |
| 软件     | 低                 | 极弱     | 低       | 低     | **极弱**   |

用法很简单：拿到一个你不熟悉的行业，从四个维度逐一分析，就能大致判断它的周期性强弱。四个维度都偏高（如煤炭），周期性极强；四个维度都偏低（如软件服务），几乎没有周期性。

## 回到那个问题：新能源算周期股吗

用四维度框架分析一下光伏和锂电：

**需求弹性：中等偏高。** 光伏装机量受政策（碳中和目标）和成本（组件价格）驱动，不完全跟随经济周期，但有自身的政策周期和技术替代周期。锂电池需求受新能源车销量驱动——汽车是可选消费，有经济周期弹性。

**供给刚性：强。** 光伏硅料产能扩张需要 1.5-2 年，锂电池正极材料产能扩张需要 1-2 年。供给响应不算特别慢，但在需求暴增时仍然会跟不上。

**固定成本：高。** 光伏组件厂、电池厂都是重资产——设备投资大、折旧高，经营杠杆效应明显。

**同质化：中高。** 光伏组件是偏同质化的产品（转换效率差异不大，买家看性价比）。锂电池有一定技术差异（宁德时代 vs 二三线），但头部以下同质化严重。

综合判断：**光伏和锂电当前是强周期行业。** 虽然长期看是成长赛道（渗透率还在提升），但短中期表现为典型的产能周期——大量资本涌入 → 产能过剩 → 价格战 → 利润暴跌 → 产能出清 → 供需恢复。2023-2024 年光伏组件价格腰斩、硅料价格暴跌 60%，就是产能周期下行的典型表现。

这和煤炭、钢铁的产能周期本质上是一回事——只是驱动因素从「基建需求」变成了「碳中和政策+技术进步」。但供需错配的机制完全相同。

## 这篇的框架怎么用

这四维度框架不是理论——它是一个可以直接用的分析工具：

**拿到一个 ETF 时**，先看它的成分股属于哪些行业，用四维度判断周期性。如果周期性强，就不能用普通 PE 估值（第 01 篇讲过，周期股低 PE 是陷阱），要结合行业自身的周期位置来判断。

**做资产配置时**，选周期性不同的行业组合在一起。煤炭（产能周期）+ 银行（信贷周期）+ 食品饮料（弱周期）——它们不同步，组合的波动率远低于单个行业。你在 [资产配置](/posts/asset-allocation/01-why-diversify) 里学过分散化的原理，这里给出的是分散化的底层依据。

**判断行业拐点时**，重点跟踪供给端信号——产能利用率、在建产能、资本开支增速。产能出清是周期见底的前提条件，产能扩张是周期见顶的预警信号。第 08 篇会详细讲怎么用这些信号判断周期位置。

下一篇，我们用这个框架解剖三个经典行业——煤炭、银行、汽车。理论和真实案例一对照，你会有更深的体感。
