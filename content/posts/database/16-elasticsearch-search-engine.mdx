## 前言

PostgreSQL 的全文搜索不够强大？需要实现复杂的搜索功能？Elasticsearch 是最流行的开源搜索引擎，提供了强大的全文搜索、分布式架构和实时分析能力。

本文将系统讲解 Elasticsearch 的核心概念和使用方法，帮你实现强大的搜索功能。

---

## 核心概念

### 为什么选择 Elasticsearch？

**Elasticsearch 优势**：

- ✅ 强大的全文搜索
- ✅ 分布式架构
- ✅ 实时性
- ✅ 可扩展性
- ✅ 丰富的查询 DSL

**vs PostgreSQL 全文搜索**：

- PostgreSQL：简单搜索场景
- Elasticsearch：复杂搜索、大数据量

---

## 核心概念

**1. Index（索引）**

- 类似关系型数据库的数据库
- 存储相似类型的文档

**2. Document（文档）**

- JSON 格式的数据记录
- 类似关系型数据库的行

**3. Field（字段）**

- 文档中的键值对
- 类似关系型数据库的列

**4. Mapping（映射）**

- 定义文档结构和字段类型
- 类似关系型数据库的 Schema

**5. Analyzer（分析器）**

- 分词器
- 用于全文搜索

## 快速开始

### 1. Docker Compose 配置

```yaml
version: "3.8"

services:
  elasticsearch:
    image: elasticsearch:8.11.0
    container_name: elasticsearch_dev
    environment:
      - discovery.type=single-node
      - xpack.security.enabled=false
      - ES_JAVA_OPTS=-Xms512m -Xmx512m
    ports:
      - "9200:9200"
    volumes:
      - es_data:/usr/share/elasticsearch/data

  kibana:
    image: kibana:8.11.0
    container_name: kibana_dev
    environment:
      - ELASTICSEARCH_HOSTS=http://elasticsearch:9200
    ports:
      - "5601:5601"
    depends_on:
      - elasticsearch

volumes:
  es_data:
```

## 基本操作

### 1. 创建索引

```typescript
await client.indices.create({
  index: "products",
  body: {
    mappings: {
      properties: {
        name: { type: "text", analyzer: "ik_max_word" },
        description: { type: "text", analyzer: "ik_max_word" },
        price: { type: "double" },
        category: { type: "keyword" },
        tags: { type: "keyword" },
        createdAt: { type: "date" },
      },
    },
  },
});
```

---

### 2. 插入文档

```typescript
// 插入单个文档
await client.index({
  index: "products",
  id: "prod-1",
  body: {
    name: "Apple iPhone 15",
    description: "最新款 iPhone，搭载 A17 芯片",
    price: 7999,
    category: "手机",
    tags: ["Apple", "iPhone", "5G"],
    createdAt: new Date(),
  },
});

// 批量插入
const operations = products.flatMap((product) => [
  { index: { _index: "products", _id: product.id } },
  product,
]);
await client.bulk({ body: operations });
```

---

### 3. 查询文档

```typescript
// 根据 ID 查询
const result = await client.get({
  index: "products",
  id: "prod-1",
});
```

## 查询 DSL

### 1. 全文搜索

```typescript
const result = await client.search({
  index: "products",
  body: {
    query: {
      multi_match: {
        query: "iPhone",
        fields: ["name^2", "description"],
        fuzziness: "AUTO",
      },
    },
    highlight: {
      fields: {
        name: {},
        description: {},
      },
    },
  },
});
```

---

### 2. 布尔查询

```typescript
const result = await client.search({
  index: "products",
  body: {
    query: {
      bool: {
        must: [{ match: { name: "iPhone" } }],
        should: [{ term: { tags: "Apple" } }, { term: { category: "手机" } }],
        filter: [{ range: { price: { lte: 10000 } } }],
      },
    },
  },
});
```

---

### 3. 聚合查询

```typescript
const result = await client.search({
  index: "products",
  body: {
    size: 0,
    aggregations: {
      categories: {
        terms: { field: "category" },
      },
      priceRanges: {
        range: {
          field: "price",
          ranges: [{ to: 1000 }, { from: 1000, to: 5000 }, { from: 5000 }],
        },
      },
      stats: {
        stats: { field: "price" },
      },
    },
  },
});
```
