## Python 渐进式学习

欢迎来到 **Python 渐进式学习系列**！

本系列面向**有其他编程语言经验**的开发者，帮助你快速掌握 Python 核心特性，并使用 FastAPI 构建生产级 Web 应用。文章采用渐进式设计，从语言特性到工程实践，再到 Web 实战，每篇都包含丰富的代码示例。

---

## 技术栈

| 类别        | 技术                     |
| ----------- | ------------------------ |
| 语言        | Python 3.12+             |
| Web 框架    | FastAPI 0.110+           |
| ASGI 服务器 | Uvicorn                  |
| ORM         | SQLAlchemy 2.0 + Alembic |
| 数据验证    | Pydantic v2              |
| 数据库      | PostgreSQL 16            |
| 缓存        | Redis 7                  |
| 测试        | pytest + httpx           |
| 类型检查    | mypy                     |
| 包管理      | uv / poetry              |
| 容器化      | Docker + Docker Compose  |
