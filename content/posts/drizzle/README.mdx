## Drizzle ORM 类型安全的数据库访问实战

欢迎来到 **Drizzle ORM 实战系列**！

本系列将带你从零开始，深入掌握 Drizzle ORM，构建类型安全、高性能的数据库应用，特别适合 Edge Computing 环境。

---

## 系列概述

**技术栈**：

- **ORM**：Drizzle ORM
- **数据库**：PostgreSQL、SQLite（D1）
- **框架**：tRPC、Next.js、Nuxt
- **边缘平台**：Cloudflare Workers、Deno Deploy、Bun
- **缓存**：Cloudflare KV、Redis
- **存储**：Cloudflare R2

**核心优势**：

- 🚀 **极致性能**：比 Prisma 快 10x+
- 📦 **极小体积**：~50KB vs Prisma ~2MB
- 🔒 **类型安全**：100% TypeScript 类型覆盖
- ⚡ **Edge 完美**：Cloudflare Workers、Deno、Bun 完美支持
- 💪 **SQL 控制力**：类 SQL 语法，精细控制查询
