# MyBatis 渐进式学习计划

## 背景

MyBatis 是国内 Java 企业级开发中最常用的持久层框架，约 80% 的国内公司使用 MyBatis 作为数据访问层。相比 JPA，MyBatis 提供了更灵活的 SQL 控制，更适合复杂的业务场景和性能优化需求。

**为什么选择 MyBatis？**

- 🔥 **国内主流**：阿里巴巴、腾讯、百度等大厂首选
- 🎯 **SQL 灵活**：完全掌控 SQL，适合复杂查询
- 🚀 **性能优秀**：优秀的性能表现，易于优化
- 📚 **学习曲线平缓**：相比 Hibernate 更容易上手
- 💼 **就业必备**：国内企业招聘的必备技能

## 技术栈

- **MyBatis 3.5+**：最新稳定版本
- **MyBatis-Plus 3.5+**：MyBatis 增强工具
- **Spring Boot 3.x**：应用框架
- **MySQL 8.0+**：关系型数据库
- **Maven**：构建工具
- **PageHelper**：分页插件

## 学习路径

```
基础入门 → 映射进阶 → 开发方式 → 实战增强 → 项目实战
```

## 写作规范

### 代码示例要求

1. **完整可运行**：每个示例都是完整的代码
2. **Java + XML**：展示两种配置方式
3. **对比 JDBC**：展示 MyBatis 的优势
4. **Spring Boot 集成**：使用现代开发方式
5. **MySQL 数据库**：使用 MySQL 作为示例数据库

### 内容要求

- 通俗易懂，对比 JDBC 的繁琐
- 结合国内企业实际应用
- 提供完整可运行的示例
- 强调 MyBatis vs JPA 的选择

### 格式要求

- 使用 MDX 格式
- 代码块指定语言（`java、`xml、`yaml、`sql 等）
- 对比表格展示差异
- 实战场景驱动
