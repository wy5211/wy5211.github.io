## 前言

MyBatis 的核心功能是将数据库结果映射到 Java 对象。本文将深入讲解结果映射的各种方式。

---

## resultType：自动映射

### 基础用法

```xml
<!-- 列名和属性名一致时 -->
<select id="selectById" resultType="com.example.entity.User">
    SELECT id, username, email, age FROM users WHERE id = #{id}
</select>
```

### 驼峰命名转换

```xml
<!-- 开启驼峰转换 -->
<setting name="mapUnderscoreToCamelCase" value="true"/>

<!-- 数据库列名：user_name -->
<!-- Java 属性名：userName -->
<select id="selectById" resultType="User">
    SELECT user_name FROM users WHERE id = #{id}
</select>
```

---

## resultMap：手动映射

### 定义映射规则

```xml
<resultMap id="BaseResultMap" type="com.example.entity.User">
    <!-- 主键映射 -->
    <id column="id" property="id"/>

    <!-- 普通字段映射 -->
    <result column="username" property="username"/>
    <result column="email" property="email"/>
    <result column="age" property="age"/>
    <result column="create_time" property="createTime"/>
</resultMap>

<!-- 使用 resultMap -->
<select id="selectById" resultMap="BaseResultMap">
    SELECT * FROM users WHERE id = #{id}
</select>
```

---

## 复杂类型映射

### 嵌套对象

```java
public class User {
    private Long id;
    private String username;
    private Address address;  // 嵌套对象
}

public class Address {
    private String province;
    private String city;
}
```

```xml
<resultMap id="UserWithAddressMap" type="User">
    <id column="id" property="id"/>
    <result column="username" property="username"/>

    <!-- 嵌套对象映射 -->
    <association property="address" javaType="Address">
        <result column="province" property="province"/>
        <result column="city" property="city"/>
    </association>
</resultMap>
```

---

## 最佳实践

✅ 列名和属性名一致时用 resultType
✅ 复杂映射用 resultMap
✅ 开启驼峰命名转换

---

## 总结

✅ resultType 自动映射
✅ resultMap 手动映射
✅ 复杂类型处理
