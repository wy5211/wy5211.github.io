# 数组详解

## 前言

数组是 Java 中最基本的数据结构，用于存储固定大小的同类型元素。理解数组对于后续学习集合框架和算法至关重要。本文将全面介绍数组的使用方法和常见操作。

## 数组概述

### 什么是数组

数组是一个容器，可以存储多个相同类型的数据。数组中的每个元素都有一个索引（下标），从 0 开始。

```
数组示意图：
┌─────┬─────┬─────┬─────┬─────┐
│  10 │  20 │  30 │  40 │  50 │
└─────┴─────┴─────┴─────┴─────┘
   0     1     2     3     4
   ↑                           ↑
索引0                        索引4（长度-1）
```

**数组的特点**：

- 固定长度：创建后长度不可改变
- 同类型：所有元素必须是相同类型
- 连续内存：元素在内存中连续存储
- 索引访问：通过索引快速访问元素

```java
import java.util.Arrays;

public class ArraySorting {
    public static void main(String[] args) {
        // 冒泡排序
        int[] arr1 = {64, 34, 25, 12, 22, 11, 90};
        System.out.println("排序前：" + Arrays.toString(arr1));

        bubbleSort(arr1);
        System.out.println("冒泡排序：" + Arrays.toString(arr1));

        // 选择排序
        int[] arr2 = {64, 34, 25, 12, 22, 11, 90};
        selectionSort(arr2);
        System.out.println("选择排序：" + Arrays.toString(arr2));

        // 使用 Arrays.sort()
        int[] arr3 = {64, 34, 25, 12, 22, 11, 90};
        Arrays.sort(arr3);
        System.out.println("Arrays.sort：" + Arrays.toString(arr3));

        // 降序排序（使用包装类）
        Integer[] arr4 = {64, 34, 25, 12, 22, 11, 90};
        Arrays.sort(arr4, (a, b) -> b - a);  // Lambda 表达式
        System.out.println("降序排序：" + Arrays.toString(arr4));
    }

    // 冒泡排序
    public static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - 1 - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    // 交换
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }

    // 选择排序
    public static void selectionSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            // 交换
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }
}
```

### 数组查找

```java
import java.util.Arrays;

public class ArraySearch {
    public static void main(String[] args) {
        int[] numbers = {5, 3, 8, 2, 9, 1, 7};

        // 线性查找
        int target = 8;
        int index = linearSearch(numbers, target);
        if (index != -1) {
            System.out.println("线性查找：" + target + " 在索引 " + index);
        } else {
            System.out.println("线性查找：" + target + " 不存在");
        }

        // 二分查找（需要已排序的数组）
        int[] sorted = {1, 3, 5, 7, 9, 11, 13};
        target = 7;
        index = Arrays.binarySearch(sorted, target);
        System.out.println("二分查找：" + target + " 在索引 " + index);

        // 自定义二分查找
        index = binarySearch(sorted, target);
        System.out.println("自定义二分查找：" + target + " 在索引 " + index);

        // 查找最大值和最小值
        System.out.println("\n最大值：" + findMax(numbers));
        System.out.println("最小值：" + findMin(numbers));
    }

    // 线性查找
    public static int linearSearch(int[] arr, int target) {
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] == target) {
                return i;
            }
        }
        return -1;
    }

    // 二分查找
    public static int binarySearch(int[] arr, int target) {
        int left = 0;
        int right = arr.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (arr[mid] == target) {
                return mid;
            } else if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return -1;
    }

    // 查找最大值
    public static int findMax(int[] arr) {
        int max = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }
        return max;
    }

    // 查找最小值
    public static int findMin(int[] arr) {
        int min = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] < min) {
                min = arr[i];
            }
        }
        return min;
    }
}
```

```java
import java.util.Scanner;

/**
 * 统计数组中的正数、负数和零的个数
 */
public class ArrayStatistics {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 数组统计程序 ===");
        System.out.print("请输入数组大小：");
        int size = scanner.nextInt();

        int[] numbers = new int[size];
        System.out.println("请输入 " + size + " 个整数：");

        for (int i = 0; i < size; i++) {
            numbers[i] = scanner.nextInt();
        }

        // 统计
        int positiveCount = 0;
        int negativeCount = 0;
        int zeroCount = 0;
        int sum = 0;
        int max = numbers[0];
        int min = numbers[0];

        for (int num : numbers) {
            if (num > 0) {
                positiveCount++;
            } else if (num < 0) {
                negativeCount++;
            } else {
                zeroCount++;
            }

            sum += num;

            if (num > max) max = num;
            if (num < min) min = num;
        }

        double average = (double) sum / size;

        // 输出结果
        System.out.println("\n=== 统计结果 ===");
        System.out.println("数组：" + java.util.Arrays.toString(numbers));
        System.out.println("正数个数：" + positiveCount);
        System.out.println("负数个数：" + negativeCount);
        System.out.println("零的个数：" + zeroCount);
        System.out.println("总和：" + sum);
        System.out.println("平均值：" + average);
        System.out.println("最大值：" + max);
        System.out.println("最小值：" + min);

        scanner.close();
    }
}
```

### 示例 2：数组去重

```java
import java.util.Arrays;

/**
 * 删除数组中的重复元素
 */
public class ArrayDeduplication {

    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 2, 4, 5, 3, 6, 7, 5};

        System.out.println("原数组：" + Arrays.toString(numbers));

        // 方式一：去重后保持原顺序
        int[] unique1 = removeDuplicates1(numbers);
        System.out.println("去重（保持顺序）：" + Arrays.toString(unique1));

        // 方式二：排序后去重
        int[] unique2 = removeDuplicates2(numbers);
        System.out.println("去重（排序后）：" + Arrays.toString(unique2));
    }

    // 方式一：保持原顺序去重
    public static int[] removeDuplicates1(int[] arr) {
        int[] temp = new int[arr.length];
        int index = 0;

        for (int i = 0; i < arr.length; i++) {
            boolean isDuplicate = false;
            for (int j = 0; j < index; j++) {
                if (arr[i] == temp[j]) {
                    isDuplicate = true;
                    break;
                }
            }
            if (!isDuplicate) {
                temp[index++] = arr[i];
            }
        }

        return Arrays.copyOf(temp, index);
    }

    // 方式二：排序后去重
    public static int[] removeDuplicates2(int[] arr) {
        int[] sorted = arr.clone();
        Arrays.sort(sorted);

        int[] temp = new int[sorted.length];
        int index = 0;

        for (int i = 0; i < sorted.length; i++) {
            if (i == 0 || sorted[i] != sorted[i - 1]) {
                temp[index++] = sorted[i];
            }
        }

        return Arrays.copyOf(temp, index);
    }
}
```

```java
            }
        }
```

### 示例 4：数组旋转

```java
import java.util.Arrays;

/**
 * 将数组向右旋转 k 个位置
 */
public class ArrayRotation {

    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5, 6, 7};
        int k = 3;

        System.out.println("原数组：" + Arrays.toString(numbers));
        System.out.println("旋转 " + k + " 位");

        // 方式一：使用额外数组
        int[] rotated1 = rotate1(numbers.clone(), k);
        System.out.println("方式一：" + Arrays.toString(rotated1));

        // 方式二：逐个移动
        int[] rotated2 = rotate2(numbers.clone(), k);
        System.out.println("方式二：" + Arrays.toString(rotated2));

        // 方式三：反转法（最优）
        int[] rotated3 = rotate3(numbers.clone(), k);
        System.out.println("方式三：" + Arrays.toString(rotated3));
    }

    // 方式一：使用额外数组
    public static int[] rotate1(int[] nums, int k) {
        int n = nums.length;
        k = k % n;  // 处理 k 大于 n 的情况

        int[] rotated = new int[n];
        for (int i = 0; i < n; i++) {
            rotated[(i + k) % n] = nums[i];
        }

        return rotated;
    }

    // 方式二：逐个移动
    public static int[] rotate2(int[] nums, int k) {
        int n = nums.length;
        k = k % n;

        for (int i = 0; i < k; i++) {
            int temp = nums[n - 1];
            for (int j = n - 1; j > 0; j--) {
                nums[j] = nums[j - 1];
            }
            nums[0] = temp;
        }

        return nums;
    }

    // 方式三：反转法
    public static int[] rotate3(int[] nums, int k) {
        int n = nums.length;
        k = k % n;

        // 反转整个数组
        reverse(nums, 0, n - 1);
        // 反转前 k 个元素
        reverse(nums, 0, k - 1);
        // 反转剩余元素
        reverse(nums, k, n - 1);

        return nums;
    }

    // 反转数组的指定范围
    private static void reverse(int[] nums, int start, int end) {
        while (start < end) {
            int temp = nums[start];
            nums[start] = nums[end];
            nums[end] = temp;
            start++;
            end--;
        }
    }
}
```

### 示例 5：杨辉三角

```java
import java.util.Arrays;

/**
 * 打印杨辉三角
 */
public class PascalsTriangle {

    public static void main(String[] args) {
        int rows = 10;

        System.out.println("=== 杨辉三角（" + rows + "行）===");

        int[][] triangle = generatePascalsTriangle(rows);

        // 打印杨辉三角
        for (int i = 0; i < rows; i++) {
            // 打印前导空格
            for (int j = 0; j < rows - i - 1; j++) {
                System.out.print("  ");
            }

            // 打印数字
            for (int j = 0; j <= i; j++) {
                System.out.printf("%4d", triangle[i][j]);
            }
            System.out.println();
        }

        // 验证：每行数字之和等于 2^n
        System.out.println("\n验证（每行之和）:");
        for (int i = 0; i < rows; i++) {
            int sum = 0;
            for (int num : triangle[i]) {
                sum += num;
            }
            System.out.printf("第%d行: %d = 2^%d%n", i + 1, sum, i);
        }
    }

    // 生成杨辉三角
    public static int[][] generatePascalsTriangle(int rows) {
        int[][] triangle = new int[rows][];

        for (int i = 0; i < rows; i++) {
            triangle[i] = new int[i + 1];
            triangle[i][0] = 1;  // 每行第一个元素为1
            triangle[i][i] = 1;  // 每行最后一个元素为1

            // 计算中间元素
            for (int j = 1; j < i; j++) {
                triangle[i][j] = triangle[i - 1][j - 1] + triangle[i - 1][j];
            }
        }

        return triangle;
    }
}
```

## 常见问题

### Q1：数组越界异常如何避免？

**答**：访问数组前检查索引范围。

```java
if (index >= 0 && index < array.length) {
    value = array[index];
}
```

### Q2：如何动态扩展数组？

**答**：数组长度固定，需要创建新数组并复制元素。

```java
int[] oldArray = {1, 2, 3};
int[] newArray = Arrays.copyOf(oldArray, oldArray.length * 2);
```

或者使用 `ArrayList`（后续章节介绍）。

### Q3：数组初始化有什么区别？

```java
int[] arr1 = new int[5];      // 默认值为 0
int[] arr2 = new int[]{1,2,3}; // 初始化为指定值
int[] arr3 = {1, 2, 3};        // 简写形式
```

### Q4：如何比较两个数组是否相等？

**答**：使用 `Arrays.equals()` 而不是 `==`。

```java
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};

System.out.println(arr1 == arr2);              // false（比较引用）
System.out.println(Arrays.equals(arr1, arr2)); // true（比较内容）
```

### Q5：二维数组必须每行长度相同吗？

**答**：不一定。Java 支持不规则二维数组（锯齿数组）。

```java
int[][] jagged = {
    {1, 2},
    {3, 4, 5},
    {6}
};
```
