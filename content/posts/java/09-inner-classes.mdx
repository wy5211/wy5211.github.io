# 内部类与枚举

## 内部类概述

Java 支持在一个类内部定义另一个类，这称为内部类。

```
内部类分类：
┌─────────────────────┐
│   成员内部类         │
│   静态内部类         │
│   局部内部类         │
│   匿名内部类         │
└─────────────────────┘
```

## 成员内部类

```java
public class OuterClass {

    private int outerVar = 10;

    // 成员内部类
    public class InnerClass {

        private int innerVar = 20;

        public void display() {
            // 访问外部类成员
            System.out.println("外部类变量：" + outerVar);
            System.out.println("内部类变量：" + innerVar);
        }
    }

    public void useInner() {
        InnerClass inner = new InnerClass();
        inner.display();
    }
}
```

## 静态内部类

```java
public class OuterClass {

    private static int staticVar = 10;
    private int instanceVar = 20;

    // 静态内部类
    public static class StaticInnerClass {

        public void display() {
            // 只能访问外部类的静态成员
            System.out.println("静态变量：" + staticVar);
            // System.out.println(instanceVar); // 编译错误
        }
    }
}
```

## 局部内部类

```java
public class OuterClass {

    public void method() {
        // 局部内部类（在方法内定义）
        class LocalInner {

            public void display() {
                System.out.println("局部内部类");
            }
        }

        LocalInner local = new LocalInner();
        local.display();
    }
}
```

## 匿名内部类

```java
public interface OnClickListener {

    void onClick();
}

public class Button {

    public void setOnClickListener(OnClickListener listener) {
        listener.onClick();
    }

    public static void main(String[] args) {
        Button button = new Button();

        // 匿名内部类
        button.setOnClickListener(new OnClickListener() {
            @Override
            public void onClick() {
                System.out.println("按钮被点击");
            }
        });
    }
}
```

## 枚举

### 枚举定义

```java
public enum DayOfWeek {

    MONDAY, TUESDAY, WEDNESDAY, THURSDAY,
    FRIDAY, SATURDAY, SUNDAY
}
```

### 枚举使用

```java
public class EnumDemo {

    public static void main(String[] args) {
        // 获取枚举值
        DayOfWeek today = DayOfWeek.MONDAY;

        // 枚举比较
        if (today == DayOfWeek.MONDAY) {
            System.out.println("今天是星期一");
        }

        // 遍历所有枚举值
        for (DayOfWeek day : DayOfWeek.values()) {
            System.out.println(day);
        }

        // 获取枚举名称和序号
        System.out.println("名称：" + today.name());
        System.out.println("序号：" + today.ordinal());
    }
}
```

### 枚举高级用法

```java
public enum Level {

    LOW(1) {
        @Override
        public String getDescription() {
            return "低级别";
        }
    },
    MEDIUM(2) {
        @Override
        public String getDescription() {
            return "中级别";
        }
    },
    HIGH(3) {
        @Override
        public String getDescription() {
            return "高级别";
        }
    };

    private int value;

    Level(int value) {
        this.value = value;
    }

    public int getValue() {
        return value;
    }

    public abstract String getDescription();
}
```

## 实战演练

### 示例 1：迭代器

```java
public class MyContainer {

    private Object[] items;
    private int size;

    public MyContainer(int capacity) {
        items = new Object[capacity];
        size = 0;
    }

    public void add(Object item) {
        if (size < items.length) {
            items[size++] = item;
        }
    }

    // 成员内部类：迭代器
    public class Iterator {

        private int current = 0;

        public boolean hasNext() {
            return current < size;
        }

        public Object next() {
            return items[current++];
        }
    }

    public Iterator iterator() {
        return new Iterator();
    }

    public static void main(String[] args) {
        MyContainer container = new MyContainer(5);
        container.add("A");
        container.add("B");
        container.add("C");

        Iterator iterator = container.iterator();
        while (iterator.hasNext()) {
            System.out.println(iterator.next());
        }
    }
}
```

### 示例 2：状态机

```java
public enum State {

    READY {
        @Override
        public State next() {
            return RUNNING;
        }
    },
    RUNNING {
        @Override
        public State next() {
            return PAUSED;
        }
    },
    PAUSED {
        @Override
        public State next() {
            return RUNNING;
        }
    },
    STOPPED {
        @Override
        public State next() {
            return STOPPED;
        }
    };

    public abstract State next();
}

public class StateMachine {

    private State state;

    public StateMachine() {
        this.state = State.READY;
    }

    public void transition() {
        state = state.next();
        System.out.println("当前状态：" + state);
    }

    public static void main(String[] args) {
        StateMachine machine = new StateMachine();
        machine.transition();
        machine.transition();
    }
}
```

## 最佳实践

### 1. 优先使用枚举替代常量

```java
// ❌ 不好的做法
public static final int STATUS_PENDING = 1;
public static final int STATUS_APPROVED = 2;

// ✅ 好的做法
public enum Status {
    PENDING, APPROVED
}
```

### 2. 合理使用内部类

```java
// 成员内部类：需要访问外部类实例
// 静态内部类：不需要访问外部类实例
// 局部内部类：仅在方法内使用
// 匿名内部类：一次性使用
```

## 常见问题

### Q1：内部类可以访问外部类的私有成员吗？

**答**：可以，内部类可以访问外部类的所有成员。

### Q2：枚举可以是抽象的吗？

**答**：可以，抽象枚举可以包含抽象方法。

### Q3：枚举实现接口？

**答**：可以，枚举可以实现接口。
