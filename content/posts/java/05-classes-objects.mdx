# 类与对象基础

## 前言

从本章开始，我们将进入 Java 最核心的部分——面向对象编程（OOP）。面向对象是一种编程范式，它将现实世界的事物抽象为"对象"，通过对象之间的交互来构建程序。理解类与对象，是掌握 Java 的关键。

## 面向对象编程概述

### 什么是面向对象

面向对象编程（Object-Oriented Programming，OOP）是一种编程范式，它使用"对象"来设计软件。

**面向过程 vs 面向对象**：

```java
// 面向过程：关注怎么做
// 洗衣服的过程：1.放水 2.放衣服 3.放洗衣粉 4.搓洗 5.漂洗 6.晾晒

// 面向对象：关注谁来做
// 洗衣机对象.洗衣服()
```

### OOP 的三大特征

1. **封装（Encapsulation）**：隐藏实现细节，暴露必要接口
2. **继承（Inheritance）**：子类继承父类的属性和方法
3. **多态（Polymorphism）**：同一方法在不同对象上有不同行为

## 类与对象

### 什么是类

类是对象的模板或蓝图，定义了对象的属性和行为。

```
类 → 对象的关系

类（图纸）          对象（房子）
┌─────────┐         ┌─────┐ ┌─────┐ ┌─────┐
│  房子   │    →    │ 房子 │ │ 房子 │ │ 房子 │
│  设计图 │         │  A  │ │  B  │ │  C  │
└─────────┘         └─────┘ └─────┘ └─────┘
```

### 什么是对象

对象是类的实例化，是实际存在的实体。

```java
// 类：手机（概念）
// 对象：你的 iPhone（具体）

// 类：学生（概念）
// 对象：张三（具体）
```

### 成员变量 vs 局部变量

```java
public class VariableComparison {

    // 成员变量（在类中，方法外）
    int instanceVar = 10;      // 实例变量
    static int staticVar = 20; // 静态变量（类变量）

    public void method() {
        // 局部变量（在方法中）
        int localVar = 30;

        System.out.println("实例变量：" + instanceVar);
        System.out.println("静态变量：" + staticVar);
        System.out.println("局部变量：" + localVar);
    }

    public static void main(String[] args) {
        VariableComparison obj = new VariableComparison();
        obj.method();

        // 实例变量通过对象访问
        System.out.println("通过对象访问实例变量：" + obj.instanceVar);

        // 静态变量通过类名访问（推荐）
        System.out.println("通过类名访问静态变量：" + VariableComparison.staticVar);
    }
}
```

**区别对比**：

| 特性     | 成员变量       | 局部变量             |
| -------- | -------------- | -------------------- |
| 位置     | 类中，方法外   | 方法中或代码块中     |
| 作用域   | 整个类         | 方法或代码块内       |
| 生命周期 | 与对象共存     | 方法执行期间         |
| 默认值   | 有默认值       | 无默认值，必须初始化 |
| 修饰符   | 可用访问修饰符 | 不能用访问修饰符     |

### 方法重载（Overload）

同一个类中，方法名相同但参数列表不同。

```java
public class MethodOverloading {

    // 重载方法：参数个数不同
    public int add(int a, int b) {
        return a + b;
    }

    public int add(int a, int b, int c) {
        return a + b + c;
    }

    // 重载方法：参数类型不同
    public double add(double a, double b) {
        return a + b;
    }

    // 重载方法：参数顺序不同
    public void print(String name, int age) {
        System.out.println("姓名：" + name + "，年龄：" + age);
    }

    public void print(int age, String name) {
        System.out.println("年龄：" + age + "，姓名：" + name);
    }

    public static void main(String[] args) {
        MethodOverloading obj = new MethodOverloading();

        System.out.println(obj.add(1, 2));        // 调用两个参数的版本
        System.out.println(obj.add(1, 2, 3));     // 调用三个参数的版本
        System.out.println(obj.add(1.5, 2.5));    // 调用 double 版本

        obj.print("张三", 20);
        obj.print(20, "李四");
    }
}
```

### 可变参数方法

```java
public class VarargsExample {

    // 可变参数方法（参数个数可变）
    public int sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {
            total += num;
        }
        return total;
    }

    // 可变参数必须放在最后
    public void display(String prefix, int... numbers) {
        System.out.print(prefix + ": ");
        for (int num : numbers) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        VarargsExample obj = new VarargsExample();

        System.out.println(obj.sum());                    // 0
        System.out.println(obj.sum(1));                   // 1
        System.out.println(obj.sum(1, 2));                // 3
        System.out.println(obj.sum(1, 2, 3, 4, 5));       // 15

        obj.display("数字", 1, 2, 3);
    }
}
```

## 构造方法

### 构造方法的特点

- 方法名与类名相同
- 没有返回值类型（连 void 都没有）
- 用于初始化对象
- 创建对象时自动调用

## this 关键字

`this` 关键字引用当前对象。

```java
public class ThisKeyword {

    private String name;
    private int age;

    public ThisKeyword() {
        // 调用其他构造方法（必须是第一条语句）
        this("默认姓名", 0);
    }

    public ThisKeyword(String name, int age) {
        // this.name 指向成员变量，name 指向参数
        this.name = name;
        this.age = age;
    }

    // this 返回当前对象
    public ThisKeyword setName(String name) {
        this.name = name;
        return this;  // 链式调用
    }

    public ThisKeyword setAge(int age) {
        this.age = age;
        return this;
    }

    public void display() {
        // this 可省略
        System.out.println("姓名：" + this.name + "，年龄：" + this.age);
    }

    // this 作为参数传递
    public void compareWith(ThisKeyword other) {
        if (this.age > other.age) {
            System.out.println(this.name + " 比 " + other.name + " 大");
        } else {
            System.out.println(this.name + " 比 " + other.name + " 小");
        }
    }

    public static void main(String[] args) {
        ThisKeyword obj1 = new ThisKeyword("张三", 25);
        ThisKeyword obj2 = new ThisKeyword("李四", 30);

        obj1.display();
        obj2.display();

        obj1.compareWith(obj2);

        // 链式调用
        obj1.setName("王五").setAge(35).display();
    }
}
```

## 实战演练

### 示例 1：银行账户类

```java
/**
 * 银行账户类
 */
public class BankAccount {

    private String accountNumber;  // 账号
    private String accountHolder;  // 账户持有人
    private double balance;        // 余额

    // 构造方法
    public BankAccount(String accountNumber, String accountHolder, double initialBalance) {
        this.accountNumber = accountNumber;
        this.accountHolder = accountHolder;
        this.balance = initialBalance;
        System.out.println("账户创建成功：" + accountNumber);
    }

    // 存款
    public void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
            System.out.printf("存入 %.2f 元，当前余额：%.2f 元%n", amount, balance);
        } else {
            System.out.println("存款金额必须大于0");
        }
    }

    // 取款
    public void withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            System.out.printf("取出 %.2f 元，当前余额：%.2f 元%n", amount, balance);
        } else if (amount > balance) {
            System.out.println("余额不足！");
        } else {
            System.out.println("取款金额必须大于0");
        }
    }

    // 查询余额
    public double getBalance() {
        return balance;
    }

    // 转账
    public void transfer(BankAccount targetAccount, double amount) {
        if (amount > 0 && amount <= balance) {
            this.balance -= amount;
            targetAccount.balance += amount;
            System.out.printf("转账 %.2f 元到账户 %s 成功%n", amount, targetAccount.accountNumber);
            System.out.printf("当前余额：%.2f 元%n", this.balance);
        } else if (amount > balance) {
            System.out.println("余额不足，转账失败");
        } else {
            System.out.println("转账金额必须大于0");
        }
    }

    // 显示账户信息
    public void displayInfo() {
        System.out.println("=== 账户信息 ===");
        System.out.println("账号：" + accountNumber);
        System.out.println("持有人：" + accountHolder);
        System.out.printf("余额：%.2f 元%n", balance);
    }

    public static void main(String[] args) {
        // 创建两个账户
        BankAccount account1 = new BankAccount("622200001", "张三", 1000.0);
        BankAccount account2 = new BankAccount("622200002", "李四", 500.0);

        System.out.println();

        // 存款操作
        account1.deposit(500.0);

        // 取款操作
        account1.withdraw(200.0);

        // 转账操作
        account1.transfer(account2, 300.0);

        System.out.println();

        // 显示账户信息
        account1.displayInfo();
        System.out.println();
        account2.displayInfo();
    }
}
```

### 示例 2：矩形计算器

```java
/**
 * 矩形类
 */
public class RectangleCalculator {

    private double length;  // 长
    private double width;   // 宽

    // 无参构造
    public RectangleCalculator() {
        this(1.0, 1.0);
    }

    // 有参构造
    public RectangleCalculator(double length, double width) {
        this.length = length;
        this.width = width;
    }

    // Getter 和 Setter 方法
    public double getLength() {
        return length;
    }

    public void setLength(double length) {
        if (length > 0) {
            this.length = length;
        } else {
            System.out.println("长度必须大于0");
        }
    }

    public double getWidth() {
        return width;
    }

    public void setWidth(double width) {
        if (width > 0) {
            this.width = width;
        } else {
            System.out.println("宽度必须大于0");
        }
    }

    // 计算面积
    public double calculateArea() {
        return length * width;
    }

    // 计算周长
    public double calculatePerimeter() {
        return 2 * (length + width);
    }

    // 判断是否为正方形
    public boolean isSquare() {
        return length == width;
    }

    // 比较面积
    public int compareArea(RectangleCalculator other) {
        double thisArea = this.calculateArea();
        double otherArea = other.calculateArea();

        if (thisArea > otherArea) {
            return 1;
        } else if (thisArea < otherArea) {
            return -1;
        } else {
            return 0;
        }
    }

    // 显示信息
    public void displayInfo() {
        System.out.println("矩形信息：");
        System.out.println("长度：" + length);
        System.out.println("宽度：" + width);
        System.out.println("面积：" + calculateArea());
        System.out.println("周长：" + calculatePerimeter());
        System.out.println("是否为正方形：" + (isSquare() ? "是" : "否"));
    }

    public static void main(String[] args) {
        // 创建矩形对象
        RectangleCalculator rect1 = new RectangleCalculator(5, 3);
        RectangleCalculator rect2 = new RectangleCalculator(4, 4);

        // 显示信息
        rect1.displayInfo();
        System.out.println();
        rect2.displayInfo();

        // 比较面积
        int result = rect1.compareArea(rect2);
        if (result > 0) {
            System.out.println("\n矩形1的面积大于矩形2");
        } else if (result < 0) {
            System.out.println("\n矩形1的面积小于矩形2");
        } else {
            System.out.println("\n两个矩形面积相等");
        }
    }
}
```

```java
/**
 * 图书管理系统

public class LibrarySystem {

    private Book[] books;
    private int bookCount;

    public LibrarySystem(int capacity) {
        books = new Book[capacity];
        bookCount = 0;
    }

    // 添加图书
    public void addBook(Book book) {
        if (bookCount < books.length) {
            books[bookCount++] = book;
            System.out.println("添加图书成功：" + book.getTitle());
        } else {
            System.out.println("图书库已满！");
        }
    }

    // 查找图书
    public Book findBook(String title) {
        for (int i = 0; i < bookCount; i++) {
            if (books[i].getTitle().equals(title)) {
                return books[i];
            }
        }
        return null;
    }

    // 显示所有图书
    public void displayAllBooks() {
        System.out.println("=== 图书列表 ===");
        for (int i = 0; i < bookCount; i++) {
            System.out.println("\n图书 " + (i + 1) + "：");
            books[i].displayInfo();
        }
    }

    public static void main(String[] args) {
        LibrarySystem library = new LibrarySystem(10);

        // 添加图书
        library.addBook(new Book("978-7-111", "Java编程思想", "Bruce Eckel", 108.00, 5));
        library.addBook(new Book("978-7-121", "Effective Java", "Joshua Bloch", 89.00, 3));

        // 显示所有图书
        library.displayAllBooks();

        System.out.println("\n=== 借书操作 ===");
        Book book = library.findBook("Java编程思想");
        if (book != null) {
            book.borrow();
            book.displayInfo();
        }

        System.out.println("\n=== 还书操作 ===");
        book.returnBook();
        book.displayInfo();
    }
}
```

## 最佳实践

### 1. 类名使用大驼峰命名

```java
// ✅ 好的做法
public class UserService {
}

public class BankAccount {
}

// ❌ 不好的做法
public class userService {
}

public class bank_account {
}
```

### 2. 一个类一个文件（public 类）

```java
// ✅ 好的做法：每个 public 类单独一个文件
// 文件名：Student.java
public class Student {
}

// 文件名：Teacher.java
public class Teacher {
}
```

### 3. 封装字段，提供访问方法

```java
// ✅ 好的做法
public class Person {
    private String name;  // 私有字段

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

// ❌ 不好的做法：直接暴露字段
public class Person {
    public String name;  // 公共字段
}
```

### 4. 方法名使用小驼峰命名

```java
// ✅ 好的做法
public void calculateArea() {
}

public void getUserName() {
}

// ❌ 不好的做法
public void CalculateArea() {
}

public void GetUserName() {
}
```

## 常见问题

### Q1：类和对象有什么区别？

**答**：

- **类**：模板/蓝图，抽象的概念
- **对象**：类的实例，具体的实体

```java
// 类：汽车设计图
// 对象：具体的某辆车
```

### Q2：构造方法和普通方法有什么区别？

**答**：

- 构造方法：方法名与类名相同，无返回值类型，创建对象时自动调用
- 普通方法：有返回值类型，需要显式调用

### Q3：this 关键字有什么用？

**答**：

- 区分成员变量和局部变量
- 调用其他构造方法
- 返回当前对象
- 作为参数传递当前对象

### Q4：什么时候使用静态方法？

**答**：

- 不需要访问对象状态
- 工具类方法
- 常用计算方法

```java
public static int max(int a, int b) {
    return a > b ? a : b;
}
```

### Q5：对象数组如何初始化？

```java
// 创建对象数组
Student[] students = new Student[3];

// 必须逐个初始化
students[0] = new Student();
students[1] = new Student();
students[2] = new Student();

// 或者使用初始化块
Student[] students = {
    new Student(),
    new Student(),
    new Student()
};
```
