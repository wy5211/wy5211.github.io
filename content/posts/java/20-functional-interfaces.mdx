# 函数式接口

## 前言

函数式接口是 Lambda 表达式的基础。Java 提供了丰富的内置函数式接口，涵盖了常见的使用场景。本文将详细介绍这些接口及其应用。

## 函数式接口概述

### 什么是函数式接口

只包含一个抽象方法的接口称为函数式接口。

```java
@FunctionalInterface
interface MyFunction {
    int apply(int x);
}
```

### @FunctionalInterface 注解

- 可选的，但建议使用
- 编译器检查是否只有一个抽象方法
- 可以包含默认方法和静态方法

```java
@FunctionalInterface
interface Calculator {
    int calculate(int a, int b);

    // 默认方法不影响函数式接口
    default void print() {
        System.out.println("计算器");
    }

    // 静态方法也不影响
    static int add(int a, int b) {
        return a + b;
    }
}
```

## 核心函数式接口

### Predicate 泛型接口

断言型接口，接受参数返回布尔值。

```java
import java.util.function.*;

public class PredicateDemo {

    public static void main(String[] args) {
        // 判断是否为偶数
        Predicate<Integer> isEven = n -> n % 2 == 0;
        System.out.println("4 是偶数？" + isEven.test(4));  // true
        System.out.println("3 是偶数？" + isEven.test(3));  // false

        // 判断字符串是否为空
        Predicate<String> isEmpty = String::isEmpty;
        System.out.println("空字符串？" + isEmpty.test(""));  // true

        // 组合条件
        Predicate<Integer> isPositive = n -> n > 0;
        Predicate<Integer> isLessThan10 = n -> n < 10;

        // and：两个条件都满足
        Predicate<Integer> condition = isPositive.and(isLessThan10);
        System.out.println("5 符合条件？" + condition.test(5));  // true

        // or：任一条件满足
        Predicate<Integer> condition2 = isPositive.or(isLessThan10);

        // negate：取反
        Predicate<Integer> condition3 = isPositive.negate();
    }
}
```

### Function 函数型接口

函数型接口，接受一个类型参数，返回另一个类型结果。

```java
import java.util.function.*;

public class FunctionDemo {

    public static void main(String[] args) {
        // 字符串转长度
        Function<String, Integer> length = String::length;
        System.out.println("长度：" + length.apply("Hello"));  // 5

        // 字符串转大写
        Function<String, String> toUpper = String::toUpperCase;
        System.out.println("大写：" + toUpper.apply("hello"));  // HELLO

        // 组合函数
        Function<String, Integer> f1 = String::length;
        Function<Integer, Integer> f2 = x -> x * 2;

        // compose：先执行参数函数
        Function<String, Integer> composed = f2.compose(f1);
        System.out.println("组合结果：" + composed.apply("Hello"));  // 10

        // andThen：先执行当前函数
        Function<String, Integer> andThen = f1.andThen(f2);
        System.out.println("AndThen 结果：" + andThen.apply("Hello"));  // 10

        // 恒等函数
        Function<String, String> identity = Function.identity();
        System.out.println("恒等：" + identity.apply("Test"));  // Test
    }
}
```

### Consumer 消费型接口

消费型接口，接受参数无返回值。

```java
import java.util.function.*;

public class ConsumerDemo {

    public static void main(String[] args) {
        // 打印字符串
        Consumer<String> printer = System.out::println;
        printer.accept("Hello");  // Hello

        // 列表操作
        Consumer<List<Integer>> addFirst = list -> list.add(0);
        Consumer<List<Integer>> addLast = list -> list.add(99);

        // andThen：链式操作
        Consumer<List<Integer>> operations = addFirst.andThen(addLast);

        List<Integer> numbers = new ArrayList<>();
        operations.accept(numbers);
        System.out.println(numbers);  // [0, 99]

        // Map 操作
        Consumer<Map<String, Integer>> printMap = map -> {
            map.forEach((k, v) -> System.out.println(k + "=" + v));
        };
    }
}
```

### Supplier 供应型接口

供应型接口，无参数返回结果。

```java
import java.util.function.*;
import java.util.*;

public class SupplierDemo {

    public static void main(String[] args) {
        // 生成随机数
        Supplier<Double> random = Math::random;
        System.out.println("随机数：" + random.get());

        // 创建对象
        Supplier<List<String>> listSupplier = ArrayList::new;
        List<String> list = listSupplier.get();

        // 延迟计算
        Supplier<String> lazyMessage = () -> {
            System.out.println("执行复杂计算");
            return "结果";
        };

        // 只有调用 get 才执行
        String result = lazyMessage.get();
    }
}
```

### UnaryOperator 一元运算接口

一元运算接口，接受和返回相同类型。

```java
import java.util.function.*;

public class UnaryOperatorDemo {

    public static void main(String[] args) {
        // 平方
        UnaryOperator<Integer> square = x -> x * x;
        System.out.println("平方：" + square.apply(5));  // 25

        // 字符串反转
        UnaryOperator<String> reverse = s -> new StringBuilder(s).reverse().toString();
        System.out.println("反转：" + reverse.apply("Hello"));  // olleH

        // 用于 List.replaceAll
        List<String> words = new ArrayList<>(Arrays.asList("apple", "banana"));
        words.replaceAll(String::toUpperCase);
        System.out.println(words);  // [APPLE, BANANA]
    }
}
```

### BinaryOperator 二元运算接口

二元运算接口，接受两个相同类型参数，返回相同类型结果。

```java
import java.util.function.*;

public class BinaryOperatorDemo {

    public static void main(String[] args) {
        // 加法
        BinaryOperator<Integer> add = Integer::sum;
        System.out.println("相加：" + add.apply(3, 5));  // 8

        // 最大值
        BinaryOperator<Integer> max = BinaryOperator.maxBy(Integer::compareTo);
        System.out.println("最大值：" + max.apply(3, 5));  // 5

        // 最小值
        BinaryOperator<Integer> min = BinaryOperator.minBy(Integer::compareTo);
        System.out.println("最小值：" + min.apply(3, 5));  // 3

        // 用于 Stream.reduce
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
        int sum = numbers.stream()
            .reduce(0, Integer::sum);
        System.out.println("总和：" + sum);  // 15
    }
}
```

## 原始类型特化

为了避免装箱/拆箱开销，Java 提供了原始类型的函数式接口。

```java
import java.util.function.*;

public class PrimitiveSpecialization {

    public static void main(String[] args) {
        // IntPredicate
        IntPredicate isEven = n -> n % 2 == 0;
        System.out.println("4 是偶数？" + isEven.test(4));

        // IntFunction
        IntFunction<String> intToString = n -> "数字：" + n;
        System.out.println(intToString.apply(10));

        // IntConsumer
        IntConsumer printer = n -> System.out.println("数字：" + n);
        printer.accept(5);

        // IntSupplier
        IntSupplier supplier = () -> 42;
        System.out.println("供应：" + supplier.getAsInt());

        // IntUnaryOperator
        IntUnaryOperator increment = n -> n + 1;
        System.out.println("递增：" + increment.applyAsInt(5));

        // IntBinaryOperator
        IntBinaryOperator add = (a, b) -> a + b;
        System.out.println("相加：" + add.applyAsInt(3, 5));
    }
}
```

## 自定义函数式接口

```java
// 三参数函数式接口示例
@FunctionalInterface
interface TriFunction<T, U, V, R> {
    R apply(T t, U u, V v);
}

public class CustomFunctionalInterface {

    public static void main(String[] args) {
        // 三参数函数
        TriFunction<Integer, Integer, Integer, Integer> sum =
            (a, b, c) -> a + b + c;

        System.out.println("三数和：" + sum.apply(1, 2, 3));  // 6

        // 字符串连接
        TriFunction<String, String, String, String> join =
            (a, b, c) -> a + b + c;

        System.out.println("连接：" + join.apply("Hello", " ", "World"));
    }
}
```

## 实战应用

### 示例 1：策略模式

```java
import java.util.function.*;
import java.util.*;

public class StrategyPattern {

    // 验证策略
    public static class Validator {

        private List<Predicate<String>> rules = new ArrayList<>();

        public void addRule(Predicate<String> rule) {
            rules.add(rule);
        }

        public boolean validate(String input) {
            return rules.stream().allMatch(rule -> rule.test(input));
        }
    }

    public static void main(String[] args) {
        Validator passwordValidator = new Validator();

        // 添加验证规则
        passwordValidator.addRule(s -> s.length() >= 8);
        passwordValidator.addRule(s -> s.matches(".*[A-Z].*"));  // 包含大写
        passwordValidator.addRule(s -> s.matches(".*[0-9].*"));  // 包含数字

        String password = "Password123";
        boolean valid = passwordValidator.validate(password);
        System.out.println("密码有效？" + valid);
    }
}
```

### 示例 2：函数组合

```java
import java.util.function.*;

public class FunctionComposition {

    public static void main(String[] args) {
        // 定义函数
        Function<Integer, Integer> multiply = x -> x * 2;
        Function<Integer, Integer> add = x -> x + 3;
        Function<Integer, Integer> square = x -> x * x;

        // 组合函数：(x + 3) * 2
        Function<Integer, Integer> combined = add.andThen(multiply);
        System.out.println("组合结果：" + combined.apply(5));  // (5+3)*2 = 16

        // 复杂组合：((x * 2) + 3)^2
        Function<Integer, Integer> complex =
            multiply.andThen(add).andThen(square);
        System.out.println("复杂组合：" + complex.apply(2));  // ((2*2)+3)^2 = 49
    }
}
```

### 示例 3：缓存装饰器

```java
import java.util.function.*;
import java.util.*;

public class Memoization {

    public static <T, R> Function<T, R> memoize(Function<T, R> function) {
        Map<T, R> cache = new HashMap<>();

        return input -> cache.computeIfAbsent(input, function);
    }

    public static void main(String[] args) {
        // 计算斐波那契数列（带缓存）
        Function<Integer, Long> fib = Memoization.memoize(n -> {
            if (n <= 1) {
                return (long) n;
            }
            return fib.apply(n - 1) + fib.apply(n - 2);
        });

        System.out.println("Fibonacci(10)：" + fib.apply(10));  // 55
        System.out.println("Fibonacci(20)：" + fib.apply(20));  // 6765
    }
}
```

## Lambda 与 Stream 结合

```java
import java.util.*;
import java.util.function.*;
import java.util.stream.*;

public class LambdaWithStream {

    public static void main(String[] args) {
        List<String> words = Arrays.asList("apple", "banana", "orange", "grape");

        // 使用 Predicate 过滤
        Predicate<String> isLong = s -> s.length() > 5;
        List<String> longWords = words.stream()
            .filter(isLong)
            .collect(Collectors.toList());

        // 使用 Function 转换
        Function<String, String> toUpper = String::toUpperCase;
        List<String> upperWords = words.stream()
            .map(toUpper)
            .collect(Collectors.toList());

        // 使用 Consumer 处理
        Consumer<String> printer = System.out::println;
        words.forEach(printer);

        // 使用 BinaryOperator 归约
        BinaryOperator<Integer> add = Integer::sum;
        int sum = Arrays.asList(1, 2, 3, 4, 5).stream()
            .reduce(0, add);
    }
}
```

## 最佳实践

### 1. 使用标准函数式接口

```java
// ✅ 好的做法：使用标准接口
Function<String, Integer> parser = Integer::parseInt;

// ❌ 不好的做法：重复造轮子
@FunctionalInterface
interface StringToInt {
    int apply(String s);
}
```

### 2. 方法引用优先

```java
// ✅ 好的做法
list.forEach(System.out::println);

// ✅ 也是可以的
list.forEach(s -> System.out.println(s));
```

### 3. 保持 Lambda 简洁

```java
// ✅ 好的做法：简洁明了
list.filter(item -> item.isValid())

// ❌ 不好的做法：过于复杂
list.filter(item -> {
    // 很多行代码
    // 应该提取为方法
})
```

## 常见问题

### Q1：函数式接口可以有多个方法吗？

**答**：可以有多个默认方法和静态方法，但只能有一个抽象方法。

### Q2：什么时候自定义函数式接口？

**答**：标准接口无法满足需求时，如需要多个参数或特殊返回类型。

### Q3：如何选择函数式接口？

**答**：

- 需要布尔值 → Predicate
- 需要转换 → Function
- 需要消费 → Consumer
- 需要供应 → Supplier
- 需要运算 → UnaryOperator/BinaryOperator
