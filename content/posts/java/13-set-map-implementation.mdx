# Set 与 Map 实现

## 前言

Set 用于存储不重复的元素，Map 用于存储键值对。它们是基于哈希表和红黑树实现的高效数据结构。理解它们的工作原理对于编写高效代码至关重要。

## Set 实现

### HashSet

基于哈希表实现，提供 O(1) 的添加、删除和查找操作。

```java
import java.util.*;

public class HashSetExample {

    public static void main(String[] args) {
        Set<String> set = new HashSet<>();

        // 添加元素（自动去重）
        set.add("Java");
        set.add("Python");
        set.add("Java");  // 重复，不会被添加

        System.out.println("大小：" + set.size());  // 2

        // 判断包含
        System.out.println("包含 Java？" + set.contains("Java"));

        // 删除元素
        set.remove("Python");

        // 遍历（无序）
        for (String item : set) {
            System.out.println(item);
        }
    }
}
```

### LinkedHashSet

继承 HashSet，使用链表维护插入顺序。

```java
import java.util.*;

public class LinkedHashSetExample {

    public static void main(String[] args) {
        Set<String> set = new LinkedHashSet<>();

        set.add("C");
        set.add("A");
        set.add("B");

        // 保持插入顺序遍历
        for (String item : set) {
            System.out.println(item);  // C, A, B
        }
    }
}
```

### TreeSet

基于红黑树实现，元素自动排序。

```java
import java.util.*;

public class TreeSetExample {

    public static void main(String[] args) {
        Set<Integer> set = new TreeSet<>();

        set.add(5);
        set.add(2);
        set.add(8);
        set.add(1);

        // 自动排序
        for (Integer num : set) {
            System.out.println(num);  // 1, 2, 5, 8
        }

        // TreeSet 特有方法
        TreeSet<Integer> treeSet = (TreeSet<Integer>) set;
        System.out.println("第一个：" + treeSet.first());
        System.out.println("最后一个：" + treeSet.last());
        System.out.println("小于 5 的子集：" + treeSet.headSet(5));
    }
}
```

## Map 实现

### HashMap

基于哈希表实现，存储键值对，键不能重复。

```java
import java.util.*;

public class HashMapExample {

    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();

        // 添加键值对
        map.put("张三", 25);
        map.put("李四", 30);
        map.put("王五", 28);

        // 获取值
        Integer age = map.get("张三");
        System.out.println("张三的年龄：" + age);

        // 判断包含
        System.out.println("包含李四？" + map.containsKey("李四"));

        // 删除
        map.remove("王五");

        // 遍历方式一：entrySet
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey() + "：" + entry.getValue());
        }

        // 遍历方式二：keySet
        for (String key : map.keySet()) {
            System.out.println(key + "：" + map.get(key));
        }

        // 遍历方式三：values
        for (Integer value : map.values()) {
            System.out.println("年龄：" + value);
        }

        // getOrDefault（Java 8+）
        int unknownAge = map.getOrDefault("赵六", 0);
        System.out.println("赵六的年龄：" + unknownAge);

        // putIfAbsent（Java 8+）
        map.putIfAbsent("赵六", 35);
    }
}
```

### LinkedHashMap

继承 HashMap，使用链表维护插入顺序或访问顺序。

```java
import java.util.*;

public class LinkedHashMapExample {

    public static void main(String[] args) {
        // 按访问顺序排序
        Map<String, Integer> map = new LinkedHashMap<>(16, 0.75f, true);

        map.put("A", 1);
        map.put("B", 2);
        map.put("C", 3);

        // 访问某个键
        map.get("A");

        // 遍历：A 会被放到最后
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey());  // B, C, A
        }
    }
}
```

### TreeMap

基于红黑树实现，键自动排序。

```java
import java.util.*;

public class TreeMapExample {

    public static void main(String[] args) {
        Map<String, Integer> map = new TreeMap<>();

        map.put("C", 3);
        map.put("A", 1);
        map.put("B", 2);

        // 自动按键排序
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey() + "：" + entry.getValue());
        }
        // A：1, B：2, C：3

        // TreeMap 特有方法
        TreeMap<String, Integer> treeMap = (TreeMap<String, Integer>) map;
        System.out.println("第一个键：" + treeMap.firstKey());
        System.out.println("最后一个键：" + treeMap.lastKey());
    }
}
```

## 实战演练

### 示例 1：词频统计

```java
import java.util.*;

public class WordFrequency {

    public static void main(String[] args) {
        String text = "Java is great and Java is fun";

        // 统计词频
        Map<String, Integer> frequency = new HashMap<>();

        String[] words = text.split("\\s+");
        for (String word : words) {
            frequency.put(word, frequency.getOrDefault(word, 0) + 1);
        }

        // 按频率排序
        List<Map.Entry<String, Integer>> list = new ArrayList<>(frequency.entrySet());
        list.sort((e1, e2) -> e2.getValue().compareTo(e1.getValue()));

        // 输出结果
        for (Map.Entry<String, Integer> entry : list) {
            System.out.println(entry.getKey() + "：" + entry.getValue());
        }
    }
}
```

### 示例 2：缓存实现（LRU）

```java
import java.util.*;

public class LRUCache<K, V> extends LinkedHashMap<K, V> {

    private final int maxSize;

    public LRUCache(int maxSize) {
        super(16, 0.75f, true);  // accessOrder = true
        this.maxSize = maxSize;
    }

    @Override
    protected boolean removeEldestEntry(Map.Entry<K, V> eldest) {
        return size() > maxSize;
    }

    public static void main(String[] args) {
        LRUCache<String, Integer> cache = new LRUCache<>(3);

        cache.put("A", 1);
        cache.put("B", 2);
        cache.put("C", 3);
        cache.get("A");  // 访问 A
        cache.put("D", 4);  // B 被淘汰（最少使用）

        System.out.println(cache);  // {A=1, C=3, D=4}
    }
}
```

### 示例 3：去重操作

```java
import java.util.*;

public class RemoveDuplicates {

    public static void main(String[] args) {
        List<Integer> list = Arrays.asList(1, 2, 3, 2, 4, 1, 5);

        // 方法一：使用 Set 去重
        Set<Integer> set = new HashSet<>(list);
        List<Integer> unique1 = new ArrayList<>(set);
        System.out.println("去重后：" + unique1);

        // 方法二：使用 LinkedHashSet 保持顺序
        Set<Integer> linkedSet = new LinkedHashSet<>(list);
        List<Integer> unique2 = new ArrayList<>(linkedSet);
        System.out.println("保持顺序去重：" + unique2);
    }
}
```

## 最佳实践

### 1. 选择合适的 Map

```java
// 无序、快速查找 → HashMap
// 保持插入顺序 → LinkedHashMap
// 需要排序 → TreeMap
```

### 2. 遍历 Map 的效率

```java
// ✅ 最有效：entrySet
for (Map.Entry<K, V> entry : map.entrySet()) {
    K key = entry.getKey();
    V value = entry.getValue();
}

// ❌ 效率较低：keySet + get
for (K key : map.keySet()) {
    V value = map.get(key);
}
```

### 3. 初始化容量

```java
// ✅ 好的做法
Map<String, String> map = new HashMap<>(1000);
```

## 常见问题

### Q1：HashSet 如何判断元素重复？

**答**：先比较 hashCode()，相同再用 equals() 比较。

### Q2：HashMap 的扩容机制？

**答**：当元素数量 > 容量 × 负载因子（默认 0.75）时扩容。

### Q3：TreeSet 和 HashSet 的选择？

**答**：需要排序用 TreeSet，否则用 HashSet（更快）。
