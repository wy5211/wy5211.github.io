# Lambda 表达式

## 前言

Lambda 表达式是 Java 8 引入的重要特性，它允许将函数作为方法参数，使代码更加简洁。Lambda 是函数式编程的基础，也是现代 Java 开发的必备技能。

## 为什么需要 Lambda

### 传统方式的问题

```java
// 使用匿名内部类
button.addActionListener(new ActionListener() {
    @Override
    public void actionPerformed(ActionEvent e) {
        System.out.println("按钮被点击");
    }
});
```

### Lambda 的优势

```java
// 使用 Lambda
button.addActionListener(e -> System.out.println("按钮被点击"));
```

## Lambda 语法

### 基本语法

```
(参数列表) -> {方法体}
```

### Lambda 组成部分

1. **参数列表**：类似方法的参数列表
2. **箭头符号**：`->`
3. **方法体**：可以是表达式或代码块

### Lambda 简化形式

```java
// 完整形式
(int x, int y) -> {
    return x + y;
}

// 省略参数类型
(x, y) -> {
    return x + y;
}

// 单参数省略括号
x -> x * x

// 单条语句省略大括号和 return
(x, y) -> x + y

// 无参数
() -> System.out.println("Hello")
```

## 函数式接口

### 什么是函数式接口

只包含一个抽象方法的接口称为函数式接口。

```java
@FunctionalInterface
interface Calculator {
    int calculate(int a, int b);
}
```

### 常见函数式接口

```java
// Predicate<T>：断言，返回 boolean
Predicate<Integer> isEven = n -> n % 2 == 0;
boolean result = isEven.test(4);  // true

// Function<T, R>：函数，接受 T 返回 R
Function<String, Integer> length = s -> s.length();
int len = length.apply("Hello");  // 5

// Consumer<T>：消费者，接受参数无返回值
Consumer<String> printer = s -> System.out.println(s);
printer.accept("Hello");  // Hello

// Supplier<T>：供应者，无参数返回 T
Supplier<Double> random = () -> Math.random();
double value = random.get();

// UnaryOperator<T>：一元运算，接受和返回相同类型
UnaryOperator<Integer> square = x -> x * x;
int result = square.apply(5);  // 25

// BinaryOperator<T>：二元运算
BinaryOperator<Integer> add = (a, b) -> a + b;
int sum = add.apply(3, 5);  // 8
```

## Lambda 使用示例

### 集合操作

```java
import java.util.*;

public class CollectionLambda {

    public static void main(String[] args) {
        List<String> names = Arrays.asList("张三", "李四", "王五", "赵六");

        // 遍历
        names.forEach(name -> System.out.println(name));

        // 排序
        names.sort((a, b) -> a.compareTo(b));
        names.sort(Comparator.naturalOrder());

        // 过滤
        names.stream()
            .filter(name -> name.length() == 2)
            .forEach(System.out::println);

        // 映射
        names.stream()
            .map(String::toUpperCase)
            .forEach(System.out::println);
    }
}
```

### 替代匿名内部类

```java
// 线程
Runnable task = () -> System.out.println("线程执行");

// 比较器
Comparator<String> cmp = (s1, s2) -> s1.length() - s2.length();

// 监听器
button.addActionListener(e -> handleEvent(e));
```

## 方法引用

### 方法引用类型

1. **静态方法引用**：`ClassName::staticMethod`
2. **实例方法引用**：`instance::method`
3. **对象方法引用**：`ClassName::method`
4. **构造器引用**：`ClassName::new`

### 方法引用示例

```java
import java.util.*;

public class MethodReference {

    public static void main(String[] args) {
        List<String> names = Arrays.asList("apple", "banana", "orange");

        // 静态方法引用
        names.forEach(System.out::println);

        // 实例方法引用
        names.stream()
            .map(String::toUpperCase)
            .forEach(System.out::println);

        // 构造器引用
        Supplier<List<String>> listSupplier = ArrayList::new;
        List<String> list = listSupplier.get();

        // 数组构造器引用
        Function<Integer, int[]> arrayCreator = int[]::new;
        int[] array = arrayCreator.apply(10);
    }
}
```

## Lambda 与闭包

### Lambda 访问外部变量

```java
public class LambdaClosure {

    public static void main(String[] args) {
        int factor = 10;  // effectively final

        // Lambda 可以访问外部变量
        Function<Integer, Integer> multiplier = x -> x * factor;

        // factor = 20;  // 编译错误：factor 必须是 final 或 effectively final

        System.out.println(multiplier.apply(5));  // 50
    }
}
```

### this 关键字

```java
public class LambdaThis {

    private int value = 10;

    public void test() {
        Runnable runnable = () -> {
            // Lambda 中的 this 指向外部对象
            System.out.println(this.value);  // 10
        };

        runnable.run();
    }
}
```

## 实战演练

### 示例 1：学生排序

```java
import java.util.*;
import java.util.stream.*;

class Student {
    private String name;
    private int age;
    private double score;

    public Student(String name, int age, double score) {
        this.name = name;
        this.age = age;
        this.score = score;
    }

    public String getName() { return name; }
    public int getAge() { return age; }
    public double getScore() { return score; }

    @Override
    public String toString() {
        return name + "(" + age + ", " + score + ")";
    }
}

public class StudentSort {

    public static void main(String[] args) {
        List<Student> students = Arrays.asList(
            new Student("张三", 20, 85.5),
            new Student("李四", 21, 92.0),
            new Student("王五", 22, 78.5),
            new Student("赵六", 20, 88.0)
        );

        // 按成绩排序
        students.sort(Comparator.comparingDouble(Student::getScore).reversed());
        System.out.println("按成绩排序：" + students);

        // 按年龄分组
        Map<Integer, List<Student>> grouped = students.stream()
            .collect(Collectors.groupingBy(Student::getAge));
        System.out.println("按年龄分组：" + grouped);

        // 过滤并转换
        List<String> names = students.stream()
            .filter(s -> s.getScore() >= 80)
            .map(Student::getName)
            .collect(Collectors.toList());
        System.out.println("优秀学生：" + names);
    }
}
```

### 示例 2：事件处理

```java
import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class EventHandler {

    public static void main(String[] args) {
        JButton button = new JButton("点击我");

        // Lambda 替代匿名内部类
        button.addActionListener(e -> {
            System.out.println("按钮被点击");
        });

        // 使用方法引用
        button.addActionListener(EventHandler::handleClick);
    }

    private static void handleClick(ActionEvent e) {
        System.out.println("处理点击事件");
    }
}
```

### 示例 3：延迟计算

```java
import java.util.function.*;

public class LazyEvaluation {

    public static void main(String[] args) {
        // 延迟计算
        Supplier<Double> lazyValue = () -> {
            System.out.println("执行复杂计算");
            return Math.sqrt(Math.random() * 10000);
        };

        // 只有调用 get 时才执行
        double value = lazyValue.get();
        System.out.println("结果：" + value);
    }
}
```

## Lambda 最佳实践

### 1. 保持简洁

```java
// ✅ 好的做法：简洁明了
list.forEach(item -> System.out.println(item));

// ❌ 不好的做法：过于复杂
list.forEach(item -> {
    // 很多行代码
    // 应该提取为方法
});
```

### 2. 使用方法引用

```java
// ✅ 好的做法
list.forEach(System.out::println);

// ✅ 也是可以的
list.forEach(item -> System.out.println(item));
```

### 3. 避免副作用

```java
// ❌ 不好的做法：有副作用
List<Integer> numbers = Arrays.asList(1, 2, 3);
numbers.stream()
    .map(n -> n * 2)
    .forEach(n -> numbers.add(n));  // 不要修改源数据

// ✅ 好的做法：无副作用
List<Integer> doubled = numbers.stream()
    .map(n -> n * 2)
    .collect(Collectors.toList());
```

## 常见问题

### Q1：Lambda 和匿名内部类的区别？

**答**：Lambda 更简洁，但作用域不同，this 指向外部对象。

### Q2：何时使用方法引用？

**答**：Lambda 体只是调用一个方法时，可以使用方法引用。

### Q3：effectively final 是什么？

**答**：变量虽然没有声明为 final，但实际上没有被修改。
