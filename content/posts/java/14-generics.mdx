# 泛型深入理解

## 前言

泛型是 Java 5 引入的重要特性，它允许在编译时进行类型检查，避免运行时类型转换错误。理解泛型对于编写安全、可复用的代码至关重要。

## 为什么需要泛型

### 不使用泛型的问题

```java
// 不使用泛型（Java 5 之前）
List list = new ArrayList();
list.add("Hello");
list.add(123);  // 可以添加不同类型

// 需要类型转换
String str = (String) list.get(0);  // 可能抛出 ClassCastException
```

### 使用泛型的优势

```java
// 使用泛型
List<String> list = new ArrayList<>();
list.add("Hello");
// list.add(123);  // 编译错误，类型安全

String str = list.get(0);  // 不需要类型转换
```

## 泛型类

```java
// 泛型类定义
public class Box<T> {

    private T content;

    public void set(T content) {
        this.content = content;
    }

    public T get() {
        return content;
    }

    public static void main(String[] args) {
        // 创建 String 类型的 Box
        Box<String> stringBox = new Box<>();
        stringBox.set("Hello");
        String str = stringBox.get();

        // 创建 Integer 类型的 Box
        Box<Integer> integerBox = new Box<>();
        integerBox.set(123);
        Integer num = integerBox.get();
    }
}
```

## 多个类型参数

```java
public class Pair<K, V> {

    private K key;
    private V value;

    public Pair(K key, V value) {
        this.key = key;
        this.value = value;
    }

    public K getKey() {
        return key;
    }

    public V getValue() {
        return value;
    }

    public static void main(String[] args) {
        Pair<String, Integer> pair = new Pair<>("年龄", 25);
        System.out.println(pair.getKey() + "：" + pair.getValue());
    }
}
```

## 泛型接口

```java
// 泛型接口
public interface Generator<T> {

    T next();
}

// 实现泛型接口
public class NumberGenerator implements Generator<Integer> {

    private int current = 0;

    @Override
    public Integer next() {
        return current++;
    }

    public static void main(String[] args) {
        Generator<Integer> generator = new NumberGenerator();
        for (int i = 0; i < 5; i++) {
            System.out.println(generator.next());
        }
    }
}
```

## 泛型方法

```java
public class GenericMethods {

    // 泛型方法
    public static <T> void printArray(T[] array) {
        for (T item : array) {
            System.out.println(item);
        }
    }

    // 多个类型参数的泛型方法
    public static <K, V> void printPair(K key, V value) {
        System.out.println(key + " = " + value);
    }

    // 有界类型参数
    public static <T extends Comparable<T>> T max(T x, T y, T z) {
        T max = x;
        if (y.compareTo(max) > 0) {
            max = y;
        }
        if (z.compareTo(max) > 0) {
            max = z;
        }
        return max;
    }

    public static void main(String[] args) {
        // 使用泛型方法
        String[] strings = {"A", "B", "C"};
        printArray(strings);

        Integer[] integers = {1, 2, 3};
        printArray(integers);

        printPair("姓名", "张三");

        System.out.println("最大值：" + max(3, 5, 2));
    }
}
```

## 类型通配符

### 无界通配符 `?`

```java
import java.util.*;

public class WildcardDemo {

    // 使用无界通配符
    public static void printList(List<?> list) {
        for (Object item : list) {
            System.out.println(item);
        }
    }

    public static void main(String[] args) {
        List<String> strings = Arrays.asList("A", "B", "C");
        List<Integer> integers = Arrays.asList(1, 2, 3);

        printList(strings);
        printList(integers);
    }
}
```

### 上界通配符 `? extends`

```java
import java.util.*;

public class UpperBoundDemo {

    // 上界通配符：可以接受 Number 及其子类
    public static double sum(List<? extends Number> list) {
        double sum = 0;
        for (Number num : list) {
            sum += num.doubleValue();
        }
        return sum;
    }

    public static void main(String[] args) {
        List<Integer> integers = Arrays.asList(1, 2, 3);
        List<Double> doubles = Arrays.asList(1.5, 2.5, 3.5);

        System.out.println("整数和：" + sum(integers));
        System.out.println("浮点数和：" + sum(doubles));
    }
}
```

### 下界通配符 `? super`

```java
import java.util.*;

public class LowerBoundDemo {

    // 下界通配符：可以接受 Integer 及其父类
    public static void addNumbers(List<? super Integer> list) {
        list.add(10);
        list.add(20);
    }

    public static void main(String[] args) {
        List<Number> numbers = new ArrayList<>();
        addNumbers(numbers);

        List<Object> objects = new ArrayList<>();
        addNumbers(objects);

        System.out.println(numbers);
    }
}
```

## 类型擦除

Java 的泛型在运行时会被擦除，称为类型擦除。

```java
import java.util.*;

public class TypeErasure {

    public static void main(String[] args) {
        List<String> stringList = new ArrayList<>();
        List<Integer> integerList = new ArrayList<>();

        // 运行时类型相同
        System.out.println(stringList.getClass() == integerList.getClass());
        // 输出：true

        // 不能使用泛型创建数组
        // List<String>[] array = new List<String>[10];  // 编译错误
        List<?>[] array = new List<?>[10];  // 可以使用通配符
    }
}
```

## 泛型的限制

1. **不能实例化类型参数**

   ```java
   // T item = new T();  // 编译错误
   ```

2. **不能创建泛型数组**

   ```java
   // T[] array = new T[10];  // 编译错误
   ```

3. **不能使用基本类型**

   ```java
   // List<int> list = new ArrayList<>();  // 编译错误
   List<Integer> list = new ArrayList<>();  // 使用包装类
   ```

4. **不能使用 instanceof**
   ```java
   // if (obj instanceof T) { }  // 编译错误
   ```

## 实战演练

### 示例 1：泛型栈

```java
public class GenericStack<T> {

    private List<T> stack = new ArrayList<>();

    public void push(T item) {
        stack.add(item);
    }

    public T pop() {
        if (stack.isEmpty()) {
            throw new EmptyStackException();
        }
        return stack.remove(stack.size() - 1);
    }

    public T peek() {
        if (stack.isEmpty()) {
            throw new EmptyStackException();
        }
        return stack.get(stack.size() - 1);
    }

    public boolean isEmpty() {
        return stack.isEmpty();
    }

    public static void main(String[] args) {
        GenericStack<String> stringStack = new GenericStack<>();
        stringStack.push("A");
        stringStack.push("B");

        GenericStack<Integer> intStack = new GenericStack<>();
        intStack.push(1);
        intStack.push(2);
    }
}
```

### 示例 2：泛型工具类

```java
import java.util.*;

public class CollectionUtils {

    // 交换列表中两个元素的位置
    public static <T> void swap(List<T> list, int i, int j) {
        T temp = list.get(i);
        list.set(i, list.get(j));
        list.set(j, temp);
    }

    // 查找最大元素
    public static <T extends Comparable<T>> T max(Collection<T> collection) {
        if (collection.isEmpty()) {
            throw new IllegalArgumentException("集合为空");
        }

        Iterator<T> iterator = collection.iterator();
        T max = iterator.next();

        while (iterator.hasNext()) {
            T current = iterator.next();
            if (current.compareTo(max) > 0) {
                max = current;
            }
        }

        return max;
    }

    // 复制集合
    public static <T> List<T> copy(Collection<T> collection) {
        return new ArrayList<>(collection);
    }

    public static void main(String[] args) {
        List<Integer> list = Arrays.asList(3, 1, 4, 1, 5);
        System.out.println("最大值：" + max(list));

        List<String> strings = new ArrayList<>(Arrays.asList("A", "B", "C"));
        swap(strings, 0, 2);
        System.out.println("交换后：" + strings);
    }
}
```

## 最佳实践

### 1. 泛型命名

```java
// T - Type（类型）
// E - Element（元素）
// K - Key（键）
// V - Value（值）
// N - Number（数字）
// S, U, V - 第二、第三、第四个类型
```

### 2. PECS 原则

```java
// Producer Extends, Consumer Super

// 生产者：使用 ? extends
public void process(List<? extends Number> list) {
    Number num = list.get(0);  // 可以读取
    // list.add(10);  // 不能写入
}

// 消费者：使用 ? super
public void fill(List<? super Integer> list) {
    list.add(10);  // 可以写入
    // Integer num = list.get(0);  // 读取类型是 Object
}
```

### 3. 避免原始类型

```java
// ❌ 不好的做法
List list = new ArrayList();

// ✅ 好的做法
List<String> list = new ArrayList<>();
```

## 常见问题

### Q1：泛型和数组的区别？

**答**：泛型是编译时检查，数组是运行时检查。

### Q2：为什么不能创建泛型数组？

**答**：因为类型擦除，运行时无法确定泛型类型。

### Q3：`<T>` 和 `<?>` 的区别？

**答**：`<T>` 是具体类型参数，`<?>` 是通配符。
