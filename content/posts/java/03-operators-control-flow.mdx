# 运算符与控制流

## 前言

程序的核心在于逻辑控制。运算符让我们能够进行各种计算和比较，而控制流语句则让程序能够根据条件执行不同的代码路径。掌握这些基础知识，你就能写出真正有用的程序了。

## 运算符

Java 运算符按照功能可以分为几类：算术运算符、关系运算符、逻辑运算符、位运算符、赋值运算符和三元运算符。

### 算术运算符

用于进行基本的数学运算。

| 运算符 | 描述         | 示例     | 结果          |
| ------ | ------------ | -------- | ------------- |
| `+`    | 加法         | `5 + 3`  | 8             |
| `-`    | 减法         | `5 - 3`  | 2             |
| `*`    | 乘法         | `5 \* 3` | 15            |
| `/`    | 除法         | `5 / 2`  | 2（整数除法） |
| `%`    | 取模（余数） | `5 % 2`  | 1             |
| `++`   | 自增         | a++      | `a = a + 1`   |
| `--`   | 自减         | a--      | `a = a - 1`   |

```java
public class ArithmeticOperators {
    public static void main(String[] args) {
        int a = 10, b = 3;

        System.out.println("=== 基本算术运算 ===");
        System.out.println(a + " + " + b + " = " + (a + b));  // 13
        System.out.println(a + " - " + b + " = " + (a - b));  // 7
        System.out.println(a + " * " + b + " = " + (a * b));  // 30
        System.out.println(a + " / " + b + " = " + (a / b));  // 3（整数除法）
        System.out.println(a + " % " + b + " = " + (a % b));  // 1

        System.out.println("\n=== 浮点数除法 ===");
        System.out.println("10.0 / 3 = " + (10.0 / b));       // 3.333...
        System.out.println("10 / 3.0 = " + (a / 3.0));        // 3.333...

        System.out.println("\n=== 自增自减 ===");
        int x = 5;
        System.out.println("x++: " + (x++));  // 先使用后自增，输出 5
        System.out.println("x: " + x);         // x 变为 6

        int y = 5;
        System.out.println("++y: " + (++y));  // 先自增后使用，输出 6
        System.out.println("y: " + y);         // y 为 6

        System.out.println("\n=== 自减运算 ===");
        int m = 10;
        System.out.println("m--: " + (m--));  // 输出 10，m 变为 9
        System.out.println("--m: " + (--m));  // m 变为 8，输出 8
    }
}
```

### 关系运算符

用于比较两个值，结果为布尔值（true 或 false）。

| 运算符 | 描述     | 示例             |
| ------ | -------- | ---------------- |
| `==`   | 等于     | `5 == 5` → true  |
| `!=`   | 不等于   | `5 != 3` → true  |
| `>`    | 大于     | `5 > 3` → true   |
| `<`    | 小于     | `5 < 3` → false  |
| `>=`   | 大于等于 | `5 >= 5` → true  |
| `<=`   | 小于等于 | `5 <= 3` → false |

```java
public class RelationalOperators {
    public static void main(String[] args) {
        int a = 5, b = 10, c = 5;

        System.out.println("a = " + a + ", b = " + b + ", c = " + c);
        System.out.println("a == b: " + (a == b));  // false
        System.out.println("a == c: " + (a == c));  // true
        System.out.println("a != b: " + (a != b));  // true
        System.out.println("a > b: " + (a > b));    // false
        System.out.println("a < b: " + (a < b));    // true
        System.out.println("a >= c: " + (a >= c));  // true
        System.out.println("b <= a: " + (b <= a));  // false

        // 字符比较（比较 Unicode 值）
        char ch1 = 'A', ch2 = 'B';
        System.out.println("\n'A' < 'B': " + (ch1 < ch2));  // true

        // 注意：基本类型用 ==，引用类型用 equals()
        String s1 = "Hello";
        String s2 = new String("Hello");
        System.out.println("\ns1 == s2: " + (s1 == s2));          // false
        System.out.println("s1.equals(s2): " + s1.equals(s2));  // true
    }
}
```

### 逻辑运算符

用于组合多个条件，操作数必须是布尔值。

| 运算符 | 描述             | 规则                      | 示例                     |
| ------ | ---------------- | ------------------------- | ------------------------ |
| `&&`   | 逻辑与（短路与） | 两者都为 true 才返回 true | `true && false` → false  |
| `\|\|` | 逻辑或（短路或） | 任一为 true 就返回 true   | `true \|\| false` → true |
| `!`    | 逻辑非           | 取反                      | `!true` → false          |
| `&`    | 逻辑与（不短路） | 两者都为 true 才返回 true | `true & false` → false   |
| `\|`   | 逻辑或（不短路） | 任一为 true 就返回 true   | `true \| false` → true   |
| `^`    | 逻辑异或         | 相同为 false，不同为 true | `true ^ false` → true    |

```java
public class LogicalOperators {
    public static void main(String[] args) {
        boolean a = true, b = false;

        System.out.println("=== 基本逻辑运算 ===");
        System.out.println("a && b: " + (a && b));   // false
        System.out.println("a || b: " + (a || b));   // true
        System.out.println("!a: " + (!a));           // false
        System.out.println("a ^ b: " + (a ^ b));     // true

        System.out.println("\n=== 短路运算演示 ===");
        // &&：如果第一个为 false，第二个不会执行
        int x = 5, y = 10;
        boolean result1 = (x > y) && (++x > 0);
        System.out.println("(x > y) && (++x > 0): " + result1);
        System.out.println("x 的值: " + x);  // x 仍为 5（++x 未执行）

        // ||：如果第一个为 true，第二个不会执行
        x = 5;
        boolean result2 = (x < y) || (++x > 0);
        System.out.println("(x < y) || (++x > 0): " + result2);
        System.out.println("x 的值: " + x);  // x 仍为 5（++x 未执行）

        System.out.println("\n=== 不短路的运算 ===");
        x = 5;
        boolean result3 = (x > y) & (++x > 0);
        System.out.println("x 的值: " + x);  // x 变为 6（++x 执行了）
    }
}
```

### 位运算符

直接对整数的二进制位进行操作。

| 运算符 | 描述           | 示例                    |
| ------ | -------------- | ----------------------- |
| `&`    | 按位与         | `5 & 3 = 1`             |
| `\|`   | 按位或         | `5 \| 3 = 7`            |
| `^`    | 按位异或       | `5 ^ 3 = 6`             |
| `~`    | 按位取反       | `~5 = -6`               |
| `<<`   | 左移           | `5 << 2 = 20`           |
| `>>`   | 右移（带符号） | `5 >> 1 = 2`            |
| `>>>`  | 右移（无符号） | `-5 >>> 1 = 2147483645` |

```java
public class BitwiseOperators {
    public static void main(String[] args) {
        int a = 5;  // 二进制：0101
        int b = 3;  // 二进制：0011

        System.out.println("a = " + a + " (二进制: " + Integer.toBinaryString(a) + ")");
        System.out.println("b = " + b + " (二进制: " + Integer.toBinaryString(b) + ")");

        System.out.println("\n=== 位运算结果 ===");
        System.out.println("a & b = " + (a & b) + " (二进制: " + Integer.toBinaryString(a & b) + ")");
        // 0101 & 0011 = 0001

        System.out.println("a | b = " + (a | b) + " (二进制: " + Integer.toBinaryString(a | b) + ")");
        // 0101 | 0011 = 0111

        System.out.println("a ^ b = " + (a ^ b) + " (二进制: " + Integer.toBinaryString(a ^ b) + ")");
        // 0101 ^ 0011 = 0110

        System.out.println("~a = " + (~a));  // 取反

        System.out.println("\n=== 移位运算 ===");
        System.out.println("a << 2 = " + (a << 2));   // 左移2位 = 5 * 4 = 20
        System.out.println("a >> 1 = " + (a >> 1));   // 右移1位 = 5 / 2 = 2
        System.out.println("-5 >> 1 = " + (-5 >> 1)); // -3（带符号）
        System.out.println("-5 >>> 1 = " + (-5 >>> 1)); // 2147483645（无符号）

        // 实际应用：判断奇偶数
        System.out.println("\n=== 应用：判断奇偶 ===");
        int num = 7;
        System.out.println(num + " 是" + ((num & 1) == 0 ? "偶数" : "奇数"));
    }
}
```

### 赋值运算符

用于给变量赋值。

| 运算符 | 描述           | 等价于                 |
| ------ | -------------- | ---------------------- |
| `=`    | 赋值           | `a = 5`                |
| `+=`   | 加后赋值       | `a += 5 → a = a + 5`   |
| `-=`   | 减后赋值       | `a -= 5 → a = a - 5`   |
| `*=`   | 乘后赋值       | `a *= 5 → a = a * 5`   |
| `/=`   | 除后赋值       | `a /= 5 → a = a / 5`   |
| `%=`   | 取模后赋值     | `a %= 5 → a = a % 5`   |
| `&=`   | 按位与后赋值   | `a &= 5 → a = a & 5`   |
| `\|=`  | 按位或后赋值   | `a \|= 5 → a = a \| 5` |
| `^=`   | 按位异或后赋值 | `a ^= 5 → a = a ^ 5`   |
| `<<=`  | 左移后赋值     | `a <<= 5 → a = a << 5` |
| `>>=`  | 右移后赋值     | `a >>= 5 → a = a >> 5` |

```java
public class AssignmentOperators {
    public static void main(String[] args) {
        int a = 10;

        System.out.println("初始值: a = " + a);

        a += 5;  // a = a + 5
        System.out.println("a += 5: " + a);  // 15

        a -= 3;  // a = a - 3
        System.out.println("a -= 3: " + a);  // 12

        a *= 2;  // a = a * 2
        System.out.println("a *= 2: " + a);  // 24

        a /= 4;  // a = a / 4
        System.out.println("a /= 4: " + a);  // 6

        a %= 4;  // a = a % 4
        System.out.println("a %= 4: " + a);  // 2

        // 字符串拼接
        String message = "Hello";
        message += " World";
        System.out.println("\n字符串拼接: " + message);  // Hello World
    }
}
```

### 三元运算符

唯一的三元运算符，常用于简化 if-else 语句。

**语法**：`条件表达式 ? 表达式1 : 表达式2`

```java
public class TernaryOperator {
    public static void main(String[] args) {
        int a = 10, b = 20;

        // 基本使用
        int max = (a > b) ? a : b;
        System.out.println("最大值: " + max);  // 20

        // 嵌套使用
        int x = 15, y = 25, z = 10;
        int maxOfThree = (x > y) ?
                         ((x > z) ? x : z) :
                         ((y > z) ? y : z);
        System.out.println("三个数的最大值: " + maxOfThree);  // 25

        // 实际应用
        int score = 85;
        String grade = (score >= 90) ? "A" :
                      (score >= 80) ? "B" :
                      (score >= 70) ? "C" :
                      (score >= 60) ? "D" : "F";
        System.out.println("成绩 " + score + " 对应等级: " + grade);  // B

        // 与 if-else 对比
        int num = 5;
        String result;

        // 三元运算符（简洁）
        result = (num % 2 == 0) ? "偶数" : "奇数";

        // if-else（更清晰）
        if (num % 2 == 0) {
            result = "偶数";
        } else {
            result = "奇数";
        }
        System.out.println(num + " 是" + result);
    }
}
```

```

```

## 条件语句

### if-else 语句

最基本的条件判断语句。

```java
public class IfElseStatement {
    public static void main(String[] args) {
        int score = 75;

        // 基本形式
        if (score >= 60) {
            System.out.println("及格了！");
        }

        // if-else 形式
        if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }

        // if-else if-else 形式
        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");
        } else if (score >= 70) {
            System.out.println("中等");
        } else if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }

        // 省略大括号（仅单行语句时，不推荐）
        if (score >= 60)
            System.out.println("及格");

        // 嵌套 if
        int age = 20;
        boolean hasTicket = true;

        if (age >= 18) {
            if (hasTicket) {
                System.out.println("可以入场");
            } else {
                System.out.println("需要购票");
            }
        } else {
            System.out.println("未成年禁止入场");
        }
    }
}
```

#### Java 14+ 新 switch 语法

```java
public class ModernSwitch {
    public static void main(String[] args) {
        // 新语法：使用箭头和表达式
        int dayOfWeek = 3;
        String dayName = switch (dayOfWeek) {
            case 1 -> "星期一";
            case 2 -> "星期二";
            case 3 -> "星期三";
            case 4 -> "星期四";
            case 5 -> "星期五";
            case 6 -> "星期六";
            case 7 -> "星期日";
            default -> "无效的日期";
        };
        System.out.println("今天是：" + dayName);

        // 多个 case 合并
        int month = 3;
        String season = switch (month) {
            case 12, 1, 2 -> "冬季";
            case 3, 4, 5 -> "春季";
            case 6, 7, 8 -> "夏季";
            case 9, 10, 11 -> "秋季";
            default -> "无效月份";
        };
        System.out.println("季节：" + season);

        // 使用 yield 返回值
        int score = 85;
        String grade = switch (score / 10) {
            case 10, 9:
                System.out.println("优秀！");
                yield "A";
            case 8:
                yield "B";
            case 7:
                yield "C";
            case 6:
                yield "D";
            default:
                yield "F";
        };
        System.out.println("等级：" + grade);
    }
}
```

## 循环语句

### for 循环

最常用的循环，适合已知循环次数的场景。

```java
public class ForLoop {
    public static void main(String[] args) {
        // 基本形式
        System.out.println("=== 基本形式 ===");
        for (int i = 1; i <= 5; i++) {
            System.out.println("第 " + i + " 次循环");
        }

        // 多个初始化和更新
        System.out.println("\n=== 多变量控制 ===");
        for (int i = 0, j = 10; i < j; i++, j--) {
            System.out.println("i = " + i + ", j = " + j);
        }

        // 省略部分（无限循环）
        System.out.println("\n=== 省略初始化 ===");
        int i = 1;
        for (; i <= 3; i++) {
            System.out.println("i = " + i);
        }

        // 增强型 for 循环（for-each）
        System.out.println("\n=== for-each 循环 ===");
        int[] numbers = {1, 2, 3, 4, 5};
        for (int num : numbers) {
            System.out.println("数字：" + num);
        }

        // 嵌套循环
        System.out.println("\n=== 九九乘法表 ===");
        for (int m = 1; m <= 9; m++) {
            for (int n = 1; n <= m; n++) {
                System.out.printf("%d×%d=%-2d ", n, m, m * n);
            }
            System.out.println();
        }

        // 使用 break 和 continue
        System.out.println("\n=== break 和 continue ===");
        for (int k = 1; k <= 10; k++) {
            if (k == 5) {
                continue;  // 跳过 5
            }
            if (k == 8) {
                break;     // 在 8 处停止
            }
            System.out.print(k + " ");
        }
        System.out.println();
    }
}
```

### while 循环

适合不确定循环次数的场景，先判断后执行。

```java
public class WhileLoop {
    public static void main(String[] args) {
        // 基本形式
        System.out.println("=== 基本形式 ===");
        int count = 1;
        while (count <= 5) {
            System.out.println("count = " + count);
            count++;
        }

        // 计算阶乘
        System.out.println("\n=== 计算阶乘 ===");
        int n = 5;
        int factorial = 1;
        int i = 1;
        while (i <= n) {
            factorial *= i;
            i++;
        }
        System.out.println(n + "! = " + factorial);

        // 反转数字
        System.out.println("\n=== 反转数字 ===");
        int num = 12345;
        int reversed = 0;
        while (num != 0) {
            int digit = num % 10;
            reversed = reversed * 10 + digit;
            num /= 10;
        }
        System.out.println("反转后：" + reversed);
    }
}
```

### do-while 循环

先执行后判断，保证至少执行一次。

```java
public class DoWhileLoop {
    public static void main(String[] args) {
        // 基本形式
        System.out.println("=== do-while 示例 ===");
        int i = 1;
        do {
            System.out.println("i = " + i);
            i++;
        } while (i <= 5);

        // 至少执行一次的场景
        System.out.println("\n=== 菜单选择 ===");
        java.util.Scanner scanner = new java.util.Scanner(System.in);
        int choice;

        do {
            System.out.println("\n=== 菜单 ===");
            System.out.println("1. 选项一");
            System.out.println("2. 选项二");
            System.out.println("0. 退出");
            System.out.print("请选择：");

            choice = scanner.nextInt();

            switch (choice) {
                case 1:
                    System.out.println("你选择了选项一");
                    break;
                case 2:
                    System.out.println("你选择了选项二");
                    break;
                case 0:
                    System.out.println("再见！");
                    break;
                default:
                    System.out.println("无效选择！");
            }
        } while (choice != 0);

        scanner.close();
    }
}
```

### 循环对比

```java
public class LoopComparison {
    public static void main(String[] args) {
        // 相同功能的比较
        System.out.println("=== for 循环 ===");

        while (x < 5) {
            System.out.println("while: x = " + x);
            x++;
        }

        int y = 10;
        // do-while：至少执行一次
        do {
            System.out.println("do-while: y = " + y);
            y++;
        } while (y < 5);
    }
```

```java
import java.util.Scanner;
import java.util.Random;

/**
 * 猜数字游戏：1-100 之间的随机数
 */
public class GuessNumber {

    public static void main(String[] args) {
        Random random = new Random();
        Scanner scanner = new Scanner(System.in);

        int target = random.nextInt(100) + 1;  // 生成 1-100 的随机数
        int attempts = 0;
        boolean hasGuessed = false;

        System.out.println("=== 猜数字游戏 ===");
        System.out.println("我已经想好了一个 1-100 之间的数字，你来猜猜看！");

        while (!hasGuessed && attempts < 10) {
            System.out.print("\n请输入你的猜测 (1-100): ");

            if (scanner.hasNextInt()) {
                int guess = scanner.nextInt();
                attempts++;

                if (guess < 1 || guess > 100) {
                    System.out.println("请输入 1-100 之间的数字！");
                    attempts--;  // 不计入尝试次数
                } else if (guess == target) {
                    System.out.println("🎉 恭喜你猜对了！");
                    System.out.println("数字确实是 " + target);
                    System.out.println("你用了 " + attempts + " 次猜中。");
                    hasGuessed = true;
                } else if (guess < target) {
                    System.out.println("太小了！再大一点。");
                    int remaining = 10 - attempts;
                    if (remaining > 0) {
                        System.out.println("还有 " + remaining + " 次机会。");
                    }
                } else {
                    System.out.println("太大了！再小一点。");
                    int remaining = 10 - attempts;
                    if (remaining > 0) {
                        System.out.println("还有 " + remaining + " 次机会。");
                    }
                }
            } else {
                System.out.println("请输入有效的数字！");
                scanner.next();  // 清除无效输入
            }
        }

        if (!hasGuessed) {
            System.out.println("\n很遗憾，你没有在 10 次内猜中。");
            System.out.println("正确答案是：" + target);
        }

        scanner.close();
    }
}
```

```java
import java.util.Scanner;

/**
 * 简单的计算器：支持四则运算
 */
public class SimpleCalculator {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 简单计算器 ===");
        System.out.println("支持运算：+、-、*、/");

            System.out.print("请输入第一个数字: ");
            String input1 = scanner.nextLine();

            if (input1.equalsIgnoreCase("exit")) {
                break;
            }

            try {
                double num1 = Double.parseDouble(input1);

                System.out.print("请输入运算符 (+, -, *, /): ");
                String operator = scanner.nextLine();

                if (operator.length() != 1 ||
                    (!operator.equals("+") && !operator.equals("-") &&
                     !operator.equals("*") && !operator.equals("/"))) {
                    System.out.println("无效的运算符！\n");
                    continue;
                }

                System.out.print("请输入第二个数字: ");
                String input2 = scanner.nextLine();
                double num2 = Double.parseDouble(input2);

                double result;
                switch (operator) {
                    case "+":
                        result = num1 + num2;
                        System.out.printf("结果: %.2f %s %.2f = %.2f\n\n",
                            num1, operator, num2, result);
                        break;
                    case "-":
                        result = num1 - num2;
                        System.out.printf("结果: %.2f %s %.2f = %.2f\n\n",
                            num1, operator, num2, result);
                        break;
                    case "*":
                        result = num1 * num2;
                        System.out.printf("结果: %.2f %s %.2f = %.2f\n\n",
                            num1, operator, num2, result);
                        break;
                    case "/":
                        if (num2 == 0) {
                            System.out.println("错误：除数不能为零！\n");
                        } else {
                            result = num1 / num2;
                            System.out.printf("结果: %.2f %s %.2f = %.2f\n\n",
                                num1, operator, num2, result);
                        }
                        break;
                }
            } catch (NumberFormatException e) {
                System.out.println("输入错误！请输入有效的数字。\n");
            }
        }
```

```java
import java.util.Scanner;

/**
 * 判断一个数是否为质数
 */
public class PrimeNumberChecker {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 质数判断器 ===");
        System.out.print("请输入一个正整数: ");

        if (scanner.hasNextInt()) {
            int num = scanner.nextInt();

            if (num < 2) {
                System.out.println(num + " 不是质数。");
            } else if (num == 2) {
                System.out.println("2 是最小的质数。");
            } else if (num % 2 == 0) {
                System.out.println(num + " 不是质数（是偶数）。");
            } else {
                boolean isPrime = true;

                // 只需要检查到 √num
                for (int i = 3; i * i <= num; i += 2) {
                    if (num % i == 0) {
                        isPrime = false;
                        System.out.println(num + " 可以被 " + i + " 整除。");
                        break;
                    }
                }

                if (isPrime) {
                    System.out.println(num + " 是质数！");
                } else {
                    System.out.println(num + " 不是质数。");
                }
            }

            // 显示指定范围内的所有质数
            System.out.print("\n是否要显示 1-100 的所有质数？(y/n): ");
            scanner.nextLine();  // 消耗换行符
            String choice = scanner.nextLine();

            if (choice.equalsIgnoreCase("y")) {
                System.out.println("\n1-100 的质数：");
                int count = 0;
                for (int i = 2; i <= 100; i++) {
                    boolean prime = true;
                    for (int j = 2; j * j <= i; j++) {
                        if (i % j == 0) {
                            prime = false;
                            break;
                        }
                    }
                    if (prime) {
                        System.out.print(i + " ");
                        count++;
                        if (count % 10 == 0) {
                            System.out.println();
                        }
                    }
                }
                System.out.println("\n共 " + count + " 个质数。");
            }
        } else {
            System.out.println("请输入有效的整数！");
        }

        scanner.close();
    }
}
```

## 最佳实践

### 1. 使用括号明确优先级

```java
// ❌ 不好的做法：依赖优先级
boolean result = a > b && c < d || e == f;

// ✅ 好的做法：使用括号
boolean result = (a > b && c < d) || (e == f);
```

### 2. 避免过深的嵌套

```java
// ❌ 不好的做法：嵌套过深
if (condition1) {
    if (condition2) {
        if (condition3) {
            // 做某事
        }
    }
}

// ✅ 好的做法：提前返回
if (!condition1) return;
if (!condition2) return;
if (!condition3) return;
// 做某事
```

### 3. 使用 for-each 遍历集合

```java
// ❌ 不好的做法：使用索引
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}

// ✅ 好的做法：使用 for-each
for (String item : list) {
    System.out.println(item);
}
```

### 4. switch 使用 default

```java
// ❌ 不好的做法：缺少 default
switch (day) {
    case 1:
        // ...
        break;
    case 2:
        // ...
        break;
}

// ✅ 好的做法：包含 default
switch (day) {
    case 1:
        // ...
        break;
    case 2:
        // ...
        break;
    default:
```

## 常见问题

### Q1：&& 和 & 有什么区别？

**答**：

- `&&`：短路与，如果左边为 false，右边不会执行
- `&`：逻辑与，两边都会执行

```java
boolean a = false;
int x = 0;

boolean result1 = a && (x++ > 0);  // x 不会递增
boolean result2 = a & (x++ > 0);   // x 会递增
```

**建议**：优先使用 `&&`，性能更好。

### Q2：break 和 continue 有什么区别？

**答**：

- `break`：完全退出循环
- `continue`：跳过本次循环，继续下一次

```java
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;  // 跳过 3，继续 4、5
    if (i == 5) break;     // 在 5 处停止
    System.out.println(i);
}
// 输出：1 2 4
```

### Q3：什么时候用 for，什么时候用 while？

**答**：

- **for**：知道循环次数，或有明确的起始和结束条件
- **while**：不确定循环次数，只需要循环条件
- **do-while**：需要至少执行一次的情况

### Q4：switch 能判断哪些类型？

**答**：

- byte、short、char、int
- 包装类 Byte、Short、Character、Integer
- 枚举（enum）
- 字符串（String）（Java 7+）

### Q5：如何避免无限循环？

**答**：

1. 确保循环条件最终会变为 false
2. 在循环体内更新循环变量
3. 设置最大迭代次数作为保护

```java
int maxAttempts = 1000;
int attempts = 0;

while (condition && attempts < maxAttempts) {
    // 做某事
    attempts++;
    // 更新 condition
}
```
