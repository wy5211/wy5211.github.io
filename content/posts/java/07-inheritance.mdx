# 继承与多态

## 前言

继承和多态是面向对象编程的核心特性。继承让代码可以复用，多态让程序更加灵活。理解这两个概念，是掌握 Java 面向对象编程的关键。

## 继承

### 什么是继承

继承是从已有的类创建新类的过程。新类自动拥有已有类的属性和方法。

```
继承关系图：

        Animal（父类/基类）
         △
         │
    ┌────┴────┐
    │         │
  Dog       Cat（子类/派生类）
```

### 继承的优点

1. **代码复用**：子类继承父类的功能
2. **扩展性**：在不修改父类的情况下扩展功能
3. **维护性**：修改父类自动影响所有子类

### extends 关键字

```java
// 父类
public class Animal {

    protected String name;
    protected int age;

    public Animal(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void eat() {
        System.out.println(name + " 在吃东西");
    }

    public void sleep() {
        System.out.println(name + " 在睡觉");
    }

    public void displayInfo() {
        System.out.println("动物信息：" + name + "，" + age + "岁");
    }
}

// 子类
public class Dog extends Animal {

    private String breed;  // 品种

    public Dog(String name, int age, String breed) {
        super(name, age);  // 调用父类构造方法
        this.breed = breed;
    }

    // 子类特有方法
    public void bark() {
        System.out.println(name + " 在汪汪叫");
    }

    public void fetch() {
        System.out.println(name + " 在捡球");
    }
}

// 另一个子类
public class Cat extends Animal {

    private boolean isIndoor;  // 是否室内猫

    public Cat(String name, int age, boolean isIndoor) {
        super(name, age);
        this.isIndoor = isIndoor;
    }

    // 子类特有方法
    public void meow() {
        System.out.println(name + " 在喵喵叫");
    }

    public void scratch() {
        System.out.println(name + " 在抓东西");
    }
}
```

### super 关键字

`super` 用于引用父类的成员。

```java
public class SuperDemo {

    public static void main(String[] args) {
        Dog dog = new Dog("旺财", 3, "金毛");
        dog.displayInfo();
        dog.bark();
        dog.eat();  // 继承的方法
    }
}

class Dog extends Animal {
    private String breed;

    public Dog(String name, int age, String breed) {
        super(name, age);  // ① 调用父类构造方法
        this.breed = breed;
    }

    @Override
    public void displayInfo() {
        // ② 调用父类方法
        super.displayInfo();
        System.out.println("品种：" + breed);
    }

    public void test() {
        // ③ 访问父类字段
        System.out.println("父类名字：" + super.name);
    }
}
```

### 方法重写（Override）

子类可以重新定义父类的方法。

```java
public class MethodOverride {

    public static void main(String[] args) {
        Animal animal = new Animal("动物", 5);
        animal.eat();  // 动物 在吃东西

        Dog dog = new Dog("旺财", 3, "金毛");
        dog.eat();  // 旺财 狼吞虎咽地吃东西
    }
}

class Animal {
    public void eat() {
        System.out.println("动物在吃东西");
    }
}

class Dog extends Animal {
    @Override  // 注解，表示重写父类方法
    public void eat() {
        System.out.println("狗狼吞虎咽地吃东西");
    }
}
```

**重写规则**：

- 方法名、参数列表必须相同
- 返回类型相同或是子类型
- 访问权限不能更严格
- 不能抛出更多异常

### 重写 vs 重载

```java
public class OverloadVsOverride {

    public static void main(String[] args) {
        Parent parent = new Parent();
        Child child = new Child();

        // 重载：编译时多态
        parent.test();          // 无参方法
        parent.test("Hello");   // 有参方法

        // 重写：运行时多态
        parent.show();  // 父类的 show 方法
        child.show();   // 子类的 show 方法
    }
}

class Parent {
    public void test() {
        System.out.println("父类无参方法");
    }

    public void test(String message) {
        System.out.println("父类有参方法：" + message);
    }

    public void show() {
        System.out.println("父类 show 方法");
    }
}

class Child extends Parent {
    // 重载
    public void test(int number) {
        System.out.println("子类有参方法：" + number);
    }

    // 重写
    @Override
    public void show() {
        System.out.println("子类 show 方法");
    }
}
```

## 多态

### 什么是多态

多态是同一个方法在不同对象上有不同行为。

```java
// 多态示例
Animal animal1 = new Dog();   // 向上转型
Animal animal2 = new Cat();   // 向上转型

animal1.makeSound();  // 汪汪叫
animal2.makeSound();  // 喵喵叫
```

### 多态的实现条件

1. **继承**：必须有继承关系
2. **重写**：子类重写父类方法
3. **向上转型**：父类引用指向子类对象

### 向上转型

```java
public class UpcastingDemo {

    public static void main(String[] args) {
        // 向上转型：子类对象赋给父类引用
        Animal animal = new Dog("旺财", 3, "金毛");

        // 只能访问父类中定义的方法
        animal.eat();      // ✅ 可以
        animal.sleep();    // ✅ 可以

        // animal.bark();   // ❌ 编译错误，父类没有此方法

        // 如果需要调用子类特有方法，需要向下转型
        if (animal instanceof Dog) {
            Dog dog = (Dog) animal;  // 向下转型
            dog.bark();  // ✅ 现在可以了
        }
    }
}
```

### 向下转型

```java
public class DowncastingDemo {

    public static void main(String[] args) {
        Animal animal = new Dog("旺财", 3, "金毛");

        // 检查类型后再转型
        if (animal instanceof Dog) {
            Dog dog = (Dog) animal;
            dog.bark();
            System.out.println("成功转换为 Dog");
        }

        if (animal instanceof Cat) {
            Cat cat = (Cat) animal;  // 不会执行
            cat.meow();
        } else {
            System.out.println("不是 Cat 类型");
        }
    }
}
```

### instanceof 操作符

用于检查对象是否是某个类的实例。

```java
public class InstanceofDemo {

    public static void main(String[] args) {
        Object obj = new Dog("旺财", 3, "金毛");

        // 检查对象类型
        if (obj instanceof Dog) {
            System.out.println("是 Dog 类型");
        }

        if (obj instanceof Animal) {
            System.out.println("是 Animal 类型");
        }

        if (obj instanceof Object) {
            System.out.println("是 Object 类型");
        }
    }
}
```

### 动态绑定

方法在运行时根据实际对象类型调用。

```java
public class DynamicBinding {

    public static void main(String[] args) {
        Animal animal = new Dog("旺财", 3, "金毛");
        animal.makeSound();  // 运行时调用 Dog 的 makeSound
    }
}

class Animal {
    public void makeSound() {
        System.out.println("动物发出声音");
    }
}

class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("汪汪汪");
    }
}
```

## 实战演练

### 示例 1：员工工资系统

```java
// 员工基类
public class Employee {

    protected String name;
    protected double baseSalary;

    public Employee(String name, double baseSalary) {
        this.name = name;
        this.baseSalary = baseSalary;
    }

    public double calculateSalary() {
        return baseSalary;
    }

    public void displayInfo() {
        System.out.printf("员工：%s，工资：%.2f 元%n", name, calculateSalary());
    }
}

// 经理类
class Manager extends Employee {

    private double bonus;

    public Manager(String name, double baseSalary, double bonus) {
        super(name, baseSalary);
        this.bonus = bonus;
    }

    @Override
    public double calculateSalary() {
        return baseSalary + bonus;
    }
}

// 程序员类
class Programmer extends Employee {

    private int overtimeHours;
    private double overtimeRate;

    public Programmer(String name, double baseSalary,
                     int overtimeHours, double overtimeRate) {
        super(name, baseSalary);
        this.overtimeHours = overtimeHours;
        this.overtimeRate = overtimeRate;
    }

    @Override
    public double calculateSalary() {
        return baseSalary + (overtimeHours * overtimeRate);
    }
}

// 测试类
class EmployeeSystem {
    public static void main(String[] args) {
        Employee[] employees = {
            new Employee("张三", 5000),
            new Manager("李四", 8000, 2000),
            new Programmer("王五", 7000, 20, 100)
        };

        System.out.println("=== 员工工资单 ===");
        double totalSalary = 0;
        for (Employee emp : employees) {
            emp.displayInfo();
            totalSalary += emp.calculateSalary();
        }
        System.out.printf("总工资支出：%.2f 元%n", totalSalary);
    }
}
```

### 示例 2：图形计算

```java
// 图形基类
abstract class Shape {

    protected String color;

    public Shape(String color) {
        this.color = color;
    }

    // 抽象方法
    public abstract double getArea();
    public abstract double getPerimeter();

    public void display() {
        System.out.println("颜色：" + color);
        System.out.printf("面积：%.2f%n", getArea());
        System.out.printf("周长：%.2f%n", getPerimeter());
    }
}

// 矩形
class Rectangle extends Shape {

    private double width;
    private double height;

    public Rectangle(String color, double width, double height) {
        super(color);
        this.width = width;
        this.height = height;
    }

    @Override
    public double getArea() {
        return width * height;
    }

    @Override
    public double getPerimeter() {
        return 2 * (width + height);
    }
}

// 圆形
class Circle extends Shape {

    private double radius;

    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }

    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }

    @Override
    public double getPerimeter() {
        return 2 * Math.PI * radius;
    }
}

// 测试
class ShapeDemo {
    public static void main(String[] args) {
        Shape[] shapes = {
            new Rectangle("红色", 5, 3),
            new Circle("蓝色", 4)
        };

        for (Shape shape : shapes) {
            shape.display();
            System.out.println();
        }
    }
}
```

### 示例 3：支付系统

```java
// 支付接口
interface PaymentMethod {
    void pay(double amount);
    void refund(double amount);
}

// 信用卡支付
class CreditCard implements PaymentMethod {

    private String cardNumber;

    public CreditCard(String cardNumber) {
        this.cardNumber = cardNumber;
    }

    @Override
    public void pay(double amount) {
        System.out.printf("使用信用卡 %s 支付 %.2f 元%n",
            maskCardNumber(cardNumber), amount);
    }

    @Override
    public void refund(double amount) {
        System.out.printf("退款 %.2f 元到信用卡 %s%n",
            amount, maskCardNumber(cardNumber));
    }

    private String maskCardNumber(String number) {
        return "****-****-****-" + number.substring(number.length() - 4);
    }
}

// 支付宝
class Alipay implements PaymentMethod {

    private String account;

    public Alipay(String account) {
        this.account = account;
    }

    @Override
    public void pay(double amount) {
        System.out.printf("使用支付宝 %s 支付 %.2f 元%n", account, amount);
    }

    @Override
    public void refund(double amount) {
        System.out.printf("退款 %.2f 元到支付宝 %s%n", amount, account);
    }
}

// 微信支付
class WeChatPay implements PaymentMethod {

    private String account;

    public WeChatPay(String account) {
        this.account = account;

    public void pay(double amount) {
```

## 最佳实践

### 1. 优先使用组合而非继承

```java
// ❌ 不好的做法：过度继承
class Dog extends Animal, Swimmable, Flyable { ... }

// ✅ 好的做法：使用组合
class Dog extends Animal {
    private Swimmable swimmable;
    private Flyable flyable;
}
```

### 2. 遵循里氏替换原则

```java
// 子类应该可以替换父类
Animal animal = new Dog();  // ✅ 应该正常工作
```

### 3. 避免过深的继承层次

```java
// ❌ 不好的做法
A → B → C → D → E → F

// ✅ 好的做法
A → B → C
```

### 4. 使用 @Override 注解

```java
@Override
public void eat() {
    // 方法实现
}
```

## 常见问题

### Q1：重写和重载的区别？

**答**：

- **重写**：子类重新定义父类方法，方法签名相同
- **重载**：同一个类中方法名相同但参数不同

### Q2：什么时候使用继承？

**答**：当子类是父类的一种时（is-a 关系）。

### Q3：final 关键字的作用？

**答**：

- final 类：不能被继承
- final 方法：不能被重写
- final 变量：不能被修改

### Q4：动态绑定何时发生？

**答**：运行时，根据实际对象类型调用方法。

### Q5：如何选择继承还是组合？

**答**：

- is-a 关系：使用继承
- has-a 关系：使用组合
