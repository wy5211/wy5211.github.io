# 异常处理机制

## 前言

异常是程序运行时发生的错误。良好的异常处理可以让程序更加健壮和用户友好。Java 提供了完善的异常处理机制，本文将详细介绍如何正确处理异常。

## 异常层次结构

```
                    Throwable
                      /    \
                    /        \
                 Error      Exception
                            /    \
                    Checked   Unchecked
                       (编译时) (运行时)
```

### Error vs Exception

- **Error**：严重错误，程序无法处理（如 OutOfMemoryError）
- **Exception**：异常，程序可以处理

### Checked vs Unchecked

- **Checked 异常**：编译时必须处理
- **Unchecked 异常**：运行时异常（RuntimeException）

## try-catch-finally

### 基本语法

```java
public class ExceptionDemo {

    public static void main(String[] args) {
        try {
            // 可能抛出异常的代码
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            // 捕获并处理异常
            System.out.println("算术异常：" + e.getMessage());
        } finally {
            // 无论是否异常都会执行
            System.out.println("finally 块");
        }
    }
}
```

### 多个 catch 块

```java
public class MultiCatchDemo {

    public static void main(String[] args) {
        try {
            String[] array = {"A", "B", "C"};
            System.out.println(array[5]);  // 数组越界
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("数组越界：" + e.getMessage());
        } catch (NullPointerException e) {
            System.out.println("空指针异常：" + e.getMessage());
        } catch (Exception e) {
            System.out.println("其他异常：" + e.getMessage());
        }
    }
}
```

### try-with-resources（Java 7+）

```java
public class TryWithResources {

    public static void main(String[] args) {
        // 自动关闭资源
        try (FileReader reader = new FileReader("test.txt")) {
            // 读取文件
        } catch (IOException e) {
            System.out.println("文件读取失败：" + e.getMessage());
        }
    }
}
```

## throw 和 throws

### throw 抛出异常

```java
public class ThrowDemo {

    public static void setAge(int age) {
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("年龄范围：0-150");
        }
    }

    public static void main(String[] args) {
        try {
            setAge(-5);
        } catch (IllegalArgumentException e) {
            System.out.println("捕获异常：" + e.getMessage());
        }
    }
}
```

### throws 声明异常

```java
import java.io.*;

public class ThrowsDemo {

    // 声明可能抛出的异常
    public static void readFile(String filename) throws IOException {
        FileReader reader = new FileReader(filename);
        // 读取操作
        reader.close();
    }

    public static void main(String[] args) {
        try {
            readFile("test.txt");
        } catch (IOException e) {
            System.out.println("文件处理失败");
        }
    }
}
```

## 自定义异常

```java
// 自定义 Checked 异常
public class InsufficientFundsException extends Exception {

    private double amount;
    private double balance;

    public InsufficientFundsException(double amount, double balance) {
        super(String.format("余额不足：尝试取款 %.2f，余额 %.2f", amount, balance));
        this.amount = amount;
        this.balance = balance;
    }

    public double getAmount() {
        return amount;
    }

    public double getBalance() {
        return balance;
    }
}

// 使用自定义异常
public class BankAccount {

    private double balance;

    public void withdraw(double amount) throws InsufficientFundsException {
        if (amount > balance) {
            throw new InsufficientFundsException(amount, balance);
        }
        balance -= amount;
    }
}
```

## 实战演练

### 示例 1：用户输入验证

```java
import java.util.Scanner;

public class InputValidation {

    public static int readPositiveInt(Scanner scanner, String prompt) {
        while (true) {
            try {
                System.out.print(prompt);
                String input = scanner.nextLine();
                int number = Integer.parseInt(input);

                if (number < 0) {
                    System.out.println("请输入正数！");
                    continue;
                }

                return number;
            } catch (NumberFormatException e) {
                System.out.println("输入错误，请输入整数！");
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int age = readPositiveInt(scanner, "请输入年龄：");
        System.out.println("年龄：" + age);
        scanner.close();
    }
}
```

### 示例 2：文件处理

```java
import java.io.*;
import java.util.*;

public class FileProcessor {

    public static List<String> readFile(String filename) {
        List<String> lines = new ArrayList<>();

        // try-with-resources 自动关闭资源
        try (BufferedReader reader = new BufferedReader(new FileReader(filename))) {
            String line;
            while ((line = reader.readLine()) != null) {
                lines.add(line);
            }
        } catch (FileNotFoundException e) {
            System.out.println("文件不存在：" + filename);
        } catch (IOException e) {
            System.out.println("读取文件失败：" + e.getMessage());
        }

        return lines;
    }

    public static void writeFile(String filename, List<String> lines) {
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filename))) {
            for (String line : lines) {
                writer.write(line);
                writer.newLine();
            }
            System.out.println("文件写入成功");
        } catch (IOException e) {
            System.out.println("写入文件失败：" + e.getMessage());
        }
    }
}
```

## 最佳实践

### 1. 具体异常优先捕获

```java
// ✅ 好的做法
try {
    // 代码
} catch (NullPointerException e) {
    // 处理空指针
} catch (IllegalArgumentException e) {
    // 处理非法参数
} catch (Exception e) {
    // 处理其他异常
}
```

### 2. 不要捕获 Throwable

```java
// ❌ 不好的做法
try {
    // 代码
} catch (Throwable t) {
    // 捕获太多
}

// ✅ 好的做法
try {
    // 代码
} catch (Exception e) {
    // 只捕获异常
}
```

### 3. 使用 try-with-resources

```java
// ✅ 好的做法（Java 7+）
try (FileReader reader = new FileReader("test.txt")) {
    // 自动关闭
}
```

## 常见问题

### Q1：Checked 和 Unchecked 异常的区别？

**答**：

- Checked：编译时必须处理
- Unchecked：运行时异常，可不处理

### Q2：finally 什么时候不执行？

**答**：

- System.exit() 之前
- JVM 崩溃
- 无限循环

### Q3：多个异常如何捕获？

**答**：使用多重 catch 或 Java 7+ 的多异常捕获。
