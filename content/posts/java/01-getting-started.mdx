# 环境搭建与第一个程序

## 前言

欢迎来到 Java 的世界！在开始编程之旅前，我们需要先搭建好开发环境。Java 是一门跨平台的编程语言，"一次编写，到处运行"（Write Once, Run Anywhere）是它的核心理念。让我们从环境搭建开始，一步步走进 Java 的世界。

## Java 的核心概念

### JDK、JRE、JVM 的关系

在学习 Java 前，需要理解三个重要的概念：

```
┌─────────────────────────────────────────────────────────────┐
│                        JDK (Java Development Kit)           │
│                    Java 开发工具包（用于开发）                │
│  ┌───────────────────────────────────────────────────────┐  │
│  │              JRE (Java Runtime Environment)            │  │
│  │              Java 运行时环境（用于运行）                │  │
│  │   ┌─────────────────────────────────────────────────┐  │  │
│  │   │          JVM (Java Virtual Machine)             │  │  │
│  │   │          Java 虚拟机（核心引擎）                 │  │  │
│  │   └─────────────────────────────────────────────────┘  │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

- **JDK（Java Development Kit）**：Java 开发工具包，包含开发 Java 程序所需的所有工具（编译器、调试器等）
- **JRE（Java Runtime Environment）**：Java 运行时环境，包含运行 Java 程序所需的核心组件
- **JVM（Java Virtual Machine）**：Java 虚拟机，负责执行 Java 字节码

**关系**：JDK 包含 JRE，JRE 包含 JVM

### Java 程序的执行流程

```
源代码文件(.java)
       ↓
    编译(javac)
       ↓
字节码文件(.class)
       ↓
    加载到JVM
       ↓
   解释/编译为本机代码
       ↓
    程序运行
```

## 环境搭建

### 1. 安装 JDK 17

推荐使用 **Java 17 LTS（长期支持版本）**，这是目前最稳定的企业级版本。

**下载方式**：

1. **Oracle JDK**：访问 [Oracle官网](https://www.oracle.com/java/technologies/downloads/)
2. **OpenJDK**：访问 [Adoptium](https://adoptium.net/)（推荐，免费开源）

**安装步骤**（以 Adoptium OpenJDK 17 为例）：

1. 下载适合你操作系统的安装包
2. 运行安装程序，按提示完成安装
3. 记住安装路径（Windows 默认：`C:\Program Files\Eclipse Adoptium\jdk-17.x.x`）

### 2. 配置环境变量

**Windows 系统**：

1. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
2. 新建系统变量：
   - 变量名：`JAVA_HOME`
   - 变量值：JDK 安装路径（如 `C:\Program Files\Eclipse Adoptium\jdk-17.0.9.101`）
3. 编辑 Path 变量，添加：`%JAVA_HOME%\bin`

**macOS/Linux 系统**：

编辑 `~/.zshrc` 或 `~/.bashrc` 文件，添加：

```bash
export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH
```

保存后执行：`source ~/.zshrc`

### 3. 验证安装

打开终端/命令提示符，执行以下命令：

```bash
java -version
javac -version
```

如果显示版本信息，说明安装成功！

### 4. 选择开发工具

**推荐 IDE**：

- **IntelliJ IDEA**（推荐）：功能强大，智能提示完善
  - 下载：https://www.jetbrains.com/idea/
  - Community 版本免费

- **VSCode**：轻量级，插件丰富
  - 下载：https://code.visualstudio.com/
  - 安装 "Extension Pack for Java" 插件

- **Eclipse**：老牌 IDE，功能全面

## 编写第一个 Java 程序

### 使用文本编写

创建文件 `HelloWorld.java`：

```java
/**
 * 我的第一个 Java 程序
 */
public class HelloWorld {

    // 程序的入口方法
    public static void main(String[] args) {
        // 向控制台输出消息
        System.out.println("Hello, World!");
        System.out.println("欢迎来到 Java 的世界！");
    }
}
```

### 代码解析

让我们逐行理解这个程序：

```java
public class HelloWorld {  // 定义一个名为 HelloWorld 的公共类
    // 类的内容放在花括号内
}
```

**类的命名规则**：

- 类名必须与文件名相同（HelloWorld.java）
- 使用大驼峰命名法（每个单词首字母大写）
- public 修饰符表示这个类可以被任何其他类访问

```java
public static void main(String[] args) {
    // 主方法，程序执行的入口点
}
```

**main 方法详解**：

- `public`：公共方法，可以从任何地方访问
- `static`：静态方法，不需要创建对象即可调用
- `void`：无返回值
- `main`：方法名，JVM 查找的特定名称
- `String[] args`：命令行参数数组

```java
System.out.println("Hello, World!");
```

**输出语句**：

- `System`：java.lang 包中的系统类
- `out`：System 类的静态成员，表示标准输出
- `println()`：打印文本并换行
- 还有 `print()` 方法：打印文本但不换行

### 编译与运行

**命令行方式**：

```bash
# 编译：将 .java 文件编译成 .class 文件（字节码）
javac HelloWorld.java

# 运行：使用 JVM 执行字节码
java HelloWorld
```

**输出结果**：

```
Hello, World!
欢迎来到 Java 的世界！
```

**注意**：运行时使用类名 `HelloWorld`，而不是 `HelloWorld.class`

## 包与导入语句

### 包（Package）

包用于组织类，避免命名冲突。

```java
package com.example;  // 声明包，必须是文件的第一条语句

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello from package!");
    }
}
```

**包的命名规范**：

- 通常使用公司域名的倒序（如 `com.companyname.projectname`）
- 全部小写
- 用点号分隔

**目录结构**：

```
项目根目录/
└── com/
    └── example/
        └── HelloWorld.java
```

编译和运行带包的类：

```bash
javac com/example/HelloWorld.java
java com.example.HelloWorld
```

### 导入（Import）

使用其他包中的类需要导入：

```java
package com.example;

import java.util.Scanner;  // 导入 Scanner 类
import java.util.*;       // 导入 java.util 包下的所有类

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);  // 创建 Scanner 对象
        System.out.println("请输入你的名字：");
        String name = scanner.nextLine();  // 读取一行输入
        System.out.println("你好，" + name + "！");
        scanner.close();  // 关闭 scanner
    }
}
```

## 实战演练

### 示例 1：接收命令行参数

```java
/**
 * 命令行参数示例
 */
public class CommandLineArgs {

    public static void main(String[] args) {
        System.out.println("接收到 " + args.length + " 个参数");

        // 遍历所有参数
        for (int i = 0; i < args.length; i++) {
            System.out.println("参数 " + (i + 1) + ": " + args[i]);
        }

        // 使用增强 for 循环（for-each）
        System.out.println("\n使用 for-each 遍历：");
        for (String arg : args) {
            System.out.println(arg);
        }
    }
}
```

**运行方式**：

```bash
java CommandLineArgs Java Python C++
```

**输出**：

```
接收到 3 个参数
参数 1: Java
参数 2: Python
参数 3: C++

使用 for-each 遍历：
Java
Python
C++
```

### 示例 2：简单的加法计算器

```java
import java.util.Scanner;

/**
 * 简单的加法计算器
 */
public class SimpleCalculator {

    public static void main(String[] args) {
        // 创建 Scanner 对象，用于读取用户输入
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 简单加法计算器 ===");

        // 提示用户输入第一个数字
        System.out.print("请输入第一个数字：");
        double num1 = scanner.nextDouble();  // 读取一个 double 类型数字

        // 提示用户输入第二个数字
        System.out.print("请输入第二个数字：");
        double num2 = scanner.nextDouble();

        // 计算两数之和
        double sum = num1 + num2;

        // 输出结果
        System.out.println("计算结果：" + num1 + " + " + num2 + " = " + sum);

        // 格式化输出（保留两位小数）
        System.out.printf("格式化结果：%.2f + %.2f = %.2f%n", num1, num2, sum);

        // 关闭 scanner
        scanner.close();
    }
}
```

**运行示例**：

```
=== 简单加法计算器 ===
请输入第一个数字：12.5
请输入第二个数字：8.3
计算结果：12.5 + 8.3 = 20.8
格式化结果：12.50 + 8.30 = 20.80
```

**代码解释**：

- `scanner.nextDouble()`：读取用户输入的下一个 double 值
- `+`：字符串拼接符，将多个字符串连接在一起
- `printf()`：格式化输出，`%.2f` 表示保留两位小数，`%n` 表示换行

## 最佳实践

### 1. 代码注释规范

```java
/**
 * 类级别的文档注释
 * 用于描述类的功能、作者、版本等信息
 *
 * @author 作者名
 * @version 1.0
 */
public class CommentExample {

    /**
     * 方法级别的文档注释
     * 描述方法的功能、参数、返回值等
     *
     * @param num1 第一个数字
     * @param num2 第二个数字
     * @return 两数之和
     */
    public static int add(int num1, int num2) {
        // 单行注释：解释关键代码
        int sum = num1 + num2;
        return sum;
    }

    /*
     * 多行注释
     * 用于注释大段代码或详细说明
     */
}
```

### 2. 命名规范

| 类型   | 命名规则             | 示例                         |
| ------ | -------------------- | ---------------------------- |
| 类名   | 大驼峰（PascalCase） | `HelloWorld`, `StringBuffer` |
| 方法名 | 小驼峰（camelCase）  | `main`, `nextInt`, `println` |
| 变量名 | 小驼峰（camelCase）  | `userName`, `totalCount`     |
| 常量名 | 全大写下划线分隔     | `MAX_SIZE`, `DEFAULT_VALUE`  |
| 包名   | 全部小写，点分隔     | `java.util`, `com.example`   |

### 3. 代码格式化

- 使用 4 个空格缩进（不使用 Tab）
- 左大括号不换行
- 操作符两边添加空格
- 适当使用空行分隔逻辑块

```java
// 良好的格式
public class GoodFormat {
    public static void main(String[] args) {
        int sum = 0;

        for (int i = 1; i <= 100; i++) {
            sum = sum + i;
        }

        System.out.println("Sum: " + sum);
    }
}
```

## 常见问题

### Q1：编译时提示 "javac 不是内部或外部命令"

**原因**：环境变量配置不正确

**解决**：

1. 检查 `JAVA_HOME` 是否正确设置
2. 检查 Path 是否包含 `%JAVA_HOME%\bin`
3. 重新打开命令提示符使配置生效

### Q2：运行时提示 "找不到或无法加载主类"

**原因**：类名与文件名不匹配，或包结构错误

**解决**：

1. 确保文件名与 public 类名完全一致（包括大小写）
2. 如果使用了 package，确保目录结构与包结构一致
3. 使用完整类名运行：`java com.example.HelloWorld`

### Q3：中文输出乱码

**原因**：文件编码与控制台编码不一致

**解决**：

1. 保存源文件时使用 UTF-8 编码
2. 编译时指定编码：`javac -encoding UTF-8 HelloWorld.java`
3. 在 IDE 中设置项目文件编码为 UTF-8

### Q4：println 和 print 有什么区别？

**答**：

- `println()`：打印内容后换行
- `print()`：打印内容但不换行
- `printf()`：格式化输出

```java
System.out.print("Hello");
System.out.print("World");
// 输出：HelloWorld

System.out.println("Hello");
System.out.println("World");
// 输出：
// Hello
// World
```
