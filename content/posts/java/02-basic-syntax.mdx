# 基础语法与数据类型

## 前言

数据类型是编程语言的基石。Java 是一门强类型语言，每个变量都必须声明类型。理解 Java 的数据类型系统对于编写正确、高效的代码至关重要。本文将详细介绍 Java 的八种基本数据类型和变量使用。

## 变量

### 什么是变量

变量是内存中存储数据的容器，可以通过变量名访问存储的数据。

```java
// 变量声明语法：数据类型 变量名 = 初始值;
int age = 25;           // 声明一个整型变量 age，值为 25
double salary = 8500.5; // 声明一个双精度浮点型变量 salary
char grade = 'A';       // 声明一个字符型变量 grade
boolean isActive = true;// 声明一个布尔型变量 isActive
```

### 标识符命名规则

标识符是变量、方法、类等的名称。

**命名规则**：

- 只能包含字母、数字、下划线（\_）和美元符号（$）
- 不能以数字开头
- 不能是 Java 关键字
- 区分大小写

**合法示例**：

```java
int userName;       // ✅ 合法
int _count;         // ✅ 合法
int $value;         // ✅ 合法
int score1;         // ✅ 合法
int class;          // ❌ 非法，class 是关键字
int 2ndPlace;       // ❌ 非法，不能以数字开头
int user-name;      // ❌ 非法，不能包含连字符
```

**命名建议**：

- 见名知意：用有意义的名称
- 驼峰命名法：userName、totalCount
- 常量全大写：MAX_SIZE、DEFAULT_VALUE

### 变量的声明与初始化

```java
public class VariableDemo {
    public static void main(String[] args) {
        // 方式一：先声明，后赋值
        int x;
        x = 10;
        System.out.println("x = " + x);

        // 方式二：声明的同时赋值
        int y = 20;
        System.out.println("y = " + y);

        // 方式三：同时声明多个变量
        int a = 5, b = 10, c = 15;
        System.out.println("a = " + a + ", b = " + b + ", c = " + c);

        // 方式四：同时声明多个同类型变量（不推荐，可读性差）
        int m, n, p;
        m = 1;
        n = 2;
        p = 3;

        // 局部变量必须初始化才能使用
        int num;
        // System.out.println(num); // ❌ 编译错误：变量可能未初始化
        num = 100;
        System.out.println("num = " + num); // ✅ 正确
    }
}
```

Java 有八种基本数据类型（primitive types），分为四类：

### 整型（4种）

| 类型    | 大小           | 取值范围       | 默认值 | 使用场景               |
| ------- | -------------- | -------------- | ------ | ---------------------- |
| `byte`  | 1 字节（8位）  | -128 ~ 127     | 0      | 节省内存，处理字节数据 |
| `short` | 2 字节（16位） | -32768 ~ 32767 | 0      | 较少使用               |
| `int`   | 4 字节（32位） | -2³¹ ~ 2³¹-1   | 0      | **最常用**的整数类型   |
| `long`  | 8 字节（64位） | -2⁶³ ~ 2⁶³-1   | 0L     | 大数值（时间戳、ID等） |

```java
public class IntegerTypes {
    public static void main(String[] args) {
        // byte 类型
        byte byteVar = 100;
        System.out.println("byte: " + byteVar);

        // short 类型
        short shortVar = 10000;
        System.out.println("short: " + shortVar);

        // int 类型（最常用）
        int intVar = 100000;
        System.out.println("int: " + intVar);

        // long 类型（需要加 L 或 l 后缀）
        long longVar = 10000000000L;
        System.out.println("long: " + longVar);

        // 整数的不同进制表示
        int decimal = 100;      // 十进制
        int octal = 0144;       // 八进制（以 0 开头）
        int hex = 0x64;         // 十六进制（以 0x 开头）
        int binary = 0b1100100; // 二进制（以 0b 开头，Java 7+）

        System.out.println("十进制: " + decimal);
        System.out.println("八进制: " + octal);
        System.out.println("十六进制: " + hex);
        System.out.println("二进制: " + binary);

        // 数字字面量使用下划线分隔（Java 7+），提高可读性
        int billion = 1_000_000_000;
        System.out.println("十亿: " + billion);
    }
}
```

### 浮点型（2种）

| 类型     | 大小           | 精度                    | 默认值 | 使用场景             |
| -------- | -------------- | ----------------------- | ------ | -------------------- |
| `float`  | 4 字节（32位） | 单精度（6-7位有效数字） | 0.0f   | 图形计算、节省内存   |
| `double` | 8 字节（64位） | 双精度（15位有效数字）  | 0.0    | **最常用**的浮点类型 |

```java
public class FloatingPointTypes {
    public static void main(String[] args) {
        // float 类型（需要加 f 或 F 后缀）
        float floatVar = 3.14f;
        System.out.println("float: " + floatVar);

        // double 类型（最常用，可以加 d 或 D 后缀，也可以不加）
        double doubleVar = 3.141592653589793;
        System.out.println("double: " + doubleVar);

        // 科学计数法
        double scientific1 = 1.23e10;  // 1.23 × 10¹⁰ = 12300000000
        double scientific2 = 1.23e-5;  // 1.23 × 10⁻⁵ = 0.0000123
        System.out.println("科学计数法1: " + scientific1);
        System.out.println("科学计数法2: " + scientific2);

        // 浮点数的精度问题
        System.out.println("\n浮点数精度问题：");
        double a = 0.1;
        double b = 0.2;
        double c = a + b;
        System.out.println("0.1 + 0.2 = " + c); // 0.30000000000000004
        System.out.printf("格式化输出: %.2f%n", c); // 0.30

        // 特殊值
        System.out.println("\n浮点数特殊值：");
        System.out.println("正无穷: " + (1.0 / 0.0));   // Infinity
        System.out.println("负无穷: " + (-1.0 / 0.0));  // -Infinity
        System.out.println("非数字: " + (0.0 / 0.0));   // NaN
    }
}
```

### 字符型（1种）

| 类型   | 大小           | 取值范围                  | 默认值   |
| ------ | -------------- | ------------------------- | -------- |
| `char` | 2 字节（16位） | 0 ~ 65535（Unicode 字符） | '\u0000' |

```java
public class CharType {
    public static void main(String[] args) {
        // 字符的多种表示方式
        char char1 = 'A';           // 直接字符
        char char2 = 65;            // 十进制 ASCII 码
        char char3 = '\u0041';      // Unicode 表示
        char char4 = '\101';        // 八进制表示

        System.out.println("char1: " + char1); // A
        System.out.println("char2: " + char2); // A
        System.out.println("char3: " + char3); // A
        System.out.println("char4: " + char4); // A

        // 特殊字符（转义字符）
        System.out.println("\n转义字符示例：");
        System.out.println("换行符\n新的一行");
        System.out.println("制表符\t内容");
        System.out.println("反斜杠\\");
        System.out.println("单引号\'");
        System.out.println("双引号\"");
        System.out.println("回车符\r前面内容被覆盖");

        // 中文字符
        char chineseChar = '中';
        System.out.println("\n中文字符: " + chineseChar);
        System.out.println("Unicode 值: " + (int) chineseChar); // 20013

        // 字符运算
        char letter1 = 'A';
        char letter2 = 'B';
        System.out.println("\n字符运算：");
        System.out.println("'A' + 1 = " + (letter1 + 1));       // 66
        System.out.println("(char)('A' + 1) = " + (char)(letter1 + 1)); // B
    }
}
```

### 布尔型（1种）

| 类型      | 大小                     | 取值        | 默认值 |
| --------- | ------------------------ | ----------- | ------ |
| `boolean` | 1 字节（JVM 规范未明确） | true、false | false  |

```java
public class BooleanType {
    public static void main(String[] args) {
        boolean flag1 = true;
        boolean flag2 = false;

        System.out.println("flag1: " + flag1);
        System.out.println("flag2: " + flag2);

        // 布尔值常用于条件判断
        int age = 18;
        boolean isAdult = age >= 18;
        System.out.println("isAdult: " + isAdult);

        // 布尔运算
        boolean a = true;
        boolean b = false;
        System.out.println("\n布尔运算：");

        System.out.println("a || b = " + (a || b)); // true（或）
        System.out.println("!a = " + (!a));         // false（非）
    }
}
```

## 数据类型转换

Java 中数据类型转换分为两种：**自动类型转换**和**强制类型转换**。

### 自动类型转换（隐式）

从小类型到大类型的自动转换，不会丢失数据。

```
byte → short → int → long → float → double
     char → int → long → float → double
```

```java
public class AutoConversion {
    public static void main(String[] args) {
        // byte → int
        byte byteValue = 10;
        int intValue = byteValue;  // 自动转换
        System.out.println("byte → int: " + intValue);

        // int → long
        long longValue = intValue; // 自动转换
        System.out.println("int → long: " + longValue);

        // long → float
        float floatValue = longValue; // 自动转换
        System.out.println("long → float: " + floatValue);

        // float → double
        double doubleValue = floatValue; // 自动转换
        System.out.println("float → double: " + doubleValue);

        // char → int
        char charValue = 'A';
        int charToInt = charValue; // 自动转换，得到 ASCII 码
        System.out.println("char → int: " + charToInt);

        // 表达式中的自动提升
        byte a = 10;
        byte b = 20;
        // byte c = a + b; // ❌ 编译错误：byte 运算结果为 int
        int c = a + b; // ✅ 正确
        System.out.println("byte + byte = int: " + c);
    }
}
```

### 强制类型转换（显式）

从大类型到小类型的转换，可能丢失精度或溢出。

```java
public class ForceConversion {
    public static void main(String[] args) {
        // double → int
        double pi = 3.14159;
        int intPi = (int) pi; // 强制转换，丢弃小数部分
        System.out.println("double → int: " + intPi); // 3

        // int → byte（可能溢出）
        int largeValue = 130;
        byte byteValue = (byte) largeValue; // 强制转换
        System.out.println("int → byte: " + byteValue); // -126（溢出）

        // float → long
        float f = 123.45f;
        long l = (long) f; // 强制转换
        System.out.println("float → long: " + l); // 123

        // 字符与数字的转换
        int num = 65;
        char ch = (char) num; // int → char
        System.out.println("int → char: " + ch); // A

        char digit = '9';
        int digitValue = digit - '0'; // char → int（得到数字值）
        System.out.println("'9' - '0' = " + digitValue); // 9

        // 强制转换的注意事项
        System.out.println("\n强制转换的边界情况：");
        System.out.println("(int)1.7 = " + (int)1.7);   // 1
        System.out.println("(int)-1.7 = " + (int)-1.7); // -1
        System.out.println("(char)65536 = " + (char)65536); // 0（溢出）
    }
}
```

### 类型转换常见场景

```java
public class ConversionScenarios {
    public static void main(String[] args) {
        // 场景1：整数除法
        int a = 7;
        int b = 2;
        System.out.println("7 / 2 = " + (a / b));        // 3（整数除法）
        System.out.println("7.0 / 2 = " + (7.0 / b));    // 3.5（浮点除法）
        System.out.println("(double)a / b = " + ((double)a / b)); // 3.5

        // 场景2：运算时的类型提升
        byte x = 10;
        byte y = 20;
        byte result = (byte)(x + y); // 需要强制转换
        System.out.println("byte 运算: " + result);

        // 场景3：字符运算
        char c1 = 'A';
        char c2 = 'B';
        System.out.println("'A' + 'B' = " + (c1 + c2)); // 131（数值相加）
        System.out.println("'' + 'A' + 'B' = " + ("" + c1 + c2)); // AB（字符串拼接）

        // 场景4：避免溢出
        long maxInt = Integer.MAX_VALUE; // 2147483647
        System.out.println("maxInt + 1 = " + (maxInt + 1)); // -2147483648（溢出）
        System.out.println("maxInt + 1L = " + (maxInt + 1L)); // 2147483648（正确）
    }
}
```

## 常量

常量是值不能改变的变量，使用 `final` 关键字声明。

```java
public class Constants {
    // 声明常量的规范：全大写，下划线分隔
    public static final int MAX_SIZE = 100;
    public static final double PI = 3.141592653589793;
    public static final String DEFAULT_NAME = "Guest";

    public static void main(String[] args) {
        // 使用常量
        System.out.println("MAX_SIZE = " + MAX_SIZE);
        System.out.println("PI = " + PI);
        System.out.println("DEFAULT_NAME = " + DEFAULT_NAME);

        // 常量的特点
        final int DAYS_IN_WEEK = 7;
        System.out.println("\n一周有 " + DAYS_IN_WEEK + " 天");

        // DAYS_IN_WEEK = 8; // ❌ 编译错误：无法为 final 变量赋值

        // 常量通常用于配置和不变的值
        final double DISCOUNT_RATE = 0.1;
        double price = 100;
        double discountedPrice = price * (1 - DISCOUNT_RATE);
        System.out.printf("原价: %.2f, 折后价: %.2f%n", price, discountedPrice);
    }
}
```

## 字符串（String）基础

虽然 String 不是基本数据类型，但它是使用最频繁的类型。

```java
public class StringBasics {
    public static void main(String[] args) {
        // 字符串的创建方式
        String str1 = "Hello";              // 字符串字面量

        // 字符串的常用方法

        System.out.println("\n字符串常用方法：");

        System.out.println("首字母大写: " + text.toUpperCase()); // HELLO WORLD

        String s2 = "Hello";
        String s3 = new String("Hello");

        System.out.println("s1 == s2: " + (s1 == s2));     // true（指向同一对象）
        System.out.println("s1 == s3: " + (s1 == s3));     // false（不同对象）
        System.out.println("s1.equals(s3): " + s1.equals(s3)); // true（内容相同）

        // 字符串不可变性
        String original = "Hello";
        original = original + " World"; // 创建新对象，原对象不变
        System.out.println("\n不可变性: " + original);
    }
}
```

## 实战演练

### 示例 1：温度转换器

```java
import java.util.Scanner;

/**
 * 温度转换器：摄氏度与华氏度互相转换
 */
public class TemperatureConverter {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 温度转换器 ===");
        System.out.println("1. 摄氏度 → 华氏度");
        System.out.println("2. 华氏度 → 摄氏度");
        System.out.print("请选择转换类型 (1/2): ");

        int choice = scanner.nextInt();

        if (choice == 1) {
            System.out.print("请输入摄氏度: ");
            double celsius = scanner.nextDouble();
            double fahrenheit = celsius * 9 / 5 + 32;
            System.out.printf("%.2f°C = %.2f°F%n", celsius, fahrenheit);
        } else if (choice == 2) {
            System.out.print("请输入华氏度: ");
            double fahrenheit = scanner.nextDouble();
            double celsius = (fahrenheit - 32) * 5 / 9;
            System.out.printf("%.2f°F = %.2f°C%n", fahrenheit, celsius);
        } else {
            System.out.println("无效选择！");
        }

        scanner.close();
    }
}
```

### 示例 2：计算圆的面积和周长

```java
import java.util.Scanner;

/**
 * 计算圆的面积和周长
 */
public class CircleCalculator {

    public static void main(String[] args) {
        // 定义常量 PI
        final double PI = 3.141592653589793;

        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 圆的计算器 ===");
        System.out.print("请输入圆的半径: ");

        // 读取半径
        double radius = scanner.nextDouble();

        // 验证输入
        if (radius <= 0) {
            System.out.println("半径必须大于 0！");
        } else {
            // 计算周长：C = 2πr
            double circumference = 2 * PI * radius;

            // 计算面积：A = πr²
            double area = PI * radius * radius;

            // 输出结果
            System.out.println("\n计算结果：");
            System.out.printf("半径: %.2f%n", radius);
            System.out.printf("周长: %.2f%n", circumference);
            System.out.printf("面积: %.2f%n", area);
        }

        scanner.close();
    }
}
```

### 示例 4：秒数转换为时间格式

```java
import java.util.Scanner;

/**
 * 将秒数转换为 HH:MM:SS 格式
 */
public class SecondsToTime {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("=== 秒数转时间格式 ===");
        System.out.print("请输入秒数: ");

        int totalSeconds = scanner.nextInt();

        // 计算小时、分钟、秒
        int hours = totalSeconds / 3600;           // 总小时数
        int remainingSeconds = totalSeconds % 3600; // 剩余秒数
        int minutes = remainingSeconds / 60;        // 分钟数
        int seconds = remainingSeconds % 60;        // 剩余秒数

        // 格式化输出
        System.out.printf("%d 秒 = %02d:%02d:%02d%n",
            totalSeconds, hours, minutes, seconds);

        // 计算总天数
        int days = hours / 24;
        int remainingHours = hours % 24;
        if (days > 0) {
            System.out.printf("即 %d 天 %d 小时 %d 分钟 %d 秒%n",
                days, remainingHours, minutes, seconds);
        }

        scanner.close();
    }
}
```

```
3665 秒 = 01:01:05
即 1 天 1 小时 1 分钟 5 秒
```

## 最佳实践

### 1. 选择合适的数据类型

```java
// ❌ 不好的做法：使用过大或过小的类型
byte age = 25;                    // 容易溢出
long userId = 100;                // 浪费空间

// ✅ 好的做法：根据实际需求选择
int age = 25;                     // 年龄用 int 足够
long userId = 1000000000000L;     // 大 ID 使用 long
double price = 99.99;             // 价格用 double 确保精度
```

### 2. 避免精度丢失

```java
// ❌ 不好的做法：可能导致精度丢失
float result = 1.0f / 3.0f;       // 精度有限

// ✅ 好的做法：使用 double 提高精度
double result = 1.0 / 3.0;        // 更精确

// 需要高精度时，使用 BigDecimal
java.math.BigDecimal preciseResult =
    new java.math.BigDecimal("1.0").divide(new java.math.BigDecimal("3.0"));
```

### 3. 字符串比较使用 equals()

```java
String s1 = "Hello";
String s2 = new String("Hello");

// ❌ 错误：比较引用地址
if (s1 == s2) {
    // 条件不成立
}

// ✅ 正确：比较内容
if (s1.equals(s2)) {
    // 条件成立
}
```

### 4. 使用常量替代魔法数字

```java
// ❌ 不好的做法：魔法数字
if (score >= 60) {
    System.out.println("及格");
}

// ✅ 好的做法：使用常量
final int PASSING_SCORE = 60;
if (score >= PASSING_SCORE) {
    System.out.println("及格");
}
```

## 常见问题

### Q1：float 和 double 有什么区别？

**答**：

- `float`：4 字节，单精度，精度较低（约 6-7 位有效数字）
- `double`：8 字节，双精度，精度较高（约 15 位有效数字）

**建议**：日常开发中优先使用 `double`，精度更高且性能差异不大。

### Q2：为什么 char 类型可以参与数学运算？

**答**：因为 `char` 本质上是一个无符号的 16 位整数。每个字符都有对应的 Unicode 编码值，所以可以参与数学运算。

```java
char ch = 'A';  // Unicode 值为 65
int num = ch + 1;  // 结果为 66
System.out.println((char)num);  // 输出 'B'
```

### Q3：String 是基本数据类型吗？

**答**：不是。`String` 是引用类型（类），但它是使用最频繁的类型，所以有特殊的字面量表示方式。

### Q4：什么时候使用常量？

**答**：

- 配置值（如数据库连接参数）
- 数学常量（如 π）
- 业务规则中的固定值（如及格分数 60）
- 避免在代码中重复出现的"魔法数字"

### Q5：如何避免整数除法丢失小数？

**答**：将其中一个操作数转换为浮点类型。

```java
int a = 7;
int b = 2;

// ❌ 整数除法，结果为 3
double result1 = a / b;

// ✅ 浮点除法，结果为 3.5
double result2 = (double)a / b;
double result3 = a / (double)b;
double result4 = 1.0 * a / b;
```
