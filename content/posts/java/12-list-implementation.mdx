# List 接口与实现

## 前言

List 是最常用的集合接口，它是有序集合，允许重复元素。本文将详细介绍 List 接口的两种主要实现：ArrayList 和 LinkedList。

## List 接口

### List 特点

- 有序：元素按插入顺序存储
- 可重复：允许相同元素
- 可索引：通过索引访问元素

### List 常用方法

```java
import java.util.*;

public class ListMethods {

    public static void main(String[] args) {
        List<String> list = new ArrayList<>();

        // 添加元素
        list.add("A");
        list.add("B");
        list.add(1, "C");  // 在指定位置插入

        // 访问元素
        String item = list.get(1);  // "C"

        // 修改元素
        list.set(1, "D");

        // 删除元素
        list.remove(1);  // 按索引删除
        list.remove("A");  // 按元素删除

        // 查找元素
        int index = list.indexOf("B");

        // 截取子列表
        List<String> subList = list.subList(0, 2);

        // 转换为数组
        String[] array = list.toArray(new String[0]);
    }
}
```

## ArrayList

### 内部实现

基于动态数组，默认容量为 10，扩容时约为原来的 1.5 倍。

### ArrayList 示例

```java
import java.util.*;

public class ArrayListExample {

    public static void main(String[] args) {
        // 创建 ArrayList
        List<String> languages = new ArrayList<>();

        // 添加元素
        languages.add("Java");
        languages.add("Python");
        languages.add("JavaScript");

        // 使用 for 循环遍历
        for (int i = 0; i < languages.size(); i++) {
            System.out.println(languages.get(i));
        }

        // 使用 for-each 遍历
        for (String lang : languages) {
            System.out.println(lang);
        }

        // 使用 Iterator 遍历
        Iterator<String> iterator = languages.iterator();
        while (iterator.hasNext()) {
            System.out.println(iterator.next());
        }

        // 转换为数组
        String[] array = languages.toArray(new String[0]);
    }
}
```

### ArrayList 排序

```java
import java.util.*;

public class ArrayListSort {

    public static void main(String[] args) {
        List<Integer> numbers = new ArrayList<>();
        numbers.add(5);
        numbers.add(2);
        numbers.add(8);
        numbers.add(1);

        // 自然排序
        Collections.sort(numbers);
        System.out.println("升序：" + numbers);

        // 降序排序
        Collections.sort(numbers, Collections.reverseOrder());
        System.out.println("降序：" + numbers);

        // 对象排序
        List<Student> students = new ArrayList<>();
        students.add(new Student("张三", 25));
        students.add(new Student("李四", 20));

        // 按年龄排序
        students.sort(Comparator.comparingInt(Student::getAge));
    }
}

class Student {
    private String name;
    private int age;

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public int getAge() {
        return age;
    }
}
```

## LinkedList

### 内部实现

基于双向链表，每个节点包含数据、前驱指针和后继指针。

### LinkedList 特有方法

```java
import java.util.*;

public class LinkedListExample {

    public static void main(String[] args) {
        LinkedList<String> list = new LinkedList<>();

        // 添加元素
        list.add("A");
        list.addFirst("B");  // 在头部添加
        list.addLast("C");   // 在尾部添加

        // 获取元素
        String first = list.getFirst();
        String last = list.getLast();

        // 删除元素
        list.removeFirst();
        list.removeLast();

        // 实现栈
        list.push("D");      // 入栈
        String top = list.pop();  // 出栈
        String peek = list.peek();  // 查看栈顶

        // 实现队列
        list.offer("E");    // 入队
        String front = list.poll();  // 出队
    }
}
```

## ArrayList vs LinkedList

| 特性     | ArrayList | LinkedList |
| -------- | --------- | ---------- |
| 底层结构 | 动态数组  | 双向链表   |
| 随机访问 | O(1)      | O(n)       |
| 插入删除 | O(n)      | O(1)       |
| 内存占用 | 较少      | 较多       |
| 使用场景 | 频繁查询  | 频繁增删   |

## 实战演练

### 示例 1：待办事项列表

```java
import java.util.*;

public class TodoList {

    private List<String> tasks = new ArrayList<>();

    public void addTask(String task) {
        tasks.add(task);
        System.out.println("添加任务：" + task);
    }

    public void completeTask(int index) {
        if (index >= 0 && index < tasks.size()) {
            String completed = tasks.remove(index);
            System.out.println("完成任务：" + completed);
        } else {
            System.out.println("无效的任务编号");
        }
    }

    public void showTasks() {
        if (tasks.isEmpty()) {
            System.out.println("没有待办任务");
        } else {
            for (int i = 0; i < tasks.size(); i++) {
                System.out.println((i + 1) + ". " + tasks.get(i));
            }
        }
    }

    public static void main(String[] args) {
        TodoList todoList = new TodoList();
        todoList.addTask("学习 Java");
        todoList.addTask("完成作业");
        todoList.showTasks();
        todoList.completeTask(0);
        todoList.showTasks();
    }
}
```

### 示例 2：实现简单栈

```java
import java.util.*;

public class SimpleStack<T> {

    private LinkedList<T> stack = new LinkedList<>();

    public void push(T item) {
        stack.push(item);
    }

    public T pop() {
        if (stack.isEmpty()) {
            throw new EmptyStackException();
        }
        return stack.pop();
    }

    public T peek() {
        if (stack.isEmpty()) {
            throw new EmptyStackException();
        }
        return stack.peek();
    }

    public boolean isEmpty() {
        return stack.isEmpty();
    }

    public int size() {
        return stack.size();
    }

    public static void main(String[] args) {
        SimpleStack<Integer> stack = new SimpleStack<>();

        stack.push(1);
        stack.push(2);
        stack.push(3);

        System.out.println("栈大小：" + stack.size());
        System.out.println("栈顶：" + stack.peek());

        while (!stack.isEmpty()) {
            System.out.println("弹出：" + stack.pop());
        }
    }
}
```

## 最佳实践

### 1. 预估容量

```java
// ✅ 好的做法
List<String> list = new ArrayList<>(1000);

// ❌ 可能导致多次扩容
List<String> list = new ArrayList<>();
```

### 2. 使用增强 for 循环

```java
// ✅ 好的做法
for (String item : list) {
    System.out.println(item);
}

// ✅ 传统 for 循环（需要索引时）
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}
```

### 3. 遍历时删除元素

```java
// ❌ 错误：使用 for-each 删除
for (String item : list) {
    if (item.equals("A")) {
        list.remove(item);  // ConcurrentModificationException
    }
}

// ✅ 正确：使用 Iterator
Iterator<String> iterator = list.iterator();
while (iterator.hasNext()) {
    String item = iterator.next();
    if (item.equals("A")) {
        iterator.remove();
    }
}
```

## 常见问题

### Q1：什么时候用 ArrayList？

**答**：需要频繁访问元素，较少在中间插入删除。

### Q2：ArrayList 的扩容机制？

**答**：默认容量 10，扩容时 newCapacity = oldCapacity + (oldCapacity >> 1)。

### Q3：LinkedList 实现了哪些接口？

**答**：List、Deque，可以当作列表、栈、队列使用。
