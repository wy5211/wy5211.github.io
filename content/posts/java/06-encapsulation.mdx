# 封装与访问控制

## 前言

封装是面向对象编程的三大特征之一。它隐藏对象的实现细节，只暴露必要的接口。良好的封装可以提高代码的安全性和可维护性。本文将详细介绍 Java 的访问控制机制和封装的实现方式。

## 封装的概念

### 什么是封装

封装是将数据（属性）和操作数据的方法（行为）绑定在一起，并隐藏内部实现细节。

```
封装示意图：

外部世界                  对象
┌────────┐              ┌────────────────────┐
│        │  ──请求──→   │  私有数据          │
│  用户   │              │  ─────────────    │
│        │  ←──响应───  │  公共方法          │
└────────┘              └────────────────────┘
                         不能直接访问内部数据
```

### 封装的好处

1. **隐藏实现细节**：外部不知道内部如何实现
2. **数据安全**：防止数据被意外修改
3. **易于维护**：修改内部实现不影响外部代码
4. **降低耦合**：模块间依赖减少

## 访问控制符

Java 提供了四种访问控制符：

| 修饰符                | 同一类 | 同一包 | 子类 | 所有类 |
| --------------------- | ------ | ------ | ---- | ------ |
| `private`             | ✅     | ❌     | ❌   | ❌     |
| `default`（无修饰符） | ✅     | ✅     | ❌   | ❌     |
| `protected`           | ✅     | ✅     | ✅   | ❌     |
| `public`              | ✅     | ✅     | ✅   | ✅     |

### 访问级别演示

```java
// 文件：com/example/AccessDemo.java
package com.example;

public class AccessDemo {
    public int publicVar = 1;        // 公共的
    protected int protectedVar = 2;  // 受保护的
    int defaultVar = 3;              // 默认的
    private int privateVar = 4;      // 私有的

    public void publicMethod() {
        System.out.println("公共方法");
    }

    protected void protectedMethod() {
        System.out.println("受保护方法");
    }

    void defaultMethod() {
        System.out.println("默认方法");
    }

    private void privateMethod() {
        System.out.println("私有方法");
    }

    // 同一类中可以访问所有成员
    public void testAccess() {
        System.out.println(publicVar);
        System.out.println(protectedVar);
        System.out.println(defaultVar);
        System.out.println(privateVar);

        publicMethod();
        protectedMethod();
        defaultMethod();
        privateMethod();
    }
}
```

```java
// 同一包中的类
package com.example;

public class SamePackageTest {
    public static void main(String[] args) {
        AccessDemo demo = new AccessDemo();

        // ✅ 可以访问 public、protected、default
        System.out.println(demo.publicVar);
        System.out.println(demo.protectedVar);
        System.out.println(demo.defaultVar);

        // ❌ 不能访问 private
        // System.out.println(demo.privateVar);  // 编译错误
    }
}
```

```java
// 不同包中的子类
package com.other;

import com.example.AccessDemo;

public class ChildClass extends AccessDemo {
    public void test() {
        // ✅ 可以访问 public、protected
        System.out.println(publicVar);
        System.out.println(protectedVar);

        // ❌ 不能访问 default、private
        // System.out.println(defaultVar);   // 编译错误
        // System.out.println(privateVar);   // 编译错误
    }
}
```

### private 的使用

```java
public class BankAccount {

    private double balance;  // 私有字段，外部不能直接访问

    public void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
        }
    }

    public void withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
        }
    }

    public double getBalance() {
        return balance;  // 只读访问
    }
}
```

### protected 的使用

```java
public class Parent {

    protected String familyName = "张";

    protected void protectedMethod() {
        System.out.println("受保护的方法");
    }
}

public class Child extends Parent {

    public void test() {
        // 子类可以访问父类的 protected 成员
        System.out.println("姓：" + familyName);
        protectedMethod();
    }
}
```

## 封装的实现

### 封装步骤

1. 将属性设为 `private`
2. 提供公共的 `getter` 和 `setter` 方法
3. 在方法中添加验证逻辑

### 示例：学生类

```java
public class Student {

    // 私有属性
    private String name;
    private int age;
    private double score;

    // 构造方法
    public Student(String name, int age) {
        setName(name);  // 使用 setter 进行验证
        setAge(age);
    }

    // Getter 方法
    public String getName() {
        return name;
    }

    // Setter 方法：添加验证
    public void setName(String name) {
        if (name == null || name.trim().isEmpty()) {
            throw new IllegalArgumentException("姓名不能为空");
        }
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("年龄范围：0-150");
        }
        this.age = age;
    }

    public double getScore() {
        return score;
    }

    public void setScore(double score) {
        if (score < 0 || score > 100) {
            throw new IllegalArgumentException("成绩范围：0-100");
        }
        this.score = score;
    }

    // 业务方法
    public void displayInfo() {
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.printf("成绩：%.1f%n", score);
    }
}
```

### 封装的好处演示

```java
public class EncapsulationDemo {

    public static void main(String[] args) {
        Student student = new Student("张三", 20);

        // ✅ 使用 setter 方法，会进行验证
        student.setScore(85);

        // ❌ 直接设置无效值会抛出异常
        try {
            student.setAge(-5);  // 非法年龄
        } catch (IllegalArgumentException e) {
            System.out.println("捕获异常：" + e.getMessage());
        }

        // ✅ 使用 getter 方法安全访问
        System.out.println("年龄：" + student.getAge());

        // ❌ 无法直接访问私有字段
        // student.age = -10;  // 编译错误
        // student.name = "李四";  // 编译错误
    }
}
```

## static 关键字

`static` 关键字用于创建类级别的成员，而不是实例成员。

### 静态变量

```java
public class Counter {

    private int count = 0;           // 实例变量
    private static int totalCount = 0;  // 静态变量（类变量）

    public Counter() {
        count++;
        totalCount++;
        System.out.println("实例计数：" + count);
        System.out.println("总计数：" + totalCount);
    }

    public static void main(String[] args) {
        Counter c1 = new Counter();
        Counter c2 = new Counter();
        Counter c3 = new Counter();

        // 静态变量被所有对象共享
        System.out.println("总访问次数：" + Counter.totalCount);
    }
}
```

### 静态方法

```java
public class MathUtils {

    // 静态方法：不需要创建对象即可调用
    public static int add(int a, int b) {
        return a + b;
    }

    public static int max(int a, int b) {
        return a > b ? a : b;
    }

    // 静态方法只能访问静态成员
    private static int count = 0;

    public static int getCount() {
        return count;  // ✅ 可以访问静态变量
        // return instanceVar;  // ❌ 不能访问实例变量
    }

    public static void main(String[] args) {
        // 直接通过类名调用
        int sum = MathUtils.add(5, 3);
        int max = MathUtils.max(10, 20);

        System.out.println("和：" + sum);
        System.out.println("最大值：" + max);
    }
}
```

### 静态代码块

```java
public class StaticBlock {

    private static int[] lookupTable;

    // 静态代码块：类加载时执行一次
    static {
        System.out.println("初始化静态代码块");
        lookupTable = new int[10];
        for (int i = 0; i < 10; i++) {
            lookupTable[i] = i * i;
        }
    }

    public static int getSquare(int num) {
        if (num >= 0 && num < 10) {
            return lookupTable[num];
        }
        return num * num;
    }

    public static void main(String[] args) {
        System.out.println("平方表：");
        for (int i = 0; i < 5; i++) {
            System.out.println(i + "² = " + getSquare(i));
        }
    }
}
```

### 静态导入（Java 5+）

```java
import static java.lang.Math.*;
import static java.lang.System.out;

public class StaticImport {

    public static void main(String[] args) {
        // 直接使用静态方法和字段
        out.println("PI = " + PI);
        out.println("sqrt(16) = " + sqrt(16));
        out.println("pow(2, 3) = " + pow(2, 3));
    }
}
```

## 单例模式

单例模式确保一个类只有一个实例。

### 饿汉式

```java
public class EagerSingleton {

    // 类加载时就创建实例
    private static final EagerSingleton INSTANCE = new EagerSingleton();

    // 私有构造方法
    private EagerSingleton() {
        System.out.println("创建单例对象");
    }

    // 提供公共访问方法
    public static EagerSingleton getInstance() {
        return INSTANCE;
    }

    public void doSomething() {
        System.out.println("执行业务逻辑");
    }
}
```

### 懒汉式（线程安全）

```java
public class LazySingleton {

    private static LazySingleton instance;

    // 私有构造方法
    private LazySingleton() {
        System.out.println("创建单例对象");
    }

    // 同步方法，线程安全
    public static synchronized LazySingleton getInstance() {
        if (instance == null) {
            instance = new LazySingleton();
        }
        return instance;
    }

    public void doSomething() {
        System.out.println("执行业务逻辑");
    }
}
```

### 双重检查锁（推荐）

```java
public class DoubleCheckSingleton {

    // volatile 确保多线程环境下的可见性
    private static volatile DoubleCheckSingleton instance;

    private DoubleCheckSingleton() {
        System.out.println("创建单例对象");
    }

    public static DoubleCheckSingleton getInstance() {
        if (instance == null) {  // 第一次检查
            synchronized (DoubleCheckSingleton.class) {
                if (instance == null) {  // 第二次检查
                    instance = new DoubleCheckSingleton();
                }
            }
        }
        return instance;
    }
}
```

### 静态内部类（最优方案）

```java
public class StaticInnerSingleton {

    private StaticInnerSingleton() {
        System.out.println("创建单例对象");
    }

    // 静态内部类持有实例
    private static class Holder {
        private static final StaticInnerSingleton INSTANCE =
            new StaticInnerSingleton();
    }

    public static StaticInnerSingleton getInstance() {
        return Holder.INSTANCE;
    }
}
```

## 实战演练

### 示例 1：配置管理器

```java
import java.util.Properties;
import java.io.*;

/**
 * 单例配置管理器
 */
public class ConfigManager {

    private static ConfigManager instance;
    private Properties config;

    private ConfigManager() {
        config = new Properties();
        loadConfig();
    }

    public static ConfigManager getInstance() {
        if (instance == null) {
            synchronized (ConfigManager.class) {
                if (instance == null) {
                    instance = new ConfigManager();
                }
            }
        }
        return instance;
    }

    private void loadConfig() {
        // 模拟加载配置
        config.setProperty("app.name", "MyApplication");
        config.setProperty("app.version", "1.0.0");
        config.setProperty("db.url", "jdbc:mysql://localhost:3306/mydb");
        System.out.println("配置加载完成");
    }

    public String getProperty(String key) {
        return config.getProperty(key);
    }

    public void setProperty(String key, String value) {
        config.setProperty(key, value);
    }

    public void displayConfig() {
        System.out.println("=== 配置信息 ===");
        for (String key : config.stringPropertyNames()) {
            System.out.println(key + " = " + config.getProperty(key));
        }
    }

    public static void main(String[] args) {
        ConfigManager config1 = ConfigManager.getInstance();
        config1.displayConfig();

        ConfigManager config2 = ConfigManager.getInstance();
        System.out.println("两个实例相同：" + (config1 == config2));

        // 修改配置
        config2.setProperty("user.name", "张三");
        System.out.println("用户名：" + config1.getProperty("user.name"));
    }
}
```

### 示例 2：计数器工具

```java
/**
 * 计数器工具类
 */
public class CounterUtils {

    private static int totalCount = 0;
    private static int successCount = 0;
    private static int failureCount = 0;

    // 私有构造方法，防止实例化
    private CounterUtils() {
    }

    public static void incrementTotal() {
        totalCount++;
    }

    public static void incrementSuccess() {
        successCount++;
        totalCount++;
    }

    public static void incrementFailure() {
        failureCount++;
        totalCount++;
    }

    public static int getTotalCount() {
        return totalCount;
    }

    public static int getSuccessCount() {
        return successCount;
    }

    public static int getFailureCount() {
        return failureCount;
    }

    public static double getSuccessRate() {
        if (totalCount == 0) {
            return 0.0;
        }
        return (double) successCount / totalCount * 100;
    }

    public static void displayStatistics() {
        System.out.println("=== 统计信息 ===");
        System.out.println("总次数：" + totalCount);
        System.out.println("成功次数：" + successCount);
        System.out.println("失败次数：" + failureCount);
        System.out.printf("成功率：%.2f%%%n", getSuccessRate());
    }

    public static void reset() {
        totalCount = 0;
        successCount = 0;
        failureCount = 0;
        System.out.println("计数器已重置");
    }

    public static void main(String[] args) {
        // 模拟操作
        for (int i = 0; i < 100; i++) {
            if (Math.random() > 0.3) {
                incrementSuccess();
            } else {
                incrementFailure();
            }
        }

        displayStatistics();
    }
}
```

### 示例 3：工具类集合

```java
/**
 * 字符串工具类
 */
public class StringUtils {

    // 防止实例化
    private StringUtils() {
        throw new UnsupportedOperationException("工具类不能实例化");
    }

    // 判断字符串是否为空
    public static boolean isEmpty(String str) {
        return str == null || str.trim().isEmpty();
    }

    // 判断字符串是否不为空
    public static boolean isNotEmpty(String str) {
        return !isEmpty(str);
    }

    // 反转字符串
    public static String reverse(String str) {
        if (str == null) {
            return null;
        }
        return new StringBuilder(str).reverse().toString();
    }

    // 统计子串出现次数
    public static int countOccurrences(String str, String sub) {
        if (isEmpty(str) || isEmpty(sub)) {
            return 0;
        }

        int count = 0;
        int index = 0;
        while ((index = str.indexOf(sub, index)) != -1) {
            count++;
            index += sub.length();
        }
        return count;
    }

    public static void main(String[] args) {
        String text = "hello world hello java";

        System.out.println("是否为空：" + isEmpty(""));
        System.out.println("是否不为空：" + isNotEmpty(text));
        System.out.println("反转：" + reverse(text));
        System.out.println("hello 出现次数：" + countOccurrences(text, "hello"));
    }
}
```

## 最佳实践

### 1. 除非必要，否则字段应该是 private

```java
// ✅ 好的做法
public class Person {
    private String name;
    private int age;
}

// ❌ 不好的做法
public class Person {
    public String name;
    public int age;
}
```

### 2. 工具类应该私有化构造方法

```java
// ✅ 好的做法
public final class Utils {
    private Utils() {
        throw new UnsupportedOperationException();
    }

    public static void utilityMethod() {
    }
}
```

### 3. 静态方法只访问静态成员

```java
public class Example {
    private static int staticVar;
    private int instanceVar;

    public static void staticMethod() {
        System.out.println(staticVar);  // ✅
        // System.out.println(instanceVar);  // ❌ 编译错误
    }
}
```

### 4. 谨慎使用可变静态变量

```java
// ❌ 可能导致线程安全问题
public class BadExample {
    public static int counter;
}

// ✅ 使用线程安全的方式
public class GoodExample {
    private static final AtomicInteger counter = new AtomicInteger(0);
}
```

## 常见问题

### Q1：什么时候使用 static？

**答**：

- 需要在所有对象间共享数据
- 不需要访问对象状态的工具方法
- 常量定义
- 单例模式

### Q2：静态方法能访问实例成员吗？

**答**：不能。静态方法属于类，实例成员属于对象。

### Q3：private 和 protected 有什么区别？

**答**：

- `private`：只能在同类中访问
- `protected`：可以在同类、同包、子类中访问

### Q4：为什么要使用 getter/setter？

**答**：

- 控制访问权限
- 添加验证逻辑
- 便于维护和修改
- 支持序列化框架

### Q5：单例模式的最佳实现？

**答**：静态内部类方式最优，既保证线程安全，又实现懒加载。
