## 前言

当应用需要处理耗时操作（如发送邮件、生成报表）时，使用同步方式会阻塞请求。异步任务队列是解决方案。

**本文将带你：**

1. 理解任务队列的概念
2. 配置 BullMQ + Redis
3. 创建任务处理器
4. 实现邮件发送队列
5. 任务重试和错误处理

---

## 一、安装依赖

```bash
pnpm add @nestjs/bullmq bullmq ioredis
```

## 二、配置 Redis

使用 Docker 启动 Redis：

```yaml
# docker-compose.yml 添加
redis:
  image: redis:7-alpine
  ports:
    - "6379:6379"
```

## 三、创建 BullMQ 模块

### 3.1 配置 BullMQ

```typescript
import { BullMQModule } from "@nestjs/bullmq";

@Module({
  imports: [
    BullMQModule.forRoot({
      connection: {
        host: "localhost",
        port: 6379,
      },
    }),
  ],
})
export class AppModule {}
```

### 3.2 创建队列

```typescript
import { Module } from "@nestjs/common";
import { BullMQModule } from "@nestjs/bullmq";

@Module({
  imports: [
    BullMQModule.registerQueue({
      name: "email",
      defaultJobOptions: {
        attempts: 3,
        backoff: {
          type: "exponential",
          delay: 1000,
        },
      },
    }),
  ],
})
export class QueueModule {}
```

## 四、创建任务处理器

### 4.1 邮件处理器

```typescript
import { Processor, Process } from "@nestjs/bullmq";
import { Job } from "bullmq";
import { Logger } from "@nestjs/common";

@Processor("email")
export class EmailProcessor {
  private readonly logger = new Logger(EmailProcessor.name);

  @Process("welcome")
  async handleWelcomeEmail(job: Job<any>) {
    this.logger.log(`发送欢迎邮件给: ${job.data.email}`);

    // 模拟发送邮件
    await new Promise((resolve) => setTimeout(resolve, 2000));

    this.logger.log(`邮件发送成功: ${job.data.email}`);
    return { success: true };
  }

  @Process("reset-password")
  async handleResetPasswordEmail(job: Job<any>) {
    this.logger.log(`发送密码重置邮件给: ${job.data.email}`);

    await new Promise((resolve) => setTimeout(resolve, 2000));

    return { success: true };
  }
}
```

## 五、添加任务

```typescript
import { InjectQueue } from "@nestjs/bullmq";
import { Queue } from "bullmq";

@Injectable()
export class EmailService {
  constructor(@InjectQueue("email") private emailQueue: Queue) {}

  async sendWelcomeEmail(email: string) {
    await this.emailQueue.add("welcome", { email });
  }

  async sendResetPasswordEmail(email: string, token: string) {
    await this.emailQueue.add("reset-password", { email, token });
  }
}
```
