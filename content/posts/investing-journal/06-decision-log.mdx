## 你的记忆在骗你

回忆一下你过去一年最满意的一次投资决策。你大概很快就能想起来——"那次我在低点加仓了红利低波，后来涨了10%"。

再回忆一下你过去一年最后悔的一次决策。也需要想一想吗？

这不是偶然。**人的记忆会系统性地"自我美化"——你记住了判断对的那次，忘记了判断错的那十次。**

如果你问一个投资者"你的投资水平怎么样"，绝大多数人会高估自己。因为他们回忆中的成功案例比实际多，失败案例比实际少。

这不是道德问题，是认知偏差——"确认偏差"（Confirmation Bias）让你记住支持"我是个好投资者"的证据，忽略反面证据。

决策日志就是用来对抗这种记忆偏差的。

## 极简决策日志

决策日志不需要复杂。一个手机备忘录就够了。每次买入/卖出时，花一分钟写六行字：

```
日期：2025-08-15
操作：买入 红利低波ETF 5万
标的：563020
理由：PE到历史18%分位，按规则加仓
情绪：有点紧张（之前刚亏了3万）
预案：如果再跌15%怎么办→继续加，预算还有5万
```

就这六行。多了是负担，少了信息不够。

**关键字段说明：**

**"理由"必须具体。** 不能写"觉得该买了"——这是情绪不是理由。要写"PE到18%分位"或"偏离设定权重12%触发再平衡"。具体到可以回溯验证。

**"情绪"必须诚实。** 如果你是恐慌中加仓的，写"恐惧但按规则执行了"。如果你是追涨买入的，写"FOMO了，知道不该买但还是买了"。诚实的情绪记录让你在复盘时看清自己到底是在"执行策略"还是在"被情绪驱动"。

**"预案"是关键。** 在操作的同时写下"如果判断错了怎么办"。这个预案会在未来回来看时极其有价值——它记录了你做决策时的认知边界。

## 每季度复盘

每三个月拿出决策日志，问三个问题：

**问题一：决策逻辑有没有问题？**

回看三个月前的决策，以"后见之明"的视角审视。当时的理由现在看还成立吗？有没有遗漏的信息？

如果你写了"PE到18%分位加仓"，现在回头看PE确实在18%——逻辑没问题。如果你写了"感觉要反弹了"——这不是逻辑，是赌博。

**问题二：情绪有没有影响决策？**

看"情绪"字段。如果你的情绪频繁是"FOMO""恐慌""冲动"——说明你的决策被情绪主导了。如果大多是"冷静""按规则执行"——纪律在起作用。

**问题三：如果重来一次会怎么做？**

这个问题不是为了"纠错"——你改不了过去。是为了发现你的决策模式。如果你每次复盘都说"应该多买一点"——说明你系统性地过于保守。如果每次都说"不该追涨"——说明你的纪律执行有问题。

## 年度复盘的三个指标

每年年底，用三个量化指标评估自己的投资表现：

**指标一：实际收益率 vs 基准**

你的组合赚了多少？同期你的"基准"（比如70%红利指数+30%纳指100的定投组合）赚了多少？

如果实际收益跑输基准超过3个百分点，说明你的操作（择时/选标的）在拖后腿——不如老老实实定投指数。

如果跑赢基准，不要得意——先确认是alpha（你的判断对了）还是beta（市场整体涨了）。大多数时候是后者。

**指标二：换手率**

你一年交易了多少次？如果超过20次——大概率交易太频繁了。每次交易有成本（手续费+税费），还有判断失误的概率。大多数长期跑赢市场的投资者，年换手率低于5次。

**指标三：最大的对/错决策**

各选一个——今年做得最好的一次决策和最差的一次。分析原因。

好的决策通常有这些特征：基于规则而非情绪、在逆势中执行（别人恐惧时买入）、逻辑事后被验证。

差的决策通常有这些特征：追涨杀跌、基于"感觉"而非数据、超出自己能力圈。

## 从日志到系统

经过1-2年的记录，你会发现一个规律：**你反复犯同一类错误。**

也许是"每次市场大跌都想清仓"——恐惧型。也许是"每次涨了一波就想加仓"——贪婪型。也许是"总是找到'这次不一样'的理由不执行规则"——合理化型。

发现这个规律是决策日志最大的价值。不是让你"下次做对"——那太乐观了。是让你**知道自己是怎么犯错的**。认知自己的模式，比消除错误更重要。

因为错误永远无法完全消除。但如果你知道"我每次在X情况下容易犯错"，你可以在X出现时暂停一下——"等等，上次这种情况我犯错了，这次按规则来，不凭感觉。"

这就是从"经验"到"系统"的进化路径：记录 → 发现模式 → 在弱点出现时主动对抗 → 逐渐减少犯错频率。

## 一个真实的复盘案例

以下是模拟的年度复盘（基于红利低波+纳指100组合的投资者）：

```
2025年度复盘

指标一：实际收益 +3.2% vs 基准 +5.8%
→ 跑输2.6个百分点。主要拖累：3月在纳指回调时恐慌减仓，错过了后续反弹。

指标二：换手率 14次
→ 偏高。其中8次是"调仓试探"——加一点、减一点、再加一点。本质上是在犹豫。

指标三：
最佳决策：8月红利低波PE到15%分位时加仓，之后反弹了8%
最差决策：3月纳指回调时"先跑再说"，减仓20%，错过了20%的反弹

模式发现：
- 恐惧型决策占主导——14次操作中有6次是恐慌驱动的
- "先跑再说"是我最大的认知盲点——每次大跌都想先卖再买回来，但实际上几乎从来没有成功"买回来"
- 下年度改进：设"强制冷静期"——任何卖出操作必须延迟24小时执行
```

这种复盘的价值不在于"明年的收益率会更高"（不一定）。在于**你对自己的投资行为有了清晰认知**——知道自己擅长什么、不擅长什么。这个认知本身就能帮你少亏钱。
