## 评分 25 分，梭哈吗

上一篇 [多因子评分](/posts/valuation-toolkit/04-multi-factor-score) 给了你一个工具：把 PE/PB/PS 的分位数加权打分，输出 0-100 的估值温度计。

假设你刚给自己的红利低波 ETF 跑了一遍评分——25 分，低估区。你的第一反应可能是：「这么便宜，赶紧加仓。」

但等一下。在按下买入键之前，还有几个问题没回答：

- 你现在持有多少红利低波？占总仓位的 30% 还是 5%？
- 你的买入成本是多少？浮盈 20% 还是浮亏 15%？
- 如果买了之后它再跌 10%（低估还能再跌），你能不能扛住？
- 你的钱是从哪来的？卖了科创 50 来买，还是新增资金？

**估值告诉你「贵不贵」，但「买不买、买多少」是另一个问题。** 这篇就是搭从「判断贵贱」到「做出行动」之间的桥。

## 估值判断和买卖决策之间隔着什么

简单画一下决策链：

```
估值评分 → 仓位检查 → 风险评估 → 行动方案
```

大多数人只做了第一步（看估值），直接跳到第四步（买或卖）。中间的仓位和风险两步被跳过了——这就是为什么「估值对了，操作错了」。

### 仓位检查

你持有的这只 ETF 占总资产的比例，直接决定你的行动空间。

如果红利低波 ETF 已经占你总仓位的 40%，即使评分 25 分，你也不应该继续加仓——单一标的仓位过高的风险，比「估值便宜」更重要。

合理的单只 ETF 仓位上限取决于你的总资产和风险承受力。一般来说：

| 持仓占比 | 加仓空间         |
| -------- | ---------------- |
| 低于 10% | 可以加仓         |
| 10%-20%  | 小幅加仓         |
| 20%-30%  | 谨慎加仓或持有   |
| 超过 30% | 不加仓，考虑减仓 |

### 风险评估

假设你决定加仓红利低波 ETF。问自己：**如果买入后它再跌 15%，你能接受吗？**

不能接受 → 要么不买，要么少买。能接受 → 可以买。

这个问题的本质是**压力测试**。不管估值多低，如果后续下跌会严重影响你的生活和心态，那仓位就太大了。之前在 [回撤心理学](/posts/investing-journal/02-drawdown-psychology) 里拆过这个问题——你的真实回撤承受力通常比你以为的低。

## 三个常见决策场景

### 场景一：空仓看到低估（评分低于 30）

最理想的情况：你手里没这只 ETF，但评分告诉你它便宜。怎么买？

**不要一次性满仓。** 低估不等于马上涨——2022 年银行 ETF 评分就很低，但持有三年还是亏的（上一篇 [估值陷阱](/posts/valuation-toolkit/03-valuation-traps) 里讲过）。

分批建仓的具体方案：

```
第1个月：投入计划总金额的 30%
第2个月：投入 30%
第3个月：投入 40%
```

为什么是三个月而不是一天？因为三个月内如果价格继续跌，你的平均成本会被拉低。如果价格上涨了，你已经有了 30% 的仓位不会踏空。

这个策略的学名是「时间分散化」——和定投的逻辑一样：用时间换取更好的平均买入价格。

### 场景二：持仓看到高估（评分高于 85）

你持有科创 50 ETF，评分 75 分，接近高估区。怎么处理？

**不要清仓。** 高估不等于马上跌——2024 年纳指 100 的估值一直很高，但它就是一直在涨。提前卖掉可能错过最大的涨幅。

分批减仓的方案：

```
评分进入70-80区间：减仓 20%
评分进入80-90区间：再减 20%
评分超过90：再减 20%
保留至少40%底仓
```

为什么要保留底仓？因为「高估」可以持续很久。2021 年初创业板 PE 超过 60 倍，很多人清仓后看着它继续涨到 70 倍。保留底仓让你在高估的上涨中仍然能吃到一部分收益。

止盈纪律比止盈点位更重要。有规则地分批减仓，比「我感觉涨到顶了」靠谱得多。

### 场景三：持仓看到合理（评分 30-70）

这是最常见的情况。沪深 300 评分 50 分——不便宜也不贵。怎么办？

**什么都不做。**

这可能是投资中最难的事——不是因为它复杂，而是因为你忍不住想操作。

合理区间的正确打开方式：

- 评分在 30-50（偏低）：可以定投，但不急
- 评分在 50-70（偏高）：持有，不加仓

唯一可能触发操作的是**趋势信号**：估值合理 + 价格在 200 日均线之上 = 继续持有甚至小幅加仓；估值合理 + 价格跌破 200 日均线 = 可能是趋势转弱的信号，观望。

## 估值 + 趋势的组合拳

单看估值容易「左侧买太早、右侧卖太晚」。加上一个简单的趋势指标可以过滤掉一部分噪音。

**200 日均线（MA200）**是最常用的趋势指标。逻辑很简单：

- 价格在 MA200 之上 → 趋势向上，偏多头
- 价格在 MA200 之下 → 趋势向下，偏空头

把估值评分和趋势组合起来：

| 估值评分        | 价格 vs MA200 | 建议                                 |
| --------------- | ------------- | ------------------------------------ |
| 低于 30（低估） | 在 MA200 之上 | 加仓（低估 + 趋势好 = 最佳买点）     |
| 低于 30（低估） | 在 MA200 之下 | 小幅定投（低估但趋势差，可能是陷阱） |
| 30-70（合理）   | 在 MA200 之上 | 持有（趋势向好）                     |
| 30-70（合理）   | 在 MA200 之下 | 观望（趋势不明）                     |
| 高于 70（高估） | 在 MA200 之上 | 持有或小幅减仓（趋势好但估值偏高）   |
| 高于 70（高估） | 在 MA200 之下 | 减仓（高估 + 趋势差 = 双重警告）     |

这个框架不追求「最优买卖点」（没人能做到），追求的是**减少重大错误**。最大的错误是两类：在估值陷阱里越跌越买（低估 + 趋势差），和在泡沫顶部继续追涨（高估 + 趋势看似还在）。

## 一个完整的决策流程

把上面所有的内容串起来，当你看到一只 ETF 的估值评分时，按这个流程走：

```
Step 1：看评分
  → 低于 30：可能加仓
  → 30-70：持有
  → 高于 70：可能减仓

Step 2：检查仓位
  → 当前占比低于 20%？ → 有操作空间
  → 当前占比超过 30%？ → 不加仓，可能减仓

Step 3：检查趋势
  → 价格在 MA200 之上？ → 趋势支持操作
  → 价格在 MA200 之下？ → 趋势不支持，降低操作力度

Step 4：确定行动
  → 低估 + 仓位低 + 趋势好 = 分 3 个月加仓
  → 高估 + 仓位高 + 趋势差 = 分批减仓
  → 其他组合 = 持有，不做操作

Step 5：记录决策
  → 买了什么、为什么买、计划持有多久
  → 卖了什么、为什么卖、卖后资金放哪
```

Step 5 的「记录决策」容易被忽视，但它可能是整个流程中最重要的一步。之前在 [决策日志](/posts/investing-journal/06-decision-log) 里解释过原因：**没有记录就没有复盘，没有复盘就没有进步。**

## 这篇的核心结论

估值判断到买卖决策之间需要经过三道关卡：

- **仓位检查**：单一标的不要超过总仓位的 30%
- **风险评估**：假设买入后再跌 15%，你能接受吗？
- **趋势过滤**：用 200 日均线判断价格趋势，避免在陷阱里越跌越买

三个核心场景的操作原则：

- **空仓 + 低估**：分 3 个月定投建仓，不要梭哈
- **持仓 + 高估**：分批减仓但保留底仓，不要清仓
- **持仓 + 合理**：什么都不做，最难但也最重要

最后一条——**估值只是输入之一，仓位管理和情绪控制才是决定你最终赚多少的关键。** 下一篇把所有知识整合起来，搭一个实用的 [估值追踪体系](/posts/valuation-toolkit/06-build-valuation-dashboard)。
