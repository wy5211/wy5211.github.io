## 你真的了解纳指100 吗

买纳指100 ETF 的人很多，但能说清楚里面装了什么的，不多。

打开国内任意一只纳指100 ETF（如 513100）的持仓页，你会看到 100 只股票的名字。但你大概率只认识前十几个——苹果、微软、英伟达、亚马逊、谷歌、Meta、博通、特斯拉、Costco、百事可乐。

这十只股票占了整个指数权重的 50% 以上。

换句话说，你买的"纳指100"，有一半的钱压在了十家公司身上。这不是分散投资，这是精选持仓加了一层包装。

这篇要做的事：把纳指100 拆开给你看——成分、权重、行业分布、收益来源——让你知道自己每天盯着涨跌的那个数字，到底在代表什么。

## 纳指100 不是纳斯达克综合指数

先解决一个最基础的混淆。

**纳斯达克综合指数（IXIC / Nasdaq Composite）**：在纳斯达克交易所上市的**所有**股票（超过 3000 只），只要没退市就包含在内。这是"宽基"。

**纳斯达克100 指数（NDX / Nasdaq 100）**：从纳斯达克交易所上市的**非金融**公司里，选出市值最大、流动性最好的 **100 只**。这是"精选"。

区别在于：综合指数包含了一大堆小公司、仙股、甚至快退市的垃圾股。纳指100 只留大块头。你买的 ETF（不管是美国的 QQQ 还是国内的 513100）追踪的是纳指100，不是综合指数。

还有一个容易混的：纳指100 **排除了金融股**。所以银行、保险、证券公司统统不在里面。这点和标普500 不一样——标普500 是全行业覆盖。

## 成分拆解：一半的钱在科技

纳指100 的行业分布（截至 2026 年初的近似比例）：

| 行业       | 权重   | 代表公司                 |
| ---------- | ------ | ------------------------ |
| 信息技术   | 约 50% | 苹果、微软、英伟达、博通 |
| 通信服务   | 约 15% | 谷歌、Meta               |
| 非必需消费 | 约 15% | 亚马逊、特斯拉、Costco   |
| 医疗保健   | 约 5%  | 安进、吉利德             |
| 必需消费   | 约 5%  | 百事可乐、怪物饮料       |
| 公用事业   | 约 3%  |                          |
| 其他       | 约 7%  |                          |

科技（含通信服务）合计超过 60%。**纳指100 本质上是一只超级科技 ETF**，不是"美国股市的缩影"。如果你想分散配置，把纳指100 当宽基用，等于是重仓科技。

前十大成分股（近似权重）：

| 排名 | 公司            | 权重    |
| ---- | --------------- | ------- |
| 1    | 苹果            | 约 8-9% |
| 2    | 微软            | 约 8-9% |
| 3    | 英伟达          | 约 7-8% |
| 4    | 亚马逊          | 约 5-6% |
| 5    | 谷歌 (Alphabet) | 约 5-6% |
| 6    | Meta (Facebook) | 约 4-5% |
| 7    | 博通            | 约 4-5% |
| 8    | 特斯拉          | 约 3-4% |
| 9    | Costco          | 约 2-3% |
| 10   | 百事可乐        | 约 2%   |

前十大合计约 50-55%。这个集中度和科创50 差不多，但成分质量完全不同——科创50 的重仓股是中芯国际、金山办公这些还在成长期的公司，纳指100 的重仓股是全球市值最大的科技巨头。

## 纳指100 vs 科创50：同样叫成长，基因不同

之前在 [科创50 和创业板](/posts/growth-chip/01-what-are-they) 里拆过 A 股的成长指数。现在和纳指100 并排放，差异更明显：

| 维度          | 纳指100                | 科创50               |
| ------------- | ---------------------- | -------------------- |
| 成分股成熟度  | 成熟巨头，年利润百亿级 | 多数还在成长期或亏损 |
| PE            | 约 30-35 倍            | 约 170 倍            |
| 前十大权重    | 约 50-55%              | 约 45-50%            |
| 最大行业      | 科技（60%+）           | 半导体（40%）        |
| 回购文化      | 每年回购数千亿美元     | 几乎没有回购         |
| 近2年收益     | +49.8%                 | +197.5%              |
| 近2年最大回撤 | -24.0%                 | -19.3%               |
| 年化波动率    | 24.6%                  | 37.5%                |

两个观察：

**科创50 收益远高于纳指100，但波动率也远高（37.5% vs 24.6%）。** 科创50 涨得多是因为起点低、弹性大，但大跌的时候也狠。纳指100 的夏普比率（风险调整后收益）反而更稳健。

**纳指100 的 PE 只有科创50 的五分之一。** 不是因为纳指100 "便宜"，是因为它的成分股已经在赚大钱了——苹果每年净利润近千亿美元，微软也是。利润基数大，PE 天然低。科创50 的很多成分股利润极小甚至亏损，PE 被拉到天文数字。这个在 [PE 用好是利器](/posts/valuation/03-pe-deep-dive) 里详细讲过：PE 跨市场比没有意义，要在各自的历史区间里比。

## 收益拆解：涨了50%，钱从哪来

纳指100 ETF 近两年涨了 49.8%。这个收益怎么来的？

股票收益可以拆成三个来源：**盈利增长 + 估值变化 + 股票回购**。

**盈利增长**：纳指100 的成分股（尤其是苹果、微软、英伟达、谷歌）过去两年利润持续增长。英伟达因为 AI 芯片需求爆发，净利润从 2023 年的不到 300 亿美元飙到 2025 年的超过 700 亿美元。盈利增长是股价上涨最健康的动力——公司真的在赚更多钱。

**估值变化**：PE 从 2023 年底的约 25 倍扩张到 2026 年初的约 30-35 倍。市场愿意给更高的估值，部分原因是 AI 叙事——投资者相信这些公司的未来利润会更高。但估值扩张是"市场情绪"驱动的，如果叙事破裂，这部分涨幅会吐回去。

**股票回购**：这个是 A 股投资者最不熟悉的部分。美股公司有一个传统——赚了钱之后大量回购自家股票。苹果 2024 年宣布了 1100 亿美元的回购计划，谷歌和微软每年也回购数百亿美元。回购减少流通股数量，即使公司利润不变，每股收益（EPS）也会自动上升。**这是美股独有的"增长引擎"**，A 股公司几乎没有这个习惯（反而喜欢增发圈钱）。

三者的贡献大致是：盈利增长贡献了约一半的涨幅，估值扩张贡献了约三分之一，回购贡献了剩余部分。这意味着纳指100 的上涨有真实的盈利支撑，不全是泡沫——但估值扩张的部分确实是市场情绪，需要警惕。

## QQQ vs QQQM：追踪同一个指数，费率差三倍

如果你通过美股账户买纳指100，有两个选择：

- **QQQ**：最早发行的纳指100 ETF（1999 年上市），费率 0.20%，流动性最好，期权市场最活跃。
- **QQQM**： Invesco 2020 年推出的低费率版本，追踪同一个指数，费率 0.15%。

两者追踪的指数一模一样，区别只在费率。QQQM 是 Invesco 专门为了和 QQQ 做区分而推出的——同样的东西，每年少收 0.05%。

如果你是长期持有（不是频繁交易、不玩期权），买 QQQM 更划算。十年下来 0.05% 的费率差，在复利效应下不是小数目。

国内的纳指100 ETF（513100 等）费率通常在 0.5-0.8% 之间，比 QQQ/QQQM 贵不少，但优势是不需要美股账户、用人民币就能买。具体渠道对比后面会专门讲。

## 纳指100 和标普500 重复了多大

很多人同时持有纳指100 和标普500，但没意识到两者的重合度很高。

标普500 市值加权，前七大成分股和纳指100 的前七大几乎一样——苹果、微软、英伟达、亚马逊、谷歌、Meta、特斯拉。这七家在标普500 里占了约 30% 的权重，在纳指100 里占了约 45%。

如果你 50% 标普500 + 50% 纳指100，实际上你对这七家公司的暴露是 30%×0.5 + 45%×0.5 = 37.5%。**看起来分散在两个指数里，实际有超过三分之一的钱压在同七家公司身上。**

这算不算问题？取决于你的目标。如果你认为这七家公司未来会继续领跑（AI 革命的核心受益者），那重合是"集中优势"。如果你担心集中度风险，那同时持有两个指数的分散效果，没有你想的那么好。

集中度风险这件事，后面会用一整篇来拆。

## 这篇的核心结论

纳指100 是一只优秀的 ETF，但它不是万金油。买之前至少要知道三件事：

**它是科技集中持仓，不是美国全市场分散。** 60% 压在科技上，50% 压在十只股票上。如果你想分散，不能只靠它。

**它的上涨有真实盈利支撑，但也有估值扩张的水分。** 盈利增长是实打实的（尤其是英伟达），但 PE 从 25 扩到 30+ 是市场情绪，这部分涨幅随时可能回撤。

**它和标普500 高度重叠。** 同时持有两个不等于分散，反而可能让你在七只股票上的集中度更高。
