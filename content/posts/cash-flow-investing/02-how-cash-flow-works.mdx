上一篇聊了现金流指数为什么突然火了。这篇拆最底层的问题：**它到底用什么指标选股，为什么这个指标比我们熟悉的PE更好用？**

如果你连自由现金流是什么都不清楚，建议先去看红利系列[第6篇](/posts/dividend-investing/06-fcf-vs-dividend)。这里直接进入进阶内容。

## FCF Yield：一句话定义

自由现金流率（Free Cash Flow Yield，简称 FCF Yield）：

> **FCF Yield = 自由现金流 ÷ 企业价值**

分子很简单——自由现金流（经营现金流 − 资本支出），上一篇说过了。难点在分母：**企业价值（Enterprise Value，简称 EV）是什么？**

大部分人选股看PE（市盈率），分母用的是市值（股价 × 总股本）。FCF Yield用的是EV，两者有本质区别。

## 企业价值：PE看不到的那笔债

先看公式：

> **企业价值 = 市值 + 净负债**
>
> 净负债 = 总负债 − 现金及现金等价物

为什么要加净负债？用一个例子你就懂了。

假设你要收购一家公司。你付的钱不仅是买下它的股权（市值），还要承担它的债务。如果这家公司市值100亿，但账上还欠银行50亿，你实际上要花150亿才能完全拥有它。反过来，如果它市值100亿，账上还躺着50亿现金，你实际只花了50亿（100亿买下来，但手里多了50亿现金）。

**所以企业价值衡量的是"收购这家公司真正要花多少钱"，比市值更接近真实成本。**

这为什么重要？因为PE用市值做分母，会忽略公司的债务结构。两家公司市值一样、利润一样、PE一样，但一家负债累累、一家现金充裕——PE告诉你它们"一样便宜"，但FCF Yield会告诉你现金充裕的那家更便宜（EV更低，FCF Yield更高）。

这恰恰是A股最容易踩的坑之一：**看起来PE很低、很"便宜"的房地产和建筑公司，很多都是高负债撑起来的。** 用PE看它们很便宜，用FCF Yield看就知道它们不便宜。

## 用两家公司拆解

讲概念不如看数据。选两家你一定听过的A股公司，拆一下它们的数据。

**公司A：贵州茅台（600519）**

茅台2024年的关键数据（简化版，单位：亿元）：

| 指标         | 数值                             |
| ------------ | -------------------------------- |
| 净利润       | ~860                             |
| 经营现金流   | ~930                             |
| 资本支出     | ~50                              |
| 自由现金流   | ~880                             |
| 市值         | ~17,000                          |
| 净负债       | 负数（净现金，账上现金远超负债） |
| 企业价值(EV) | ~16,000                          |
| PE           | ~20x                             |
| FCF Yield    | ~5.5%                            |

茅台是典型的"现金奶牛"。经营现金流930亿，资本支出只需要50亿（酿酒不需要大规模建厂），自由现金流880亿。而且账上现金堆积如山，几乎没有有息负债——EV比市值还低。

FCF Yield大约5.5%，不算特别高，但考虑到茅台的确定性，这是一个很扎实的数据。

**公司B：某头部房企（我们不点名，你懂的）**

同样是2021年巅峰期的数据（简化版）：

| 指标                  | 数值     |
| --------------------- | -------- |
| 净利润                | ~400     |
| 经营现金流            | ~100     |
| 资本支出（拿地+在建） | ~200     |
| 自由现金流            | **-100** |
| 市值                  | ~2,000   |
| 净负债                | ~1,500   |
| 企业价值(EV)          | ~3,500   |
| PE                    | ~5x      |
| FCF Yield             | **负数** |

看到区别了吗？

PE只有5倍——看起来"极度便宜"，无数价值投资者在2021年用PE选股时买入了地产股。但FCF Yield是负的：经营现金流只有100亿（利润400亿里有300亿是没收回来的钱），资本支出要200亿，自由现金流直接是负的。

**用PE看，这家公司"便宜得离谱"。用FCF Yield看，这家公司根本不赚钱——它是在烧钱。**

后来的故事大家都知道了。

这就是FCF Yield的核心价值：**它能帮你避开PE制造的价值陷阱。** 那些利润很高、PE很低、但现金流稀烂的公司，在FCF Yield的筛选下无所遁形。

## 现金流指数的完整选股流程

理解了FCF Yield，就理解了现金流指数的选股逻辑。以中证800自由现金流指数为例，完整流程是这样的：

**第一步：确定样本空间。**

中证800 = 沪深300 + 中证500，即A股市值最大的800只股票。国证自由现金流指数和中证全指自由现金流指数的样本空间更大——覆盖整个A股市场。

**第二步：质量筛选。**

- **剔除金融行业**（银行、保险、券商）。为什么？因为金融行业的现金流结构特殊——银行的"经营现金流"主要是存款变化，和实体企业完全不是一回事。把金融放进来，FCF Yield的数值就没有可比性了。
- **剔除自由现金流为负的公司**。连续亏现金的公司，不管其他指标多好看，都不进候选池。

**第三步：计算FCF Yield，排序选股。**

对剩余的股票，计算每只的FCF Yield = 自由现金流 ÷ 企业价值。从高到低排序，选前100只。

**第四步：加权。**

这里有个细节值得注意：不同指数的加权方式不一样。

- **国证自由现金流指数**：按自由现金流金额加权。现金流越大的公司权重越高。这意味着大市值的现金奶牛（比如茅台、神华）会占据更大权重。
- **中证800自由现金流指数**：类似逻辑，但样本池限定在800只大盘股里。

加权方式直接影响指数的行业暴露和收益特征。下一章对比三大指数时会详细展开。

**第五步：定期调仓。**

季度调仓。每三个月重新计算一遍FCF Yield，踢掉排名下滑的，纳入新冒出来的。这个机制让指数自带"高抛低吸"——一只股票价格涨太多（EV变大），FCF Yield就会下降，排名靠后就会被踢出；一只股票跌太多（EV变小），FCF Yield就会上升，排名靠前就会被纳入。

这和红利低波的"股息率加权+定期调仓"逻辑如出一辙——都是用估值指标做锚，规则性地低买高卖。

## FCF Yield vs PE：一张表说清区别

|                    | PE（市盈率）                 | FCF Yield                                     |
| ------------------ | ---------------------------- | --------------------------------------------- |
| **分子**           | 净利润（权责发生制，可调节） | 自由现金流（收付实现制，难造假）              |
| **分母**           | 市值（忽略负债结构）         | 企业价值（包含净负债）                        |
| **能发现什么**     | 盈利相对于股价是否便宜       | 现金创造力相对于全盘成本是否划算              |
| **最大盲区**       | 高负债公司、利润注水公司     | 资本密集型行业（资本支出大，FCF天然低）       |
| **适合什么公司**   | 盈利稳定、负债低的成熟企业   | 现金流稳定、资本支出可控的企业                |
| **不适合什么公司** | 周期股（利润波动大，PE失真） | 成长股（高增长需要高资本支出，FCF天然低或负） |

最后一点值得展开：**FCF Yield不适合评估成长股。** 一家正在高速扩张的公司，每年砸大量资本支出建产能、铺渠道，自由现金流可能是负的——但它不一定是坏公司。亚马逊在2010年代的前几年，FCF很低甚至为负，但后来成了全球市值最高的公司之一。

所以现金流指数天然偏向**成熟、稳定、资本支出低的行业**——这就是为什么它们重仓能源、公用事业、消费，而不碰科技成长股。这不是缺陷，是特征。你买现金流指数，买的就是这个特征。

## 真正重要的结论

这篇的核心不是让你去手算每只股票的FCF Yield，而是建立一种直觉：

1. **看公司别只看利润。** 利润是观点，现金流才是事实。一只PE很低但FCF Yield为负的股票，大概率是价值陷阱。
2. **企业价值比市值更诚实。** 高负债公司用PE看很便宜，用EV看就露馅了。
3. **FCF Yield有适用边界。** 它适合成熟行业的现金奶牛，不适合评估高增长的成长股。
