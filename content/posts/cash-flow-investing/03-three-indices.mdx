## 三个发现

**发现一：前三大重仓高度一致，但排序不同。**

三个指数的前三名都是上汽集团、中国海油、格力电器。这说明这三家公司的FCF Yield在全市场都是顶级的——不管你怎么选样本池，它们都会排在最前面。

但排序有差异：国证版把上汽集团排第一（9.90%），中证800和全指版把中国海油排第一。这背后是加权方式的细微差别。

**发现二：从第四名开始，三个指数走向了不同的路。**

国证版（159201）的第四名是中远海控（4.56%），权重骤降——前三名各占~9%，第四名突然跌到4.5%，集中度很高。

中证800版（159119）和全指版（159232）的第四名是**中国电信**，权重高达8.20%和7.66%——前三名加上中国电信，前四就占了37%。中国电信在国证版的前十里完全不见踪影。

这是一个重大差异。中国电信自由现金流确实不错（经营现金流稳定，资本支出在运营商中不算高），但国证指数可能用了不同的筛选规则把它排在了后面。

**发现三：159119和159232几乎一模一样。**

对比中证800版和全指版的前十大，10只股票完全相同，权重差异也在0.5个百分点以内。这其实不意外——中证全指虽然样本空间比中证800大（包含中小盘），但自由现金流率排在前面的本来就是大盘股，中小盘很难挤进前列。所以两个指数的实际持仓高度趋同。

**换句话说，159119和159232虽然跟踪不同指数，但实际投资效果几乎一样。** 如果你要在这两个里选，差异不在持仓，而在费率、规模和流动性（下一篇详聊）。

## 三个指数到底差在哪

理解了持仓差异，来看背后的编制规则差异。三个指数的核心区别有三个维度：

### 维度一：样本空间

| 指数               | 样本空间          | 大约覆盖               |
| ------------------ | ----------------- | ---------------------- |
| 中证800自由现金流  | 沪深300 + 中证500 | A股市值前800的大中盘   |
| 中证全指自由现金流 | 中证全指成分股    | 几乎全部A股（~5000只） |
| 国证自由现金流     | 国证2000等        | 全市场，覆盖面更广     |

样本空间的差异决定了**中小盘股能不能入选**。中证800版只能在最大的800只股票里选，中证全指和国证版理论上可以选入中小盘。

但实际效果我们看到——159119和159232的持仓几乎一样。这说明FCF Yield排名靠前的公司，天然以大盘股为主。中小盘公司即使FCF Yield高，因为绝对现金流金额小，在大盘股面前竞争力有限。

### 维度二：成分股数量

| 指数               | 成分股数量 |
| ------------------ | ---------- |
| 中证800自由现金流  | 100只      |
| 中证全指自由现金流 | 100只      |
| 国证自由现金流     | 100只      |

三个指数都选100只。这个数量设计是参考了海外的COWZ（也是100只），在分散度和因子纯度之间取了平衡。

### 维度三：加权方式

这是最关键的差异。三个指数在加权上都用了**自由现金流金额加权**（FCF越大的公司权重越高），但具体细节有差异：

- **国证版**：前三名集中度更高（上汽、海油、格力各~9.5%），第四名骤降到4.5%——说明它可能对权重上限做了更严格的控制，或者计算方式导致前三名和其他公司之间有断层。
- **中证800/全指版**：前四名权重接近（前三~9%，第四中国电信~8%），集中度稍低但头部更均匀。

加权方式直接决定了指数的收益特征：**集中度高的指数，在头部成分股表现好时涨得多，但跌的时候也跌得多。**

## 对你来说意味着什么

说了这么多，落到实操层面，有三个判断：

**第一，如果你选的是国证指数产品（159201、159222、159221等），你买的是一种"集中下注头部现金奶牛"的策略。** 前三大占了近30%，上汽、海油、格力的表现决定你大部分收益。集中度高的好处是因子纯度高，坏处是波动可能更大。

**第二，如果你选的是中证800或全指指数产品（159119、159232），持仓更分散，但和中国电信绑定更深。** 159119和159232的前四占了36%，其中中国电信一独就占了8%。如果你对中国电信的未来现金流不乐观，这个权重可能让你不舒服。

**第三，159119和159232在持仓层面几乎没区别。** 如果你在它们之间纠结，不用看持仓——比费率、规模和流动性就行了。

但不管选哪个，有一个共同特征需要注意：**这些指数的前十大持仓高度集中在能源、汽车、家电、航运、钢铁——全是周期性行业。** 这意味着现金流指数的收益和宏观经济周期高度绑定：经济好的时候这些公司现金流猛增，指数大涨；经济差的时候现金流萎缩，指数大跌。

这和红利低波重仓银行、煤炭是不同的周期暴露。下一篇聊产品全景对比时会更详细地拆这个点。
