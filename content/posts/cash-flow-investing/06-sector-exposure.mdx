上一篇聊了三个因子的逻辑差异。这篇往更深处走一步——**行业暴露。**

行业暴露决定了你的投资组合在经济周期的哪个阶段赚钱、哪个阶段亏钱。同样是"价值因子"，重仓银行和重仓制造业，面对同一个宏观事件，反应可能完全相反。

## 现金流指数的行业版图

从第3篇的前十大持仓数据，按行业归类一下159201（国证自由现金流，华夏）的头部持仓：

| 行业         | 代表持仓                                       | 合计权重 |
| ------------ | ---------------------------------------------- | -------- |
| 汽车及零部件 | 上汽集团(9.9%)、长城汽车(3.6%)、潍柴动力(2.9%) | ~16%     |
| 能源         | 中国海油(9.8%)                                 | ~10%     |
| 家电         | 格力电器(9.6%)                                 | ~10%     |
| 钢铁及有色   | 宝钢股份(3.3%)、中国铝业(3.0%)                 | ~6%      |
| 航运物流     | 中远海控(4.6%)                                 | ~5%      |
| 通信         | 中国联通(3.0%)                                 | ~3%      |
| 电气设备     | 正泰电器(2.7%)                                 | ~3%      |

前十大合计约53%。加上90只其他成分股，剩余47%分散在各行业中。

中证800版（159119）和全指版（159232）的行业版图类似，但多了一个显眼的板块——**通信**：中国电信权重高达8.2%，加上中国联通，通信行业合计超过10%。

## 一个关键特征：零金融

这是现金流指数和红利低波最大的行业分水岭。

红利低波（以中证红利低波动指数为例）的行业权重里，**银行一家就占30%以上**。工商银行、建设银行、交通银行、农业银行——四大行的分红是红利低波收益的核心来源。

现金流指数呢？**银行股零持仓。** 保险公司零持仓。券商零持仓。整个金融板块被一刀切掉。

原因在上一篇讲过——金融行业的现金流结构特殊（银行的"经营现金流"主要是存款变动，和实体企业完全不是一回事），放进来FCF Yield的数值就没有可比性。

这个差异的实际影响有多大？

2025年下半年，A股银行板块逆势走强（降息周期中银行净息差企稳、高股息资金抱团银行），红利低波因为30%+的银行仓位而获得了支撑。同期现金流指数没有银行仓位，少了这层缓冲，跌幅更大。

反过来，如果未来出现银行股回调（比如息差继续收窄、地产坏账暴露），红利低波会受到拖累，而现金流指数不受影响。

**你选红利还是现金流，在很大程度上是在选"要不要押注银行"。**

## 另一个特征：高度周期性

看现金流指数的前十大，有一个共同特征——**全是周期性行业**。

- **中国海油**：油价涨它赚得多，油价跌它赚得少
- **上汽集团、长城汽车**：汽车销量跟着经济周期和消费信心走
- **格力电器**：地产周期影响空调销售
- **中远海控**：航运运价和全球贸易周期高度绑定
- **宝钢股份、中国铝业**：大宗商品价格决定利润
- **牧原股份**：猪周期

这些公司有一个共同点：**它们的自由现金流在经济扩张期暴增，在经济收缩期骤降。**

这意味着现金流指数的收益弹性很大——经济好的时候，这些公司的FCF Yield飙升，指数涨得猛；经济差的时候，FCF骤降，股价杀估值，指数跌得也猛。

对比红利低波的重仓行业：

|               | 现金流指数 | 红利低波 |
| ------------- | ---------- | -------- |
| **银行**      | 0%         | 30%+     |
| **煤炭/能源** | ~10%       | ~15%     |
| **高速公路**  | 0%         | ~10%     |
| **制造业**    | ~25%       | ~5%      |
| **家电/消费** | ~10%       | ~5%      |
| **通信**      | ~10%       | 0%       |
| **钢铁/有色** | ~6%        | ~5%      |
| **航运/物流** | ~8%        | ~0%      |

红利低波的行业特征是"防御"——银行（利差收入稳定）、高速公路（收费站在那收钱）、煤炭（长协价锁定利润）。这些行业的现金流波动小，所以叫"低波"。

现金流指数的行业特征是"进攻型价值"——制造业、航运、有色，这些行业现金流绝对值大，但波动也大。指数不叫"低波"是有原因的。

从风险指标也能看出来。159201（国证FCF）近一年的年化标准差是**15.70%**，而红利低波ETF（563020）同期大约12-13%。差距3个百分点左右——听起来不多，但落实到实际体验上，15%的波动意味着一年内可能上下波动15%，而13%是13%。

## 互补还是重叠：一个具体场景

理解了行业差异，来看一个实际的问题：**同时持有红利低波和现金流指数，分散效果如何？**

想象三种市场场景：

**场景一：银行股大涨（如2025年下半年）**

- 红利低波：银行30%+仓位直接受益，涨
- 现金流指数：零银行仓位，不受益
- 组合效果：红利部分赚钱，现金流部分不赚不亏

**场景二：制造业复苏（如经济刺激政策出台）**

- 红利低波：制造业仓位少，小幅受益
- 现金流指数：制造业25%+仓位直接受益，涨
- 组合效果：现金流部分赚钱，红利部分小赚

**场景三：成长股牛市（如AI行情虹吸资金）**

- 红利低波：资金流出价值股，跌
- 现金流指数：同样被抽血，跌
- 组合效果：两个一起跌——这是最痛苦的场景

**场景四：全面熊市**

- 红利低波：低波属性提供一定保护，跌得少
- 现金流指数：周期性暴露使它跌得更多
- 组合效果：红利提供缓冲，现金流拖后腿

结论是：**红利和现金流在场景一和场景二里是互补的——一个涨另一个可能不涨，但至少不同跌。但在场景三和场景四里，它们是同跌的——因为本质上都是价值因子。**

这不是一个完美的对冲组合，但也不是简单的重复。它的价值在于**覆盖了更广的行业版图**——从银行到制造业，从高速公路到航运，两个因子加在一起，你基本上持有了A股"成熟企业"的全光谱。

## 真正的行业集中度风险

聊到这里，必须指出一个被很多人忽略的风险：**现金流指数自身的行业集中度偏高。**

前三大持仓（上汽+海油+格力）占了近30%。这三个分别代表汽车、能源、家电——如果同时碰到"汽车销量下滑 + 油价下跌 + 地产低迷拖累家电"的宏观组合，指数会承受巨大压力。

这不是假想的场景。2025年底到2026年初，这三件事在不同程度上同时发生了——汽车行业价格战侵蚀利润、油价从高位回落、地产链持续低迷。结果就是我们在第1篇看到的：所有现金流ETF都经历了20%以上的回撤。

**红利低波的行业集中（银行30%+）有风险，现金流指数的行业集中（汽车+能源+家电30%）也有风险——只是风险的来源不同。** 银行的风险是息差和坏账，制造业的风险是周期和需求。两个加在一起，不能消除行业集中风险，只是把它分散到了更多的行业上。

## 一个值得关注的特征：大盘价值

天天基金的风险分析显示，159201连续四个季度（2025Q2-2026Q1）的投资风格都是**"大盘价值"**。这不是巧合——FCF Yield排序天然选出的是大市值的成熟企业，因为只有大公司才能产生绝对值足够高的自由现金流。

这意味着现金流指数和沪深300有较高的重合度（前十大持仓基本都在沪深300里），但两者的加权逻辑完全不同——沪深300按市值加权（大公司权重高），现金流指数按FCF金额加权（现金奶牛权重高）。

在效果上，现金流指数相当于沪深300的一个"精选版"——从800只大盘股里选出100只现金流最好的，集中持有。收益弹性和波动都比沪深300大。

## 这篇的核心结论

1. **现金流指数的行业画像清晰：制造业+能源+家电+通信，零金融。** 和红利低波（银行+煤炭+高速）形成鲜明对比。
2. **两个因子互补的是行业暴露，不是风格暴露。** 行业上互补（银行vs制造业），但风格上同向（都是价值因子，成长牛市时一起跌）。
3. **现金流指数自身的行业集中度不低。** 前三大占30%，且全在周期性行业，经济下行时回撤可能比红利低波更大。
4. **同时持有红利和现金流 = 覆盖更广的价值股版图。** 但不能指望它们互相对冲——它们是"同一个大类里的不同细分"。
