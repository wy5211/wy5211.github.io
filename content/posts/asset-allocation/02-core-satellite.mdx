## 从100万开始

假设你有 100 万要配置。手里可选的标的有：纳指100、标普500、红利低波、现金流ETF、科创50。

上一篇讲了"为什么要分散"——降低波动率让你拿得住。这篇解决更具体的问题：**怎么分配比例。**

大多数人的做法是凭感觉。红利放多点，因为稳；纳指放少点，因为波动大。具体多少？看心情。

"核心-卫星策略"给了你一个框架来替代直觉。它不复杂，核心思想一句话：**大部分钱放低波动、长期持有的标的（核心），少部分钱放高弹性、有观点的标的（卫星）。** 关键是比例怎么定——不是拍脑袋，是先算清楚你能承受多大亏损。

## 核心仓位：让你在熊市不割肉

核心仓位的目标只有一个：**在市场暴跌的时候，组合整体回撤控制在你能承受的范围内。**

什么是"能承受"？每个人的答案不同，但可以用一个简单的测试：如果明天你的组合跌了 15%，你会不会恐慌卖出？如果会，那你的核心仓位比例不够。如果不会但每天焦虑到睡不着，那也不够。

核心标的的特征：

- **低波动**：年化波动率低于 15%。红利低波的年化波动约 14.5%，符合要求。纳指100 波动 24.6%，不符合。
- **长期正收益**：不需要每年赚，但拉长到 3-5 年应该有正回报。红利低波虽然近两年亏了 7%，但拉长到 5-10 年看是稳步向上的。
- **分红现金流**：分红提供"心理补偿"——即使价格下跌，每年还有分红到账，让你不至于觉得"一分钱没赚还亏了"。

你的红利低波 ETF 完全符合核心仓位的标准。标普500 虽然波动偏高（20%），但因为长期收益好，也可以放在核心里。

## 卫星仓位：用你能承受的亏损博取超额

卫星仓位的目标是：**在核心仓位"死不了"的前提下，用一小部分钱博取高收益。**

卫星标的的特征：

- **高弹性**：涨起来猛，跌起来也猛。科创50 年化波动 37.5%，纳指100 波动 24.6%，都适合做卫星。
- **有观点**：你买它不是因为它"稳"，是因为你对它有判断——比如"AI 叙事还没结束"或者"科创板国产替代是长期趋势"。
- **有止损纪律**：卫星仓位亏损了，你得有计划什么时候减仓或清仓，不能死扛。

关键认知：**卫星仓位的核心不是"选什么"，而是"放多少"。** 哪怕你选了世界上最棒的成长 ETF，如果放了 80% 进去，一旦回撤 25%，你大概率扛不住。

## 比例怎么定：从最大亏损倒推

这里给一个实用的方法。分三步：

**第一步：定最大可承受亏损。**

问自己一个问题：如果一年内组合整体亏了 X%，你不会割肉、不会焦虑到失眠、能继续正常生活。这个 X 就是你能承受的最大亏损。

对大多数人来说，X 在 10% 到 20% 之间。太低了（比如 5%）意味着你几乎不能承受任何风险，应该多放债券。太高了（比如 40%）意味着你可能高估了自己的承受能力——真跌的时候大概率拿不住。

假设你定的 X = 15%。

**第二步：估算极端场景下各标的的回撤。**

用历史数据做参考：

| 标的     | 历史最大回撤 | 悲观假设 |
| -------- | ------------ | -------- |
| 红利低波 | -15.3%       | -15%     |
| 标普500  | -24.9%       | -30%     |
| 纳指100  | -24.0%       | -35%     |
| 科创50   | -19.3%       | -30%     |

"悲观假设"要取比历史最大回撤更保守的值，因为历史总在更新。

**第三步：倒推比例。**

假设组合只有核心（红利低波）和卫星（纳指100）。核心占比 c，卫星占比 s，c + s = 1。

极端场景下组合最大亏损 = c × 15% + s × 35%

设这个值等于你能承受的最大亏损 15%：

c × 15% + s × 35% = 15%

因为 c = 1 - s：

(1-s) × 15% + s × 35% = 15%
15% - 15%s + 35%s = 15%
20%s = 0%
s = 0%

等等——这说明如果你只能承受 15% 的亏损，纳指100 放任何比例都会超限？不完全是。因为现实中红利低波不会和纳指100 同时跌到最大回撤（相关性低），实际组合回撤会比线性叠加小。

更现实的做法是直接看历史回测。2024-2026 这两年，如果 70% 红利 + 30% 纳指100：

红利低波 -7.0%，纳指100 +49.8%，组合收益 = 0.7 × (-7%) + 0.3 × 49.8% = +10.2%

再看回撤。两者相关性约 0.3，组合最大回撤大约在 12-15% 之间（远低于全仓纳指100 的 24%）。如果 60/40 组合，最大回撤约 14-17%。

**70/30 或 60/40（核心/卫星）是大多数风险偏好适中的人的甜区。** 你能拿到成长的部分收益，同时把回撤控制在可承受范围内。

## 美股在组合里的特殊角色

你的组合里有个独特之处：**跨市场。** 红利低波是 A 股，纳指100/标普500 是美股。

跨市场配置的优势不是"两个都赚钱"，而是"两个不同步"。A 股和美股的相关性长期在 0.2-0.4 之间，远低于 A 股内部各指数之间的相关性（0.7-0.9）。

这意味着什么？A 股跌的时候美股不一定跌，美股跌的时候 A 股不一定跌。你的组合波动率因此大幅降低——比"全仓 A 股"或"全仓美股"都低。

| 配置                | 近2年收益 | 估算最大回撤 |
| ------------------- | --------- | ------------ |
| 100% 红利低波       | -7.0%     | -15.3%       |
| 100% 纳指100        | +49.8%    | -24.0%       |
| 50% 红利 + 50% 纳指 | +21.4%    | 约 -12%      |
| 70% 红利 + 30% 纳指 | +10.2%    | 约 -10%      |

50/50 组合收益 21.4%，远好于全仓红利的 -7%，而最大回撤（约-12%）反而比全仓红利或全仓纳指都低。**这就是分散的价值——1+1 真的大于 2。**

## 现金流 ETF 放哪

现金流 ETF 是个介于核心和卫星之间的东西。

从波动率看，它和红利低波差不多（都偏低波动），适合放核心。但从因子角度看，它和红利有重叠——都是"价值因子"的变体。同时持有红利和现金流，分散效果不如一个加纳指100。

建议：如果你已经持有红利低波做核心，现金流 ETF 可以作为核心仓位的补充（不超过总仓位的 15-20%），但不要替代红利。两者有互补性（现金流因子看的是自由现金流，红利因子看的是分红率），但也有重叠（高分红的公司通常现金流也好）。

## 执行纪律：最难的部分

框架是清晰的，难的是执行。

人性最大的敌人是：**涨了想加仓，跌了想割肉。** 纳指100 在 2024-2025 年涨了 50%+，很多原本设定 30% 卫星仓位的人，看到收益这么好会忍不住加仓——从 30% 加到 40%、50%，最后核心-卫星变成了"全仓纳指100"。

然后回撤来了。

这就是为什么下一篇要讲"相关性陷阱"和后面要讲"再平衡"——**设定好比例之后，最难的不是选标的，是在涨跌面前守住比例。** 再平衡就是用规则强制你"卖出涨的、买入跌的"，对抗人性本能。

## 这篇的核心结论

核心-卫星策略的本质，不是简单的"钱分两堆"，而是**先定风险承受能力，再倒推配置比例**。

- 核心 60-70%：红利低波 + 标普500，任务是让你在熊市不割肉
- 卫星 30-40%：纳指100 + 科创/创业板，任务是用你能承受的亏损博取超额
- 跨市场（A股+美股）是你组合里天然的分散优势，不要浪费

但框架有个隐患：比例定好了，涨跌会让它慢慢漂移。怎么守住比例，是后面三篇要解决的核心问题。
