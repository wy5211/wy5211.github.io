## 前言

运算符是构建表达式的基本元素。Go 提供了丰富的运算符，包括算术、关系、逻辑、位运算等。

- ✅ 掌握算术运算符

- ✅ 学会逻辑运算符

- ✅ 掌握运算符优先级

## 算术运算符

### 基本算术运算

```go
package main

import "fmt"

func main() {
    a := 10
    b := 3

    fmt.Printf("%d + %d = %d\n", a, b, a+b)   // 加法
    fmt.Printf("%d - %d = %d\n", a, b, a-b)   // 减法
    fmt.Printf("%d * %d = %d\n", a, b, a*b)   // 乘法
    fmt.Printf("%d / %d = %d\n", a, b, a/b)   // 除法（整数除法）
    fmt.Printf("%d %% %d = %d\n", a, b, a%b)  // 取模
}
```

### 自增和自减

```go
package main

import "fmt"

func main() {
    x := 10

    // 后置自增（先使用，再自增）
    fmt.Println("x:", x)   // 10
    y := x++
    fmt.Println("y:", y)   // 10
    fmt.Println("x:", x)   // 11

    // 前置自增（先自增，再使用）
    z := ++x
    fmt.Println("z:", z)   // 12
    fmt.Println("x:", x)   // 12
}
```

**注意**：Go 不支持 `x++` 作为表达式使用：

```go
// ❌ 错误
y := x++

// ✅ 正确
x++
y := x
```

## 关系运算符

### 比较运算

```go
package main

import "fmt"

func main() {
    a := 10
    b := 20

    fmt.Printf("%d == %d: %t\n", a, b, a == b)  // 等于
    fmt.Printf("%d != %d: %t\n", a, b, a != b)  // 不等于
    fmt.Printf("%d < %d: %t\n", a, b, a < b)    // 小于
    fmt.Printf("%d > %d: %t\n", a, b, a > b)    // 大于
    fmt.Printf("%d <= %d: %t\n", a, b, a <= b)  // 小于等于
    fmt.Printf("%d >= %d: %t\n", a, b, a >= b)  // 大于等于
}
```

### 字符串比较

```go
package main

import "fmt"

func main() {
    s1 := "apple"
    s2 := "banana"

    // 字符串按字典序比较
    fmt.Printf("%s < %s: %t\n", s1, s2, s1 < s2)
    fmt.Printf("%s == %s: %t\n", s1, s2, s1 == s2)
}
```

## 逻辑运算符

### 逻辑与、或、非

```go
package main

import "fmt"

func main() {
    a := true
    b := false

    fmt.Printf("%t && %t = %t\n", a, b, a && b)  // 逻辑与
    fmt.Printf("%t || %t = %t\n", a, b, a || b)  // 逻辑或
    fmt.Printf("!%t = %t\n", a, !a)               // 逻辑非

    // 短路求值
    x := 0
    result := (x != 0) && (1/x > 0)
    fmt.Printf("结果: %t\n", result)  // false（不会除零错误）
}
```

## 位运算符

### 位运算操作

```go
package main

import "fmt"

func main() {
    a := 10  // 二进制: 1010
    b := 6   // 二进制: 0110

    fmt.Printf("%d & %d = %d\n", a, b, a & b)   // 按位与: 0010 (2)
    fmt.Printf("%d | %d = %d\n", a, b, a | b)   // 按位或: 1110 (14)
    fmt.Printf("%d ^ %d = %d\n", a, b, a ^ b)   // 按位异或: 1100 (12)
    fmt.Printf("^%d = %d\n", a, ^a)              // 按位取反
    fmt.Printf("%d << 1 = %d\n", a, a << 1)     // 左移: 10100 (20)
    fmt.Printf("%d >> 1 = %d\n", a, a >> 1)     // 右移: 0101 (5)
}
```

## 赋值运算符

### 复合赋值

```go
package main

import "fmt"

func main() {
    x := 10

    x += 5   // 等价于 x = x + 5
    fmt.Println("x += 5:", x)  // 15

    x -= 3   // 等价于 x = x - 3
    fmt.Println("x -= 3:", x)  // 12

    x *= 2   // 等价于 x = x * 2
    fmt.Println("x *= 2:", x)  // 24

    x /= 4   // 等价于 x = x / 4
    fmt.Println("x /= 4:", x)  // 6

    x %= 4   // 等价于 x = x % 4
    fmt.Println("x %= 4:", x)  // 2
}
```

### 多重赋值

```go
package main

import "fmt"

func main() {
    // 交换变量
    a, b := 10, 20
    a, b = b, a
    fmt.Printf("a=%d, b=%d\n", a, b)

    // 函数返回值
    min, max := minMax(3, 7)
    fmt.Printf("min=%d, max=%d\n", min, max)
}

func minMax(a, b int) (int, int) {
    if a < b {
        return a, b
    }
    return b, a
}
```

## 运算符优先级

从高到低：

```
1. 后置运算符      ++ -- ( )
2. 一元运算符      + - ! ^ * & <- (type)
3. 乘法类         * / % << >> & &^
4. 加法类         + - | ^
5. 关系运算符     == != < <= > >=
6. 逻辑与        &&
7. 逻辑或        ||
8. 赋值运算符     = += -= *= /= %= <<= >>= &= ^= |=
```

## 实战示例

### 示例：计算器

```go
package main

import "fmt"

func main() {
    var a, b float64
    var operator string

    fmt.Print("输入第一个数: ")
    fmt.Scanln(&a)

    fmt.Print("输入运算符 (+, -, *, /): ")
    fmt.Scanln(&operator)

    fmt.Print("输入第二个数: ")
    fmt.Scanln(&b)

    var result float64
    switch operator {
    case "+":
        result = a + b
    case "-":
        result = a - b
    case "*":
        result = a * b
    case "/":
        if b == 0 {
            fmt.Println("错误：除数不能为零")
            return
        }
        result = a / b
    default:
        fmt.Println("无效的运算符")
        return
    }

    fmt.Printf("%.2f %s %.2f = %.2f\n", a, operator, b, result)
}
```
