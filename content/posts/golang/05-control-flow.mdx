## 前言

控制流语句决定了程序的执行顺序。Go 提供了简洁高效的控制流结构。

---

## if-else 语句

### 基本用法

```go
package main

import "fmt"

func main() {
    score := 85

    if score >= 90 {
        fmt.Println("优秀")
    } else if score >= 60 {
        fmt.Println("及格")
    } else {
        fmt.Println("不及格")
    }
}
```

### if 初始化语句

```go
package main

import "fmt"

func main() {
    // if 可以包含初始化语句
    if num := 10; num%2 == 0 {
        fmt.Println(num, "是偶数")
    } else {
        fmt.Println(num, "是奇数")
    }

    // 验证 Map 键
    m := map[string]int{"a": 1}
    if value, exists := m["a"]; exists {
        fmt.Println("a的值:", value)
    }
}
```

## switch 语句

### 基本用法

```go
package main

import "fmt"

func main() {
    day := 3

    switch day {
    case 1:
        fmt.Println("星期一")
    case 2:
        fmt.Println("星期二")
    case 3:
        fmt.Println("星期三")
    default:
        fmt.Println("其他")
    }
}
```

### 多值匹配

```go
package main

import "fmt"

func main() {
    month := 1

    switch month {
    case 1, 3, 5, 7, 8, 10, 12:
        fmt.Println("31天")
    case 4, 6, 9, 11:
        fmt.Println("30天")
    case 2:
        fmt.Println("28或29天")
    }
}
```

### 无表达式 switch

```go
package main

import "fmt"

func main() {
    score := 85

    switch {
    case score >= 90:
        fmt.Println("A")
    case score >= 80:
        fmt.Println("B")
    case score >= 60:
        fmt.Println("C")
    default:
        fmt.Println("D")
    }
}
```

### fallthrough

```go
package main

import "fmt"

func main() {
    num := 1

    switch num {
    case 1:
        fmt.Println("1")
        fallthrough  // 继续执行下一个 case
    case 2:
        fmt.Println("2")
    default:
        fmt.Println("default")
    }
}
```

## for 循环

### Go 只有一种循环：for

```go
package main

import "fmt"

func main() {
    // 标准循环
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }

    // while 循环形式
    j := 0
    for j < 5 {
        fmt.Println(j)
        j++
    }

    // 无限循环
    k := 0
    for {
        if k >= 5 {
            break
        }
        fmt.Println(k)
        k++
    }
}
```

### range 遍历

```go
package main

import "fmt"

func main() {
    // 遍历切片
    nums := []int{1, 2, 3, 4, 5}
    for index, value := range nums {
        fmt.Printf("索引: %d, 值: %d\n", index, value)
    }

    // 遍历 map
    m := map[string]string{
        "name": "张三",
        "city": "北京",
    }
    for key, value := range m {
        fmt.Printf("%s: %s\n", key, value)
    }

    // 遍历字符串
    for index, char := range "Hello" {
        fmt.Printf("位置: %d, 字符: %c\n", index, char)
    }
}
```

## break 和 continue

```go
package main

import "fmt"

func main() {
    // break 跳出循环
    for i := 0; i < 10; i++ {
        if i == 5 {
            break
        }
        fmt.Println(i)
    }

    // continue 跳过本次迭代
    for i := 0; i < 10; i++ {
        if i%2 == 0 {
            continue
        }
        fmt.Println(i)
    }
}
```

## goto 语句

```go
package main

import "fmt"

func main() {
    i := 0

start:
    if i < 5 {
        fmt.Println(i)
        i++
        goto start
    }
}
```

## 实战示例

### 示例：打印九九乘法表

```go
package main

import "fmt"

func main() {
    for i := 1; i <= 9; i++ {
        for j := 1; j <= i; j++ {
            fmt.Printf("%d×%d=%-2d ", j, i, i*j)
        }
        fmt.Println()
    }
}
```
