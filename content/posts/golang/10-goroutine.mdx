## 前言

goroutine 是 Go 并发的核心。它是一种轻量级线程，由 Go 运行时管理，可以轻松创建成千上万个 goroutine。

---

## goroutine 基础

### 创建 goroutine

```go
package main

import (
    "fmt"
    "time"
)

func sayHello(name string) {
    fmt.Printf("Hello, %s!\n", name)
}

func main() {
    // 启动 goroutine
    go sayHello("张三")
    go sayHello("李四")

    // 等待 goroutine 完成
    time.Sleep(time.Second)
}
```

### 匿名 goroutine

```go
package main

import (
    "fmt"
    "time"
)

func main() {
    go func() {
        fmt.Println("匿名 goroutine")
    }()

    time.Sleep(time.Millisecond * 500)
}
```

## goroutine 同步

### 使用 WaitGroup

```go
package main

import (
    "fmt"
    "sync"
    "time"
)

func worker(id int, wg *sync.WaitGroup) {
    defer wg.Done()

    fmt.Printf("Worker %d 开始\n", id)
    time.Sleep(time.Second)
    fmt.Printf("Worker %d 完成\n", id)
}

func main() {
    var wg sync.WaitGroup

    for i := 1; i <= 3; i++ {
        wg.Add(1)
        go worker(i, &wg)
    }

    wg.Wait()
    fmt.Println("所有 worker 完成")
}
```

## goroutine 通信

### 共享变量（不推荐）

```go
package main

import (
    "fmt"
    "sync"
)

func main() {
    var counter int
    var wg sync.WaitGroup
    var mu sync.Mutex

    for i := 0; i < 1000; i++ {
        wg.Add(1)
        go func() {
            defer wg.Done()
            mu.Lock()
            counter++
            mu.Unlock()
        }()
    }

    wg.Wait()
    fmt.Println("Counter:", counter)
}
```

### 使用 channel（推荐）

```go
package main

import "fmt"

func worker(id int, jobs <-chan int, results chan<- int) {
    for j := range jobs {
        fmt.Printf("Worker %d 处理任务 %d\n", id, j)
        results <- j * 2
    }
}

func main() {
    jobs := make(chan int, 100)
    results := make(chan int, 100)

    // 启动 3 个 worker
    for w := 1; w <= 3; w++ {
        go worker(w, jobs, results)
    }

    // 发送 5 个任务
    for j := 1; j <= 5; j++ {
        jobs <- j
    }
    close(jobs)

    // 接收结果
    for j := 1; j <= 5; j++ {
        <-results
    }
}
```

## goroutine 泄漏

### 常见泄漏场景

```go
// ❌ goroutine 泄漏
func leakyFunction() {
    ch := make(chan int)

    go func() {
        val := <-ch  // 如果没有发送者，会永远阻塞
        fmt.Println(val)
    }()
}

// ✅ 使用 context 取消
func correctFunction(ctx context.Context) {
    ch := make(chan int)

    go func() {
        select {
        case val := <-ch:
            fmt.Println(val)
        case <-ctx.Done():
            return
        }
    }()
}
```

## 最佳实践

### 不要共享内存，通过通信共享

```go
// ✅ 推荐：使用 channel
func worker(ch chan int) {
    for val := range ch {
        fmt.Println(val)
    }
}

// ❌ 避免：使用共享内存 + 锁
func worker(mu *sync.Mutex, data *[]int) {
    mu.Lock()
    *data = append(*data, 1)
    mu.Unlock()
}
```
