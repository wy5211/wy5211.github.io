## 前言

欢迎来到 Go 语言的世界！Go（又称 Golang）是 Google 开发的一门开源编程语言，以其简洁、高效、并发支持强大而著称。

**为什么选择 Go？**

- 🚀 **性能卓越**：接近 C/C++ 的执行性能
- 🔄 **并发原生**： goroutine 和 channel 让并发编程变得简单
- 📦 **部署简单**：编译为单一可执行文件，无依赖
- 🎯 **语法简洁**：学习曲线平缓，代码清晰易读
- 🔧 **工具链完善**：内置格式化、测试、性能分析等工具
- 💼 **应用广泛**：Docker、Kubernetes、Prometheus 等知名项目都用 Go 编写

## Go 语言核心概念

### Go 的设计哲学

Go 的设计遵循以下原则：

1. **简单性**：语法简洁，关键字少（25个）
2. **安全性**：强类型、内存安全、垃圾回收
3. **并发性**：内置并发原语
4. **高效性**：快速编译、高效执行
5. **可扩展性**：适合大型项目

## 环境搭建

### 第一步：安装 Go

**Windows 系统**：

1. 访问 [Go 官网下载页](https://go.dev/dl/)
2. 下载 Windows 安装包（.msi）
3. 运行安装程序，按提示完成安装
4. 默认安装路径：`C:\Program Files\Go`

**macOS 系统**：

```bash
# 使用 Homebrew 安装（推荐）
brew install go

# 或下载安装包
# 访问 https://go.dev/dl/
# 下载 macOS 安装包（.pkg）
```

**Linux 系统**：

```bash
# 下载 Go
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# 解压到 /usr/local
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# 添加到 PATH
export PATH=$PATH:/usr/local/go/bin
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
```

### 第二步：验证安装

```bash
go version
```

输出示例：

```
go version go1.22.0 darwin/amd64
```

### 第三步：配置环境变量

**macOS/Linux**：

编辑 `~/.bashrc` 或 `~/.zshrc`：

```bash
# Go 安装路径
export GOROOT=/usr/local/go

# Go 工作空间
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Go 模块代理（中国用户）
export GOPROXY=https://goproxy.cn,direct

# 启用 Go Modules
export GO111MODULE=on
```

保存后执行：

```bash
source ~/.bashrc  # 或 source ~/.zshrc
```

**Windows**：

1. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
2. 新建系统变量：
   - `GOROOT`: Go 安装路径（如 `C:\Program Files\Go`）
   - `GOPATH`: 工作空间（如 `C:\Users\YourName\go`）
   - `GOPROXY`: `https://goproxy.cn,direct`
3. 编辑 Path 变量，添加：
   - `%GOROOT%\bin`
   - `%GOPATH%\bin`

### 第四步：验证配置

```bash
go env
```

输出示例：

## Go 工作空间结构

### 传统 GOPATH 模式

```
GOPATH/
├── src/           # 源代码
│   ├── github.com/
│   │   └── user/
│   │       └── project/
│   │           └── main.go
├── pkg/           # 编译后的包文件
└── bin/           # 编译后的可执行文件
```

### 现代 Go Modules 模式（推荐）

```
your-project/
├── go.mod         # 模块定义文件
├── go.sum         # 依赖校验文件
├── main.go        # 主程序
└── internal/      # 内部包
```

## 第一个 Go 程序

### Hello World

创建文件 `hello.go`：

```go
// 包声明：每个 Go 程序都必须有包声明
package main

// 导入格式化输出包
import "fmt"

// 主函数：程序入口
func main() {
    // 打印消息到控制台
    fmt.Println("Hello, World!")
    fmt.Println("欢迎来到 Go 语言的世界！")
}
```

### 运行程序

```bash
# 方式 1：直接运行
go run hello.go

# 方式 2：编译后运行
go build hello.go
./hello          # Linux/macOS
# 或
hello.exe        # Windows
```

**输出**：

```
Hello, World!
欢迎来到 Go 语言的世界！
```

### 代码解析

```go
// 1. 包声明
package main

// 每个 Go 程序都必须属于一个包
// main 包是特殊的，它定义了一个独立可执行程序
```

```go
// 2. 导入包
import "fmt"

// fmt 是格式化输入输出包
// 用于打印、扫描、格式化字符串等
```

```go
// 3. 函数定义
func main() {
    // main 函数是程序入口
    // 不接受参数，也不返回值
}
```

```go
// 4. 语句
fmt.Println("Hello, World!")

// Println 打印内容并换行
// 注意：Go 的语句不需要分号结尾
```

## Go 命令工具

### 常用命令

```bash
# 运行程序
go run main.go

# 编译程序
go build main.go

# 格式化代码
go fmt main.go

# 运行测试
go test

# 下载依赖
go get <package-url>

# 初始化模块
go mod init <module-name>

# 整理依赖
go mod tidy

# 查看环境变量
go env

# 查看文档
go doc fmt.Println
```

### go run vs go build

| 命令       | 说明               | 适用场景 |
| ---------- | ------------------ | -------- |
| `go run`   | 编译并直接运行     | 开发调试 |
| `go build` | 编译生成可执行文件 | 生产部署 |

```bash
# go run 示例
go run main.go

# go build 示例
go build -o myapp main.go
./myapp
```

## 实战演练

### 示例 1：个人信息打印

```go
package main

import "fmt"

func main() {
    // 字符串拼接
    name := "张三"
    age := 25
    occupation := "Go 工程师"

    // 使用 fmt.Printf 格式化输出
    fmt.Printf("姓名: %s\n", name)
    fmt.Printf("年龄: %d\n", age)
    fmt.Printf("职业: %s\n", occupation)

    // 使用 + 拼接字符串
    fmt.Println("\n个人信息:")
    fmt.Println("姓名: " + name)
    fmt.Println("年龄: " + string(age))
}
```

**格式化动词**：

- `%s` - 字符串
- `%d` - 整数
- `%f` - 浮点数
- `%t` - 布尔值
- `%v` - 任何值的默认格式

### 示例 2：简单计算器

```go
package main

import "fmt"

func main() {
    // 定义变量
    var num1, num2 int
    num1 = 10
    num2 = 20

    // 计算
    sum := num1 + num2
    difference := num1 - num2
    product := num1 * num2
    quotient := num1 / num2

    // 输出结果
    fmt.Println("=== 简单计算器 ===")
    fmt.Printf("数1: %d\n", num1)
    fmt.Printf("数2: %d\n", num2)
    fmt.Println("----------------")
    fmt.Printf("和: %d\n", sum)
    fmt.Printf("差: %d\n", difference)
    fmt.Printf("积: %d\n", product)
    fmt.Printf("商: %d\n", quotient)
}
```

### 示例 3：变量交换

```go
package main

import "fmt"

func main() {
    // 方式 1：使用临时变量
    a, b := 10, 20
    fmt.Printf("交换前: a=%d, b=%d\n", a, b)

    temp := a
    a = b
    b = temp
    fmt.Printf("交换后: a=%d, b=%d\n", a, b)

    // 方式 2：Go 的多重赋值（推荐）
    x, y := 100, 200
    fmt.Printf("\n交换前: x=%d, y=%d\n", x, y)

    x, y = y, x  // 一行代码完成交换
    fmt.Printf("交换后: x=%d, y=%d\n", x, y)
}
```

## Go 代码风格

### 自动格式化

Go 有严格的代码格式要求，使用 `gofmt` 工具：

```bash
# 格式化当前目录所有 Go 文件
go fmt .

# 或使用 gofmt
gofmt -w main.go
```

### 命名规范

| 类型 | 规范                           | 示例                      |
| ---- | ------------------------------ | ------------------------- |
| 包名 | 小写，单数                     | `fmt`, `strings`          |
| 常量 | 大驼峰或全大写下划线分隔       | `MaxSize` 或 `MAX_SIZE`   |
| 变量 | 小驼峰                         | `userName`, `totalCount`  |
| 函数 | 大驼峰（导出）或小驼峰（私有） | `GetUser`, `calculateSum` |
| 接口 | 通常 -er 结尾                  | `Reader`, `Writer`        |

### 注释规范

```go
// 单行注释

/*
 * 多行注释
 * 用于大段说明
 */

// 文档注释（用于生成 godoc）
// Package main 提供了示例程序
package main

// Add 函数返回两个整数的和
func Add(a, b int) int {
    return a + b
}
```

## 常见问题

**Q: Go 适合做什么？**

A: Go 适合：

- 云原生应用（Docker、K8s）
- 微服务架构
- 网络服务
- 命令行工具
- 分布式系统

**Q: Go 和 Python 怎么选？**

A:

- **选 Go**：需要高性能、高并发、编译型部署
- **选 Python**：快速开发、数据科学、AI/ML

**Q: Go 难学吗？**

A: Go 语法简单，学习曲线平缓。有编程基础的人通常 1-2 周就能上手。

**Q: Windows 和 macOS/Linux 代码能通用吗？**

A: 完全通用！Go 代码跨平台，只需在目标平台编译即可。
