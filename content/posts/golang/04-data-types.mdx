## 前言

Go 提供了丰富的数据类型，包括基本类型和复合类型。本文将详细介绍数组、切片、映射等重要类型。

---

## 数组

### 数组声明和初始化

```go
package main

import "fmt"

func main() {
    // 声明数组
    var arr1 [5]int
    fmt.Println(arr1)  // [0 0 0 0 0]

    // 初始化数组
    arr2 := [5]int{1, 2, 3, 4, 5}
    fmt.Println(arr2)

    // 让编译器计算长度
    arr3 := [...]int{1, 2, 3}
    fmt.Println(arr3)

    // 指定索引初始化
    arr4 := [5]int{1: 10, 3: 30}
    fmt.Println(arr4)  // [0 10 0 30 0]
}
```

### 数组操作

```go
package main

import "fmt"

func main() {
    arr := [5]int{10, 20, 30, 40, 50}

    // 访问元素
    fmt.Println("第一个元素:", arr[0])
    fmt.Println("最后一个元素:", arr[len(arr)-1])

    // 修改元素
    arr[0] = 100
    fmt.Println(arr)

    // 遍历数组
    for i, v := range arr {
        fmt.Printf("索引 %d: %d\n", i, v)
    }
}
```

## 切片（Slice）

切片是 Go 中最重要的数据结构之一。

### 切片基础

```go
package main

import "fmt"

func main() {
    // 基于数组创建切片
    arr := [5]int{1, 2, 3, 4, 5}
    slice := arr[1:4]  // [2 3 4]
    fmt.Println(slice)

    // 直接声明切片
    slice1 := []int{1, 2, 3}
    fmt.Println(slice1)

    // 使用 make 创建切片
    slice2 := make([]int, 5)       // 长度为5
    slice3 := make([]int, 3, 10)   // 长度3，容量10
    fmt.Println(slice2, slice3)
}
```

### 切片操作

```go
package main

import "fmt"

func main() {
    slice := []int{1, 2, 3, 4, 5}

    // 添加元素
    slice = append(slice, 6)
    fmt.Println(slice)

    // 添加多个元素
    slice = append(slice, 7, 8, 9)
    fmt.Println(slice)

    // 合并切片
    slice2 := []int{10, 11}
    slice = append(slice, slice2...)
    fmt.Println(slice)

    // 复制切片
    sliceCopy := make([]int, len(slice))
    copy(sliceCopy, slice)
    fmt.Println(sliceCopy)

    // 切片的切片
    subSlice := slice[1:4]
    fmt.Println(subSlice)
}
```

## 映射（Map）

Map 是 Go 的哈希表实现。

### Map 基础

```go
package main

import "fmt"

func main() {
    // 声明 map
    var m1 map[string]int
    fmt.Println(m1 == nil)  // true

    // 初始化 map
    m2 := make(map[string]int)
    m2["apple"] = 5
    m2["banana"] = 10
    fmt.Println(m2)

    // 字面量方式
    m3 := map[string]int{
        "red":    1,
        "green":  2,
        "blue":   3,
    }
    fmt.Println(m3)
}
```

### Map 操作

```go
package main

import "fmt"

func main() {
    m := make(map[string]int)

    // 添加/修改
    m["a"] = 1
    m["b"] = 2

    // 获取值
    val := m["a"]
    fmt.Println("a的值:", val)

    // 检查键是否存在
    val, exists := m["c"]
    if exists {
        fmt.Println("c的值:", val)
    } else {
        fmt.Println("c不存在")
    }

    // 删除键
    delete(m, "a")
    fmt.Println(m)

    // 遍历
    for key, value := range m {
        fmt.Printf("%s: %d\n", key, value)
    }
}
```

## 指针

Go 支持指针，但不支持指针运算。

### 指针基础

```go
package main

import "fmt"

func main() {
    x := 42

    // 获取地址
    p := &x
    fmt.Println("x的地址:", p)
    fmt.Println("通过指针访问x:", *p)

    // 修改值
    *p = 100
    fmt.Println("x的新值:", x)
}
```

### 指针作为函数参数

```go
package main

import "fmt"

func modifyValue(x *int) {
    *x = 100
}

func main() {
    value := 10
    fmt.Println("修改前:", value)

    modifyValue(&value)
    fmt.Println("修改后:", value)
}
```

## 结构体

### 结构体定义

```go
package main

import "fmt"

// 定义结构体
type Person struct {
    Name string
    Age  int
    City string
}

func main() {
    // 创建结构体实例
    p1 := Person{
        Name: "张三",
        Age:  25,
        City: "北京",
    }
    fmt.Println(p1)

    // 顺序初始化
    p2 := Person{"李四", 30, "上海"}
    fmt.Println(p2)

    // 使用 new
    p3 := new(Person)
    p3.Name = "王五"
    p3.Age = 28
    fmt.Println(*p3)

    // 访问字段
    fmt.Println("姓名:", p1.Name)
}
```

### 结构体方法

```go
package main

import "fmt"

type Rectangle struct {
    Width  float64
    Height float64
}

// 值接收者
func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

// 指针接收者
func (r *Rectangle) Scale(factor float64) {
    r.Width *= factor
    r.Height *= factor
}

func main() {
    rect := Rectangle{Width: 3, Height: 4}

    fmt.Println("面积:", rect.Area())

    rect.Scale(2)
    fmt.Println("缩放后:", rect)
}
```
