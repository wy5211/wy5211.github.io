## 前言

Go Modules 是 Go 1.11+ 的依赖管理系统，取代了旧的 GOPATH 模式。

---

## 初始化模块

### 创建新模块

```bash
# 创建新模块
go mod init github.com/yourname/project

# 目录结构
# project/
# ├── go.mod
# ├── go.sum
# └── main.go
```

### go.mod 文件

```go
module github.com/yourname/project

go 1.22

require (
    github.com/gin-gonic/gin v1.9.1
    github.com/stretchr/testify v1.8.4
)
```

---

## 依赖管理

### 添加依赖

```bash
# 自动添加依赖
go get github.com/gin-gonic/gin

# 指定版本
go get github.com/gin-gonic/gin@v1.9.0

# 更新所有依赖
go get -u ./...

# 整理依赖
go mod tidy
```

### 查看依赖

```bash
# 查看所有依赖
go list -m all

# 查看依赖图
go mod graph

# 为什么需要某个包
go mod why github.com/gin-gonic/gin
```

## 导入包

### 标准库导入

```go
package main

import (
    "fmt"      // 导入包
    "math/rand"
)

func main() {
    fmt.Println("随机数:", rand.Intn(100))
}
```

### 第三方包导入

```go
import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()
    r.GET("/", func(c *gin.Context) {
        c.String(200, "Hello")
    })
    r.Run()
}
```

### 别名导入

```go
import (
    stdfmt "fmt"
    myfmt "myproject/fmt"
)

func main() {
    stdfmt.Println("标准库")
    myfmt.Println("自定义")
}
```

## 创建和使用包

### 创建包

```go
// math/math.go
package math

func Add(a, b int) int {
    return a + b
}

func Subtract(a, b int) int {
    return a - b
}
```

### 使用包

```go
// main.go
package main

import (
    "fmt"
    "github.com/yourname/project/math"
)

func main() {
    result := math.Add(10, 20)
    fmt.Println(result)
}
```

## 可见性

### 导出标识符

```go
package mypackage

// 导出函数（大写开头）
func PublicFunction() {}

// 私有函数（小写开头）
func privateFunction() {}

// 导出变量
var PublicVar int

// 私有变量
var privateVar int

// 导出结构体
type PublicStruct struct{}

// 私有结构体
type privateStruct struct{}
```

## 最佳实践

### 包命名

```go
// ✅ 推荐：短小、全小写、无下划线
package http
package fmt
package strings

// ❌ 避免
package httpServer
package my_strings
```

### 导入顺序

```go
import (
    // 标准库
    "fmt"
    "net/http"

    // 第三方库
    "github.com/gin-gonic/gin"

    // 本地包
    "myproject/models"
)
```
