## 前言

反射是程序在运行时检查类型和操作对象的能力。Go 的反射通过 reflect 包实现。

---

## 反射基础

### 反射三大定律

1. 反射可以将接口类型转换为反射对象
2. 反射对象可以还原为接口类型
3. 要修改反射对象，值必须是可设置的

### 类型检查

```go
package main

import (
    "fmt"
    "reflect"
)

func main() {
    var x interface{} = 3.14

    // 获取反射对象
    v := reflect.ValueOf(x)
    t := reflect.TypeOf(x)

    fmt.Println("类型:", t)
    fmt.Println("种类:", v.Kind())
    fmt.Println("值:", v)
}
```

## 反射操作

### 获取结构体信息

```go
package main

import (
    "fmt"
    "reflect"
)

type Person struct {
    Name string `json:"name"`
    Age  int    `json:"age"`
}

func main() {
    p := Person{"张三", 25}

    // 获取类型
    t := reflect.TypeOf(p)

    // 遍历字段
    for i := 0; i < t.NumField(); i++ {
        field := t.Field(i)
        fmt.Printf("字段名: %s, 类型: %v, 标签: %s\n",
            field.Name, field.Type, field.Tag.Get("json"))
    }
}
```

### 动态调用方法

```go
package main

import (
    "fmt"
    "reflect"
)

type Math struct{}

func (m Math) Add(a, b int) int {
    return a + b
}

func main() {
    math := &Math{}

    // 获取方法
    v := reflect.ValueOf(math)
    method := v.MethodByName("Add")

    // 调用方法
    args := []reflect.Value{reflect.ValueOf(10), reflect.ValueOf(20)}
    result := method.Call(args)

    fmt.Println("结果:", result[0].Int())
}
```

## 反射的代价

### 性能考虑

反射比直接调用慢，应谨慎使用：

```go
// ❌ 避免：在性能关键路径使用反射
func Process(data interface{}) {
    v := reflect.ValueOf(data)
    // ...
}

// ✅ 推荐：使用类型断言或接口
func Process(data Processor) {
    data.Process()
}
```
