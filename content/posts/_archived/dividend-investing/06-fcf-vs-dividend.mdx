上一篇聊了分红的本质。这篇聊一个和分红紧密相关、但更底层的东西——自由现金流。

为什么要专门讲它？因为2025年底，一只叫「800现金流ETF」（159119）的新基金上市了，跟踪的是中证800自由现金流指数。这个指数被很多人称为「下一代价值因子」，有人说它会取代红利因子。

这到底是革命还是噱头？现金流策略和红利策略有什么本质区别？你应该换吗？

## 利润和现金流的区别：一个真实的陷阱

先讲一个能帮你避坑的例子。

假设有两家公司，A和B，当年都报告了10亿元净利润。看起来一模一样。

但你看它们的现金流量表：

|                | A公司   | B公司    |
| -------------- | ------- | -------- |
| 净利润         | 10亿    | 10亿     |
| 经营现金流     | 9亿     | 2亿      |
| 应收账款增加   | 1亿     | 8亿      |
| 资本支出       | 3亿     | 3亿      |
| **自由现金流** | **6亿** | **-1亿** |

（自由现金流 = 经营现金流 − 资本支出）

A公司赚了10亿利润，经营活动收回了9亿现金，减去3亿资本支出，还剩6亿自由现金流。这6亿是真金白银，可以分红、可以还债、可以回购。

B公司也赚了10亿利润，但经营活动只收回了2亿现金——剩下的8亿全是应收账款（货卖出去了，钱还没收到）。减去3亿资本支出，自由现金流是**负的1亿**。也就是说，B公司账面上赚钱了，但实际是在贴钱运营。

**哪家公司更有钱分红？** 毫无疑问是A。

但如果你只看净利润，两家公司一模一样。这就是为什么**利润是观点，现金流才是事实**。

净利润基于权责发生制——交易发生就记账，不管钱到没到。这给了公司操纵空间：提前确认收入、推迟确认费用、调节折旧。历史上那些财务造假（康美药业虚增300亿货币资金、瑞幸虚构收入），造的都是利润表。

自由现金流基于收付实现制——钱真进了银行账户才算流入。银行流水造假成本极高，所以现金流很难伪造。

## 现金流策略的选股逻辑

理解了现金流的含金量，就理解了为什么有人要用它来做选股因子。

中证800自由现金流指数的编制逻辑：

1. **样本空间**：中证800指数成分股（沪深300+中证500，即最大的800只A股）
2. **质量筛选**：剔除金融行业（银行现金流结构特殊）、剔除自由现金流为负的公司
3. **排序加权**：按**自由现金流率**（自由现金流 ÷ 企业价值）排序，选前100只
4. **加权方式**：自由现金流金额加权——现金流越大的公司权重越高
5. **季度调仓**

和红利低波对比一下：

|              | 红利低波                         | 自由现金流     |
| ------------ | -------------------------------- | -------------- |
| **选股指标** | 股息率                           | 自由现金流率   |
| **筛选门槛** | 连续3年现金分红                  | 自由现金流为正 |
| **加权方式** | 股息率×波动率倒数                | 自由现金流金额 |
| **行业限制** | 无（但不选金融以外的低分红行业） | 剔除金融       |
| **调仓频率** | 季度                             | 季度           |

核心区别在于：**红利低波选的是"愿意把钱分给你的公司"，现金流选的是"账上真有钱的公司"。**

## 两个策略会选出来不同的公司吗

会，而且差异不小。

红利低波重仓银行、煤炭、高速公路、房地产——这些行业分红慷慨但未必自由现金流最高（银行的现金流结构特殊；煤炭行业现金流好但波动大）。

自由现金流策略剔除金融后，更多选入的是制造业龙头、消费、公用事业——这些行业自由现金流稳定，但分红率未必最高（有些公司赚了钱但喜欢留存）。

打个比方：红利低波选的是「大方的朋友」（赚了钱愿意请你吃饭的），现金流选的是「真有钱的朋友」（银行卡里真有余额的）。最理想的状态当然是两者兼得——但现实中，有些"大方"的人其实是在透支（高股息但低现金流），有些"真有钱"的人反而不爱花（高现金流但低分红）。

**红利低波的问题**：可能选到"打肿脸充胖子"的公司——分红率很高，但自由现金流撑不住。一旦盈利下滑，分红就断了。

**现金流策略的优势**：更难造假，更贴近企业真实的"赚钱能力"。

**现金流策略的劣势**：自由现金流波动大（一次大额资本支出就能让当年FCF暴跌），不如股息率稳定。

## 现金流会取代红利吗

不会完全取代，但会形成互补。

红利因子的优势在于**直接**：分红就是真金白银到你的账户，不需要你判断"这个现金流是真的还是假的"。对于不擅长读财报的普通投资者，红利因子更直观、更容易理解。

现金流因子的优势在于**底层**：它直接衡量企业的造血能力，比经过管理层分红决策的股息率更"硬核"。但它的门槛也更高——你需要理解自由现金流的计算和含义。

从实际效果看，两个策略在长期收益上可能接近（都是价值因子的不同实现），但在行业暴露上互补：红利偏银行煤炭，现金流偏制造业消费。**如果你同时持有红利低波和现金流策略，相当于覆盖了更广的价值股池。**

但对大多数人来说，持有红利低波已经够了。现金流策略更适合那些：

- 对红利低波的行业集中度不放心，想分散
- 想要一个"更难造假"的选股因子
- 愿意花时间理解更复杂的指标

## 159119 现金流ETF值得买吗

说完理论，回到实操。

800现金流ETF（159119）2025年11月才上市，是市场上最早的一批现金流策略ETF。截至2026年3月，规模只有2.17亿——偏小，但作为新概念ETF还有增长空间。

费率方面，0.20%（和563020易方达红利低波一样低），这个没问题。

**需要注意的风险**：

1. **规模偏小**。2.17亿不算大，如果规模继续缩水到1亿以下，清盘风险会上升。建议观察1-2个季度，看规模趋势。

2. **没有历史验证**。中证800自由现金流指数是新指数，没有经过完整牛熊周期的考验。理论上现金流因子应该有效，但实际表现需要时间检验。

3. **和红利低波高度相关**。两个策略虽然选股逻辑不同，但都是价值因子，在风格周期上走势接近——2025年红利低波下跌时，现金流指数也在跌。持有现金流策略并不能帮你分散"价值风格走弱"的风险。

**我的判断**：现金流策略值得关注，但现阶段更适合作为红利低波的**补充**而非替代。如果你已经有红利低波仓位，可以少量配置现金流ETF作为分散（比如红利70% + 现金流30%），但没必要全换。

## 真正重要的结论

这篇的核心不是让你去买现金流ETF，而是帮你建立一种思维习惯：**看一家公司值不值钱，别只看利润，要看现金流。**

这个思维不仅适用于选ETF，也适用于选个股。当你看到某家公司净利润暴增、PE很低、看起来很便宜时，去看一眼它的现金流量表——如果经营现金流远低于净利润，或者自由现金流为负，你就该警惕了。

利润可以美化，现金很难造假。**利润是观点，现金流才是事实——记住这句话，能帮你避开A股80%的财务陷阱。**
