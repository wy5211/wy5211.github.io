## 前言

传统的 HTTP 请求是单向的——客户端发送请求，服务器返回响应。但在许多场景下，我们需要服务器主动向客户端推送数据，比如实时聊天、股票行情、在线游戏等。

**WebSocket** 是一种在单个 TCP 连接上进行全双工通信的协议，它允许服务器主动向客户端推送消息，实现真正的实时通信。

Hono 对 WebSocket 提供了良好的支持，可以轻松实现实时功能。

本篇文章将带你：

- 🌐 理解 WebSocket 协议和工作原理
- 🔌 建立 WebSocket 连接
- 💬 实现实时聊天功能
- 🔔 构建通知推送系统
- 🏠 实现房间和频道机制
- 🔄 处理连接和重连

## WebSocket 基础

### WebSocket vs HTTP

| 特性         | HTTP               | WebSocket      |
| ------------ | ------------------ | -------------- |
| **通信方式** | 单向（请求-响应）  | 双向（全双工） |
| **连接**     | 短暂               | 持久           |
| **开销**     | 每次请求带完整头部 | 初始握手后极小 |
| **状态**     | 无状态             | 有状态         |
| **适用场景** | REST API           | 实时应用       |

### WebSocket 连接流程

```
1. 客户端发送 HTTP 请求（升级为 WebSocket）
   GET /ws HTTP/1.1
   Upgrade: websocket
   Connection: Upgrade

2. 服务器返回 101 Switching Protocols
   HTTP/1.1 101 Switching Protocols
   Upgrade: websocket

3. 建立 WebSocket 连接，双向通信开始
```

```typescript
import { Hono } from "hono";
import { createWebSocketStream } from "@hono/node-server/websocket";

const app = new Hono();

app.get("/ws", async (c) => {
  if (c.req.header("upgrade") !== "websocket") {
    return c.text("Expected websocket", 400);
  }

  return c.streamText(async (stream) => {
    const wsStream = createWebSocketStream(c);

    wsStream.readable.pipeTo(
      new WritableStream({
        async write(chunk) {
          const message = new TextDecoder().decode(chunk);

          // Echo 消息回客户端
          await stream.write(message);
        },
      })
    );
  });
});

export default app;
```

## 实时聊天系统

### 连接管理

```typescript
// src/websocket/connection-manager.ts
interface Client {
  id: string;
  userId: string;
  socket: WebSocket;
  rooms: Set<string>;
}

export class ConnectionManager {
  private clients: Map<string, Client> = new Map();
  private userClients: Map<string, Set<string>> = new Map();

  add(client: Client) {
    this.clients.set(client.id, client);

    if (!this.userClients.has(client.userId)) {
      this.userClients.set(client.userId, new Set());
    }
    this.userClients.get(client.userId)!.add(client.id);
  }

  remove(clientId: string) {
    const client = this.clients.get(clientId);
    if (!client) return;

    this.clients.delete(clientId);

    const userClients = this.userClients.get(client.userId);
    if (userClients) {
      userClients.delete(clientId);
      if (userClients.size === 0) {
        this.userClients.delete(client.userId);
      }
    }
  }

  get(clientId: string): Client | undefined {
    return this.clients.get(clientId);
  }

  getByUserId(userId: string): Client[] {
    const clientIds = this.userClients.get(userId);
    if (!clientIds) return [];

    return Array.from(clientIds)
      .map((id) => this.clients.get(id))
      .filter(Boolean) as Client[];
  }

  getAll(): Client[] {
    return Array.from(this.clients.values());
  }

  broadcast(message: any, excludeClientId?: string) {
    const data = JSON.stringify(message);

    for (const [id, client] of this.clients) {
      if (
        id !== excludeClientId &&
        client.socket.readyState === WebSocket.OPEN
      ) {
        client.socket.send(data);
      }
    }
  }
}

export const connectionManager = new ConnectionManager();
```

### 聊天路由

```typescript
// src/routes/chat.ts
import { Hono } from "hono";
import { createWebSocketStream } from "@hono/node-server/websocket";
import { verify } from "hono/jwt";
import { connectionManager } from "../websocket/connection-manager";

const chat = new Hono();

// WebSocket 聊天连接
chat.get("/ws", async (c) => {
  if (c.req.header("upgrade") !== "websocket") {
    return c.text("Expected websocket", 400);
  }

  // 验证 Token（从查询参数获取）
  const token = c.req.query("token");
  if (!token) {
    return c.text("Token required", 401);
  }

  try {
    const payload = await verify(token, process.env.JWT_SECRET!);
    const userId = payload.sub as string;

    return c.streamText(async (stream) => {
      const wsStream = createWebSocketStream(c);
      const clientId = crypto.randomUUID();

      // 创建客户端
      const client = {
        id: clientId,
        userId,
        socket: wsStream as any,
        rooms: new Set(),
      };

      connectionManager.add(client);

      // 发送欢迎消息
      wsStream.writable.write(
        JSON.stringify({
          type: "system",
          message: "Connected to chat server",
          clientId,
        })
      );

      // 处理消息
      wsStream.readable.pipeTo(
        new WritableStream({
          async write(chunk) {
            try {
              const message = JSON.parse(new TextDecoder().decode(chunk));
              await handleChatMessage(client, message);
            } catch (error) {
              console.error("Error handling message:", error);
            }
          },
        })
      );

      // 清理连接
      stream.close();
      connectionManager.remove(clientId);
    });
  } catch (error) {
    return c.text("Invalid token", 401);
  }
});

// 处理聊天消息
async function handleChatMessage(client: any, message: any) {
  switch (message.type) {
    case "chat":
      // 广播消息给所有用户
      connectionManager.broadcast({
        type: "chat",
        data: {
          id: crypto.randomUUID(),
          userId: client.userId,
          username: message.username,
          content: message.content,
          timestamp: new Date().toISOString(),
        },
      });
      break;

    case "typing":
      // 广播输入状态
      connectionManager.broadcast({
        type: "typing",
        data: {
          userId: client.userId,
          username: message.username,
          isTyping: true,
        },
      });
      break;
  }
}

export default chat;
```

## 房间系统

### 加入和离开房间

```typescript
interface Room {
  id: string;
  name: string;
  members: Set<string>;
}

export class RoomManager {
  private rooms: Map<string, Room> = new Map();

  create(id: string, name: string): Room {
    const room = {
      id,
      name,
      members: new Set(),
    };
    this.rooms.set(id, room);
    return room;
  }

  join(roomId: string, clientId: string): boolean {
    const room = this.rooms.get(roomId);
    if (!room) return false;

    room.members.add(clientId);
    return true;
  }

  leave(roomId: string, clientId: string): boolean {
    const room = this.rooms.get(roomId);
    if (!room) return false;

    room.members.delete(clientId);

    // 如果房间为空，删除房间
    if (room.members.size === 0) {
      this.rooms.delete(roomId);
    }

    return true;
  }

  getMembers(roomId: string): Set<string> | undefined {
    const room = this.rooms.get(roomId);
    return room?.members;
  }

  broadcastToRoom(roomId: string, message: any): void {
    const members = this.getMembers(roomId);
    if (!members) return;

    const data = JSON.stringify(message);

    for (const clientId of members) {
      const client = connectionManager.get(clientId);
      if (client && client.socket.readyState === WebSocket.OPEN) {
        client.socket.send(data);
      }
    }
  }
}

export const roomManager = new RoomManager();
```

```typescript
// 在聊天处理中添加房间功能
async function handleChatMessage(client: any, message: any) {
  switch (message.type) {
    case "join-room":
      roomManager.join(message.roomId, client.id);
      client.rooms.add(message.roomId);

      // 通知房间其他人
      roomManager.broadcastToRoom(message.roomId, {
        type: "user-joined",
        data: {
          userId: client.userId,
          username: message.username,
        },
      });
      break;

    case "leave-room":
      roomManager.leave(message.roomId, client.id);
      client.rooms.delete(message.roomId);

      roomManager.broadcastToRoom(message.roomId, {
        type: "user-left",
        data: {
          userId: client.userId,
          username: message.username,
        },
      });
      break;

    case "room-chat":
      roomManager.broadcastToRoom(message.roomId, {
        type: "chat",
        data: {
          roomId: message.roomId,
          userId: client.userId,
          username: message.username,
          content: message.content,
          timestamp: new Date().toISOString(),
        },
      });
      break;
  }
}
```

### 通知管理器

```typescript
// src/services/notification.service.ts
interface Notification {
  id: string;
  userId: string;
  type: "like" | "comment" | "follow" | "mention";
  title: string;
  message: string;
  data?: any;
  read: boolean;
  createdAt: Date;
}

export class NotificationService {
  // 推送通知给用户
  pushToUser(
    userId: string,
    notification: Omit<Notification, "id" | "createdAt">
  ) {
    const clients = connectionManager.getByUserId(userId);

    if (clients.length === 0) {
      // 用户不在线，保存到数据库
      this.saveToDatabase(userId, notification);
      return;
    }

    // 用户在线，实时推送
    const fullNotification: Notification = {
      id: crypto.randomUUID(),
      ...notification,
      createdAt: new Date(),
    };

    for (const client of clients) {
      if (client.socket.readyState === WebSocket.OPEN) {
        client.socket.send(
          JSON.stringify({
            type: "notification",
            data: fullNotification,
          })
        );
      }
    }
  }

  // 点赞通知
  notifyLike(postAuthorId: string, likerName: string, postId: string) {
    this.pushToUser(postAuthorId, {
      userId: postAuthorId,
      type: "like",
      title: "New Like",
      message: `${likerName} liked your post`,
      data: { postId },
      read: false,
    });
  }

  // 评论通知
  notifyComment(
    postAuthorId: string,
    commenterName: string,
    postId: string,
    comment: string
  ) {
    this.pushToUser(postAuthorId, {
      userId: postAuthorId,
      type: "comment",
      title: "New Comment",
      message: `${commenterName} commented: ${comment.substring(0, 50)}...`,
      data: { postId },
      read: false,
    });
  }

  // 提及通知
  notifyMention(userId: string, mentionerName: string, postId: string) {
    this.pushToUser(userId, {
      userId,
      type: "mention",
      title: "You were mentioned",
      message: `${mentionerName} mentioned you in a post`,
      data: { postId },
      read: false,
    });
  }

  private async saveToDatabase(
    userId: string,
    notification: Omit<Notification, "id" | "createdAt">
  ) {
    // 保存到数据库实现
  }
}

export const notificationService = new NotificationService();
```

### 自动重连

```typescript
class ReconnectingWebSocket {
  private url: string;
  private ws: WebSocket | null = null;
  private reconnectDelay: number = 1000;
  private maxReconnectDelay: number = 30000;
  private reconnectAttempts: number = 0;
  private maxReconnectAttempts: number = 10;

  constructor(url: string) {
    this.url = url;
    this.connect();
  }

  connect() {
    this.ws = new WebSocket(this.url);

    this.ws.onopen = () => {
      console.log("Connected");
      this.reconnectAttempts = 0;
      this.reconnectDelay = 1000;
    };

    this.ws.onclose = () => {
      console.log("Disconnected");
      this.reconnect();
    };

    this.ws.onerror = (error) => {
      console.error("WebSocket error:", error);
    };

    this.ws.onmessage = (event) => {
      this.handleMessage(JSON.parse(event.data));
    };
  }

  reconnect() {
    if (this.reconnectAttempts >= this.maxReconnectAttempts) {
      console.error("Max reconnect attempts reached");
      return;
    }

    this.reconnectAttempts++;
    console.log(`Reconnecting... (attempt ${this.reconnectAttempts})`);

    setTimeout(() => {
      this.connect();
    }, this.reconnectDelay);

    this.reconnectDelay = Math.min(
      this.reconnectDelay * 2,
      this.maxReconnectDelay
    );
  }

  send(data: any) {
    if (this.ws && this.ws.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(data));
    }
  }

  handleMessage(message: any) {
    // 处理消息
  }
}

// 使用
const ws = new ReconnectingWebSocket("ws://localhost:3000/ws");
```

### React Hook

```typescript
// src/hooks/useWebSocket.ts
import { useEffect, useRef, useCallback } from 'react'

interface UseWebSocketOptions {
  onMessage?: (data: any) => void
  onConnect?: () => void
  onDisconnect?: () => void
  token?: string
}

export function useWebSocket(url: string, options: UseWebSocketOptions = {}) {
  const wsRef = useRef<WebSocket | null>(null)
  const reconnectTimeoutRef = useRef<NodeJS.Timeout>()

  const connect = useCallback(() => {
    const wsUrl = options.token ? `${url}?token=${options.token}` : url
    const ws = new WebSocket(wsUrl)

    ws.onopen = () => {
      console.log('WebSocket connected')
      options.onConnect?.()
    }

    ws.onmessage = (event) => {
      const data = JSON.parse(event.data)
      options.onMessage?.(data)
    }

    ws.onclose = () => {
      console.log('WebSocket disconnected')
      options.onDisconnect?.()

      // 自动重连
      reconnectTimeoutRef.current = setTimeout(() => {
        connect()
      }, 3000)
    }

    ws.onerror = (error) => {
      console.error('WebSocket error:', error)
    }

    wsRef.current = ws
  }, [url, options])

  const disconnect = useCallback(() => {
    wsRef.current?.close()
    clearTimeout(reconnectTimeoutRef.current)
  }, [])

  const send = useCallback((data: any) => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      wsRef.current.send(JSON.stringify(data))
    }
  }, [])

  useEffect(() => {
    connect()

    return () => {
      disconnect()
    }
  }, [connect, disconnect])

  return { send, disconnect }
}

// 使用示例
function ChatComponent() {

          break
        case 'notification':
          // 显示通知
          break
      }
    },
  })

    send({
      type: 'chat',

    })
  }

  return (
    <div>
      {/* 聊天界面 */}

  )
}
```
