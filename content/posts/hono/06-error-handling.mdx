## 前言

在任何应用程序中，错误处理都是至关重要的部分。一个良好的错误处理机制不仅能提升用户体验，还能大大简化开发和调试过程。想象一下，当用户提交表单时，服务器返回 500 错误却不提供任何有用信息，或者在生产环境中暴露了敏感的堆栈跟踪——这些都是糟糕的错误处理示例。

Hono 提供了强大而灵活的错误处理机制，包括内置的 `HTTPException`、全局错误处理器、自定义异常类等。通过这些工具，你可以构建统一、友好且安全的错误处理系统。

本篇文章将带你：

- 🎯 理解错误处理的重要性和最佳实践
- 🛡️ 使用 HTTPException 处理 HTTP 错误
- 🔧 创建自定义业务异常类
- 🌐 实现全局错误处理器
- 📝 记录和监控错误
- 🔒 根据环境提供不同的错误响应

## 错误处理基础

### 为什么需要良好的错误处理

```typescript
// ❌ 糟糕的错误处理
app.get("/users/:id", async (c) => {
  const user = await db.query.users.findFirst({
    where: eq(users.id, c.req.param("id")),
  });

  // 没有检查 user 是否存在
  return c.json(user);
  // 如果 user 是 null，返回 null 给客户端
  // 客户端不知道发生了什么
});

// ❌ 糟糕的错误处理 - 暴露敏感信息
app.get("/users/:id", async (c) => {
  try {
    const user = await findUser(c.req.param("id"));
    return c.json(user);
  } catch (error) {
    // 直接返回错误对象，可能暴露敏感信息
    return c.json({ error: error.message }, 500);
    // 可能暴露数据库结构、文件路径等
  }
});

// ✅ 良好的错误处理
app.get("/users/:id", async (c) => {
  const user = await findUser(c.req.param("id"));

  if (!user) {
    throw new HTTPException(404, {
      message: "User not found",
      res: c.json(
        {
          success: false,
          error: "USER_NOT_FOUND",
          message: "The requested user does not exist",
        },
        404
      ),
    });
  }

  return c.json({ success: true, data: user });
});
```

### 错误处理的原则

1. **统一性**：所有错误使用统一的响应格式
2. **安全性**：不在生产环境暴露敏感信息
3. **友好性**：提供清晰、可操作的错误消息
4. **可追踪性**：记录足够的日志用于调试
5. **HTTP 语义**：使用正确的 HTTP 状态码

## HTTPException

Hono 提供了内置的 `HTTPException` 类用于处理 HTTP 错误。

### 基本使用

```typescript
import { HTTPException } from "hono/http-exception";

app.get("/users/:id", async (c) => {
  const user = await findUser(c.req.param("id"));

  if (!user) {
    // 抛出 HTTPException
    throw new HTTPException(404, {
      message: "User not found",
    });
  }

  return c.json(user);
});

// Hono 会自动捕获 HTTPException 并返回响应
// {
//   "message": "User not found"
// }
```

### 自定义响应

```typescript
app.get("/users/:id", async (c) => {
  const user = await findUser(c.req.param("id"));

  if (!user) {
    throw new HTTPException(404, {
      res: c.json(
        {
          success: false,
          error: "USER_NOT_FOUND",
          message: "The requested user does not exist",
          documentation: "https://api.example.com/docs/errors#user-not-found",
        },
        404
      ),
    });
  }

  return c.json({ success: true, data: user });
});
```

### 常见 HTTP 状态码


## 自定义异常类

对于业务逻辑中的特定错误，创建自定义异常类可以让代码更加清晰和易于维护。

### 基础自定义异常

```typescript
// 自定义业务异常基类
export class BusinessException extends Error {
  constructor(
    public code: string,
    public message: string,
    public statusCode: number = 400
  ) {
    super(message);
    this.name = "BusinessException";
  }
}

// 使用自定义异常
app.post("/users", async (c) => {
  const { email } = await c.req.json();

  const existingUser = await findUserByEmail(email);

  if (existingUser) {
    throw new BusinessException(
      "EMAIL_ALREADY_EXISTS",
      "Email already registered",
      409
    );
  }

  // 创建用户...
});

// 在全局错误处理器中处理
app.onError((err, c) => {
  if (err instanceof BusinessException) {
    return c.json(
      {
        success: false,
        code: err.code,
        message: err.message,
      },
      err.statusCode
    );
  }

  // 其他错误...
});
```

### 完整的异常体系

```typescript
// exceptions/index.ts
import { HTTPException } from "hono/http-exception";

// 基础业务异常
export class BaseException extends Error {
  constructor(
    public code: string,
    public message: string,
    public statusCode: number = 500,
    public details?: any
  ) {
    super(message);
    this.name = this.constructor.name;
    Error.captureStackTrace(this, this.constructor);
  }

  toResponse() {
    return {
      success: false,
      code: this.code,
      message: this.message,
      ...(this.details && { details: this.details }),
    };
  }
}

// 认证异常
export class AuthenticationException extends BaseException {
  constructor(message: string = "Authentication failed") {
    super("AUTHENTICATION_FAILED", message, 401);
  }
}

export class TokenExpiredException extends BaseException {
  constructor() {
    super("TOKEN_EXPIRED", "Token has expired", 401);
  }
}

export class InvalidTokenException extends BaseException {
  constructor() {
    super("INVALID_TOKEN", "Invalid token", 401);
  }
}

// 授权异常
export class AuthorizationException extends BaseException {
  constructor(message: string = "Insufficient permissions") {
    super("AUTHORIZATION_FAILED", message, 403);
  }
}

// 资源异常
export class NotFoundException extends BaseException {
  constructor(resource: string = "Resource") {
    super("NOT_FOUND", `${resource} not found`, 404);
  }
}

export class AlreadyExistsException extends BaseException {
  constructor(resource: string = "Resource") {
    super("ALREADY_EXISTS", `${resource} already exists`, 409);
  }
}

// 验证异常
export class ValidationException extends BaseException {
  constructor(errors: any[]) {
    super("VALIDATION_FAILED", "Validation failed", 422, { errors });
  }
}

// 速率限制异常
export class RateLimitException extends BaseException {
  constructor(retryAfter?: number) {
    super("RATE_LIMIT_EXCEEDED", "Rate limit exceeded", 429, { retryAfter });
  }
}

// 使用示例
app.post("/auth/login", async (c) => {
  const { email, password } = await c.req.json();

  const user = await findUserByEmail(email);

  if (!user) {
    throw new AuthenticationException("Invalid credentials");
  }

  const isValid = await verifyPassword(password, user.password);

  if (!isValid) {
    throw new AuthenticationException("Invalid credentials");
  }

  // 生成 token...
});

app.delete("/posts/:id", async (c) => {
  const post = await findPost(c.req.param("id"));

  if (!post) {
    throw new NotFoundException("Post");
  }

  if (post.authorId !== c.get("user").id) {
    throw new AuthorizationException("You can only delete your own posts");
  }

  // 删除文章...
});
```

## 全局错误处理器

全局错误处理器是统一处理所有错误的最佳方式。

### 基础全局错误处理器

```typescript
import { Hono } from "hono";
import { HTTPException } from "hono/http-exception";
import { BaseException } from "./exceptions";

const app = new Hono();

app.onError((err, c) => {
  // 处理自定义业务异常
  if (err instanceof BaseException) {
    return c.json(err.toResponse(), err.statusCode);
  }

  // 处理 HTTPException
  if (err instanceof HTTPException) {
    return c.json(
      {
        success: false,
        message: err.message,
      },
      err.status
    );
  }

  // 处理其他未知错误
  console.error("Unhandled error:", err);

  return c.json(
    {
      success: false,
      message: "Internal server error",
    },
    500
  );
});
```

### 完整的全局错误处理器

```typescript
interface ErrorResponse {
  success: false;
  code: string;
  message: string;
  details?: any;
  stack?: string;
  requestId?: string;
}

app.onError((err, c) => {
  const requestId = crypto.randomUUID();
  const isDevelopment = c.env?.ENV === "development";

  // 记录错误
  console.error({
    requestId,
    error: err.message,
    stack: err.stack,
    url: c.req.url,
    method: c.req.method,
    headers: Object.fromEntries(c.req.raw.headers),
    body: err.body,
  });

  // 自定义业务异常
  if (err instanceof BaseException) {
    const response: ErrorResponse = {
      success: false,
      ...err.toResponse(),
      requestId,
    };

    // 开发环境添加堆栈跟踪
    if (isDevelopment) {
      response.stack = err.stack;
    }

    return c.json(response, err.statusCode);
  }

  // HTTPException
  if (err instanceof HTTPException) {
    const response: ErrorResponse = {
      success: false,
      code: "HTTP_ERROR",
      message: err.message,
      requestId,
    };

    if (isDevelopment) {
      response.stack = err.stack;
    }

    return c.json(response, err.status);
  }

  // Zod 验证错误
  if (err.name === "ZodError") {
    const response: ErrorResponse = {
      success: false,
      code: "VALIDATION_ERROR",
      message: "Validation failed",
      details: err.errors,
      requestId,
    };

    return c.json(response, 400);
  }

  // 数据库错误
  if (err.name === "DatabaseError") {
    const response: ErrorResponse = {
      success: false,
      code: "DATABASE_ERROR",
      message: isDevelopment ? err.message : "Database operation failed",
      requestId,
    };

    if (isDevelopment) {
      response.stack = err.stack;
    }

    return c.json(response, 500);
  }

  // 未知错误
  const response: ErrorResponse = {
    success: false,
    code: "INTERNAL_ERROR",
    message: isDevelopment ? err.message : "An unexpected error occurred",
    requestId,
  };

  if (isDevelopment) {
    response.stack = err.stack;
  }

  return c.json(response, 500);
});

// 404 处理
app.notFound((c) => {
  return c.json(
    {
      success: false,
      code: "NOT_FOUND",
      message: "Route not found",
      requestId: crypto.randomUUID(),
    },
    404
  );
});
```

```typescript
import { HTTPException } from "hono/http-exception";
import { zValidator } from "@hono/zod-validator";
import { z } from "zod";

export class BaseException extends Error {
  constructor(
    public code: string,
    public message: string,
    public statusCode: number = 500,
    public details?: any
  ) {
    super(message);
    this.name = this.constructor.name;
  }

  toResponse() {
    return {
      success: false,
      code: this.code,
      message: this.message,
      ...(this.details && { details: this.details }),
    };
  }
}

export class AuthenticationException extends BaseException {
  constructor(message: string = "Authentication failed") {
    super("AUTHENTICATION_FAILED", message, 401);
  }
}

export class AuthorizationException extends BaseException {
  constructor(message: string = "Insufficient permissions") {
    super("AUTHORIZATION_FAILED", message, 403);
  }
}

export class NotFoundException extends BaseException {
  constructor(resource: string = "Resource") {
    super("NOT_FOUND", `${resource} not found`, 404);
  }
}

export class ValidationException extends BaseException {
  constructor(errors: any[]) {
    super("VALIDATION_FAILED", "Validation failed", 422, { errors });
  }
}

// ==================== 中间件 ====================

// 认证中间件
export const authMiddleware = async (c: any, next: any) => {
  const token = c.req.header("Authorization")?.replace("Bearer ", "");

  if (!token) {
    throw new AuthenticationException("Token required");
  }

  try {
    const payload = await verifyToken(token);
    const user = await findUserById(payload.sub);

    if (!user) {
      throw new AuthenticationException("User not found");
    }

    c.set("user", user);
    await next();
  } catch (error) {
    throw new AuthenticationException("Invalid token");
  }
};

// 授权中间件
export const requireRole = (...roles: string[]) => {
  return async (c: any, next: any) => {
    const user = c.get("user");

    if (!roles.includes(user.role)) {
      throw new AuthorizationException(
        `Requires one of these roles: ${roles.join(", ")}`
      );
    }

    await next();
  };
};

// ==================== 应用配置 ====================

const app = new Hono();

// 全局错误处理器
app.onError((err, c) => {
  const requestId = crypto.randomUUID();
  const isDevelopment = c.env?.ENV === "development";

  // 记录错误
  console.error({
    requestId,
    error: {
      name: err.name,
      message: err.message,
      stack: err.stack,
    },
    context: {
      url: c.req.url,
      method: c.req.method,
      userId: c.get("user")?.id,
    },
  });

  // 自定义异常
  if (err instanceof BaseException) {
    const response = {
      ...err.toResponse(),
      requestId,
    };

    if (isDevelopment) {
      (response as any).stack = err.stack;
    }

    return c.json(response, err.statusCode);
  }

  // HTTPException
  if (err instanceof HTTPException) {
    return c.json(
      {
        success: false,
        code: "HTTP_ERROR",
        message: err.message,
        requestId,
      },
      err.status
    );
  }

  // Zod 验证错误
  if (err.name === "ZodError") {
    return c.json(
      {
        success: false,
        code: "VALIDATION_ERROR",
        message: "Validation failed",
        details: err.errors,
        requestId,
      },
      400
    );
  }

  // 未知错误
  return c.json(
    {
      success: false,
      code: "INTERNAL_ERROR",
      message: isDevelopment ? err.message : "Internal server error",
      requestId,
    },
    500
  );
});

// 404 处理
app.notFound((c) => {
  return c.json(
    {
      success: false,
      code: "NOT_FOUND",
      message: "Route not found",
      requestId: crypto.randomUUID(),
    },
    404
  );
});

// ==================== 路由示例 ====================

// 认证路由
app.post("/auth/login", async (c) => {
  const { email, password } = await c.req.json();

  const user = await findUserByEmail(email);

  if (!user) {
    throw new AuthenticationException("Invalid credentials");
  }

  const isValid = await verifyPassword(password, user.password);

  if (!isValid) {
    throw new AuthenticationException("Invalid credentials");
  }

  const token = await generateToken(user);

  return c.json({
    success: true,
    data: { token, user: sanitizeUser(user) },
  });
});

// 受保护的路由
app.get("/api/users", authMiddleware, requireRole("admin"), async (c) => {
  const users = await getAllUsers();
  return c.json({ success: true, data: users });
});

app.get("/api/users/:id", authMiddleware, async (c) => {
  const user = await findUser(c.req.param("id"));

  if (!user) {
    throw new NotFoundException("User");
  }

  return c.json({ success: true, data: user });
});

app.patch(
  "/api/users/:id",
  authMiddleware,
  zValidator(
    "json",
    z.object({
      name: z.string().min(1).max(50).optional(),
      email: z.string().email().optional(),
    })
  ),
  async (c) => {
    const { id } = c.req.param();
    const data = c.req.valid("json");
    const currentUser = c.get("user");

    // 检查权限
    if (id !== currentUser.id && currentUser.role !== "admin") {
      throw new AuthorizationException("You can only update your own profile");
    }

    const updated = await updateUser(id, data);

    return c.json({ success: true, data: updated });
  }
);
```

```typescript
      body: JSON.stringify({
        text: `🚨 ${error.code}: ${error.message}`,
        attachments: [
          {
            color: "danger",
            fields: [
              { title: "Request ID", value: context.requestId, short: true },
              { title: "URL", value: context.url, short: true },
              { title: "Method", value: context.method, short: true },
              { title: "User ID", value: context.userId || "N/A", short: true },
            ],
          },
        ],
      }),
    });
  }
};
```

## 常见问题

### Q1: 如何处理异步错误？

```typescript
// ✅ 自动处理：在 async 函数中抛出的错误会被全局错误处理器捕获
app.get("/users/:id", async (c) => {
  const user = await findUser(c.req.param("id"));

  if (!user) {
    throw new NotFoundException("User");
  }

  // 如果 findUser 抛出错误，也会被全局错误处理器捕获
  return c.json({ success: true, data: user });
});

// ❌ 不要手动捕获所有错误
app.get("/users/:id", async (c) => {
  try {
    const user = await findUser(c.req.param("id"));
    return c.json({ success: true, data: user });
  } catch (error) {
    // 这样会绕过全局错误处理器
    return c.json({ error: "Something went wrong" }, 500);
  }
});
```
