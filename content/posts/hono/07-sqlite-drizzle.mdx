## 前言

在现代 Web 应用中，数据持久化是不可或缺的功能。选择合适的数据库和 ORM（对象关系映射）工具对于应用的性能、可维护性和部署方式都有着重要影响。

**Drizzle ORM** 是一个新兴的 TypeScript-first ORM，以其出色的性能、类型安全性和边缘运行时支持而备受关注。与传统的 Prisma 或 TypeORM 相比，Drizzle 采用了类似 SQL 的 API 设计，运行时开销更小，并且完美适配 Cloudflare Workers、Deno Deploy 等边缘环境。

本篇文章将带你：

- 🎯 理解 Drizzle ORM 的核心优势
- 📝 定义数据库 Schema 和类型
- 🔄 执行各种 CRUD 操作
- 🔗 处理表关系和复杂查询
- 💾 管理数据库迁移
- 🌐 在边缘环境使用数据库

## 为什么选择 Drizzle ORM

### Drizzle vs 其他 ORM

| 特性                | Drizzle     | Prisma          | TypeORM     |
| ------------------- | ----------- | --------------- | ----------- |
| **TypeScript 支持** | ✅ 原生     | ✅ 优秀         | ⚠️ 需装饰器 |
| **边缘运行**        | ✅ 完美支持 | ⚠️ 有限         | ❌ 不支持   |
| **性能**            | 🚀 极快     | 🚀 快           | 🐢 较慢     |
| **运行时开销**      | ✅ 零开销   | ❌ 需要查询引擎 | ❌ 重量级   |
| **SQL 控制**        | ✅ 完全控制 | ⚠️ 抽象较高     | ✅ 完全控制 |
| **学习曲线**        | 🟡 中等     | 🟢 平缓         | 🔴 陡峭     |
| **包大小**          | 🟢 小       | 🟡 中等         | 🔴 大       |
| **关系查询**        | ✅ 支持     | ✅ 优秀         | ✅ 支持     |

### Drizzle 的核心优势

**1. TypeScript-first**

```typescript
// 完全的类型推断
const users = await db.select().from(usersTable);
// users: User[] (自动推断类型)

// 插入时类型验证
await db.insert(usersTable).values({
  name: "John",
  email: "john@example.com",
  // age: number | undefined (根据 Schema 定义)
});
```

**2. SQL-like API**

```typescript
// 类似 SQL 的查询语法
const users = await db
  .select({
    id: users.id,
    name: users.name,
    postCount: sql<number>`count(${posts.id})`,
  })
  .from(users)
  .leftJoin(posts, eq(users.id, posts.authorId))
  .where(eq(users.role, "admin"))
  .groupBy(users.id)
  .orderBy(desc(users.createdAt));
```

**3. 边缘就绪**

```typescript
// Cloudflare Workers D1
import { drizzle } from "drizzle-orm/d1";

export default {
  async fetch(request: Request, env: any) {
    const db = drizzle(env.DB);
    // 使用数据库...
  },
};

// Deno Deploy (Deno KV)
import { drizzle } from "drizzle-orm/deno-kv";

const kv = await Deno.openKv();
const db = drizzle(kv);
```

**4. 零运行时开销**

```typescript
// 查询在编译时构建，运行时无额外开销
const query = db.select().from(users).where(eq(users.id, id));
// 生成的 SQL：SELECT * FROM users WHERE id = ?
// 运行时只是简单的参数绑定和执行
```

## 定义 Schema

### 基础表定义

```typescript
// src/db/schema.ts
import { sqliteTable, text, integer } from "drizzle-orm/sqlite-core";
import { sql } from "drizzle-orm";

// 用户表
export const users = sqliteTable("users", {
  id: text("id").primaryKey(),
  email: text("email").notNull().unique(),
  username: text("username").notNull().unique(),
  password: text("password").notNull(),
  name: text("name"),
  bio: text("bio"),
  avatar: text("avatar"),
  role: text("role", { enum: ["user", "admin", "moderator"] })
    .notNull()
    .default("user"),
  isActive: integer("is_active", { mode: "boolean" }).notNull().default(true),
  emailVerified: integer("email_verified", { mode: "boolean" })
    .notNull()
    .default(false),
  createdAt: integer("created_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
  updatedAt: integer("updated_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
});

// 文章表
export const posts = sqliteTable("posts", {
  id: text("id").primaryKey(),
  title: text("title").notNull(),
  slug: text("slug").notNull().unique(),
  content: text("content").notNull(),
  excerpt: text("excerpt"),
  coverImage: text("cover_image"),
  published: integer("published", { mode: "boolean" }).notNull().default(false),
  authorId: text("author_id")
    .notNull()
    .references(() => users.id, { onDelete: "cascade" }),
  views: integer("views").notNull().default(0),
  createdAt: integer("created_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
  updatedAt: integer("updated_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
  publishedAt: integer("published_at", { mode: "timestamp" }),
});

// 评论表
export const comments = sqliteTable("comments", {
  id: text("id").primaryKey(),
  content: text("content").notNull(),
  postId: text("post_id")
    .notNull()
    .references(() => posts.id, { onDelete: "cascade" }),
  authorId: text("author_id")
    .notNull()
    .references(() => users.id, { onDelete: "cascade" }),
  parentId: text("parent_id").references(() => comments.id),
  approved: integer("approved", { mode: "boolean" }).notNull().default(false),
  createdAt: integer("created_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
  updatedAt: integer("updated_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
});

// 标签表
export const tags = sqliteTable("tags", {
  id: text("id").primaryKey(),
  name: text("name").notNull().unique(),
  slug: text("slug").notNull().unique(),
  createdAt: integer("created_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
});

// 文章-标签关联表（多对多）
export const postTags = sqliteTable("post_tags", {
  postId: text("post_id")
    .notNull()
    .references(() => posts.id, { onDelete: "cascade" }),
  tagId: text("tag_id")
    .notNull()
    .references(() => tags.id, { onDelete: "cascade" }),
  createdAt: integer("created_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
});

// 分类表
export const categories = sqliteTable("categories", {
  id: text("id").primaryKey(),
  name: text("name").notNull(),
  slug: text("slug").notNull().unique(),
  description: text("description"),
  parentId: text("parent_id").references(() => categories.id),
  createdAt: integer("created_at", { mode: "timestamp" })
    .notNull()
    .default(sql`CURRENT_TIMESTAMP`),
});
```

### 关系定义

```typescript
// 继续在 schema.ts 中添加关系定义
import { relations } from "drizzle-orm";

// 用户关系
export const usersRelations = relations(users, ({ many }) => ({
  posts: many(posts),
  comments: many(comments),
}));

// 文章关系
export const postsRelations = relations(posts, ({ one, many }) => ({
  author: one(users, {
    fields: [posts.authorId],
    references: [users.id],
  }),
  category: one(categories, {
    fields: [posts.categoryId],
    references: [categories.id],
  }),
  comments: many(comments),
  tags: many(postTags),
}));

// 评论关系
export const commentsRelations = relations(comments, ({ one, many }) => ({
  post: one(posts, {
    fields: [comments.postId],
    references: [posts.id],
  }),
  author: one(users, {
    fields: [comments.authorId],
    references: [users.id],
  }),
  parent: one(comments, {
    fields: [comments.parentId],
    references: [comments.id],
  }),
  replies: many(comments),
}));

// 标签关系
export const tagsRelations = relations(tags, ({ many }) => ({
  posts: many(postTags),
}));

// 文章-标签关联关系
export const postTagsRelations = relations(postTags, ({ one }) => ({
  post: one(posts, {
    fields: [postTags.postId],
    references: [posts.id],
  }),
  tag: one(tags, {
    fields: [postTags.tagId],
    references: [tags.id],
  }),
}));
```

## 数据库连接

### 开发环境连接（SQLite）

```typescript
// src/db/index.ts
import { drizzle } from "drizzle-orm/better-sqlite3";
import Database from "better-sqlite3";
import * as schema from "./schema";

// 创建 SQLite 连接
const sqlite = new Database("./dev.db");

// 启用外键约束
sqlite.pragma("foreign_keys = ON");

// 创建 Drizzle 实例
export const db = drizzle(sqlite, { schema });

// 导出类型
export type Database = typeof db;
```

### 生产环境连接（Cloudflare D1）

```typescript
// src/db/index.ts (Cloudflare Workers)
import { drizzle } from "drizzle-orm/d1";
import * as schema from "./schema";

export interface Env {
  DB: D1Database;
}

export const createDb = (d1: D1Database) => {
  return drizzle(d1, { schema });
};
```

```typescript
if (process.env.NODE_ENV === "production") {
```

```typescript
app.get("/posts", async (c) => {
```

## 事务处理

### 基础事务

```typescript
// 事务转账示例
app.post("/transfer", async (c) => {
  const { fromId, toId, amount } = await c.req.json();

  await db.transaction(async (tx) => {
    // 检查余额
    const [fromAccount] = await tx
      .select()
      .from(schema.accounts)
      .where(eq(schema.accounts.id, fromId));

    if (fromAccount.balance < amount) {
      throw new Error("Insufficient balance");
    }

    // 扣款
    await tx
      .update(schema.accounts)
      .set({ balance: sql`${schema.accounts.balance} - ${amount}` })
      .where(eq(schema.accounts.id, fromId));

    // 加款
    await tx
      .update(schema.accounts)
      .set({ balance: sql`${schema.accounts.balance} + ${amount}` })
      .where(eq(schema.accounts.id, toId));
  });

  return c.json({
    success: true,
    message: "Transfer completed",
  });
});
```

### 嵌套事务

```typescript
app.post("/orders", async (c) => {
  const { items, shippingAddress } = await c.req.json();

  await db.transaction(async (tx) => {
    // 创建订单
    const [order] = await tx
      .insert(schema.orders)
      .values({
        id: crypto.randomUUID(),
        userId: c.get("user").id,
        shippingAddress,
        status: "pending",
      })
      .returning();

    // 嵌套事务：处理订单项
    await tx.transaction(async (tx2) => {
      for (const item of items) {
        // 检查库存
        const [product] = await tx2
          .select()
          .from(schema.products)
          .where(eq(schema.products.id, item.productId));

        if (product.stock < item.quantity) {
          throw new Error(`Insufficient stock for ${product.name}`);
        }

        // 减少库存
        await tx2
          .update(schema.products)
          .set({ stock: sql`${schema.products.stock} - ${item.quantity}` })
          .where(eq(schema.products.id, item.productId));

        // 创建订单项
        await tx2.insert(schema.orderItems).values({
          id: crypto.randomUUID(),
          orderId: order.id,
          productId: item.productId,
          quantity: item.quantity,
          price: product.price,
        });
      }
    });

    // 更新订单状态
    await tx
      .update(schema.orders)
      .set({ status: "confirmed" })
      .where(eq(schema.orders.id, order.id));
  });

  return c.json({
    success: true,
    message: "Order created",
  });
});
```

### 1. 使用查询构建器

```typescript
// 创建可复用的查询构建器
class UserRepository {
  constructor(private db: Database) {}

  async findAll(options?: { limit?: number; offset?: number; where?: SQL }) {
    let query = this.db.select().from(schema.users);

    if (options?.where) {
      query = query.where(options.where);
    }

    if (options?.limit) {
      query = query.limit(options.limit);
    }

    if (options?.offset) {
      query = query.offset(options.offset);
    }

    return await query;
  }

  async findById(id: string) {
    const [user] = await this.db
      .select()
      .from(schema.users)
      .where(eq(schema.users.id, id))
      .limit(1);

    return user;
  }
}
```
