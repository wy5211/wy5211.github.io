## 前言

在 Web 开发中，处理 HTTP 请求和响应是构建 API 的核心能力。无论是获取用户提交的表单数据、解析 JSON 请求体、设置 Cookie，还是返回各种格式的响应，都需要一个简洁而强大的 API。

Hono 提供了一套优雅且直观的请求/响应处理 API，让你能够轻松应对各种场景。与 Express 不同，Hono 采用了统一的 **Context（上下文）** 对象模式，所有的请求和响应操作都通过 `c` 对象完成，这种设计更加简洁和一致。

本篇文章将带你：

- 📥 深入理解 Context 对象的设计理念
- 🔍 掌握各种请求数据的读取方法
- 📤 学习不同响应类型的使用场景
- 🍪 实现 Cookie 的完整管理
- 🌊 探索流式响应和文件下载
- 💡 获取实用的最佳实践和技巧

## Context 对象详解

Hono 的每个路由处理器都会接收一个 `Context` 对象（通常命名为 `c`），它是连接请求和响应的桥梁。

### Context 的核心结构

```typescript
import { Hono } from "hono";

const app = new Hono();

app.get("/", (c) => {
  // c 是 Context 对象的实例

  // 请求相关（通过 c.req 访问）
  const url = c.req.url;
  const method = c.req.method;
  const header = c.req.header("User-Agent");

  // 响应相关（直接通过 c 调用）
  return c.json({ message: "Hello" });
});
```

### Context 的主要属性和方法

```typescript
// 请求相关
c.req.url; // 完整 URL
c.req.method; // HTTP 方法
c.req.header(); // 请求头
c.req.query(); // 查询参数
c.req.param(); // 路径参数
c.req.json(); // JSON 请求体
c.req.text(); // 文本请求体
c.req.formData(); // FormData 请求体

// 响应相关
c.json(); // 返回 JSON
c.text(); // 返回文本
c.html(); // 返回 HTML
c.xml(); // 返回 XML
c.body(); // 返回原始 body
c.header(); // 设置响应头
c.status(); // 设置状态码
c.redirect(); // 重定向

// 状态管理
c.set(); // 存储数据到 context
c.get(); // 从 context 获取数据
c.var; // 类型安全的数据存储
```

## 读取请求数据

### 1. 读取请求头

获取单个或多个请求头信息：

```typescript
import { Hono } from "hono";

const app = new Hono();

// 获取单个请求头
app.get("/", (c) => {
  const userAgent = c.req.header("User-Agent");
  const authorization = c.req.header("Authorization");

  return c.json({
    userAgent,
    hasAuth: !!authorization,
  });
});

// 获取所有请求头
app.get("/headers", (c) => {
  // 注意：header() 不传参数会返回 null
  // 需要使用特定方法获取所有头部
  const headers = Object.fromEntries(c.req.raw.headers);
  return c.json(headers);
});

// 检查请求头是否存在
app.use("/api/*", async (c, next) => {
  const apiKey = c.req.header("X-API-Key");

  if (!apiKey) {
    return c.json({ error: "API Key required" }, 401);
  }

  await next();
});
```

**实际应用场景**：

- **User-Agent 检测**：识别客户端类型（浏览器、移动端、API 调用工具）
- **认证**：从 Authorization 头获取 Bearer Token
- **CORS**：检查 Origin 处理跨域请求
- **内容协商**：根据 Accept 头返回不同格式

### 2. 读取查询参数

处理 URL 查询字符串（`?key=value&foo=bar`）：

```typescript
// 获取单个查询参数
app.get("/search", (c) => {
  const keyword = c.req.query("keyword");
  const page = c.req.query("page");

  return c.json({
    keyword, // string | undefined
    page, // string | undefined
  });
});

// 获取所有查询参数
app.get("/filter", (c) => {
  const queries = c.req.query();
  // 返回对象：{ key1: 'value1', key2: 'value2' }

  return c.json(queries);
});

// 处理数组参数（重复的 key）
// URL: /tags?tag=js&tag=ts&tag=react
app.get("/tags", (c) => {
  // 注意：req.query() 只返回第一个值
  // 需要手动解析或使用 URLSearchParams
  const url = new URL(c.req.url);
  const tags = url.searchParams.getAll("tag");

  return c.json({ tags });
  // { tags: ['js', 'ts', 'react'] }
});

// 带默认值的查询参数
app.get("/users", (c) => {
  const page = parseInt(c.req.query("page") || "1");
  const limit = parseInt(c.req.query("limit") || "10");
  const sort = c.req.query("sort") || "desc";

  return c.json({ page, limit, sort });
});
```

**最佳实践**：

```typescript
import { zValidator } from "@hono/zod-validator";
import { z } from "zod";

// 使用 Zod 验证查询参数
const querySchema = z.object({
  keyword: z.string().optional(),
  page: z.string().transform(Number).default("1"),
  limit: z.string().transform(Number).default("10"),
  sort: z.enum(["asc", "desc"]).default("desc"),
});

app.get("/posts", zValidator("query", querySchema), (c) => {
  const { keyword, page, limit, sort } = c.req.valid("query");

  // page, limit 已经是 number 类型
  // sort 是 'asc' | 'desc' 类型

  return c.json({ keyword, page, limit, sort });
});
```

### 4. 解析请求体

处理各种类型的请求体数据：

#### JSON 请求体

```typescript
import { zValidator } from "@hono/zod-validator";
import { z } from "zod";

// 解析 JSON
app.post("/users", async (c) => {
  const body = await c.req.json();
  // body: any

  const { name, email } = body;

  return c.json({ message: "User created", name, email });
});

// 类型安全的 JSON 解析
interface UserCreate {
  name: string;
  email: string;
  age?: number;
}

app.post("/users/typed", async (c) => {
  const body = await c.req.json<UserCreate>();
  // body: UserCreate

  return c.json({ name: body.name });
});

// 使用 Zod 验证
const userSchema = z.object({
  name: z.string().min(2).max(50),
  email: z.string().email(),
  age: z.number().min(18).optional(),
});

app.post("/users/validated", zValidator("json", userSchema), async (c) => {
  const user = c.req.valid("json");
  // user 类型自动推断为 { name: string; email: string; age?: number }

  // 创建用户...
  return c.json({ message: "User created" }, 201);
});
```

#### FormData 请求体

处理表单数据和文件上传：

```typescript
// 解析表单数据
app.post("/contact", async (c) => {
  const body = await c.req.formData();
  // body: FormData

  const name = body.get("name");
  const email = body.get("email");
  const message = body.get("message");

  return c.json({
    name,
    email,
    message,
  });
});

// 处理文件上传
app.post("/upload/avatar", async (c) => {
  const body = await c.req.formData();
  const file = body.get("avatar") as File;

  if (!file) {
    return c.json({ error: "No file uploaded" }, 400);
  }

  // 验证文件类型
  if (!file.type.startsWith("image/")) {
    return c.json({ error: "Only images allowed" }, 400);
  }

  // 验证文件大小（5MB）
  if (file.size > 5 * 1024 * 1024) {
    return c.json({ error: "File too large" }, 400);
  }

  // 保存文件...
  const filename = `${Date.now()}-${file.name}`;
  // await saveFile(file, filename)

  return c.json({
    filename,
    size: file.size,
    type: file.type,
  });
});

// 多文件上传
app.post("/upload/gallery", async (c) => {
  const body = await c.req.formData();

  const files: File[] = [];
  for (const [key, value] of body.entries()) {
    if (value instanceof File) {
      files.push(value);
    }
  }

  return c.json({
    count: files.length,
    files: files.map((f) => ({ name: f.name, size: f.size })),
  });
});
```

## Cookie 管理

Cookie 是在客户端存储小量数据的重要方式。Hono 提供了便捷的 Cookie 操作工具。

### 基本操作

```typescript
import { getCookie, setCookie, deleteCookie } from "hono/cookie";

// 设置 Cookie
app.get("/login", (c) => {
  setCookie(c, "token", "abc123", {
    httpOnly: true, // 仅 HTTP 访问，防止 XSS
    secure: true, // 仅 HTTPS 传输
    sameSite: "Lax", // CSRF 防护
    maxAge: 3600, // 1 小时（秒）
    path: "/", // 有效路径
    domain: ".example.com", // 有效域名
  });

  return c.text("Login successful");
});

// 获取 Cookie
app.get("/profile", (c) => {
  const token = getCookie(c, "token");

  if (!token) {
    return c.json({ error: "Not authenticated" }, 401);
  }

  return c.json({ token });
});

// 删除 Cookie
app.get("/logout", (c) => {
  deleteCookie(c, "token");
  return c.text("Logout successful");
});
```

### 实战：JWT 认证与 Cookie

```typescript
import { sign } from "hono/jwt";
import { getCookie, setCookie, deleteCookie } from "hono/cookie";

// 登录
app.post("/auth/login", async (c) => {
  const { email, password } = await c.req.json();

  // 验证用户...
  const user = await authenticateUser(email, password);

  if (!user) {
    return c.json({ error: "Invalid credentials" }, 401);
  }

  // 生成 JWT
  const token = await sign(
    { sub: user.id, email: user.email },
    process.env.JWT_SECRET!
  );

  // 设置 Cookie
  setCookie(c, "auth_token", token, {
    httpOnly: true,
    secure: process.env.NODE_ENV === "production",
    sameSite: "Lax",
    maxAge: 7 * 24 * 60 * 60, // 7 天
    path: "/",
  });

  return c.json({ message: "Login successful" });
});

// 受保护的路由
app.get("/api/me", async (c) => {
  const token = getCookie(c, "auth_token");

  if (!token) {
    return c.json({ error: "Unauthorized" }, 401);
  }

  // 验证 token
  try {
    const payload = await verify(token, process.env.JWT_SECRET!);
    const user = await db.findUser(payload.sub);

    return c.json({ user });
  } catch (error) {
    return c.json({ error: "Invalid token" }, 401);
  }
});

// 登出
app.post("/auth/logout", (c) => {
  deleteCookie(c, "auth_token");
  return c.json({ message: "Logout successful" });
});
```

### Cookie 最佳实践

```typescript
// 1. 敏感数据使用 httpOnly
setCookie(c, "session", "xxx", {
  httpOnly: true, // JavaScript 无法访问
  secure: true,
  sameSite: "Strict",
});

// 2. 非敏感数据可以不使用 httpOnly
setCookie(c, "theme", "dark", {
  maxAge: 365 * 24 * 60 * 60, // 1 年
});

// 3. 使用签名 Cookie 防止篡改
import { signCookie } from "hono/cookie";

app.post("/set-prefs", async (c) => {
  const prefs = await c.req.json();

  // 签名 Cookie
  const signed = await signCookie(prefs, "secret-key");

  setCookie(c, "preferences", signed);

  return c.json({ message: "Preferences saved" });
});

// 4. 分离认证 Cookie
// access_token：短期，频繁使用
// refresh_token：长期，用于刷新
setCookie(c, "access_token", accessToken, {
  httpOnly: true,
  secure: true,
  sameSite: "Lax",
  maxAge: 15 * 60, // 15 分钟
});

setCookie(c, "refresh_token", refreshToken, {
  httpOnly: true,
  secure: true,
  sameSite: "Lax",
  maxAge: 7 * 24 * 60 * 60, // 7 天
});
```

## 文件下载

### 基本文件下载

```typescript
import { readFile } from "fs/promises";

// 下载静态文件
app.get("/download/:filename", async (c) => {
  const filename = c.req.param("filename");
  const filepath = `./uploads/${filename}`;

  try {
    const file = await readFile(filepath);

    return c.body(file, 200, {
      "Content-Type": "application/octet-stream",
      "Content-Disposition": `attachment; filename="${filename}"`,
    });
  } catch (error) {
    return c.json({ error: "File not found" }, 404);
  }
});

// 根据 MIME 类型返回
app.get("/files/:filename", async (c) => {
  const filename = c.req.param("filename");
  const ext = filename.split(".").pop();

  const mimeTypes: Record<string, string> = {
    jpg: "image/jpeg",
    png: "image/png",
    pdf: "application/pdf",
    txt: "text/plain",
  };

  const contentType = mimeTypes[ext || ""] || "application/octet-stream";

  const file = await readFile(`./files/${filename}`);

  return c.body(file, 200, {
    "Content-Type": contentType,
    "Content-Disposition": `inline; filename="${filename}"`,
  });
});
```

### 流式文件下载

```typescript
import { createReadStream } from "fs";
import { stat } from "fs/promises";

// 大文件流式下载
app.get("/download-large/:filename", async (c) => {
  const filename = c.req.param("filename");
  const filepath = `./downloads/${filename}`;

  const stats = await stat(filepath);
  const fileSize = stats.size;
  const range = c.req.header("Range");

  // 处理断点续传
  if (range) {
    const parts = range.replace(/bytes=/, "").split("-");
    const start = parseInt(parts[0], 10);
    const end = parts[1] ? parseInt(parts[1], 10) : fileSize - 1;
    const chunkSize = end - start + 1;

    c.header("Content-Range", `bytes ${start}-${end}/${fileSize}`);
    c.header("Accept-Ranges", "bytes");
    c.header("Content-Length", chunkSize.toString());
    c.header("Content-Type", "application/octet-stream");

    const stream = createReadStream(filepath, { start, end });

    return c.stream(async (stream) => {
      stream.pipe(stream);
    });
  }

  // 正常下载
  c.header("Content-Length", fileSize.toString());
  c.header("Content-Type", "application/octet-stream");
  c.header("Content-Disposition", `attachment; filename="${filename}"`);
```

## 流式响应

流式响应可以逐步发送数据，适用于大数据集、实时更新等场景。

### Server-Sent Events (SSE)

```typescript
// SSE 推送
app.get("/events", (c) => {
  return c.streamText(async (stream) => {
    // 发送欢迎消息
    await stream.write(`data: ${JSON.stringify({ message: "Connected" })}\n\n`);

    // 定期发送数据
    let count = 0;
    const interval = setInterval(async () => {
      count++;

      await stream.write(
        `data: ${JSON.stringify({ count, time: Date.now() })}\n\n`
      );

      if (count >= 10) {
        clearInterval(interval);
        stream.close();
      }
    }, 1000);
  });
});

// 实时进度更新
app.get("/download-progress/:taskId", async (c) => {
  const { taskId } = c.req.param();

  return c.streamText(async (stream) => {
    while (true) {
      const progress = await getTaskProgress(taskId);

      await stream.write(`data: ${JSON.stringify(progress)}\n\n`);

      if (progress.status === "completed") {
        break;
      }

      await new Promise((resolve) => setTimeout(resolve, 1000));
    }

    stream.close();
  });
});
```

### Q2: 如何处理多种请求体类型？

```typescript
app.post("/data", async (c) => {
  const contentType = c.req.header("Content-Type");

  if (contentType?.includes("application/json")) {
    const json = await c.req.json();
    return c.json({ type: "json", data: json });
  }

  if (contentType?.includes("application/x-www-form-urlencoded")) {
    const formData = await c.req.formData();
    return c.json({ type: "form", data: Object.fromEntries(formData) });
  }

  if (contentType?.includes("text/plain")) {
    const text = await c.req.text();
    return c.json({ type: "text", data: text });
  }

  return c.json({ error: "Unsupported content type" }, 415);
});
```
