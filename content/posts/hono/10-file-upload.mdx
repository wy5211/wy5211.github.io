## 前言

文件上传是现代 Web 应用中非常常见的功能。用户需要上传头像、照片、文档等各种文件。然而，文件上传也是最容易出安全漏洞的地方之一——不当的文件处理可能导致服务器被攻击、存储空间被耗尽，甚至恶意文件被执行。

Hono 提供了简单而强大的文件上传处理能力，结合 Cloudflare R2、AWS S3 等云存储服务，可以构建安全可靠的文件上传系统。

本篇文章将带你：

- 📁 理解文件上传的工作原理
- 🔍 处理 multipart/form-data 请求
- ✅ 验证文件类型、大小和内容
- ☁️ 存储到本地或云服务
- 🖼️ 实现图片处理和优化
- 🔒 遵循安全最佳实践

## 基础文件上传

### 安装依赖

```bash
# Node.js 环境
npm install @hono/node-server

# 如果需要图片处理
npm install sharp
```

### 单文件上传

```typescript
import { Hono } from "hono";
import { multipart } from "@hono/node-server";

const app = new Hono();

app.post(
  "/upload/avatar",
  multipart(async (c) => {
    const body = await c.req.parseBody();
    const file = body.avatar as File;

    if (!file) {
      return c.json({ success: false, message: "No file uploaded" }, 400);
    }

    // 文件信息
    const data = {
      name: file.name,
      type: file.type,
      size: file.size,
    };

    return c.json({ success: true, data });
  })
);
```

### 多文件上传

```typescript
app.post(
  "/upload/gallery",
  multipart(async (c) => {
    const body = await c.req.parseBody();
    const files: File[] = [];

    for (const [key, value] of Object.entries(body)) {
      if (value instanceof File) {
        files.push(value);
      }
    }

    return c.json({
      success: true,
      data: {
        count: files.length,
        files: files.map((f) => ({ name: f.name, size: f.size })),
      },
    });
  })
);
```

## 文件验证

### 验证文件类型和大小

```typescript
const ALLOWED_TYPES = ["image/jpeg", "image/png", "image/webp"];
const MAX_SIZE = 5 * 1024 * 1024; // 5MB

app.post(
  "/upload",
  multipart(async (c) => {
    const body = await c.req.parseBody();
    const file = body.file as File;

    // 验证类型
    if (!ALLOWED_TYPES.includes(file.type)) {
      return c.json({ success: false, message: "Invalid file type" }, 400);
    }

    // 验证大小
    if (file.size > MAX_SIZE) {
      return c.json({ success: false, message: "File too large" }, 400);
    }

    return c.json({ success: true });
  })
);
```

## Cloudflare R2 上传

```typescript
// wrangler.toml
[[r2_buckets]];
binding = "R2";
bucket_name = "uploads";

// 上传到 R2
app.post(
  "/upload/r2",
  authMiddleware,
  multipart(async (c) => {
    const body = await c.req.parseBody();
    const file = body.file as File;

    const key = `uploads/${Date.now()}-${file.name}`;
    await c.env.R2.put(key, file.stream());

    const url = `${c.env.R2_PUBLIC_URL}/${key}`;

    return c.json({ success: true, data: { url, key } });
  })
);
```

## 安全最佳实践

1. **验证文件内容**（不只是扩展名）
2. **生成安全的文件名**（使用 UUID）
3. **限制文件大小**
4. **隔离用户文件**
5. **扫描病毒**（生产环境）
