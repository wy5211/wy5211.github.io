上一篇我们搭好了定投的框架，主线案例也启动了——年投入 20 万，月定投约 16600 元。

现在要回答一个关键问题：**这 16600 元，怎么分配到不同市场、不同指数基金上？**

你可能觉得选哪只基金最重要，但学术界有一个广泛认可的结论：**投资组合的长期收益，90% 以上由资产配置决定，而不是由选股或择时决定。**

这个结论来自 1986 年 Gary Brinson 等人发表的经典研究。他们分析了 91 只大型养老基金的 10 年数据，发现资产配置（股票/债券/现金的比例）解释了投资收益波动的 91.5%。选股和择时的贡献加起来不到 10%。

当然，这个研究被引用得太多，也有一些批评声音。但核心结论至今站得住脚：**你把钱放在哪类资产上、放多少，比你在同类资产里挑哪个重要得多。**

## 资产配置的核心逻辑

资产配置的本质就两件事：**分散** 和 **再平衡**。

### 分散：不把赌注押在一个结果上

分散不是"买很多只基金"那么简单。如果你买了 10 只科技主题基金，那不叫分散，叫"重复下注"。

真正的分散要满足一个条件：**你的各类资产之间相关性要低。** 也就是说，它们不能同涨同跌。

我们在第三篇聊过三个市场的相关性：美股和A股相关性偏低，A股和港股偏高。所以跨市场配置本身就是一种有效的分散。

### 再平衡：定期把偏离的比例调回来

假设你设定的比例是美股 60%、A股 30%、港股 10%。一年后美股大涨，变成了 70%、A股 25%、港股 5%。

这时候你需要卖掉一些美股，买入A股和港股，让比例回到 60:30:10。这就是再平衡。

再平衡有两个好处：

1. **纪律性**：强迫你在涨得多的时候卖出（止盈），在跌得多的时候买入（抄底）
2. **控制风险**：防止某个市场占比过高，组合的风险超出你的承受能力

## 先定大类：股票、债券、现金

在分配到具体市场之前，先确定大类资产的比例。

### 股票 vs 债券

股票代表经济增长，长期收益高但波动大。债券代表固定收益，收益低但稳定。

一个简单的经验法则：**你的股票比例 = 100 - 你的年龄。** 30 岁的人，70% 股票 + 30% 债券。50 岁的人，50% 股票 + 50% 债券。

这个法则比较粗，但提供了一个起点。实际上，比例取决于你的风险承受能力：

| 类型   | 股票   | 债券   | 适合人群                       |
| ------ | ------ | ------ | ------------------------------ |
| 激进型 | 80-90% | 10-20% | 年轻、收入稳定、能接受大幅波动 |
| 均衡型 | 60-70% | 30-40% | 有一定投资经验、希望稳健增长   |
| 保守型 | 40-50% | 50-60% | 接近退休、不能承受大幅亏损     |

### 现金

现金（或货币基金）不是投资，是缓冲。建议始终保持 6-12 个月生活费作为应急资金。这部分不算在投资组合里。

## 再定市场：美股、A股、港股怎么分

股票部分确定后，再在三个市场之间分配。

回顾一下各市场的特点（第三篇的结论）：

- **美股**：长期向上、确定性高 → 适合作为核心仓位
- **A股**：波动大、周期性强 → 适合在低估值时多配，高估值时少配
- **港股**：估值低、流动性风险 → 适合作为补充

基于此，一个基础的配置框架：

### 保守方案（稳健优先）

| 资产                 | 占比 | 说明     |
| -------------------- | ---- | -------- |
| 美股（标普 500）     | 40%  | 核心底仓 |
| 美股（纳斯达克 100） | 10%  | 科技敞口 |
| A股（沪深 300）      | 15%  | 大盘蓝筹 |
| A股（中证 500）      | 5%   | 中盘弹性 |
| 港股（恒生科技）     | 5%   | 少量配置 |
| 债券基金             | 25%  | 压舱石   |

### 均衡方案（成长与稳健兼顾）

| 资产                 | 占比 | 说明         |
| -------------------- | ---- | ------------ |
| 美股（标普 500）     | 30%  | 核心底仓     |
| 美股（纳斯达克 100） | 20%  | 科技敞口更大 |
| A股（沪深 300）      | 15%  | 大盘蓝筹     |
| A股（中证 500）      | 10%  | 中盘弹性     |
| 港股（恒生科技）     | 10%  | 估值洼地     |
| 债券基金             | 15%  | 少量压舱     |

### 激进方案（追求更高收益）

| 资产                 | 占比 | 说明         |
| -------------------- | ---- | ------------ |
| 美股（标普 500）     | 25%  | 核心底仓     |
| 美股（纳斯达克 100） | 25%  | 科技重仓     |
| A股（沪深 300）      | 15%  | 大盘蓝筹     |
| A股（中证 500）      | 15%  | 中盘弹性     |
| 港股（恒生科技）     | 15%  | 估值洼地加码 |
| 债券基金             | 5%   | 极少量       |

## 主线案例：选择均衡方案

回到我们的主线案例——年投入 20 万，月定投 16600 元，选择均衡方案：

| 资产         | 占比     | 月定投金额   |
| ------------ | -------- | ------------ |
| 标普 500     | 30%      | 4980 元      |
| 纳斯达克 100 | 20%      | 3320 元      |
| 沪深 300     | 15%      | 2490 元      |
| 中证 500     | 10%      | 1660 元      |
| 恒生科技     | 10%      | 1660 元      |
| 债券基金     | 15%      | 2490 元      |
| **合计**     | **100%** | **16600 元** |

这个配置的思路：

- 美股占 50%（标普 + 纳斯达克），作为组合的核心增长引擎
- A股占 25%（沪深 300 + 中证 500），利用A股的波动获取弹性收益
- 港股占 10%（恒生科技），在低估值时捕捉中国科技公司的成长空间
- 债券 15%，提供稳定性，在股市大跌时给你"抄底"的弹药

## 配置不是一成不变的

上面这个比例是一个**起点**，不是终点。好的资产配置会根据两个因素动态调整：

### 1. 估值变化

当某个市场估值明显偏低时，可以适当超配；明显偏高时，适当低配。

比如：

- 美股标普 500 的 Shiller PE 超过 35（历史极端高位）→ 美股减配到 35%
- A股沪深 300 的 PE 百分位低于 20%（历史低位）→ A股加配到 35%

这种"估值驱动的动态调整"我们在估值篇详细聊。

### 2. 人生阶段

随着年龄和财务状况的变化，配置也需要调整：

- **25-35 岁**：可以偏激进，股票比例 80%+，时间是你最大的优势
- **35-45 岁**：逐步转向均衡，股票比例 60-70%
- **45-55 岁**：偏保守，股票比例降到 50% 左右
- **55 岁以上**：以稳健为主，股票比例 30-40%，债券和现金占比提升

## 关于"为什么不全买美股"

这是很多人会问的问题：既然美股长期表现最好，为什么不 100% 买美股？

几个考虑：

1. **单一市场风险**：虽然美股过去 50 年一直向上，但未来不一定重复。日本市场就是一个反例
2. **汇率风险**：你用人民币买美股基金，美元兑人民币的波动会影响你的实际收益。如果美元大幅贬值，即使美股涨了，你的人民币收益也可能打折扣
3. **机会成本**：A股和港股虽然波动大，但在低估值时买入的潜在回报远高于当时的美股。2024 年底的A股和港股就是典型的例子
4. **心理因素**：如果 100% 美股，在美股大跌时你可能扛不住。有A股和港股的仓位，至少有一个市场可能在涨，心理上好受很多

## Trade-off

配置越分散，风险越小，但极端行情下的收益也越平庸。如果你在 2023 年初 100% 买入纳斯达克 100，那一年的收益是 55%。但如果按均衡方案配置，收益大概在 25-30%。

分散的本质是**放弃最高收益的可能性，换取最坏情况下的保护**。对于大多数人来说，这个交换是值得的。

**一个容易犯的错**：过度配置。有人买了 10 只以上的基金，覆盖 5-6 个市场，每只占比都很小。这看起来很"分散"，实际上管理成本高、调整困难，收益还被稀释了。**5-6 只基金覆盖 3 个市场，足够分散了。**
