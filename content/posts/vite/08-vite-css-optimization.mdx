## 前言

Vite 对 CSS 提供了开箱即用的支持，并提供了强大的扩展能力。本文将深入探讨 Vite 的 CSS 处理机制和优化策略。

## CSS 基础配置

### CSS 导入

```typescript
// 方式 1：导入 CSS
import './style.css';

// 方式 2：导入预处理器
import './style.scss';
import './style.less';

// 方式 3：CSS Modules
import styles from './App.module.css';

// 使用
function App() {
  return <div className={styles.container}>Hello</div>;
}
```

### CSS 配置

```typescript
// vite.config.ts
export default defineConfig({
  css: {
    // CSS Modules 配置
    modules: {
      localsConvention: 'camelCaseOnly',
      scopeBehaviour: 'local',
      generateScopedName: '[name]__[local]__[hash:base64:5]'
    },

    // PostCSS 配置
    postcss: {
      plugins: [
        require('autoprefixer'),
        require('postcss-nested')
      ]
    },

    // 预处理器配置
    preprocessorOptions: {
      scss: {
        additionalData: `@import "@/styles/variables.scss";`,
        api: 'modern-compiler'  // 使用现代编译器
      },
      less: {
        modifyVars: {
          '@primary-color': '#1890ff'
        },
        javascriptEnabled: true
      },
      styl: {
        define: {
          $primary-color: '#1890ff'
        }
      }
    },

    // CSS 代码分割
    cssCodeSplit: true,

    // 开发环境 Source Map
    devSourcemap: false
  }
});
```

## CSS Modules

### 基础使用

```css
/* App.module.css */
.container {
  padding: 20px;
  background: #f0f0f0;
}

.title {
  color: #333;
  font-size: 24px;
}
```

```typescript
// App.tsx
import styles from './App.module.css';

function App() {
  return (
    <div className={styles.container}>
      <h1 className={styles.title}>Hello</h1>
    </div>
  );
}
```

### 高级配置

```typescript
export default defineConfig({
  css: {
    modules: {
      // 命名约定
      localsConvention: "camelCaseOnly", // 'camelCase' | 'camelCaseOnly' | 'dashes' | 'dashesOnly'

      // 作用域行为
      scopeBehaviour: "local", // 'local' | 'global'

      // 生成类名
      generateScopedName: "[name]__[local]__[hash:base64:5]",

      // 全局模式
      globalModulePaths: [/node_modules/],

      // Hash 生成函数
      getHash: (content, filename) => {
        return customHash(content + filename);
      },
    },
  },
});
```

### 全局样式

```css
/* global.css */
:global(.global-class) {
  color: red;
}

/* 或使用 :global 包裹 */
:global {
  .container {
    padding: 20px;
  }
}
```

## CSS 预处理器

### Sass/SCSS

```typescript
// vite.config.ts
export default defineConfig({
  css: {
    preprocessorOptions: {
      scss: {
        // 自动导入变量
        additionalData: `@import "@/styles/variables.scss";`,

        // 编译器选项
        api: "modern-compiler",

        // 包含路径
        includePaths: ["./src/styles"],
      },
    },
  },
});
```

```scss
// _variables.scss
$primary-color: #1890ff;
$border-radius: 4px;

// _mixins.scss
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

// styles.scss
@import "variables";
@import "mixins";

.container {
  @include flex-center;
  background: $primary-color;
}
```

### Less

```typescript
export default defineConfig({
  css: {
    preprocessorOptions: {
      less: {
        // 修改变量
        modifyVars: {
          "@primary-color": "#1890ff",
          "@border-radius": "4px",
        },

        // 启用 JavaScript
        javascriptEnabled: true,

        // 其他选项
        math: "always",
        relativeUrls: true,
      },
    },
  },
});
```

### Stylus

```typescript
export default defineConfig({
  css: {
    preprocessorOptions: {
      styl: {
        // 定义变量
        define: {
          $primaryColor: "#1890ff",
          $borderRadius: "4px",
        },

        // 导入路径
        imports: ["./src/styles/variables.styl"],
      },
    },
  },
});
```

## PostCSS

### 配置 PostCSS

```javascript
// postcss.config.js
export default {
  plugins: [
    // 自动添加浏览器前缀
    require("autoprefixer")({
      overrideBrowserslist: ["> 1%", "last 2 versions"],
    }),

    // 嵌套语法
    require("postcss-nested")(),

    // 导入
    require("postcss-import")(),

    // CSS 变量
    require("postcss-custom-properties")({
      preserve: false,
    }),

    // 重置 CSS
    require("postcss-normalize")(),
  ],
};
```

### PostCSS 插件

```typescript
// vite.config.ts
export default defineConfig({
  css: {
    postcss: {
      plugins: [
        require("autoprefixer"),
        require("cssnano")({
          preset: "default",
        }),
      ],
    },
  },
});
```

## CSS 优化

### 压缩 CSS

```typescript
export default defineConfig({
  build: {
    cssCodeSplit: true,
    cssMinify: true,
    cssTarget: "chrome80",
  },
});
```

### PurgeCSS

```typescript
import { viteCommonjs } from "@originjs/vite-plugin-commonjs";

export default defineConfig({
  plugins: [
    viteCommonjs(),
    {
      name: "purgecss",
      transform(code, id) {
        if (!id.endsWith(".css")) return null;

        // 使用 PurgeCSS 移除未使用的样式
        const purged = purgeCSS(code, {
          safelist: {
            standard: [/^active/, /^disabled/],
          },
        });

        return {
          code: purged,
        };
      },
    },
  ],
});
```

### CSS 代码分割

```typescript
export default defineConfig({
  build: {
    cssCodeSplit: true,

    rollupOptions: {
      output: {
        // CSS chunk 文件名
        assetFileNames: (assetInfo) => {
          if (assetInfo.name?.endsWith(".css")) {
            return "assets/css/[name]-[hash][extname]";
          }
          return "assets/[ext]/[name]-[hash][extname]";
        },
      },
    },
  },
});
```

## 常见问题

### Q1: CSS Modules 不生效？

A: 检查配置：

```typescript
css: {
  modules: {
    generateScopedName: "[name]__[local]__[hash:base64:5]";
  }
}
```

### Q2: 全局样式被覆盖？

A: 使用 :global 或调整导入顺序：

```css
/* global.css */
:global(.container) {
  padding: 20px;
}
```

### Q3: Tailwind 生产环境样式丢失？

A: 检查 content 配置：

```javascript
content: ["./src/**/*.{js,jsx,ts,tsx}"];
```

## 总结

Vite CSS 处理要点：

1. **CSS Modules**：样式隔离
2. **预处理器**：Sass、Less、Stylus
3. **PostCSS**：强大的后处理
4. **Tailwind**：实用优先的框架
5. **CSS-in-JS**：Styled Components、Emotion
6. **优化策略**：压缩、代码分割
