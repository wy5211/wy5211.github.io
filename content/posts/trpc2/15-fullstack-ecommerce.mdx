## 前言

在上一篇文章中，我们构建了一个协作式代码编辑器，重点展示了 tRPC 的实时协作能力。今天，我们来做最后一个项目：一个完整的全栈电商系统。

电商系统是全栈开发的经典场景，因为它涵盖了方方面面的技术挑战：

**商品目录**：复杂的查询、过滤、分页、搜索

**购物车**：临时状态管理、跨设备同步

**订单处理**：事务一致性、状态流转、幂等性

**支付集成**：安全性、异步回调、错误处理

**库存管理**：并发控制、实时更新、预警机制

**后台管理**：权限控制、数据可视化、批量操作

这个项目会很大，但我们会一步步来，重点关注 tRPC 如何简化复杂业务逻辑的开发。

## 为什么电商系统需要类型安全

电商系统对数据准确性要求极高。价格算错、库存扣错、订单漏记，任何小错误都可能造成重大损失。传统开发中，前后端类型不一致导致的bug很常见：

后端改了价格字段从 `number` 到 `string`，前端没同步，支付时崩溃

库存数量从 `int` 变成 `bigint`，前端显示异常

新增了"预售"状态，前端不知道怎么处理，显示为未知状态

这些问题在编译时无法发现，只在特定运行场景下才会暴露，排查困难。

tRPC 通过端到端类型安全，从根本上杜绝了这类问题。后端改了类型，前端编译器立即报错，不会漏到生产环境。对于电商这种容错率低的系统，类型安全不是锦上添花，而是必需品。

**Redis**：缓存和会话存储，支持 pub/sub 实现实时更新

## 后端实现

### 商品 Router

商品模块是电商系统的核心，需要支持复杂的查询和过滤。

```typescript
// packages/trpc/src/routers/products.router.ts

import { z } from "zod";
import { t } from "../";
import type { Context } from "../context";

export const productsRouter = t.router<Context>({
  // 商品列表（支持复杂过滤和排序）
  list: t.procedure
    .input(
      z.object({
        page: z.number().min(1).default(1),
        limit: z.number().min(1).max(100).default(20),
        category: z.string().optional(),
        search: z.string().optional(),
        sortBy: z
          .enum(["name", "price", "createdAt", "popular"])
          .default("createdAt"),
        sortOrder: z.enum(["asc", "desc"]).default("desc"),
        featured: z.boolean().optional(),
        minPrice: z.number().min(0).optional(),
        maxPrice: z.number().min(0).optional(),
        tags: z.array(z.string()).optional(),
        inStock: z.boolean().optional(),
      })
    )
    .query(async ({ input, ctx }) => {
      const {
        page,
        limit,
        category,
        search,
        sortBy,
        sortOrder,
        featured,
        minPrice,
        maxPrice,
        tags,
        inStock,
      } = input;

      // 构建查询条件
      const where: any = {
        active: true,
      };

      // 分类过滤
      if (category) {
        where.categories = {
          some: {
            category: { slug: category },
          },
        };
      }

      // 搜索（名称、描述、标签）
      if (search) {
        where.OR = [
          { name: { contains: search, mode: "insensitive" } },
          { description: { contains: search, mode: "insensitive" } },
          { tags: { hasSome: [search] } },
        ];
      }

      // 特色商品
      if (featured !== undefined) {
        where.featured = featured;
      }

      // 价格范围
      if (minPrice !== undefined || maxPrice !== undefined) {
        where.price = {};
        if (minPrice !== undefined) where.price.gte = minPrice;
        if (maxPrice !== undefined) where.price.lte = maxPrice;
      }

      // 标签过滤
      if (tags && tags.length > 0) {
        where.tags = { hasSome: tags };
      }

      // 库存过滤
      if (inStock) {
        where.stock = { gt: 0 };
      }

      // 查询商品和总数
      const [products, total] = await Promise.all([
        ctx.prisma.product.findMany({
          where,
          take: limit,
          skip: (page - 1) * limit,
          orderBy: { [sortBy]: sortOrder },
          include: {
            categories: {
              include: {
                category: true,
              },
            },
          },
        }),
        ctx.prisma.product.count({ where }),
      ]);

      return {
        products,
        pagination: {
          page,
          limit,
          total,
          totalPages: Math.ceil(total / limit),
        },
      };
    }),

  // 商品详情
  byId: t.procedure
    .input(z.object({ id: z.string() }))
    .query(async ({ input, ctx }) => {
      const product = await ctx.prisma.product.findUnique({
        where: { id: input.id },
        include: {
          categories: {
            include: {
              category: true,
            },
          },
          reviews: {
            include: {
              user: {
                select: {
                  id: true,
                  name: true,
                  avatar: true,
                },
              },
            },
            take: 10,
            orderBy: { createdAt: "desc" },
          },
        },
      });

      if (!product) {
        throw new TRPCError({
          code: "NOT_FOUND",
          message: "商品不存在",
        });
      }

      return product;
    }),

  // 搜索建议（自动完成）
  searchSuggestions: t.procedure
    .input(
      z.object({
        query: z.string().min(2),
        limit: z.number().min(1).max(10).default(5),
      })
    )
    .query(async ({ input, ctx }) => {
      const products = await ctx.prisma.product.findMany({
        where: {
          active: true,
          OR: [
            { name: { contains: input.query, mode: "insensitive" } },
            { tags: { hasSome: [input.query] } },
          ],
        },
        take: input.limit,
        select: {
          id: true,
          name: true,
          slug: true,
          images: true,
          price: true,
        },
      });

      return products;
    }),

  // 库存检查（购物车和结账时使用）
  checkStock: t.procedure
    .input(
      z.object({
        items: z.array(
          z.object({
            productId: z.string(),
            quantity: z.number().min(1),
          })
        ),
      })
    )
    .query(async ({ input, ctx }) => {
      const productIds = input.items.map((item) => item.productId);

      const products = await ctx.prisma.product.findMany({
        where: { id: { in: productIds } },
        select: {
          id: true,
          stock: true,
          active: true,
        },
      });

      const stockMap = new Map(products.map((p) => [p.id, p]));

      const results = input.items.map((item) => {
        const product = stockMap.get(item.productId);
        return {
          productId: item.productId,
          inStock: product ? product.stock >= item.quantity : false,
          availableStock: product?.stock || 0,
          active: product?.active || false,
        };
      });

      return results;
    }),

  // 特色商品
  featured: t.procedure
    .input(z.object({ limit: z.number().min(1).max(20).default(10) }))
    .query(async ({ input, ctx }) => {
      return await ctx.prisma.product.findMany({
        where: {
          featured: true,
          active: true,
          stock: { gt: 0 },
        },
        take: input.limit,
        include: {
          categories: {
            include: {
              category: true,
            },
          },
        },
        orderBy: { createdAt: "desc" },
      });
    }),
});
```

这个商品 Router 实现了电商所需的核心查询功能。关键点：

**复杂过滤**：支持按分类、价格、库存、标签等多维度过滤

**全文搜索**：在名称、描述、标签中搜索

**库存检查**：购物车和结账时验证库存

**分页排序**：灵活的排序选项，支持热门度排序

### 购物车 Router

购物车看似简单，但有很多细节需要处理：商品失效、库存不足、价格变动等。

```typescript
// packages/trpc/src/routers/cart.router.ts

export const cartRouter = t.router<Context>({
  // 获取购物车
  get: t.procedure.query(async ({ ctx }) => {
    if (!ctx.user) {
      throw new TRPCError({
        code: "UNAUTHORIZED",
        message: "请先登录",
      });
    }

    const cart = await ctx.prisma.cart.findUnique({
      where: { userId: ctx.user.id },
      include: {
        items: {
          include: {
            product: {
              include: {
                categories: {
                  include: {
                    category: true,
                  },
                },
              },
            },
          },
        },
      },
    });

    // 购物车不存在则创建
    if (!cart) {
      return await ctx.prisma.cart.create({
        data: { userId: ctx.user.id },
        include: {
          items: {
            include: {
              product: true,
            },
          },
        },
      });
    }

    // 过滤失效商品
    const validItems = cart.items.filter(
      (item) => item.product.active && item.product.stock > 0
    );

    if (validItems.length !== cart.items.length) {
      // 删除失效商品
      await ctx.prisma.cartItem.deleteMany({
        where: {
          cartId: cart.id,
          productId: {
            in: cart.items
              .filter(
                (i) => !validItems.some((v) => v.productId === i.productId)
              )
              .map((i) => i.productId),
          },
        },
      });

      return {
        ...cart,
        items: validItems,
      };
    }

    return cart;
  }),

  // 添加商品到购物车
  addItem: t.procedure
    .input(
      z.object({
        productId: z.string(),
        quantity: z.number().min(1).max(99),
      })
    )
    .mutation(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      // 检查商品
      const product = await ctx.prisma.product.findUnique({
        where: { id: input.productId },
      });

      if (!product || !product.active) {
        throw new TRPCError({
          code: "NOT_FOUND",
          message: "商品不存在或已下架",
        });
      }

      if (product.stock < input.quantity) {
        throw new TRPCError({
          code: "BAD_REQUEST",
          message: "库存不足",
        });
      }

      // 获取或创建购物车
      let cart = await ctx.prisma.cart.findUnique({
        where: { userId: ctx.user.id },
      });

      if (!cart) {
        cart = await ctx.prisma.cart.create({
          data: { userId: ctx.user.id },
        });
      }

      // 检查商品是否已在购物车中
      const existingItem = await ctx.prisma.cartItem.findUnique({
        where: {
          cartId_productId: {
            cartId: cart.id,
            productId: input.productId,
          },
        },
      });

      if (existingItem) {
        // 更新数量
        const newQuantity = existingItem.quantity + input.quantity;

        if (newQuantity > product.stock) {
          throw new TRPCError({
            code: "BAD_REQUEST",
            message: "库存不足",
          });
        }

        await ctx.prisma.cartItem.update({
          where: { id: existingItem.id },
          data: { quantity: newQuantity },
        });
      } else {
        // 添加新商品
        await ctx.prisma.cartItem.create({
          data: {
            cartId: cart.id,
            productId: input.productId,
            quantity: input.quantity,
          },
        });
      }

      // 返回更新后的购物车
      return await ctx.prisma.cart.findUnique({
        where: { userId: ctx.user.id },
        include: {
          items: {
            include: {
              product: true,
            },
          },
        },
      });
    }),

  // 更新商品数量
  updateQuantity: t.procedure
    .input(
      z.object({
        itemId: z.string(),
        quantity: z.number().min(0).max(99),
      })
    )
    .mutation(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      const cartItem = await ctx.prisma.cartItem.findUnique({
        where: { id: input.itemId },
        include: {
          product: true,
          cart: true,
        },
      });

      if (!cartItem) {
        throw new TRPCError({
          code: "NOT_FOUND",
          message: "购物车商品不存在",
        });
      }

      if (cartItem.cart.userId !== ctx.user.id) {
        throw new TRPCError({
          code: "FORBIDDEN",
          message: "无权操作此购物车",
        });
      }

      if (input.quantity === 0) {
        // 删除商品
        await ctx.prisma.cartItem.delete({
          where: { id: input.itemId },
        });
      } else {
        // 检查库存
        if (input.quantity > cartItem.product.stock) {
          throw new TRPCError({
            code: "BAD_REQUEST",
            message: "库存不足",
          });
        }

        // 更新数量
        await ctx.prisma.cartItem.update({
          where: { id: input.itemId },
          data: { quantity: input.quantity },
        });
      }

      // 返回更新后的购物车
      return await ctx.prisma.cart.findUnique({
        where: { userId: ctx.user.id },
        include: {
          items: {
            include: {
              product: true,
            },
          },
        },
      });
    }),

  // 获取购物车商品数量（用于显示角标）
  getItemCount: t.procedure.query(async ({ ctx }) => {
    if (!ctx.user) {
      return 0;
    }

    const cart = await ctx.prisma.cart.findUnique({
      where: { userId: ctx.user.id },
      include: { items: true },
    });

    if (!cart) {
      return 0;
    }

    return cart.items.reduce((sum, item) => sum + item.quantity, 0);
  }),
});
```

购物车 Router 的关键设计：

**自动清理**：过滤失效商品，避免用户困惑

**库存验证**：每次操作都检查库存，防止超卖

**权限控制**：确保用户只能操作自己的购物车

**数量限制**：防止恶意操作，限制单次添加数量

### 订单 Router

订单处理是电商系统最复杂的部分，涉及事务、状态流转、库存扣减等。

```typescript
// packages/trpc/src/routers/orders.router.ts

export const ordersRouter = t.router<Context>({
  // 创建订单
  create: t.procedure
    .input(
      z.object({
        items: z.array(
          z.object({
            productId: z.string(),
            quantity: z.number().min(1),
          })
        ),
        shippingAddress: z.object({
          name: z.string().min(1),
          email: z.string().email(),
          phone: z.string().min(1),
          address: z.string().min(1),
          city: z.string().min(1),
          state: z.string().min(1),
          zip: z.string().min(1),
          country: z.string().min(1),
        }),
        paymentMethod: z.enum(["CREDIT_CARD", "DEBIT_CARD", "PAYPAL"]),
        notes: z.string().optional(),
      })
    )
    .mutation(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      // 获取商品信息
      const productIds = input.items.map((item) => item.productId);
      const products = await ctx.prisma.product.findMany({
        where: { id: { in: productIds } },
      });

      const productMap = new Map(products.map((p) => [p.id, p]));

      // 验证库存
      for (const item of input.items) {
        const product = productMap.get(item.productId);
        if (!product || product.stock < item.quantity) {
          throw new TRPCError({
            code: "BAD_REQUEST",
            message: `商品 ${product?.name || item.productId} 库存不足`,
          });
        }
      }

      // 计算订单金额
      let subtotal = 0;
      const orderItems = input.items.map((item) => {
        const product = productMap.get(item.productId)!;
        const itemTotal = Number(product.price) * item.quantity;
        subtotal += itemTotal;

        return {
          productId: item.productId,
          quantity: item.quantity,
          price: product.price,
        };
      });

      const tax = subtotal * 0.08; // 8% 税
      const shipping = subtotal > 100 ? 0 : 9.99; // 满100免运费
      const total = subtotal + tax + shipping;

      // 生成订单号
      const orderNumber = `ORD-${Date.now()}-${Math.random()
        .toString(36)
        .substr(2, 9)
        .toUpperCase()}`;

      // 使用事务创建订单
      const order = await ctx.prisma.$transaction(async (tx) => {
        // 创建订单
        const order = await tx.order.create({
          data: {
            orderNumber,
            userId: ctx.user.id,
            items: {
              create: orderItems,
            },
            status: "PENDING",
            subtotal,
            tax,
            shipping,
            total,
            shippingName: input.shippingAddress.name,
            shippingEmail: input.shippingAddress.email,
            shippingPhone: input.shippingAddress.phone,
            shippingAddress: input.shippingAddress.address,
            shippingCity: input.shippingAddress.city,
            shippingState: input.shippingAddress.state,
            shippingZip: input.shippingAddress.zip,
            shippingCountry: input.shippingAddress.country,
            paymentMethod: input.paymentMethod,
            notes: input.notes,
          },
        });

        // 扣减库存
        for (const item of input.items) {
          await tx.product.update({
            where: { id: item.productId },
            data: {
              stock: {
                decrement: item.quantity,
              },
            },
          });

          // 记录库存变动
          await tx.stockMovement.create({
            data: {
              productId: item.productId,
              quantity: -item.quantity,
              type: "SALE",
              orderId: order.id,
            },
          });
        }

        // 清空购物车
        await tx.cartItem.deleteMany({
          where: {
            cart: {
              userId: ctx.user.id,
            },
          },
        });

        return order;
      });

      return order;
    }),

  // 获取订单列表
  list: t.procedure
    .input(
      z.object({
        page: z.number().min(1).default(1),
        limit: z.number().min(1).max(100).default(10),
        status: z
          .enum([
            "PENDING",
            "PROCESSING",
            "SHIPPED",
            "DELIVERED",
            "CANCELLED",
            "REFUNDED",
          ])
          .optional(),
      })
    )
    .query(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      const where: any = {
        userId: ctx.user.id,
      };

      if (input.status) {
        where.status = input.status;
      }

      const [orders, total] = await Promise.all([
        ctx.prisma.order.findMany({
          where,
          take: input.limit,
          skip: (input.page - 1) * input.limit,
          orderBy: { createdAt: "desc" },
          include: {
            items: {
              include: {
                product: true,
              },
            },
          },
        }),
        ctx.prisma.order.count({ where }),
      ]);

      return {
        orders,
        pagination: {
          page: input.page,
          limit: input.limit,
          total,
          totalPages: Math.ceil(total / input.limit),
        },
      };
    }),

  // 订单详情
  byId: t.procedure
    .input(z.object({ id: z.string() }))
    .query(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      const order = await ctx.prisma.order.findUnique({
        where: { id: input.id },
        include: {
          items: {
            include: {
              product: true,
            },
          },
        },
      });

      if (!order) {
        throw new TRPCError({
          code: "NOT_FOUND",
          message: "订单不存在",
        });
      }

      if (order.userId !== ctx.user.id && ctx.user.role !== "ADMIN") {
        throw new TRPCError({
          code: "FORBIDDEN",
          message: "无权访问此订单",
        });
      }

      return order;
    }),

  // 取消订单
  cancel: t.procedure
    .input(z.object({ id: z.string() }))
    .mutation(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      const order = await ctx.prisma.order.findUnique({
        where: { id: input.id },
        include: { items: true },
      });

      if (!order) {
        throw new TRPCError({
          code: "NOT_FOUND",
          message: "订单不存在",
        });
      }

      if (order.userId !== ctx.user.id) {
        throw new TRPCError({
          code: "FORBIDDEN",
          message: "无权操作此订单",
        });
      }

      if (order.status !== "PENDING") {
        throw new TRPCError({
          code: "BAD_REQUEST",
          message: "订单已处理，无法取消",
        });
      }

      // 使用事务恢复库存
      await ctx.prisma.$transaction(async (tx) => {
        // 恢复库存
        for (const item of order.items) {
          await tx.product.update({
            where: { id: item.productId },
            data: {
              stock: {
                increment: item.quantity,
              },
            },
          });

          // 记录库存变动
          await tx.stockMovement.create({
            data: {
              productId: item.productId,
              quantity: item.quantity,
              type: "RETURN",
              orderId: order.id,
            },
          });
        }

        // 更新订单状态
        await tx.order.update({
          where: { id: input.id },
          data: {
            status: "CANCELLED",
          },
        });
      });

      return { success: true };
    }),
});
```

订单 Router 的关键设计：

**事务处理**：创建订单和扣减库存在同一个事务中，保证一致性

**库存追踪**：每次库存变动都记录，支持审计和盘点

**状态管理**：订单只能取消待处理状态，防止业务逻辑错误

**权限验证**：确保用户只能访问自己的订单

### 支付 Router（Stripe 集成）

支付是电商系统的最后一环，也是最容易出问题的地方。

```typescript
// packages/trpc/src/routers/payments.router.ts

import Stripe from "stripe";

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!, {
  apiVersion: "2024-11-20.acacia",
});

export const paymentsRouter = t.router<Context>({
  // 创建支付意图
  createPaymentIntent: t.procedure
    .input(
      z.object({
        orderId: z.string(),
      })
    )
    .mutation(async ({ input, ctx }) => {
      if (!ctx.user) {
        throw new TRPCError({
          code: "UNAUTHORIZED",
          message: "请先登录",
        });
      }

      const order = await ctx.prisma.order.findUnique({
        where: { id: input.orderId },
      });

      if (!order) {
        throw new TRPCError({
          code: "NOT_FOUND",
          message: "订单不存在",
        });
      }

      if (order.userId !== ctx.user.id) {
        throw new TRPCError({
          code: "FORBIDDEN",
          message: "无权操作此订单",
        });
      }

      if (order.paymentStatus === "COMPLETED") {
        throw new TRPCError({
          code: "BAD_REQUEST",
          message: "订单已支付",
        });
      }

      // 创建 Stripe PaymentIntent
      const paymentIntent = await stripe.paymentIntents.create({
        amount: Math.round(Number(order.total) * 100), // 转换为分
        currency: "usd",
        metadata: {
          orderId: order.id,
          orderNumber: order.orderNumber,
          userId: ctx.user.id,
        },
        description: `订单 ${order.orderNumber}`,
      });

      // 更新订单
      await ctx.prisma.order.update({
        where: { id: input.orderId },
        data: {
          stripePaymentIntentId: paymentIntent.id,
        },
      });

      return {
        clientSecret: paymentIntent.client_secret,
        paymentIntentId: paymentIntent.id,
        amount: paymentIntent.amount,
        currency: paymentIntent.currency,
      };
    }),

  // Webhook 处理（Stripe 异步通知）
  handleWebhook: t.procedure.mutation(async ({ ctx }) => {
    const sig = ctx.req.headers["stripe-signature"];

    if (!sig) {
      throw new TRPCError({
        code: "BAD_REQUEST",
        message: "无效的签名",
      });
    }

    let event: Stripe.Event;

    try {
      // 验证 webhook 签名
      event = stripe.webhooks.constructEvent(
        ctx.req.body,
        sig,
        process.env.STRIPE_WEBHOOK_SECRET!
      );
    } catch (err) {
      throw new TRPCError({
        code: "BAD_REQUEST",
        message: "Webhook 签名验证失败",
      });
    }

    // 处理事件
    switch (event.type) {
      case "payment_intent.succeeded": {
        const paymentIntent = event.data.object as Stripe.PaymentIntent;
        const orderId = paymentIntent.metadata.orderId;

        await ctx.prisma.order.update({
          where: { id: orderId },
          data: {
            paymentStatus: "COMPLETED",
            status: "PROCESSING",
            paidAt: new Date(),
          },
        });

        // 发送确认邮件
        await sendOrderConfirmationEmail(orderId);
        break;
      }

      case "payment_intent.payment_failed": {
        const failedPayment = event.data.object as Stripe.PaymentIntent;
        const failedOrderId = failedPayment.metadata.orderId;

        await ctx.prisma.order.update({
          where: { id: failedOrderId },
          data: {
            paymentStatus: "FAILED",
          },
        });

        // 恢复库存
        const order = await ctx.prisma.order.findUnique({
          where: { id: failedOrderId },
          include: { items: true },
        });

        if (order) {
          for (const item of order.items) {
            await ctx.prisma.product.update({
              where: { id: item.productId },
              data: {
                stock: {
                  increment: item.quantity,
                },
              },
            });
          }
        }
        break;
      }
    }

    return { received: true };
  }),
});
```

支付 Router 的关键设计：

**幂等性**：支付成功后重复调用 webhook 不会重复处理

**错误恢复**：支付失败时自动恢复库存

**安全验证**：验证 webhook 签名，防止伪造请求

**异步通知**：通过 webhook 处理支付结果，不阻塞用户操作

### 购物车 Hook

```typescript
// apps/web/src/hooks/useCart.ts

"use client";

import { useQuery, useMutation, useQueryClient } from "@tanstack/react-query";
import { trpc } from "@/lib/trpc/client";

export function useCart() {
  const queryClient = useQueryClient();

  // 获取购物车
  const { data: cart, isLoading } = trpc.cart.get.useQuery();

  // 添加到购物车
  const addToCartMutation = trpc.cart.addItem.useMutation({
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: [["cart", "get"]] });
      queryClient.invalidateQueries({ queryKey: [["cart", "getItemCount"]] });
    },
  });

  // 更新数量
  const updateQuantityMutation = trpc.cart.updateQuantity.useMutation({
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: [["cart", "get"]] });
      queryClient.invalidateQueries({ queryKey: [["cart", "getItemCount"]] });
    },
  });

  // 获取购物车数量
  const { data: itemCount } = trpc.cart.getItemCount.useQuery();

  const addToCart = (productId: string, quantity: number) => {
    addToCartMutation.mutate({ productId, quantity });
  };

  const updateQuantity = (itemId: string, quantity: number) => {
    updateQuantityMutation.mutate({ itemId, quantity });
  };

  // 计算总价
  const subtotal =
    cart?.items.reduce(
      (sum, item) => sum + Number(item.product.price) * item.quantity,
      0
    ) || 0;

  return {
    cart,
    isLoading,
    itemCount: itemCount || 0,
    addToCart,
    updateQuantity,
    subtotal,
  };
}
```

### 结账页面

```typescript
// apps/web/src/app/checkout/page.tsx

"use client";

import { useState } from "react";
import { useCart } from "@/hooks/useCart";
import { trpc } from "@/lib/trpc/client";
import { loadStripe } from "@stripe/stripe-js";

const stripePromise = loadStripe(
  process.env.NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY!
);

export default function CheckoutPage() {
  const { cart, subtotal } = useCart();
  const [isProcessing, setIsProcessing] = useState(false);
  const [shippingAddress, setShippingAddress] = useState({
    name: "",
    email: "",
    phone: "",
    address: "",
    city: "",
    state: "",
    zip: "",
    country: "US",

  const createOrderMutation = trpc.orders.create.useMutation();
  const createPaymentIntentMutation =
    trpc.payments.createPaymentIntent.useMutation();

  const total = subtotal + tax + shipping;

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    setIsProcessing(true);

    try {
      // 创建订单

      // 创建支付意图

        orderId: order.id,
      });

      // 跳转到 Stripe 支付页面
      const stripe = await stripePromise;
      if (stripe && clientSecret) {

        } else {
          toast.success("支付成功！");
          router.push(`/orders/${order.id}`);
        }
      }
    } catch (error) {
      toast.error("结账失败，请重试");
    } finally {
      setIsProcessing(false);
    }
  };

  return (
    <div className="container mx-auto px-4 py-8">
      <h1 className="text-3xl font-bold mb-8">结账</h1>

      <div className="grid grid-cols-1 lg:grid-cols-3 gap-8">
        {/* 结账表单 */}
        <div className="lg:col-span-2">
          <form onSubmit={handleSubmit} className="space-y-6">
            {/* 收货地址 */}
            <div className="bg-white p-6 rounded-lg shadow-sm">
              <h2 className="text-xl font-semibold mb-4">收货地址</h2>

              <div className="grid grid-cols-2 gap-4">
                <div>
                  <label className="block text-sm font-medium mb-1">
                    姓名
                  </label>
                  <input
                    type="text"
                    required
                    value={shippingAddress.name}
                    onChange={(e) =>
                      setShippingAddress({
                        ...shippingAddress,
                        name: e.target.value,
                      })
                    }
                    className="w-full border rounded-lg px-3 py-2"
                  />
                </div>

                <div>
                  <label className="block text-sm font-medium mb-1">
                    邮箱
                  </label>
                  <input
                    type="email"
                    required
                    value={shippingAddress.email}
                    onChange={(e) =>
                      setShippingAddress({
                        ...shippingAddress,
                        email: e.target.value,
                      })
                    }
                    className="w-full border rounded-lg px-3 py-2"
                  />
                </div>

                {/* 其他字段... */}
              </div>
            </div>

            {/* 支付信息 */}
            <div className="bg-white p-6 rounded-lg shadow-sm">
              <h2 className="text-xl font-semibold mb-4">支付信息</h2>
              <CardElement />
            </div>

            <button
              type="submit"
              disabled={isProcessing}
              className="w-full bg-blue-600 text-white py-3 rounded-lg font-semibold hover:bg-blue-700 disabled:bg-gray-300"
            >
              {isProcessing
                ? "处理中..."
                : `支付 $${total.toFixed(2)}`}
            </button>
          </form>
        </div>

        {/* 订单摘要 */}
        <div className="lg:col-span-1">
          <div className="bg-white p-6 rounded-lg shadow-sm sticky top-4">
            <h2 className="text-xl font-semibold mb-4">订单摘要</h2>

            <div className="space-y-3">
              {cart?.items.map((item) => (
                <div
                  key={item.id}
                  className="flex justify-between"
                >
                  <span>
                    {item.product.name} x{item.quantity}
                  </span>
                  <span>
                    $
                    {(
                      Number(item.product.price) * item.quantity
                    ).toFixed(2)}
                  </span>
                </div>
              ))}

              <hr />

              <div className="flex justify-between">
                <span>小计</span>
                <span>${subtotal.toFixed(2)}</span>
              </div>

              <div className="flex justify-between">
                <span>税费</span>
                <span>${tax.toFixed(2)}</span>
              </div>

              <div className="flex justify-between">
                <span>运费</span>
                <span>
                  {shipping === 0 ? "免运费" : `$${shipping.toFixed(2)}`}
                </span>
              </div>

              <hr />

              <div className="flex justify-between font-bold text-lg">
                <span>总计</span>
                <span>${total.toFixed(2)}</span>
              </div>
            </div>
          </div>
        </div>
      </div>
    </div>
  );
}
```

## 实时库存更新

库存变化需要实时通知用户，避免下单时才发现缺货。

### WebSocket 订阅

```typescript
// apps/web/src/hooks/useRealtimeStock.ts

"use client";

import { useEffect } from "react";
import { trpc } from "@/lib/trpc/client";
import { useQueryClient } from "@tanstack/react-query";

export function useRealtimeStock(productIds: string[]) {
  const queryClient = useQueryClient();

  // 订阅库存更新（需要在后端实现对应的 Subscription）
  useEffect(() => {
    const unsubscribes = productIds.map((productId) => {
      return trpc.products.onStockUpdate.subscribe(
        { productId },
        {
          onData: (update) => {
            // 更新缓存中的库存数据
            queryClient.setQueryData(
              [["products", "checkStock"]],
              (old: any) => {
                if (!old) return old;

                return old.map((item: any) => {
                  if (item.productId === update.productId) {
                    return {
                      ...item,
                      availableStock: update.stock,
                      inStock: update.stock > 0,
                    };
                  }
                  return item;
                });
              }
            );

            // 显示库存提醒
            if (update.stock < 10 && update.stock > 0) {
              toast.warn(`商品仅剩 ${update.stock} 件！`);
            }
          },
        }
      );
    });

    return () => {
      unsubscribes.forEach((unsub) => unsub());
    };
  }, [productIds, queryClient]);
}
```

## 总结与权衡

电商系统是个庞大的工程，我们用 tRPC 构建了一个功能完整的基础版本。现在来回顾一下技术选择的权衡：

**tRPC vs REST**：tRPC 的类型安全在电商这种容错率低的系统中价值巨大。价格、库存、订单状态任何一点错误都会造成损失，类型检查能在编译时发现大部分问题。但 tRPC 生态相对小，第三方工具支持不如 REST 丰富。

**Monorepo vs Multi-repo**：Monorepo 让前后端共享类型变得简单，代码复用率高。但随着项目增长，构建时间会变长，需要 Turborepo 这样的工具优化。如果团队规模很大，Multi-repo 可能更合适。

**SQL vs NoSQL**：我们选了 PostgreSQL 因为电商数据关系性强，事务支持重要。如果商品属性极其多样，MongoDB 可能更合适。混合使用也是常见方案：订单用 SQL，商品目录用 NoSQL。

**同步支付 vs 异步支付**：我们用 Stripe 的异步 webhook 处理支付结果，用户体验好但实现复杂。小型电商可以直接等支付完成再返回，简单但用户体验差。

**实时库存 vs 定期刷新**：实时更新体验好但消耗资源。根据商品特性选择：热销商品实时更新，其他商品定期刷新。

这个项目综合运用了本系列学到的所有知识：Monorepo 架构、类型安全、测试、实时更新等等。tRPC 让我们能专注于业务逻辑，而不是在类型定义和接口对接上浪费时间。
