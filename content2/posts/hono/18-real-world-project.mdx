- 热门文章推荐

**技术栈选择**：

- 数据库：D1（SQLite）

- 认证：JWT
- 验证：Zod
- 测试：Vitest
- 部署：Wrangler

这个技术栈非常适合博客 API：轻量、快速、可扩展、成本可控。

```typescript
import { sqliteTable, text, integer } from "drizzle-orm/sqlite-core";

export const users = sqliteTable("users", {
```

## 认证系统实现

认证是博客 API 的基础，我们使用 JWT 来实现：

```typescript
// src/routes/auth.ts
import { Hono } from "hono";
import { sign, verify } from "hono/jwt";
import { hash, compare } from "bcrypt";
import { zValidator } from "@hono/zod-validator";
import { z } from "zod";

const auth = new Hono<{ Bindings: Env }>();

const registerSchema = z.object({
  email: z.string().email(),
  username: z.string().min(3).max(20),
  password: z.string().min(8),
  name: z.string().optional(),
});

auth.post("/register", zValidator("json", registerSchema), async (c) => {
  const data = c.req.valid("json");

  // 检查邮箱是否已存在
  const existingUser = await c.env.DB.prepare(
    "SELECT id FROM users WHERE email = ?"
  )
    .bind(data.email)
    .first();

  if (existingUser) {
    return c.json({ success: false, message: "Email already exists" }, 400);
  }

  // 哈希密码
  const hashedPassword = await hash(data.password, 10);

  // 创建用户
  const userId = crypto.randomUUID();
  await c.env.DB.prepare(
    "INSERT INTO users (id, email, username, password, name, created_at) VALUES (?, ?, ?, ?, ?, ?)"
  )
    .bind(
      userId,
      data.email,
      data.username,
      hashedPassword,
      data.name,
      new Date().toISOString()
    )
    .run();

  // 生成 token
  const token = await sign({ sub: userId, role: "user" }, c.env.JWT_SECRET);

  return c.json(
    {
      success: true,
      data: {
        token,
        user: { id: userId, email: data.email, username: data.username },
      },
    },
    201
  );
});

const loginSchema = z.object({
  email: z.string().email(),
  password: z.string(),
});

auth.post("/login", zValidator("json", loginSchema), async (c) => {
  const { email, password } = c.req.valid("json");

  // 查找用户
  const user = (await c.env.DB.prepare("SELECT * FROM users WHERE email = ?")
    .bind(email)
    .first()) as any;

  if (!user || !(await compare(password, user.password))) {
    return c.json({ success: false, message: "Invalid credentials" }, 401);
  }

  // 生成 token
  const token = await sign({ sub: user.id, role: user.role }, c.env.JWT_SECRET);

  return c.json({
    success: true,
    data: {
      token,
      user: { id: user.id, email: user.email, username: user.username },
    },
  });
});

export default auth;
```

这个实现包含了注册和登录两个核心功能，密码使用 bcrypt 哈希，JWT token 用于后续请求的认证。

## 文章 CRUD 实现

文章是博客的核心，让我们实现完整的 CRUD 操作：

```typescript
// src/routes/posts.ts
import { Hono } from "hono";
import { zValidator } from "@hono/zod-validator";
import { z } from "zod";
import { authMiddleware } from "../middleware/auth";

const posts = new Hono<{ Bindings: Env }>();

const postSchema = z.object({
  title: z.string().min(1).max(200),
  content: z.string().min(1),
  excerpt: z.string().max(500).optional(),
  coverImage: z.string().url().optional(),
  published: z.boolean().default(false),
  tags: z.array(z.string()).optional(),
});

// GET /posts - 获取文章列表
posts.get("/", async (c) => {
  const { page = "1", limit = "10" } = c.req.query();

  const offset = (parseInt(page) - 1) * parseInt(limit);

  const result = await c.env.DB.prepare(
    `
    SELECT * FROM posts
    WHERE published = 1
    ORDER BY created_at DESC
    LIMIT ? OFFSET ?
  `
  )
    .bind(limit, offset)
    .all();

  return c.json({ success: true, data: result.results });
});

// GET /posts/:id - 获取文章详情
posts.get("/:id", async (c) => {
  const { id } = c.req.param();

  // 增加浏览量
  await c.env.DB.prepare("UPDATE posts SET views = views + 1 WHERE id = ?")
    .bind(id)
    .run();

  const post = await c.env.DB.prepare("SELECT * FROM posts WHERE id = ?")
    .bind(id)
    .first();

  if (!post) {
    return c.json({ success: false, message: "Post not found" }, 404);
  }

  return c.json({ success: true, data: post });
});

// POST /posts - 创建文章
posts.post("/", authMiddleware, zValidator("json", postSchema), async (c) => {
  const data = c.req.valid("json");
  const user = c.get("user");

  const postId = crypto.randomUUID();
  const slug = data.title.toLowerCase().replace(/[^a-z0-9]+/g, "-");
  const now = new Date().toISOString();

  await c.env.DB.prepare(
    `
    INSERT INTO posts (id, title, slug, content, excerpt, cover_image, published, author_id, created_at, updated_at)
    VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
  `
  )
    .bind(
      postId,
      data.title,
      slug,
      data.content,
      data.excerpt,
      data.coverImage,
      data.published ? 1 : 0,
      user.id,
      now,
      now
    )
    .run();

  // 处理标签
  if (data.tags && data.tags.length > 0) {
    for (const tagName of data.tags) {
      const tagSlug = tagName.toLowerCase().replace(/[^a-z0-9]+/g, "-");

      // 创建或获取标签
      let tag = await c.env.DB.prepare("SELECT id FROM tags WHERE slug = ?")
        .bind(tagSlug)
        .first();

      if (!tag) {
        const tagId = crypto.randomUUID();
        await c.env.DB.prepare(
          "INSERT INTO tags (id, name, slug) VALUES (?, ?, ?)"
        )
          .bind(tagId, tagName, tagSlug)
          .run();
        tag = { id: tagId };
      }

      // 关联文章和标签
      await c.env.DB.prepare(
        "INSERT INTO post_tags (post_id, tag_id) VALUES (?, ?)"
      )
        .bind(postId, tag.id)
        .run();
    }
  }

  return c.json({ success: true, data: { id: postId } }, 201);
});

export default posts;
```

这个实现包含了文章列表、详情和创建功能，还支持标签关联。更新和删除操作类似，这里就不重复了。

## 评论系统实现

评论系统支持嵌套回复，这是博客社区功能的重要组成部分：

```typescript
// src/routes/comments.ts
import { Hono } from "hono";
import { zValidator } from "@hono/zod-validator";
import { z } from "zod";
import { authMiddleware } from "../middleware/auth";

const comments = new Hono<{ Bindings: Env }>();

const commentSchema = z.object({
  content: z.string().min(1).max(1000),
  parentId: z.string().optional(),
});

// GET /posts/:postId/comments - 获取文章评论
comments.get("/posts/:postId/comments", async (c) => {
  const { postId } = c.req.param();

  const comments = await c.env.DB.prepare(
    `
    SELECT c.*, u.username, u.name, u.avatar
    FROM comments c
    JOIN users u ON c.author_id = u.id
    WHERE c.post_id = ?
    ORDER BY c.created_at ASC
  `
  )
    .bind(postId)
    .all();

  return c.json({ success: true, data: comments.results });
});

// POST /posts/:postId/comments - 发表评论
comments.post(
  "/posts/:postId/comments",
  authMiddleware,
  zValidator("json", commentSchema),
  async (c) => {
    const { postId } = c.req.param();
    const { content, parentId } = c.req.valid("json");
    const user = c.get("user");

    // 验证文章存在
    const post = await c.env.DB.prepare("SELECT id FROM posts WHERE id = ?")
      .bind(postId)
      .first();

    if (!post) {
      return c.json({ success: false, message: "Post not found" }, 404);
    }

    // 如果是回复，验证父评论存在
    if (parentId) {
      const parentComment = await c.env.DB.prepare(
        "SELECT id FROM comments WHERE id = ?"
      )
        .bind(parentId)
        .first();

      if (!parentComment) {
        return c.json(
          { success: false, message: "Parent comment not found" },
          404
        );
      }
    }

    // 创建评论
    const commentId = crypto.randomUUID();
    await c.env.DB.prepare(
      `
      INSERT INTO comments (id, content, post_id, author_id, parent_id, created_at)
      VALUES (?, ?, ?, ?, ?, ?)
    `
    )
      .bind(
        commentId,
        content,
        postId,
        user.id,
        parentId,
        new Date().toISOString()
      )
      .run();

    return c.json({ success: true, data: { id: commentId } }, 201);
  }
);

export default comments;
```

## 文件上传实现

使用 R2 存储文章封面图：

```typescript
// src/routes/upload.ts
import { Hono } from "hono";
import { authMiddleware } from "../middleware/auth";

const upload = new Hono<{ Bindings: Env }>();

// 允许的图片格式
const ALLOWED_TYPES = ["image/jpeg", "image/png", "image/webp"];
// 最大文件大小 5MB
const MAX_SIZE = 5 * 1024 * 1024;

upload.post("/", authMiddleware, async (c) => {
  const formData = await c.req.formData();
  const file = formData.get("file") as File;

  if (!file) {
    return c.json({ success: false, message: "No file provided" }, 400);
  }

  // 验证文件类型
  if (!ALLOWED_TYPES.includes(file.type)) {
    return c.json(
      {
        success: false,
        message: "Invalid file type. Only JPEG, PNG, and WebP are allowed.",
      },
      400
    );
  }

  // 验证文件大小
  if (file.size > MAX_SIZE) {
    return c.json(
      {
        success: false,
        message: "File too large. Maximum size is 5MB.",
      },
      400
    );
  }

  // 生成唯一文件名
  const ext = file.name.split(".").pop();
  const key = `covers/${crypto.randomUUID()}.${ext}`;

  // 上传到 R2
  await c.env.STORAGE.put(key, file.stream(), {
    httpMetadata: {
      contentType: file.type,
    },
  });

  // 返回访问 URL
  const url = `https://cdn.yourdomain.com/${key}`;

  return c.json({
    success: true,
    data: { url, key },
  });
});

export default upload;
```

## 缓存优化

实现一个智能的缓存服务，减少数据库查询：

```typescript
// src/services/cache.service.ts
export class CacheService {
  constructor(private kv: KVNamespace) {}

  async get<T>(key: string): Promise<T | null> {
    const value = await this.kv.get(key, "json");
    return value as T;
  }

  async set(key: string, value: any, ttl: number): Promise<void> {
    await this.kv.put(key, JSON.stringify(value), {
      expirationTtl: ttl,
    });
  }

  async delete(key: string): Promise<void> {
    await this.kv.delete(key);
  }

  async invalidatePattern(prefix: string): Promise<void> {
    const listed = await this.kv.list({ prefix });
    await Promise.all(listed.keys.map((key) => this.kv.delete(key.name)));
  }
}
```

在文章路由中使用缓存：

```typescript
const cache = new CacheService(c.env.CACHE);

app.get("/posts/:id", async (c) => {
```
